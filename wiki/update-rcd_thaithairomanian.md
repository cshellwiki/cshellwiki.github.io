# [Linux] Bash update-rc.d: [จัดการการเริ่มต้นบริการ]

## Overview
คำสั่ง `update-rc.d` ใช้ในการจัดการการเริ่มต้นของบริการในระบบ Linux โดยเฉพาะในการเพิ่มหรือเอาบริการออกจากการเริ่มต้นอัตโนมัติในระดับต่าง ๆ ของระบบปฏิบัติการ

## Usage
รูปแบบพื้นฐานของคำสั่งคือ:
```bash
update-rc.d [options] [arguments]
```

## Common Options
- `defaults`: ใช้สำหรับติดตั้งบริการด้วยค่าที่ตั้งไว้เริ่มต้น
- `remove`: ใช้เพื่อลบบริการออกจากการเริ่มต้นอัตโนมัติ
- `enable`: ใช้เพื่อเปิดใช้งานบริการในการเริ่มต้น
- `disable`: ใช้เพื่อปิดใช้งานบริการในการเริ่มต้น

## Common Examples
- **ติดตั้งบริการใหม่ด้วยค่าที่ตั้งไว้เริ่มต้น:**
  ```bash
  sudo update-rc.d myservice defaults
  ```

- **ลบบริการออกจากการเริ่มต้นอัตโนมัติ:**
  ```bash
  sudo update-rc.d myservice remove
  ```

- **เปิดใช้งานบริการในการเริ่มต้น:**
  ```bash
  sudo update-rc.d myservice enable
  ```

- **ปิดใช้งานบริการในการเริ่มต้น:**
  ```bash
  sudo update-rc.d myservice disable
  ```

## Tips
- ควรใช้คำสั่ง `sudo` เพื่อให้มีสิทธิ์ในการจัดการบริการ
- ตรวจสอบสถานะของบริการก่อนทำการเปลี่ยนแปลงเพื่อหลีกเลี่ยงปัญหาที่อาจเกิดขึ้น
- ใช้คำสั่ง `service` หรือ `systemctl` ร่วมกับ `update-rc.d` เพื่อจัดการบริการได้อย่างมีประสิทธิภาพมากขึ้น
