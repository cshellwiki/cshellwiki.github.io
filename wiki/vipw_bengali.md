# [লিনাক্স] Bash vipw ব্যবহার: পাসওয়ার্ড ফাইল সম্পাদনা করা

## Overview
`vipw` কমান্ডটি সিস্টেমের পাসওয়ার্ড ফাইল সম্পাদনা করার জন্য ব্যবহৃত হয়। এটি নিরাপদভাবে `/etc/passwd` এবং `/etc/shadow` ফাইলগুলি খুলে দেয়, যাতে ব্যবহারকারীরা তাদের তথ্য পরিবর্তন করতে পারেন।

## Usage
`vipw` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
vipw [options]
```

## Common Options
- `-s`: শ্যাডো ফাইল সম্পাদনা করতে ব্যবহৃত হয়।
- `-u`: নির্দিষ্ট ব্যবহারকারীর তথ্য সম্পাদনা করতে ব্যবহৃত হয়।

## Common Examples
1. সাধারণভাবে পাসওয়ার্ড ফাইল সম্পাদনা করতে:
   ```bash
   vipw
   ```

2. শ্যাডো ফাইল সম্পাদনা করতে:
   ```bash
   vipw -s
   ```

3. নির্দিষ্ট ব্যবহারকারীর তথ্য সম্পাদনা করতে (যেমন, "username"):
   ```bash
   vipw -u username
   ```

## Tips
- `vipw` ব্যবহার করার আগে নিশ্চিত করুন যে আপনার কাছে যথাযথ অনুমতি রয়েছে।
- পাসওয়ার্ড ফাইল সম্পাদনা করার সময় সতর্ক থাকুন, কারণ ভুল তথ্য সিস্টেমের প্রবেশাধিকারকে প্রভাবিত করতে পারে।
- পরিবর্তন করার পর ফাইলটি সঠিকভাবে সংরক্ষণ হয়েছে কিনা তা পরীক্ষা করুন।
