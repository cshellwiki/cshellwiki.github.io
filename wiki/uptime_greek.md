# [Λειτουργικό Σύστημα] Bash uptime Χρήση: [παρουσίαση χρόνου λειτουργίας συστήματος]

## Overview
Η εντολή `uptime` χρησιμοποιείται για να εμφανίσει πόσο καιρό λειτουργεί το σύστημα από την τελευταία εκκίνηση. Παρέχει πληροφορίες σχετικά με τον χρόνο λειτουργίας, τον αριθμό χρηστών που είναι συνδεδεμένοι και τη μέση φόρτιση του συστήματος.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
uptime [options]
```

## Common Options
- `-p`: Εμφανίζει τον χρόνο λειτουργίας με πιο φιλικό προς τον χρήστη τρόπο.
- `-s`: Εμφανίζει την ημερομηνία και ώρα της τελευταίας εκκίνησης του συστήματος.

## Common Examples
### Παράδειγμα 1: Απλή χρήση
Για να δείτε τον χρόνο λειτουργίας του συστήματος, απλά πληκτρολογήστε:

```bash
uptime
```

### Παράδειγμα 2: Φιλική εμφάνιση χρόνου λειτουργίας
Για να εμφανίσετε τον χρόνο λειτουργίας με πιο κατανοητό τρόπο, χρησιμοποιήστε την επιλογή `-p`:

```bash
uptime -p
```

### Παράδειγμα 3: Ημερομηνία και ώρα τελευταίας εκκίνησης
Για να δείτε πότε έγινε η τελευταία εκκίνηση του συστήματος, μπορείτε να χρησιμοποιήσετε την επιλογή `-s`:

```bash
uptime -s
```

## Tips
- Χρησιμοποιήστε την εντολή `uptime` σε τακτική βάση για να παρακολουθείτε την υγεία του συστήματος.
- Συνδυάστε την εντολή με άλλες εντολές, όπως `top` ή `htop`, για να έχετε μια πιο ολοκληρωμένη εικόνα της απόδοσης του συστήματος.
- Εξετάστε τη μέση φόρτιση του συστήματος για να καταλάβετε αν χρειάζεται βελτιστοποίηση ή αναβάθμιση.
