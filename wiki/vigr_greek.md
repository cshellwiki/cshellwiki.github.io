# [Λειτουργικό Σύστημα] Bash vigr Χρήση: Επεξεργασία αρχείων /etc/hosts

## Overview
Η εντολή `vigr` χρησιμοποιείται για την επεξεργασία του αρχείου `/etc/group` και `/etc/passwd` με ασφάλεια, διασφαλίζοντας ότι δεν θα υπάρξουν ταυτόχρονες τροποποιήσεις από άλλους χρήστες. Είναι ένα εργαλείο που σας επιτρέπει να επεξεργαστείτε αυτά τα κρίσιμα αρχεία του συστήματος με τη βοήθεια ενός επεξεργαστή κειμένου.

## Usage
Η βασική σύνταξη της εντολής είναι:

```bash
vigr [options] [arguments]
```

## Common Options
- `-c`: Ελέγχει την ορθότητα του αρχείου μετά την επεξεργασία.
- `-s`: Ενεργοποιεί την ασφαλή επεξεργασία, αποτρέποντας ταυτόχρονες τροποποιήσεις.
- `-f <file>`: Καθορίζει ένα εναλλακτικό αρχείο για επεξεργασία.

## Common Examples
### Επεξεργασία του αρχείου /etc/group
```bash
vigr
```

### Επεξεργασία ενός συγκεκριμένου αρχείου
```bash
vigr -f /etc/passwd
```

### Έλεγχος ορθότητας του αρχείου μετά την επεξεργασία
```bash
vigr -c
```

## Tips
- Χρησιμοποιήστε την επιλογή `-s` για να διασφαλίσετε ότι κανένας άλλος χρήστης δεν μπορεί να τροποποιήσει το αρχείο κατά τη διάρκεια της επεξεργασίας.
- Πάντα να κάνετε ένα αντίγραφο ασφαλείας των αρχείων `/etc/passwd` και `/etc/group` πριν από οποιαδήποτε τροποποίηση.
- Εξοικειωθείτε με τον επεξεργαστή κειμένου που χρησιμοποιεί το `vigr`, καθώς αυτό θα διευκολύνει την επεξεργασία των αρχείων.
