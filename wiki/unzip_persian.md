# [سیستم‌عامل] C Shell (csh) unzip استفاده: استخراج فایل‌های فشرده

## Overview
دستور `unzip` برای استخراج فایل‌های فشرده با فرمت ZIP استفاده می‌شود. این دستور به کاربران این امکان را می‌دهد که محتویات فایل‌های ZIP را به راحتی در دایرکتوری‌های مورد نظر خود استخراج کنند.

## Usage
سینتکس پایه دستور `unzip` به صورت زیر است:

```csh
unzip [options] [arguments]
```

## Common Options
- `-l`: نمایش لیست فایل‌های داخل آرشیو ZIP بدون استخراج آن‌ها.
- `-d [directory]`: استخراج فایل‌ها به دایرکتوری مشخص شده.
- `-o`: بازنویسی فایل‌های موجود بدون پرسش.
- `-q`: حالت ساکت، که فقط خطاها را نمایش می‌دهد و هیچ خروجی دیگری ندارد.

## Common Examples
- استخراج فایل ZIP به دایرکتوری فعلی:
  ```csh
  unzip archive.zip
  ```

- استخراج فایل ZIP به دایرکتوری مشخص:
  ```csh
  unzip archive.zip -d /path/to/directory
  ```

- نمایش لیست فایل‌های داخل آرشیو ZIP:
  ```csh
  unzip -l archive.zip
  ```

- استخراج فایل ZIP و بازنویسی فایل‌های موجود:
  ```csh
  unzip -o archive.zip
  ```

## Tips
- قبل از استخراج، همیشه با استفاده از گزینه `-l` لیست فایل‌ها را بررسی کنید تا از محتویات آرشیو مطلع شوید.
- اگر می‌خواهید از بازنویسی فایل‌های موجود جلوگیری کنید، از گزینه `-o` استفاده نکنید.
- برای مدیریت فایل‌های بزرگ، می‌توانید از گزینه `-q` برای کاهش خروجی استفاده کنید و فقط خطاها را مشاهده کنید.
