# [لینوکس] Bash while استفاده: حلقه تکرار برای اجرای دستورات

## Overview
دستور `while` در Bash برای اجرای یک بلوک از دستورات به طور مکرر تا زمانی که یک شرط خاص برقرار باشد، استفاده می‌شود. این دستور به شما امکان می‌دهد که یک فرآیند را تا زمانی که شرطی برقرار است، ادامه دهید.

## Usage
سینتکس پایه دستور `while` به شکل زیر است:

```bash
while [شرط]
do
    # دستورات
done
```

## Common Options
دستور `while` به خودی خود گزینه‌های خاصی ندارد، اما می‌توانید از دستورات و شرایط مختلف درون آن استفاده کنید. برخی از شرایط رایج شامل:

- `true`: همیشه درست است و حلقه را به طور نامحدود اجرا می‌کند.
- `false`: همیشه نادرست است و حلقه را هرگز اجرا نمی‌کند.
- `[ -e file ]`: بررسی وجود یک فایل.

## Common Examples

### مثال ۱: حلقه ساده
این مثال یک حلقه ساده را نشان می‌دهد که تا زمانی که متغیر `count` کمتر از 5 باشد، اجرا می‌شود.

```bash
count=0
while [ $count -lt 5 ]
do
    echo "Count is: $count"
    ((count++))
done
```

### مثال ۲: خواندن ورودی کاربر
در این مثال، از کاربر خواسته می‌شود تا یک عدد وارد کند و حلقه تا زمانی که کاربر عدد منفی وارد کند، ادامه می‌یابد.

```bash
number=0
while [ $number -ge 0 ]
do
    read -p "Enter a number (negative to quit): " number
    echo "You entered: $number"
done
```

### مثال ۳: بررسی وجود فایل
این مثال نشان می‌دهد که چگونه می‌توان از `while` برای بررسی وجود یک فایل استفاده کرد.

```bash
file="example.txt"
while [ ! -e $file ]
do
    echo "$file does not exist. Waiting..."
    sleep 2
done
echo "$file exists!"
```

## Tips
- همیشه از یک شرط خروجی مناسب استفاده کنید تا از حلقه‌های بی‌پایان جلوگیری کنید.
- می‌توانید از `break` برای خروج از حلقه در شرایط خاص استفاده کنید.
- از `sleep` برای جلوگیری از بارگذاری بیش از حد سیستم در حلقه‌های مکرر استفاده کنید.
