# [سیستم‌عامل] C Shell (csh) while: اجرای دستورات در حلقه

## Overview
دستور `while` در C Shell (csh) برای اجرای مکرر یک یا چند دستور تا زمانی که یک شرط خاص برقرار باشد، استفاده می‌شود. این دستور به شما امکان می‌دهد تا به راحتی حلقه‌های تکرار ایجاد کنید که به شرایط خاصی وابسته هستند.

## Usage
سینتکس پایه دستور `while` به صورت زیر است:

```csh
while ( شرط )
    دستورات
end
```

## Common Options
دستور `while` به خودی خود گزینه‌های خاصی ندارد، اما می‌توانید از عبارات شرطی مختلف استفاده کنید که شامل موارد زیر است:
- `-eq`: برابر بودن دو عدد
- `-ne`: نابرابر بودن دو عدد
- `-lt`: کمتر بودن یک عدد از دیگری
- `-gt`: بزرگتر بودن یک عدد از دیگری
- `-le`: کمتر یا برابر بودن
- `-ge`: بزرگتر یا برابر بودن

## Common Examples

### مثال ۱: شمارش از ۱ تا ۵
```csh
set i = 1
while ($i <= 5)
    echo $i
    @ i++
end
```

### مثال ۲: حلقه بی‌نهایت
```csh
while (1)
    echo "این یک حلقه بی‌نهایت است"
end
```

### مثال ۳: خواندن ورودی کاربر
```csh
set input = ""
while ($input != "خروج")
    echo "لطفا یک کلمه وارد کنید (برای خروج 'خروج' را وارد کنید):"
    set input = $<
end
```

## Tips
- همیشه از `end` برای بستن حلقه `while` استفاده کنید.
- برای جلوگیری از حلقه‌های بی‌نهایت، اطمینان حاصل کنید که شرط خروج از حلقه به درستی تنظیم شده باشد.
- می‌توانید از متغیرهای محیطی برای کنترل شرایط حلقه استفاده کنید.
