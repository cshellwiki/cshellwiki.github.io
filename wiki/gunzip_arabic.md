# [لينكس] Bash gunzip الاستخدام: فك ضغط الملفات

## Overview
يستخدم أمر `gunzip` لفك ضغط الملفات التي تم ضغطها باستخدام خوارزمية Gzip. يقوم هذا الأمر بإعادة الملفات إلى حالتها الأصلية، مما يسهل الوصول إلى المحتوى المخزن داخلها.

## Usage
تكون الصيغة الأساسية لأمر `gunzip` كما يلي:

```bash
gunzip [options] [arguments]
```

## Common Options
- `-c`: يخرج المحتوى المفكوك إلى stdout بدلاً من الكتابة إلى ملف.
- `-f`: يفرض فك الضغط حتى لو كانت الملفات موجودة بالفعل.
- `-k`: يحتفظ بالملف المضغوط بعد فك الضغط.
- `-r`: يقوم بفك ضغط الملفات في الدلائل الفرعية.

## Common Examples
إليك بعض الأمثلة العملية لاستخدام `gunzip`:

1. فك ضغط ملف مضغوط:
    ```bash
    gunzip file.txt.gz
    ```

2. فك ضغط عدة ملفات في نفس الوقت:
    ```bash
    gunzip file1.gz file2.gz file3.gz
    ```

3. إخراج المحتوى المفكوك إلى stdout:
    ```bash
    gunzip -c file.txt.gz > output.txt
    ```

4. فك ضغط ملف مع الاحتفاظ بالملف الأصلي:
    ```bash
    gunzip -k file.txt.gz
    ```

5. فك ضغط جميع الملفات في دليل وما يحتويه من دلائل فرعية:
    ```bash
    gunzip -r /path/to/directory/*.gz
    ```

## Tips
- تأكد من أن لديك مساحة كافية على القرص قبل فك ضغط الملفات الكبيرة.
- استخدم الخيار `-k` إذا كنت بحاجة للاحتفاظ بالملف المضغوط لأغراض النسخ الاحتياطي.
- إذا كنت تعمل مع ملفات متعددة، يمكنك استخدام `*` لتسهيل العملية بدلاً من كتابة أسماء الملفات بشكل فردي.
