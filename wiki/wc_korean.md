# [리눅스] Bash wc 사용법: 파일의 행, 단어, 문자 수 세기

## Overview
`wc` 명령어는 파일의 행 수, 단어 수, 문자 수를 세는 데 사용됩니다. 이 명령어는 텍스트 파일의 내용을 분석하고 요약하는 데 유용합니다.

## Usage
기본 구문은 다음과 같습니다:
```
wc [옵션] [파일]
```

## Common Options
- `-l`: 행 수를 출력합니다.
- `-w`: 단어 수를 출력합니다.
- `-c`: 문자 수를 출력합니다.
- `-m`: 문자 수를 출력하지만, 멀티바이트 문자를 고려합니다.
- `-L`: 가장 긴 행의 길이를 출력합니다.

## Common Examples
- 파일의 행 수 세기:
  ```bash
  wc -l filename.txt
  ```

- 파일의 단어 수 세기:
  ```bash
  wc -w filename.txt
  ```

- 파일의 문자 수 세기:
  ```bash
  wc -c filename.txt
  ```

- 여러 파일의 행, 단어, 문자 수 모두 세기:
  ```bash
  wc filename1.txt filename2.txt
  ```

- 표준 입력에서 데이터 세기 (예: 파이프 사용):
  ```bash
  echo "Hello World" | wc
  ```

## Tips
- 여러 파일을 한 번에 처리할 수 있어, 대량의 데이터 분석에 유용합니다.
- `wc` 명령어는 파이프와 함께 사용하여 다른 명령어의 출력을 쉽게 분석할 수 있습니다.
- 결과를 정렬하거나 필터링하려면 `sort` 또는 `grep`과 함께 사용하는 것이 좋습니다.
