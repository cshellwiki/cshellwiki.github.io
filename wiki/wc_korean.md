# [리눅스] C Shell (csh) wc 사용법: 파일의 행, 단어, 문자 수 세기

## 개요
`wc` 명령어는 파일의 행 수, 단어 수, 문자 수를 계산하는 데 사용됩니다. 이 명령어는 텍스트 파일의 내용을 분석하고 통계를 제공하는 데 유용합니다.

## 사용법
기본 구문은 다음과 같습니다:
```
wc [옵션] [파일]
```

## 일반 옵션
- `-l`: 행 수를 출력합니다.
- `-w`: 단어 수를 출력합니다.
- `-c`: 문자 수를 출력합니다.
- `-m`: 문자 수를 출력합니다 (UTF-8 문자 포함).
- `-L`: 가장 긴 행의 길이를 출력합니다.

## 일반 예제
1. 파일의 행 수 세기:
   ```bash
   wc -l filename.txt
   ```

2. 파일의 단어 수 세기:
   ```bash
   wc -w filename.txt
   ```

3. 파일의 문자 수 세기:
   ```bash
   wc -c filename.txt
   ```

4. 여러 파일의 행, 단어, 문자 수 세기:
   ```bash
   wc file1.txt file2.txt
   ```

5. 가장 긴 행의 길이 확인:
   ```bash
   wc -L filename.txt
   ```

## 팁
- 여러 옵션을 조합하여 사용할 수 있습니다. 예를 들어, `wc -l -w filename.txt`는 행 수와 단어 수를 동시에 출력합니다.
- 파이프를 사용하여 다른 명령어와 결합할 수 있습니다. 예를 들어, `cat filename.txt | wc -l`은 파일의 행 수를 출력합니다.
- 대용량 파일을 처리할 때는 결과를 파일로 리다이렉션하여 저장할 수 있습니다: `wc filename.txt > result.txt`.
