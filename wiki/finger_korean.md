# [리눅스] Bash finger 사용법: 사용자 정보 확인

## Overview
`finger` 명령어는 시스템에 로그인한 사용자에 대한 정보를 표시하는 데 사용됩니다. 이 명령어를 통해 사용자의 이름, 로그인 시간, idle 시간, 그리고 사용자가 실행 중인 프로세스 등의 정보를 확인할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:

```
finger [options] [arguments]
```

## Common Options
- `-l`: 긴 형식으로 사용자 정보를 표시합니다.
- `-m`: 사용자 이름을 대소문자 구분 없이 검색합니다.
- `-s`: 간단한 형식으로 사용자 정보를 표시합니다.

## Common Examples
- 모든 사용자 정보 확인:
    ```bash
    finger
    ```

- 특정 사용자 정보 확인:
    ```bash
    finger username
    ```

- 긴 형식으로 특정 사용자 정보 확인:
    ```bash
    finger -l username
    ```

- 대소문자 구분 없이 사용자 검색:
    ```bash
    finger -m userName
    ```

## Tips
- `finger` 명령어는 시스템에 로그인한 사용자에 대한 정보를 제공하므로, 보안상의 이유로 일부 시스템에서는 사용이 제한될 수 있습니다.
- 사용자 정보를 확인할 때는 `-l` 옵션을 사용하여 더 많은 세부 정보를 얻는 것이 유용합니다.
- `finger` 명령어는 다른 사용자와의 소통을 위해 유용하게 사용될 수 있습니다.
