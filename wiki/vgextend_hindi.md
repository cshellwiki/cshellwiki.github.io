# [Linux] Bash vgextend उपयोग: वॉल्यूम ग्रुप में भंडारण जोड़ना

## Overview
`vgextend` कमांड का उपयोग वॉल्यूम ग्रुप (VG) में अतिरिक्त भंडारण जोड़ने के लिए किया जाता है। यह कमांड Logical Volume Management (LVM) के तहत काम करता है और आपको अपने वॉल्यूम ग्रुप की क्षमता बढ़ाने की अनुमति देता है।

## Usage
`vgextend` कमांड का मूल सिंटैक्स निम्नलिखित है:

```bash
vgextend [options] [arguments]
```

## Common Options
- `-n, --noheadings`: हेडिंग्स के बिना आउटपुट दिखाता है।
- `-v, --verbose`: विस्तृत आउटपुट प्रदान करता है।
- `--test`: बिना किसी बदलाव के परीक्षण करता है कि क्या कमांड सफल होगा।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

### उदाहरण 1: वॉल्यूम ग्रुप में एक भंडारण जोड़ना
```bash
vgextend myvg /dev/sdb1
```
इस कमांड से `myvg` वॉल्यूम ग्रुप में `/dev/sdb1` भंडारण जोड़ा जाएगा।

### उदाहरण 2: एकाधिक भंडारण जोड़ना
```bash
vgextend myvg /dev/sdb1 /dev/sdc1
```
यह कमांड `myvg` में `/dev/sdb1` और `/dev/sdc1` दोनों भंडारण जोड़ता है।

### उदाहरण 3: विस्तृत आउटपुट के साथ जोड़ना
```bash
vgextend -v myvg /dev/sdb1
```
इससे `myvg` में `/dev/sdb1` जोड़ते समय विस्तृत जानकारी प्राप्त होगी।

## Tips
- हमेशा `vgextend` कमांड चलाने से पहले अपने वॉल्यूम ग्रुप की वर्तमान स्थिति की जांच करें।
- यदि आप नए भंडारण को जोड़ने से पहले टेस्ट करना चाहते हैं, तो `--test` विकल्प का उपयोग करें।
- सुनिश्चित करें कि आप सही भंडारण डिवाइस का चयन कर रहे हैं, क्योंकि गलत डिवाइस जोड़ने से डेटा हानि हो सकती है।
