# [لينكس] Bash date الاستخدام: عرض التاريخ والوقت الحالي

## Overview
أمر `date` في Bash يُستخدم لعرض التاريخ والوقت الحاليين. يمكن تخصيص تنسيق العرض حسب الحاجة، مما يجعله أداة مفيدة في العديد من السيناريوهات.

## Usage
الصيغة الأساسية للأمر هي:

```bash
date [options] [arguments]
```

## Common Options
- `+FORMAT`: لتحديد تنسيق العرض. يمكنك استخدام رموز مختلفة لتخصيص كيفية ظهور التاريخ والوقت.
- `-u`: لعرض الوقت بالتوقيت العالمي المنسق (UTC).
- `-d STRING`: لعرض التاريخ والوقت بناءً على سلسلة معينة. 

## Common Examples
- لعرض التاريخ والوقت الحاليين بالتنسيق الافتراضي:
```bash
date
```

- لعرض التاريخ والوقت بالتنسيق المخصص (على سبيل المثال، يوم/شهر/سنة):
```bash
date +"%d/%m/%Y"
```

- لعرض الوقت بالتوقيت العالمي:
```bash
date -u
```

- لعرض تاريخ محدد (مثل 1 يناير 2023):
```bash
date -d "2023-01-01"
```

## Tips
- استخدم الخيار `+FORMAT` لتخصيص عرض التاريخ والوقت حسب احتياجاتك.
- يمكنك دمج أمر `date` مع أوامر أخرى في Bash لتسجيل الأوقات أو إنشاء سجلات.
- تذكر أن تنسيق التاريخ يعتمد على الإعدادات المحلية، لذا تأكد من استخدام الرموز الصحيحة لتجنب الالتباس.
