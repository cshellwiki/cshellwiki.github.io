# [لينكس] Bash yes الاستخدام: تكرار النصوص بشكل مستمر

## Overview
يستخدم أمر `yes` في نظام التشغيل لينكس لطباعة نص معين بشكل متكرر. بشكل افتراضي، يقوم الأمر بطباعة كلمة "y" باستمرار، مما يجعله مفيدًا في حالات معينة مثل تلقين الأوامر التي تتطلب تأكيدًا.

## Usage
يمكن استخدام الأمر `yes` بالصيغة التالية:

```bash
yes [options] [arguments]
```

## Common Options
- `-h`, `--help`: عرض معلومات المساعدة حول الأمر.
- `-V`, `--version`: عرض معلومات حول إصدار الأمر.

## Common Examples
إليك بعض الأمثلة العملية لاستخدام الأمر `yes`:

1. **طباعة "y" بشكل مستمر:**
   ```bash
   yes
   ```

2. **طباعة نص مخصص بشكل مستمر:**
   ```bash
   yes "Hello, World!"
   ```

3. **استخدام `yes` مع أمر آخر:**
   يمكنك استخدام `yes` لتلقين أمر يتطلب تأكيدًا:
   ```bash
   yes | rm -i *.tmp
   ```

4. **تحديد عدد مرات الطباعة:**
   يمكنك استخدام `head` لتحديد عدد الأسطر التي تريد طباعتها:
   ```bash
   yes "Accepted" | head -n 5
   ```

## Tips
- استخدم الأمر `yes` بحذر، حيث يمكن أن يؤدي إلى طباعة نصوص بشكل لا نهائي إذا لم يتم توجيهه بشكل صحيح.
- من المفيد دمج الأمر `yes` مع أوامر أخرى تتطلب تأكيدًا لتسهيل العمليات التلقائية.
- تأكد من أن لديك طريقة لإنهاء الأمر (مثل Ctrl+C) إذا كنت تستخدمه في وضعية الطباعة المستمرة.
