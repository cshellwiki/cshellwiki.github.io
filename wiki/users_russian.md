# [Linux] Bash пользователи команды: [управление пользователями]

## Обзор
Команда `users` в Bash используется для отображения имен пользователей, которые в данный момент вошли в систему. Она предоставляет простой способ узнать, кто активен на машине в данный момент.

## Использование
Основной синтаксис команды выглядит следующим образом:

```
users [options] [arguments]
```

## Общие параметры
- `-n` : Выводит только количество уникальных пользователей.
- `-r` : Игнорирует пользователей, которые не имеют активных сеансов.

## Общие примеры
Вот несколько практических примеров использования команды `users`:

1. **Отобразить всех активных пользователей:**
   ```bash
   users
   ```

2. **Отобразить количество уникальных пользователей:**
   ```bash
   users -n
   ```

3. **Игнорировать пользователей без активных сеансов:**
   ```bash
   users -r
   ```

## Советы
- Используйте команду `who` для получения более подробной информации о пользователях, включая время входа и терминалы.
- Команда `w` также может быть полезна, так как она показывает, что делают пользователи в данный момент.
- Регулярно проверяйте активных пользователей для обеспечения безопасности системы.
