# [লিনাক্স] Bash yum ব্যবহার: প্যাকেজ ব্যবস্থাপনা কমান্ড

## Overview
`yum` (Yellowdog Updater Modified) হল একটি প্যাকেজ ব্যবস্থাপনা টুল যা RPM ভিত্তিক লিনাক্স ডিস্ট্রিবিউশনে সফটওয়্যার প্যাকেজ ইনস্টল, আপডেট এবং মুছে ফেলার জন্য ব্যবহৃত হয়। এটি স্বয়ংক্রিয়ভাবে নির্ভরতাগুলি পরিচালনা করে এবং ব্যবহারকারীদের জন্য সফটওয়্যার পরিচালনাকে সহজ করে তোলে।

## Usage
`yum` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
yum [options] [arguments]
```

## Common Options
- `install`: একটি প্যাকেজ ইনস্টল করতে ব্যবহৃত হয়।
- `remove`: একটি প্যাকেজ মুছে ফেলতে ব্যবহৃত হয়।
- `update`: ইনস্টল করা প্যাকেজগুলিকে আপডেট করতে ব্যবহৃত হয়।
- `search`: প্যাকেজের নাম বা বর্ণনা অনুসন্ধান করতে ব্যবহৃত হয়।
- `list`: ইনস্টল করা এবং উপলব্ধ প্যাকেজের তালিকা দেখায়।

## Common Examples
- একটি প্যাকেজ ইনস্টল করা:
  ```bash
  yum install package_name
  ```

- একটি প্যাকেজ মুছে ফেলা:
  ```bash
  yum remove package_name
  ```

- সমস্ত প্যাকেজ আপডেট করা:
  ```bash
  yum update
  ```

- একটি প্যাকেজ অনুসন্ধান করা:
  ```bash
  yum search package_name
  ```

- ইনস্টল করা প্যাকেজের তালিকা দেখা:
  ```bash
  yum list installed
  ```

## Tips
- সর্বদা `yum update` চালানোর মাধ্যমে আপনার সিস্টেমের প্যাকেজগুলি আপডেট রাখুন।
- নির্ভরতা সমস্যা এড়াতে, একসাথে একাধিক প্যাকেজ ইনস্টল করার সময় `yum` ব্যবহার করুন।
- `yum clean all` কমান্ডটি ক্যাশে পরিষ্কার করতে ব্যবহৃত হয়, যা স্থান মুক্ত করতে সাহায্য করে।
