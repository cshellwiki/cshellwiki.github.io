# [Linux] Bash users Χρήση: Διαχείριση χρηστών στο σύστημα

## Overview
Η εντολή `users` χρησιμοποιείται για να εμφανίσει τους χρήστες που είναι συνδεδεμένοι στο σύστημα. Παρέχει μια απλή λίστα με τα ονόματα των χρηστών που έχουν ενεργές συνεδρίες.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
users [options]
```

## Common Options
- `-n`: Εμφανίζει μόνο το όνομα του χρήστη χωρίς επαναλήψεις.
- `-r`: Εμφανίζει μόνο τους χρήστες που είναι συνδεδεμένοι σε τερματικά.

## Common Examples
### Παράδειγμα 1: Εμφάνιση όλων των χρηστών
Για να δείτε όλους τους χρήστες που είναι συνδεδεμένοι:

```bash
users
```

### Παράδειγμα 2: Εμφάνιση μοναδικών χρηστών
Για να δείτε μόνο τους μοναδικούς χρήστες που είναι συνδεδεμένοι:

```bash
users -n
```

### Παράδειγμα 3: Εμφάνιση χρηστών σε τερματικά
Για να δείτε μόνο τους χρήστες που είναι συνδεδεμένοι σε τερματικά:

```bash
users -r
```

## Tips
- Χρησιμοποιήστε την εντολή `who` για περισσότερες λεπτομέρειες σχετικά με τις συνδέσεις χρηστών.
- Συνδυάστε την εντολή `users` με άλλες εντολές όπως `wc -l` για να μετρήσετε πόσοι χρήστες είναι συνδεδεμένοι:
  
  ```bash
  users | wc -w
  ```
