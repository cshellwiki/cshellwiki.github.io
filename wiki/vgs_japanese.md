# [Linux] Bash vgs の使い方: ボリュームグループの情報を表示する

## 概要
`vgs` コマンドは、Linux の論理ボリュームマネージャ（LVM）で使用され、システム内のボリュームグループに関する情報を表示します。このコマンドを使用することで、ボリュームグループの状態やサイズ、使用状況を簡単に確認できます。

## 使用法
基本的な構文は以下の通りです。

```
vgs [オプション] [引数]
```

## 一般的なオプション
- `-o`: 表示するフィールドを指定します。
- `--units`: サイズの単位を指定します（例: k, m, g）。
- `-h`: 人間に読みやすい形式で出力します。
- `--noheadings`: ヘッダー行を表示しません。

## 一般的な例
以下に、`vgs` コマンドのいくつかの実用的な例を示します。

### 1. 基本的なボリュームグループ情報の表示
```bash
vgs
```

### 2. 特定のフィールドのみを表示
```bash
vgs -o vg_name,lv_count,vg_size
```

### 3. 人間に読みやすい形式で表示
```bash
vgs -h
```

### 4. ヘッダーなしで表示
```bash
vgs --noheadings
```

### 5. サイズの単位を指定して表示
```bash
vgs --units g
```

## ヒント
- `vgs` コマンドは、LVM を使用しているシステムでのみ機能しますので、事前に LVM がインストールされていることを確認してください。
- ボリュームグループの状態を定期的に確認することで、ディスクの使用状況を把握し、適切な管理を行うことができます。
- スクリプトで使用する場合は、`--noheadings` オプションを使うと、出力をパースしやすくなります。
