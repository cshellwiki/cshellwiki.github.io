# [日本語] Bash wc の使い方: 行数、単語数、バイト数をカウントする

## Overview
`wc` コマンドは、ファイルの行数、単語数、バイト数をカウントするためのツールです。このコマンドを使用することで、テキストファイルの内容を簡単に分析できます。

## Usage
基本的な構文は以下の通りです。

```bash
wc [options] [arguments]
```

## Common Options
- `-l`: 行数をカウントします。
- `-w`: 単語数をカウントします。
- `-c`: バイト数をカウントします。
- `-m`: 文字数をカウントします。
- `-L`: 最長行の文字数を表示します。

## Common Examples
以下は、`wc` コマンドの実用的な例です。

1. **行数をカウントする**
   ```bash
   wc -l filename.txt
   ```

2. **単語数をカウントする**
   ```bash
   wc -w filename.txt
   ```

3. **バイト数をカウントする**
   ```bash
   wc -c filename.txt
   ```

4. **複数のファイルの行数、単語数、バイト数を同時にカウントする**
   ```bash
   wc file1.txt file2.txt
   ```

5. **最長行の文字数を表示する**
   ```bash
   wc -L filename.txt
   ```

## Tips
- 複数のオプションを組み合わせて使用することができます。例えば、行数と単語数を同時にカウントするには、`wc -l -w filename.txt` とします。
- 標準入力からデータを受け取ることも可能です。例えば、`echo "Hello World" | wc -w` で単語数をカウントできます。
- `wc` コマンドは、スクリプト内でのデータ処理やログファイルの分析に非常に便利です。
