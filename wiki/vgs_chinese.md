# [Linux] Bash vgs 用法: 显示卷组信息

## 概述
`vgs` 命令用于显示逻辑卷管理器（LVM）中的卷组信息。它提供了有关系统中所有卷组的详细信息，包括名称、大小、可用空间等。

## 用法
基本语法如下：
```
vgs [options] [arguments]
```

## 常用选项
- `-o`：指定要显示的字段。
- `--units`：设置输出单位，例如 MB 或 GB。
- `-h`：以人类可读的格式显示输出。
- `--noheadings`：不显示标题行。

## 常见示例
以下是一些常见的 `vgs` 命令示例：

1. 显示所有卷组的基本信息：
   ```bash
   vgs
   ```

2. 以人类可读的格式显示卷组信息：
   ```bash
   vgs -h
   ```

3. 只显示特定字段（例如卷组名称和大小）：
   ```bash
   vgs -o vg_name,size
   ```

4. 使用特定单位显示卷组大小：
   ```bash
   vgs --units g
   ```

5. 不显示标题行，只列出卷组信息：
   ```bash
   vgs --noheadings
   ```

## 提示
- 在使用 `vgs` 命令时，可以结合其他 LVM 命令（如 `lvdisplay` 和 `pvdisplay`）来获取更全面的存储信息。
- 定期检查卷组的使用情况，以便及时扩展或调整存储资源。
- 使用 `-o` 选项自定义输出字段，可以帮助您更快速地找到所需信息。
