# [لینوکس] Bash write استفاده: ارسال پیام به کاربر دیگر

## Overview
دستور `write` در Bash به شما این امکان را می‌دهد که پیام‌هایی را به کاربر دیگری که در سیستم شما وارد شده است، ارسال کنید. این دستور به صورت زنده کار می‌کند و به شما اجازه می‌دهد تا ارتباطات فوری با دیگر کاربران برقرار کنید.

## Usage
سینتکس پایه دستور `write` به صورت زیر است:

```
write [options] [user] [tty]
```

## Common Options
- `-n`: این گزینه به شما اجازه می‌دهد تا پیام را بدون انتظار برای تأیید دریافت کنید.
- `-h`: این گزینه به شما کمک می‌کند تا نام کاربر را در پیام نشان ندهید.

## Common Examples
1. ارسال پیام به کاربر `ali`:
   ```bash
   write ali
   ```

2. ارسال پیام به کاربر `reza` در tty خاص:
   ```bash
   write reza pts/1
   ```

3. ارسال پیام بدون انتظار برای تأیید:
   ```bash
   write -n ali
   ```

## Tips
- قبل از استفاده از `write`، مطمئن شوید که کاربر مقصد در حال حاضر وارد سیستم شده است.
- برای خروج از حالت نوشتن، می‌توانید کلید `Ctrl + D` را فشار دهید.
- اگر کاربر مقصد پیام‌های شما را نمی‌بیند، ممکن است که او از حالت "مشغول" استفاده کند.
