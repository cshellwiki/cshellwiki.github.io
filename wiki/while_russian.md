# [Linux] Bash while использование: Цикл выполнения команд

## Обзор
Команда `while` в Bash используется для выполнения набора команд многократно, пока заданное условие истинно. Это позволяет автоматизировать повторяющиеся задачи и обрабатывать данные до тех пор, пока не будет достигнуто определенное состояние.

## Использование
Основной синтаксис команды `while` выглядит следующим образом:

```bash
while [условие ]; do
    # команды
done
```

## Общие параметры
- `[условие]`: Условие, которое проверяется перед каждой итерацией цикла. Если оно истинно, выполняются команды внутри блока `do`.
- `do`: Ключевое слово, указывающее начало блока команд, которые будут выполняться в цикле.
- `done`: Ключевое слово, указывающее конец блока команд.

## Общие примеры

### Пример 1: Простой счетчик
Этот пример демонстрирует, как использовать `while` для создания простого счетчика.

```bash
count=1
while [ $count -le 5 ]; do
    echo "Счетчик: $count"
    ((count++))
done
```

### Пример 2: Чтение из файла
В этом примере `while` используется для построчного чтения содержимого файла.

```bash
while IFS= read -r line; do
    echo "Строка: $line"
done < файл.txt
```

### Пример 3: Ожидание ввода пользователя
Этот пример показывает, как `while` может использоваться для ожидания ввода от пользователя.

```bash
input=""
while [ "$input" != "выход" ]; do
    read -p "Введите 'выход' для завершения: " input
done
```

## Советы
- Убедитесь, что условие в `while` корректно, чтобы избежать бесконечных циклов.
- Используйте `break` для выхода из цикла при выполнении определенных условий.
- Применяйте `sleep` внутри цикла, если необходимо замедлить выполнение для избежания чрезмерной загрузки системы.
