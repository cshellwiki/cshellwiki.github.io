# [Linux] Bash xz Использование: Сжатие и распаковка файлов

## Обзор
Команда `xz` используется для сжатия и распаковки файлов с использованием алгоритма сжатия LZMA. Она позволяет значительно уменьшить размер файлов, что полезно для экономии дискового пространства и ускорения передачи данных.

## Использование
Базовый синтаксис команды выглядит следующим образом:

```bash
xz [options] [arguments]
```

## Общие опции
- `-d`, `--decompress`: Распаковать сжатый файл.
- `-k`, `--keep`: Сохранить оригинальный файл после сжатия.
- `-f`, `--force`: Принудительно перезаписать существующие файлы.
- `-9`: Установить максимальный уровень сжатия (от 1 до 9).
- `-z`: Сжать файл (по умолчанию).

## Общие примеры
1. Сжатие файла:
   ```bash
   xz myfile.txt
   ```
   Это создаст файл `myfile.txt.xz` и удалит оригинальный файл.

2. Распаковка файла:
   ```bash
   xz -d myfile.txt.xz
   ```
   Это восстановит оригинальный файл `myfile.txt`.

3. Сжатие файла с сохранением оригинала:
   ```bash
   xz -k myfile.txt
   ```
   Оригинальный файл останется, а новый сжатый файл будет создан.

4. Принудительное сжатие, перезаписывая существующий файл:
   ```bash
   xz -f myfile.txt
   ```

5. Сжатие с максимальным уровнем:
   ```bash
   xz -9 myfile.txt
   ```

## Советы
- Используйте опцию `-k`, если хотите сохранить оригинальные файлы после сжатия.
- Для больших файлов сжатие может занять время, поэтому учитывайте это при работе с большими объемами данных.
- Проверяйте размер сжатых файлов, чтобы убедиться, что сжатие дало ожидаемый результат.
