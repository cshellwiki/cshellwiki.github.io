# [लिनक्स] Bash vgs का उपयोग: भंडारण समूहों की जानकारी प्राप्त करें

## Overview
`vgs` कमांड का उपयोग भंडारण समूहों (Volume Groups) की जानकारी प्राप्त करने के लिए किया जाता है। यह कमांड Logical Volume Manager (LVM) के अंतर्गत काम करता है और भंडारण समूहों के बारे में महत्वपूर्ण जानकारी प्रदान करता है, जैसे कि उनके आकार, उपलब्ध स्थान, और अन्य विशेषताएँ।

## Usage
`vgs` कमांड की मूल संरचना निम्नलिखित है:

```bash
vgs [options] [arguments]
```

## Common Options
- `-o` : आउटपुट में दिखाई देने वाले फ़ील्ड्स को निर्दिष्ट करता है।
- `-n` : भंडारण समूहों के नामों को केवल दिखाने के लिए उपयोग किया जाता है।
- `--units` : आकार की इकाइयों को निर्दिष्ट करता है, जैसे कि MB, GB आदि।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. सभी भंडारण समूहों की जानकारी प्राप्त करें:
   ```bash
   vgs
   ```

2. विशेष फ़ील्ड्स के साथ भंडारण समूहों की जानकारी दिखाएँ:
   ```bash
   vgs -o vg_name,vg_size,vg_free
   ```

3. केवल भंडारण समूहों के नाम दिखाएँ:
   ```bash
   vgs -n
   ```

4. आकार की इकाइयों को GB में दिखाएँ:
   ```bash
   vgs --units g
   ```

## Tips
- `vgs` कमांड का उपयोग करने से पहले सुनिश्चित करें कि LVM स्थापित और कॉन्फ़िगर किया गया है।
- यदि आप अधिक जानकारी चाहते हैं, तो `man vgs` का उपयोग करके मैन पेज देख सकते हैं।
- नियमित रूप से भंडारण समूहों की स्थिति की जांच करना एक अच्छा अभ्यास है, ताकि आप उपलब्ध स्थान का सही उपयोग कर सकें।
