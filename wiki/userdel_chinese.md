# [Linux] Bash userdel 使用方法: 删除用户账户

## 概述
`userdel` 命令用于从系统中删除用户账户。使用此命令时，您可以选择是否同时删除与该用户相关的主目录和邮件文件。

## 用法
基本语法如下：
```bash
userdel [选项] [用户名]
```

## 常用选项
- `-r`：同时删除用户的主目录及其邮件文件。
- `-f`：强制删除用户，即使用户当前正在登录。
- `-Z`：删除用户时，保留 SELinux 上的用户上下文。

## 常见示例
1. 删除用户账户：
   ```bash
   userdel username
   ```

2. 删除用户账户并同时删除其主目录：
   ```bash
   userdel -r username
   ```

3. 强制删除正在登录的用户：
   ```bash
   userdel -f username
   ```

4. 删除用户账户但保留其主目录：
   ```bash
   userdel username
   ```

## 提示
- 在删除用户之前，请确保备份重要数据，以免丢失。
- 使用 `-f` 选项时要小心，因为这会强制删除用户，即使他们当前正在使用系统。
- 定期检查系统用户账户，以确保没有不再需要的账户存在。
