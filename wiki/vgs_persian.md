# [لینوکس] Bash vgs استفاده: نمایش اطلاعات گروه‌های ولوم

## Overview
دستور `vgs` در Bash برای نمایش اطلاعات مربوط به گروه‌های ولوم (Volume Groups) در سیستم‌های مدیریت حجم منطقی (LVM) استفاده می‌شود. این دستور به کاربران کمک می‌کند تا وضعیت و ویژگی‌های گروه‌های ولوم را مشاهده کنند.

## Usage
سینتکس پایه دستور `vgs` به صورت زیر است:

```bash
vgs [options] [arguments]
```

## Common Options
- `-o`: انتخاب ستون‌های خاص برای نمایش.
- `--units`: تعیین واحدهای نمایش (مانند MB، GB).
- `-h`: نمایش خروجی به صورت قابل خواندن برای انسان (human-readable).
- `--noheadings`: حذف عنوان ستون‌ها از خروجی.

## Common Examples
1. **نمایش اطلاعات کلی گروه‌های ولوم:**
   ```bash
   vgs
   ```

2. **نمایش اطلاعات با واحدهای قابل خواندن:**
   ```bash
   vgs -h
   ```

3. **انتخاب ستون‌های خاص برای نمایش:**
   ```bash
   vgs -o vg_name,lv_count,vg_size
   ```

4. **حذف عنوان ستون‌ها از خروجی:**
   ```bash
   vgs --noheadings
   ```

5. **استفاده از واحدهای خاص:**
   ```bash
   vgs --units g
   ```

## Tips
- برای مشاهده اطلاعات دقیق‌تر، می‌توانید از گزینه‌های مختلف استفاده کنید تا فقط اطلاعات مورد نیاز خود را دریافت کنید.
- استفاده از گزینه `-h` می‌تواند به شما کمک کند تا مقادیر را به راحتی بخوانید و درک کنید.
- اگر به دنبال اطلاعات خاصی هستید، از گزینه `-o` برای فیلتر کردن خروجی استفاده کنید.
