# [ระบบปฏิบัติการ] C Shell (csh) wget การใช้งาน: ดาวน์โหลดไฟล์จากเว็บ

## Overview
คำสั่ง `wget` เป็นเครื่องมือที่ใช้ในการดาวน์โหลดไฟล์จากเว็บโดยอัตโนมัติ มันสามารถดาวน์โหลดไฟล์จาก URL ที่กำหนดได้อย่างง่ายดายและรองรับการดาวน์โหลดไฟล์หลายประเภท รวมถึงไฟล์ HTML, รูปภาพ, และไฟล์อื่น ๆ

## Usage
รูปแบบพื้นฐานของคำสั่ง `wget` คือ:

```
wget [options] [arguments]
```

## Common Options
- `-O <filename>`: กำหนดชื่อไฟล์ที่ต้องการบันทึก
- `-c`: ดำเนินการดาวน์โหลดต่อจากจุดที่หยุด
- `-r`: ดาวน์โหลดไฟล์ทั้งหมดในเว็บไซต์ (recursive)
- `-p`: ดาวน์โหลดไฟล์ที่จำเป็นสำหรับการแสดงผลหน้าเว็บ
- `--limit-rate=<rate>`: จำกัดความเร็วในการดาวน์โหลด

## Common Examples
- ดาวน์โหลดไฟล์จาก URL ที่กำหนด:
  ```bash
  wget http://example.com/file.zip
  ```

- ดาวน์โหลดไฟล์และตั้งชื่อไฟล์ใหม่:
  ```bash
  wget -O newfile.zip http://example.com/file.zip
  ```

- ดาวน์โหลดไฟล์ต่อจากที่หยุด:
  ```bash
  wget -c http://example.com/file.zip
  ```

- ดาวน์โหลดเว็บไซต์ทั้งหมด:
  ```bash
  wget -r http://example.com
  ```

- ดาวน์โหลดไฟล์และจำกัดความเร็วในการดาวน์โหลด:
  ```bash
  wget --limit-rate=200k http://example.com/file.zip
  ```

## Tips
- ใช้ `-c` เพื่อช่วยในการดาวน์โหลดไฟล์ขนาดใหญ่ที่อาจถูกขัดจังหวะ
- ตรวจสอบ URL ให้แน่ใจว่าถูกต้องก่อนการดาวน์โหลด
- ใช้ `-p` เพื่อให้แน่ใจว่าไฟล์ทั้งหมดที่จำเป็นสำหรับการแสดงผลหน้าเว็บจะถูกดาวน์โหลดด้วย
