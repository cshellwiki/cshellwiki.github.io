# [Unix系] C Shell (csh) wait の使い方: プロセスの終了を待つ

## Overview
`wait` コマンドは、シェルスクリプトやコマンドラインで実行中のプロセスが終了するのを待つために使用されます。特に、バックグラウンドで実行されているジョブの完了を確認する際に便利です。

## Usage
基本的な構文は以下の通りです。

```
wait [options] [arguments]
```

## Common Options
- `-n`: 最後に終了したバックグラウンドジョブを待ちます。
- `pid`: 特定のプロセスID（PID）を指定して、そのプロセスが終了するのを待ちます。

## Common Examples
以下に、`wait` コマンドのいくつかの実用的な例を示します。

### 例1: バックグラウンドジョブの終了を待つ
```csh
sleep 10 &
wait
echo "バックグラウンドジョブが終了しました。"
```

### 例2: 特定のプロセスIDを指定して待つ
```csh
sleep 5 &
pid=$!
echo "プロセスID: $pid"
wait $pid
echo "プロセス $pid が終了しました。"
```

### 例3: 最後のバックグラウンドジョブを待つ
```csh
sleep 3 &
sleep 4 &
wait -n
echo "最後のバックグラウンドジョブが終了しました。"
```

## Tips
- `wait` コマンドを使用することで、スクリプトの実行がバックグラウンドジョブの完了を待つようになります。これにより、リソースの競合を避けることができます。
- 複数のバックグラウンドジョブがある場合、`wait` を使ってそれぞれの終了を確認することができます。
- スクリプトのデバッグ時には、`wait` を使ってプロセスの状態を確認することが役立ちます。
