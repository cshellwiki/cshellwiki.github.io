# [Linux] Bash wait 使用法: プロセスの終了を待つ

## 概要
`wait` コマンドは、シェルスクリプトやコマンドラインで実行中のプロセスが終了するのを待つために使用されます。このコマンドは、バックグラウンドで実行されているプロセスの終了を確認し、その終了ステータスを取得するのに役立ちます。

## 使用法
基本的な構文は次のとおりです。

```
wait [options] [arguments]
```

## 一般的なオプション
- `-n` : 次に終了するプロセスを待ちます。
- `-p` : 指定したプロセスIDの終了を待ちます。

## 一般的な例
以下に、`wait` コマンドのいくつかの実用的な例を示します。

### 例 1: バックグラウンドプロセスの終了を待つ
```bash
sleep 5 &
wait
echo "バックグラウンドプロセスが終了しました。"
```

### 例 2: 特定のプロセスIDを待つ
```bash
sleep 5 &
PID=$!
wait $PID
echo "プロセス $PID が終了しました。"
```

### 例 3: 複数のバックグラウンドプロセスを待つ
```bash
sleep 3 &
sleep 5 &
wait
echo "すべてのバックグラウンドプロセスが終了しました。"
```

## ヒント
- `wait` コマンドを使用することで、スクリプトの実行を制御し、特定のプロセスが終了するまで次の処理を待機させることができます。
- プロセスの終了ステータスを確認するために、`wait` コマンドの後に `$?` を使用することができます。
- バックグラウンドで実行されるプロセスが多い場合、`wait` を使ってそれらを一括で管理することができます。
