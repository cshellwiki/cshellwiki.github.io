# [Linux] Bash unxz 使用方法: 解压缩 .xz 文件

## 概述
`unxz` 命令用于解压缩 `.xz` 格式的压缩文件。它是 XZ Utils 工具包的一部分，能够有效地将压缩文件恢复为原始未压缩状态。

## 用法
基本语法如下：
```
unxz [选项] [参数]
```

## 常用选项
- `-k`：保留原始的压缩文件。
- `-f`：强制覆盖已存在的输出文件。
- `-v`：显示详细的处理信息。

## 常见示例
以下是一些常见的 `unxz` 使用示例：

1. 解压缩一个文件：
   ```bash
   unxz example.xz
   ```

2. 解压缩并保留原始文件：
   ```bash
   unxz -k example.xz
   ```

3. 强制解压缩已存在的文件：
   ```bash
   unxz -f example.xz
   ```

4. 解压缩多个文件：
   ```bash
   unxz file1.xz file2.xz
   ```

5. 显示详细信息：
   ```bash
   unxz -v example.xz
   ```

## 提示
- 在使用 `unxz` 时，确保你有足够的磁盘空间来存放解压后的文件。
- 使用 `-k` 选项可以避免意外删除原始压缩文件，特别是在处理重要数据时。
- 如果你需要处理 `.tar.xz` 文件，建议先使用 `tar -xvf` 命令，因为它可以同时解压缩和解包。
