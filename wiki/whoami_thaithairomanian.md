# [ระบบปฏิบัติการ] Bash whoami การใช้งาน: แสดงชื่อผู้ใช้ปัจจุบัน

## Overview
คำสั่ง `whoami` ใช้เพื่อแสดงชื่อผู้ใช้ที่กำลังใช้งานอยู่ในเซสชันปัจจุบัน โดยจะคืนค่าชื่อผู้ใช้ที่เข้าสู่ระบบในขณะนั้น

## Usage
รูปแบบพื้นฐานของคำสั่งคือ:

```bash
whoami [options] [arguments]
```

## Common Options
- ไม่มีตัวเลือกที่สำคัญสำหรับคำสั่ง `whoami` เนื่องจากมันทำงานได้โดยไม่ต้องใช้ตัวเลือกเพิ่มเติม

## Common Examples
ตัวอย่างการใช้งานคำสั่ง `whoami` มีดังนี้:

1. แสดงชื่อผู้ใช้ปัจจุบัน:
   ```bash
   whoami
   ```

2. ใช้ในสคริปต์เพื่อเก็บชื่อผู้ใช้ในตัวแปร:
   ```bash
   current_user=$(whoami)
   echo "Current user is: $current_user"
   ```

3. ใช้ร่วมกับคำสั่งอื่นเพื่อแสดงข้อมูลผู้ใช้:
   ```bash
   echo "You are logged in as: $(whoami)"
   ```

## Tips
- คำสั่ง `whoami` เป็นเครื่องมือที่ง่ายและรวดเร็วในการตรวจสอบชื่อผู้ใช้ที่กำลังใช้งานอยู่
- สามารถใช้ในสคริปต์เพื่อจัดการการเข้าถึงหรือการอนุญาตตามชื่อผู้ใช้
- หากคุณต้องการข้อมูลเพิ่มเติมเกี่ยวกับผู้ใช้ เช่น UID หรือ GID คุณอาจต้องใช้คำสั่ง `id` แทน
