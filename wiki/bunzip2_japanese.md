# [Linux] Bash bunzip2 の使い方: bzip2 圧縮ファイルの解凍

## Overview
`bunzip2` コマンドは、bzip2 形式で圧縮されたファイルを解凍するためのツールです。このコマンドを使用することで、圧縮されたデータを元の状態に戻すことができます。

## Usage
基本的な構文は以下の通りです。

```bash
bunzip2 [options] [arguments]
```

## Common Options
- `-k` : 解凍後も元の圧縮ファイルを保持します。
- `-f` : 既存のファイルを強制的に上書きします。
- `-v` : 詳細情報を表示します。

## Common Examples
以下に、`bunzip2` コマンドのいくつかの実用的な例を示します。

### 例1: 単純な解凍
bzip2 形式のファイル `example.bz2` を解凍します。

```bash
bunzip2 example.bz2
```

### 例2: 解凍後に元のファイルを保持
元のファイルを保持しながら解凍します。

```bash
bunzip2 -k example.bz2
```

### 例3: 詳細情報を表示しながら解凍
解凍プロセスの詳細を表示しながら実行します。

```bash
bunzip2 -v example.bz2
```

## Tips
- 解凍したファイルの名前は、元のファイル名から `.bz2` 拡張子が削除されたものになります。
- 大きなファイルを解凍する際は、十分なディスクスペースがあることを確認してください。
- `bunzip2` は、他の圧縮形式（例えば gzip や zip）には対応していないため、適切なツールを使用する必要があります。
