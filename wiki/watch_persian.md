# [لینوکس] Bash watch استفاده: [نظارت بر خروجی دستورات]

## Overview
دستورات `watch` به شما این امکان را می‌دهد که خروجی یک دستور را به صورت مداوم و در فواصل زمانی مشخص مشاهده کنید. این ابزار برای نظارت بر تغییرات در داده‌ها یا وضعیت سیستم بسیار مفید است.

## Usage
سینتکس پایه دستور `watch` به صورت زیر است:

```bash
watch [options] [arguments]
```

## Common Options
- `-n, --interval`: تعیین فاصله زمانی (به ثانیه) بین هر بار اجرای دستور.
- `-d, --differences`: تغییرات بین دو خروجی را هایلایت می‌کند.
- `-t, --no-title`: عنوان پنجره را نمایش نمی‌دهد.
- `-x`: اجرای دستور در یک شل جدید.

## Common Examples

### 1. نظارت بر استفاده از CPU
برای مشاهده مداوم استفاده از CPU می‌توانید از دستور زیر استفاده کنید:

```bash
watch -n 2 mpstat
```

این دستور هر ۲ ثانیه اطلاعات مربوط به استفاده از CPU را نمایش می‌دهد.

### 2. نظارت بر تغییرات در یک دایرکتوری
برای مشاهده تغییرات در یک دایرکتوری خاص، می‌توانید از دستور `ls` استفاده کنید:

```bash
watch -n 5 ls -l /path/to/directory
```

این دستور هر ۵ ثانیه محتوای دایرکتوری مشخص شده را نمایش می‌دهد.

### 3. نظارت بر وضعیت شبکه
برای نظارت بر وضعیت شبکه، می‌توانید از دستور `ifconfig` استفاده کنید:

```bash
watch -n 1 ifconfig
```

این دستور هر ثانیه وضعیت شبکه را به‌روز می‌کند.

## Tips
- از گزینه `-d` برای هایلایت کردن تغییرات استفاده کنید تا متوجه تغییرات سریع‌تر شوید.
- می‌توانید از `-t` برای حذف عنوان پنجره استفاده کنید تا فضای بیشتری برای خروجی داشته باشید.
- برای دستورات طولانی، فاصله زمانی بیشتری را انتخاب کنید تا خروجی‌ها به راحتی قابل خواندن باشند.
