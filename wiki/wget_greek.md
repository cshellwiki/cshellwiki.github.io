# [Λειτουργικό Σύστημα] Bash wget Χρήση: Λήψη αρχείων από το διαδίκτυο

## Overview
Η εντολή `wget` είναι ένα εργαλείο γραμμής εντολών που χρησιμοποιείται για τη λήψη αρχείων από το διαδίκτυο. Υποστηρίζει HTTP, HTTPS και FTP πρωτόκολλα και είναι ιδιαίτερα χρήσιμη για την αυτόματη λήψη περιεχομένου από ιστότοπους.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [file]`: Καθορίζει το όνομα του αρχείου που θα αποθηκευτεί.
- `-q`: Λειτουργία σιωπηλής, χωρίς εκτύπωση μηνυμάτων.
- `-c`: Συνεχίζει μια διακοπείσα λήψη.
- `-r`: Λήψη αρχείων αναδρομικά (για ιστότοπους).
- `--limit-rate=[rate]`: Περιορίζει την ταχύτητα λήψης.

## Common Examples
### Λήψη ενός αρχείου
```bash
wget https://example.com/file.zip
```

### Λήψη αρχείου με καθορισμένο όνομα
```bash
wget -O myfile.zip https://example.com/file.zip
```

### Λήψη αρχείων αναδρομικά
```bash
wget -r https://example.com/directory/
```

### Συνεχίζοντας μια διακοπείσα λήψη
```bash
wget -c https://example.com/file.zip
```

### Περιορισμός ταχύτητας λήψης
```bash
wget --limit-rate=200k https://example.com/file.zip
```

## Tips
- Χρησιμοποιήστε την επιλογή `-q` για να κάνετε την λήψη λιγότερο ενοχλητική αν δεν χρειάζεστε πληροφορίες.
- Ελέγξτε αν η διεύθυνση URL είναι σωστή πριν ξεκινήσετε τη λήψη για να αποφύγετε σφάλματα.
- Για μεγάλες λήψεις, σκεφτείτε να χρησιμοποιήσετε την επιλογή `-c` για να μην χάσετε δεδομένα σε περίπτωση διακοπής.
