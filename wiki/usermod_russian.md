# [Linux] Bash usermod <Использование>: Управление учетными записями пользователей

## Обзор
Команда `usermod` используется для изменения учетных записей пользователей в системе Linux. С ее помощью администраторы могут редактировать различные параметры, такие как имя пользователя, домашний каталог, группы и другие атрибуты.

## Использование
Основной синтаксис команды `usermod` выглядит следующим образом:

```bash
usermod [options] [arguments]
```

## Общие опции
- `-a, --append`: Добавляет пользователя в указанные группы, не удаляя его из других групп.
- `-G, --groups`: Указывает группы, в которые должен быть добавлен пользователь.
- `-d, --home`: Задает новый домашний каталог для пользователя.
- `-l, --login`: Изменяет имя пользователя.
- `-s, --shell`: Указывает новую оболочку для пользователя.
- `-u, --uid`: Изменяет идентификатор пользователя (UID).

## Общие примеры
1. Изменение имени пользователя:
   ```bash
   usermod -l новое_имя старое_имя
   ```

2. Изменение домашнего каталога:
   ```bash
   usermod -d /новый/каталог имя_пользователя
   ```

3. Добавление пользователя в группу:
   ```bash
   usermod -aG имя_группы имя_пользователя
   ```

4. Изменение оболочки пользователя:
   ```bash
   usermod -s /bin/bash имя_пользователя
   ```

5. Изменение UID пользователя:
   ```bash
   usermod -u новый_uid имя_пользователя
   ```

## Советы
- Всегда создавайте резервные копии конфигурационных файлов перед внесением изменений.
- Используйте опцию `-a` при добавлении пользователя в группы, чтобы избежать удаления его из других групп.
- После изменения имени пользователя убедитесь, что все файлы и каталоги, принадлежащие этому пользователю, также обновлены.
- Проверяйте изменения с помощью команды `id имя_пользователя`, чтобы убедиться, что все параметры установлены правильно.
