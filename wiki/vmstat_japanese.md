# [Linux] Bash vmstat の使い方: システムのパフォーマンスを監視する

## 概要
`vmstat` コマンドは、システムのメモリ、プロセス、入出力、CPUの使用状況を監視するためのツールです。このコマンドを使用することで、システムのパフォーマンスをリアルタイムで把握することができます。

## 使用法
基本的な構文は以下の通りです。

```bash
vmstat [options] [interval] [count]
```

## 一般的なオプション
- `-a`: アクティブおよび非アクティブなメモリを表示します。
- `-m`: メモリのページング統計を表示します。
- `-s`: システムの統計を表示します。
- `-d`: ディスクの統計を表示します。
- `interval`: データを表示する間隔（秒）を指定します。
- `count`: 表示する回数を指定します。

## 一般的な例
以下にいくつかの実用的な例を示します。

### 1. 基本的な使用法
システムのパフォーマンスを1秒ごとに表示するには、次のコマンドを使用します。

```bash
vmstat 1
```

### 2. 特定の回数だけ表示
5回だけシステムの状態を表示する場合は、次のようにします。

```bash
vmstat 1 5
```

### 3. メモリの詳細を表示
アクティブおよび非アクティブなメモリを確認するには、`-a` オプションを使用します。

```bash
vmstat -a
```

### 4. ディスクの統計を表示
ディスクの入出力統計を確認するには、`-d` オプションを使用します。

```bash
vmstat -d
```

## ヒント
- `vmstat` は、システムのボトルネックを特定するのに役立ちます。CPU、メモリ、入出力の使用状況をチェックして、どこに問題があるかを見極めましょう。
- 定期的に `vmstat` を実行し、ログを取ることで、システムのパフォーマンスのトレンドを把握することができます。
- 他の監視ツール（例: `top` や `htop`）と併用することで、より詳細な情報を得ることができます。
