# [لینوکس] Bash usermod کاربرد: مدیریت کاربران در سیستم

## Overview
دستور `usermod` در لینوکس برای تغییر اطلاعات یک کاربر موجود در سیستم استفاده می‌شود. این دستور به مدیران سیستم این امکان را می‌دهد که ویژگی‌های مختلف حساب‌های کاربری را تغییر دهند، از جمله گروه‌ها، نام‌ها و دیگر مشخصات.

## Usage
سینتکس پایه دستور `usermod` به صورت زیر است:

```bash
usermod [options] [arguments]
```

## Common Options
- `-aG`: اضافه کردن کاربر به گروه‌های اضافی بدون حذف از گروه‌های قبلی.
- `-d`: تغییر دایرکتوری خانگی کاربر.
- `-l`: تغییر نام کاربری.
- `-g`: تغییر گروه اصلی کاربر.
- `-s`: تغییر شل پیش‌فرض کاربر.

## Common Examples
### 1. اضافه کردن کاربر به گروه
برای اضافه کردن کاربر به گروهی به نام `developers`، از دستور زیر استفاده کنید:

```bash
usermod -aG developers username
```

### 2. تغییر نام کاربری
برای تغییر نام کاربری از `oldname` به `newname`، از این دستور استفاده کنید:

```bash
usermod -l newname oldname
```

### 3. تغییر دایرکتوری خانگی
برای تغییر دایرکتوری خانگی کاربر به `/home/newdir`، از دستور زیر استفاده کنید:

```bash
usermod -d /home/newdir username
```

### 4. تغییر شل پیش‌فرض
برای تغییر شل پیش‌فرض کاربر به `/bin/bash`، از این دستور استفاده کنید:

```bash
usermod -s /bin/bash username
```

## Tips
- همیشه قبل از استفاده از `usermod` از اطلاعات کاربر پشتیبان‌گیری کنید.
- از گزینه `-aG` برای اضافه کردن کاربر به گروه‌های جدید استفاده کنید تا کاربر از گروه‌های قبلی حذف نشود.
- برای اعمال تغییرات، ممکن است نیاز باشد که کاربر دوباره وارد سیستم شود.
