# [سیستم‌عامل] C Shell (csh) usermod استفاده: [مدیریت کاربران]

## Overview
دستورات `usermod` برای تغییر اطلاعات کاربری در سیستم‌های یونیکس و لینوکس استفاده می‌شود. این دستور به مدیر سیستم اجازه می‌دهد تا ویژگی‌های مختلف یک کاربر را مانند نام، گروه، و دیگر تنظیمات را تغییر دهد.

## Usage
سینتکس پایه دستور `usermod` به صورت زیر است:

```
usermod [options] [arguments]
```

## Common Options
- `-l`: تغییر نام کاربری.
- `-d`: تغییر دایرکتوری خانگی کاربر.
- `-g`: تغییر گروه اصلی کاربر.
- `-aG`: اضافه کردن کاربر به گروه‌های اضافی.
- `-s`: تغییر شل پیش‌فرض کاربر.

## Common Examples
### تغییر نام کاربری
برای تغییر نام کاربری از `oldname` به `newname`:
```bash
usermod -l newname oldname
```

### تغییر دایرکتوری خانگی
برای تغییر دایرکتوری خانگی کاربر به `/new/home`:
```bash
usermod -d /new/home username
```

### تغییر گروه اصلی
برای تغییر گروه اصلی کاربر به `newgroup`:
```bash
usermod -g newgroup username
```

### اضافه کردن کاربر به گروه‌های اضافی
برای اضافه کردن کاربر به گروه `group1` و `group2`:
```bash
usermod -aG group1,group2 username
```

### تغییر شل پیش‌فرض
برای تغییر شل پیش‌فرض کاربر به `/bin/bash`:
```bash
usermod -s /bin/bash username
```

## Tips
- قبل از استفاده از `usermod`، بهتر است از اطلاعات کاربر پشتیبان‌گیری کنید.
- برای اعمال تغییرات، ممکن است نیاز به ورود مجدد کاربر باشد.
- از گزینه `-aG` برای اضافه کردن کاربر به گروه‌های جدید استفاده کنید تا گروه‌های قبلی حذف نشوند.
