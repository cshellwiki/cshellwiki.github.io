# [Linux] Bash write использование: отправка сообщений пользователям

## Обзор
Команда `write` в Bash используется для отправки текстовых сообщений другим пользователям, которые в данный момент вошли в систему. Это позволяет быстро обмениваться сообщениями в реальном времени.

## Использование
Основной синтаксис команды выглядит следующим образом:

```bash
write [пользователь] [терминал]
```

Если терминал не указан, `write` отправит сообщение на стандартный терминал пользователя.

## Общие опции
- `-n`: Не отображать имя отправителя в начале сообщения.
- `-s`: Отправить сообщение в "тихом" режиме, не отображая его на экране получателя.

## Общие примеры
1. Отправка сообщения пользователю `ivan` на его стандартный терминал:
   ```bash
   write ivan
   Привет, как дела?
   ```

2. Отправка сообщения пользователю `maria` на терминал `pts/1`:
   ```bash
   write maria pts/1
   Увидимся на встрече в 3 часа!
   ```

3. Использование опции `-n` для отправки сообщения без имени отправителя:
   ```bash
   write -n ivan
   Важно: проверь, пожалуйста, почту!
   ```

## Советы
- Убедитесь, что получатель находится в системе и не отключил прием сообщений (с помощью команды `mesg`).
- Используйте `Ctrl+D` для завершения ввода сообщения и отправки его получателю.
- Если вы хотите отправить более длинные сообщения, рассмотрите возможность использования текстового редактора и отправки файла через `cat` или `less`.
