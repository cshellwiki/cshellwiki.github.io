# [리눅스] C Shell (csh) whoami 사용법: 현재 사용자 이름 출력

## 개요
`whoami` 명령은 현재 로그인한 사용자의 이름을 출력하는 간단한 명령입니다. 이 명령은 사용자가 시스템에 로그인한 계정을 확인할 때 유용합니다.

## 사용법
기본 구문은 다음과 같습니다.

```
whoami [옵션] [인수]
```

## 일반 옵션
- `--help`: 사용법에 대한 도움말을 표시합니다.
- `--version`: 현재 `whoami` 명령의 버전을 표시합니다.

## 일반 예제
1. 현재 사용자 이름 확인하기:
   ```csh
   whoami
   ```

2. 도움말 보기:
   ```csh
   whoami --help
   ```

3. 버전 정보 확인하기:
   ```csh
   whoami --version
   ```

## 팁
- `whoami` 명령은 스크립트에서 현재 사용자 정보를 확인할 때 유용합니다.
- 여러 사용자 계정이 있는 시스템에서 작업할 때, 이 명령을 사용하여 현재 작업 중인 계정을 항상 확인하세요.
