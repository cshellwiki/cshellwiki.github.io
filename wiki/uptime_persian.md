# [لینوکس] Bash uptime معادل استفاده: [نمایش زمان فعالیت سیستم]

## Overview
دستور `uptime` در Bash برای نمایش زمان فعالیت سیستم و بار فعلی آن استفاده می‌شود. این دستور اطلاعاتی درباره مدت زمان روشن بودن سیستم، تعداد کاربران فعال و بار متوسط سیستم در ۱، ۵ و ۱۵ دقیقه گذشته ارائه می‌دهد.

## Usage
سینتکس پایه دستور `uptime` به شکل زیر است:

```bash
uptime [options]
```

## Common Options
- `-p` : نمایش مدت زمان فعالیت به صورت خوانا.
- `-s` : نمایش زمان شروع فعالیت سیستم.
- `-h` : نمایش راهنما و گزینه‌های موجود.

## Common Examples
### مثال ۱: نمایش زمان فعالیت سیستم
برای مشاهده زمان فعالیت سیستم و بار فعلی، کافیست دستور زیر را اجرا کنید:

```bash
uptime
```

### مثال ۲: نمایش مدت زمان فعالیت به صورت خوانا
برای نمایش مدت زمان فعالیت به صورت خوانا، از گزینه `-p` استفاده کنید:

```bash
uptime -p
```

### مثال ۳: نمایش زمان شروع فعالیت سیستم
برای مشاهده زمان شروع فعالیت سیستم، از گزینه `-s` استفاده کنید:

```bash
uptime -s
```

## Tips
- استفاده از `uptime` به شما کمک می‌کند تا بفهمید آیا سیستم شما تحت بار سنگین است یا خیر.
- این دستور می‌تواند به عنوان یک ابزار مفید برای نظارت بر عملکرد سرورهای شما باشد.
- می‌توانید این دستور را به صورت دوره‌ای در یک اسکریپت قرار دهید تا وضعیت سیستم را به صورت مداوم زیر نظر داشته باشید.
