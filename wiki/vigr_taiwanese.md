# [Linux] Bash vigr 使用方法: 編輯系統的群組檔案

## Overview
`vigr` 是一個用於安全編輯系統群組檔案的命令。它提供了一個編輯器介面，讓使用者能夠方便地修改 `/etc/group` 檔案，並在編輯過程中自動檢查語法錯誤，以避免損壞檔案。

## Usage
基本語法如下：
```bash
vigr [options] [arguments]
```

## Common Options
- `-c`：檢查群組檔案的語法。
- `-s`：以安全模式啟動編輯器，防止其他使用者同時編輯。
- `-h`：顯示幫助信息。

## Common Examples
1. **基本使用**：直接編輯群組檔案。
   ```bash
   vigr
   ```

2. **使用安全模式編輯**：
   ```bash
   vigr -s
   ```

3. **檢查群組檔案的語法**：
   ```bash
   vigr -c
   ```

## Tips
- 在編輯之前，建議備份 `/etc/group` 檔案，以防不小心刪除或修改重要內容。
- 使用 `-s` 選項可以避免其他使用者同時編輯，這樣可以減少衝突的風險。
- 熟悉編輯器的基本操作（如 vi 或 nano），以提高編輯效率。
