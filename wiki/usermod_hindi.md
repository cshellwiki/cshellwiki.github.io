# [Linux] Bash usermod उपयोग: उपयोगकर्ता खातों को संशोधित करना

## Overview
`usermod` कमांड का उपयोग Linux में उपयोगकर्ता खातों को संशोधित करने के लिए किया जाता है। यह आपको मौजूदा उपयोगकर्ता की जानकारी को बदलने की अनुमति देता है, जैसे कि उपयोगकर्ता का नाम, समूह, होम निर्देशिका, और अधिक।

## Usage
कमांड का मूल सिंटैक्स इस प्रकार है:
```
usermod [options] [arguments]
```

## Common Options
- `-l, --login NEW_LOGIN` : उपयोगकर्ता का नाम बदलें।
- `-d, --home NEW_HOME` : उपयोगकर्ता की होम निर्देशिका बदलें।
- `-m, --move-home` : उपयोगकर्ता की होम निर्देशिका को नए स्थान पर ले जाएं।
- `-G, --groups GROUP1[,GROUP2,...]` : उपयोगकर्ता को नए समूहों में जोड़ें।
- `-a, --append` : उपयोगकर्ता को मौजूदा समूहों में जोड़ें (बिना हटाए)।

## Common Examples
1. उपयोगकर्ता का नाम बदलना:
   ```bash
   usermod -l नया_उपयोगकर्ता पुराना_उपयोगकर्ता
   ```

2. उपयोगकर्ता की होम निर्देशिका बदलना:
   ```bash
   usermod -d /नया/पथ/होम उपयोगकर्ता
   ```

3. उपयोगकर्ता को नए समूह में जोड़ना:
   ```bash
   usermod -aG समूह नाम
   ```

4. उपयोगकर्ता की होम निर्देशिका को नए स्थान पर ले जाना:
   ```bash
   usermod -m -d /नया/पथ/होम उपयोगकर्ता
   ```

## Tips
- हमेशा सुनिश्चित करें कि आप `usermod` कमांड का उपयोग करने से पहले आवश्यक बैकअप ले लें।
- उपयोगकर्ता नाम और समूह नाम में बदलाव करते समय, सुनिश्चित करें कि नए नाम पहले से मौजूद न हों।
- `usermod` का उपयोग करने के लिए आपको सुपरयूजर (root) या sudo अधिकारों की आवश्यकता होती है।
