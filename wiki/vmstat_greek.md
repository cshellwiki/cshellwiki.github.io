# [Λειτουργικό Σύστημα] Bash vmstat Χρήση: Παρακολούθηση της απόδοσης του συστήματος

## Overview
Η εντολή `vmstat` (Virtual Memory Statistics) χρησιμοποιείται για την παρακολούθηση της απόδοσης του συστήματος, παρέχοντας πληροφορίες σχετικά με τη μνήμη, τη διαδικασία, την είσοδο/έξοδο και τη χρήση του CPU. Είναι ένα χρήσιμο εργαλείο για την ανάλυση της απόδοσης και την επίλυση προβλημάτων στο Linux.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
vmstat [options] [interval] [count]
```

## Common Options
- `-a`: Εμφανίζει πληροφορίες για τη μνήμη, συμπεριλαμβανομένων των ελεύθερων και ανενεργών σελίδων.
- `-m`: Εμφανίζει στατιστικά για τη μνήμη που χρησιμοποιείται από τα cache.
- `-s`: Εμφανίζει συνοπτικά στατιστικά για τη μνήμη και τις διαδικασίες.
- `-d`: Εμφανίζει στατιστικά για τις συσκευές δίσκου.
- `interval`: Ο χρόνος σε δευτερόλεπτα μεταξύ των αναφορών.
- `count`: Ο αριθμός των αναφορών που θα εμφανιστούν.

## Common Examples
### Παράδειγμα 1: Βασική χρήση
Για να δείτε τις στατιστικές του συστήματος σε πραγματικό χρόνο κάθε 5 δευτερόλεπτα, μπορείτε να χρησιμοποιήσετε την εντολή:

```bash
vmstat 5
```

### Παράδειγμα 2: Στατιστικά μνήμης
Για να δείτε λεπτομέρειες σχετικά με τη μνήμη, μπορείτε να χρησιμοποιήσετε:

```bash
vmstat -a 5 3
```

Αυτό θα εμφανίσει στατιστικά μνήμης κάθε 5 δευτερόλεπτα για 3 φορές.

### Παράδειγμα 3: Στατιστικά δίσκου
Για να δείτε στατιστικά για τις συσκευές δίσκου, μπορείτε να εκτελέσετε:

```bash
vmstat -d 5
```

### Παράδειγμα 4: Συνοπτικά στατιστικά
Για να δείτε συνοπτικά στατιστικά για τη μνήμη και τις διαδικασίες, χρησιμοποιήστε:

```bash
vmstat -s
```

## Tips
- Χρησιμοποιήστε την επιλογή `-a` για να έχετε μια πιο ολοκληρωμένη εικόνα της μνήμης του συστήματος.
- Συνδυάστε την `vmstat` με άλλες εντολές όπως `top` ή `iostat` για μια πιο λεπτομερή ανάλυση της απόδοσης.
- Εξετάστε τη χρήση της εντολής σε σενάρια αυτοματισμού για την παρακολούθηση της απόδοσης του συστήματος σε τακτική βάση.
