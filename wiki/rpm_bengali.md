# [লিনাক্স] Bash rpm ব্যবহার: প্যাকেজ ম্যানেজমেন্টের জন্য একটি শক্তিশালী টুল

## Overview
`rpm` কমান্ডটি লিনাক্সে প্যাকেজ ম্যানেজমেন্টের জন্য ব্যবহৃত হয়। এটি RPM (Red Hat Package Manager) ফরম্যাটে প্যাকেজ ইনস্টল, অপসারণ, আপডেট এবং তথ্য প্রদর্শনের জন্য ব্যবহৃত হয়।

## Usage
`rpm` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
rpm [options] [arguments]
```

## Common Options
- `-i`: নতুন প্যাকেজ ইনস্টল করতে ব্যবহৃত হয়।
- `-e`: একটি প্যাকেজ অপসারণ করতে ব্যবহৃত হয়।
- `-U`: একটি প্যাকেজ আপডেট করতে ব্যবহৃত হয়।
- `-q`: একটি প্যাকেজের তথ্য অনুসন্ধান করতে ব্যবহৃত হয়।
- `-v`: বিস্তারিত আউটপুট প্রদর্শন করতে ব্যবহৃত হয়।
- `--help`: কমান্ডের সাহায্য তথ্য প্রদর্শন করতে ব্যবহৃত হয়।

## Common Examples
### 1. প্যাকেজ ইনস্টল করা
নতুন RPM প্যাকেজ ইনস্টল করতে:

```bash
rpm -i package-name.rpm
```

### 2. প্যাকেজ অপসারণ করা
একটি প্যাকেজ অপসারণ করতে:

```bash
rpm -e package-name
```

### 3. প্যাকেজ আপডেট করা
একটি প্যাকেজ আপডেট করতে:

```bash
rpm -U package-name.rpm
```

### 4. প্যাকেজের তথ্য দেখা
একটি প্যাকেজের তথ্য দেখতে:

```bash
rpm -q package-name
```

### 5. ইনস্টল করা প্যাকেজের তালিকা দেখা
সকল ইনস্টল করা প্যাকেজের তালিকা দেখতে:

```bash
rpm -qa
```

## Tips
- প্যাকেজ ইনস্টল করার সময় সবসময় নিশ্চিত হন যে আপনি সঠিক প্যাকেজ ফাইলটি ব্যবহার করছেন।
- `-v` অপশন ব্যবহার করে বিস্তারিত আউটপুট পেতে পারেন, যা সমস্যা সমাধানে সহায়ক হতে পারে।
- RPM প্যাকেজের নির্ভরতা যাচাই করতে `yum` বা `dnf` ব্যবহার করা যেতে পারে, কারণ তারা স্বয়ংক্রিয়ভাবে নির্ভরতা ম্যানেজ করে।
