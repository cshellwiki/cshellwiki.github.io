# [לינוקס] Bash yum שימוש: ניהול חבילות במערכת

## Overview
הפקודה `yum` (Yellowdog Updater Modified) היא כלי לניהול חבילות במערכות הפעלה מבוססות לינוקס. היא מאפשרת למשתמשים להתקין, לעדכן ולהסיר חבילות תוכנה בקלות.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
yum [אפשרויות] [ארגומנטים]
```

## Common Options
- `install`: התקנת חבילה חדשה.
- `remove`: הסרת חבילה מהמערכת.
- `update`: עדכון חבילה קיימת לגרסה החדשה ביותר.
- `list`: הצגת רשימת חבילות זמינות או מותקנות.
- `search`: חיפוש חבילות לפי מילות מפתח.

## Common Examples
- **התקנת חבילה**:
```bash
yum install httpd
```
הפקודה הזו מתקינה את שרת ה-HTTP Apache.

- **הסרת חבילה**:
```bash
yum remove httpd
```
הפקודה הזו מסירה את שרת ה-HTTP Apache מהמערכת.

- **עדכון חבילות**:
```bash
yum update
```
הפקודה הזו מעדכנת את כל החבילות המותקנות לגרסה החדשה ביותר.

- **חיפוש חבילה**:
```bash
yum search vim
```
הפקודה הזו מחפשת חבילות שקשורות לעריכת טקסט עם Vim.

- **רשימת חבילות מותקנות**:
```bash
yum list installed
```
הפקודה הזו מציגה את כל החבילות המותקנות במערכת.

## Tips
- תמיד כדאי לבדוק אם יש עדכונים זמינים לפני התקנת חבילות חדשות.
- השתמש באפשרות `-y` כדי לאשר אוטומטית את ההתקנה או ההסרה של חבילות.
- שמור על מערכת מעודכנת כדי להבטיח אבטחה וביצועים טובים.
