# [Linux] Bash vigr: [แก้ไขไฟล์ /etc/hosts ด้วย vi]

## Overview
คำสั่ง `vigr` เป็นเครื่องมือที่ใช้ในการแก้ไขไฟล์ที่เกี่ยวข้องกับการตั้งค่าของระบบ เช่น `/etc/hosts` และ `/etc/passwd` โดยจะเปิดไฟล์เหล่านี้ในโปรแกรมแก้ไขข้อความ `vi` หรือ `vim` ซึ่งช่วยให้ผู้ใช้สามารถทำการแก้ไขไฟล์เหล่านี้ได้อย่างปลอดภัยและมีการตรวจสอบความถูกต้องก่อนที่จะบันทึกการเปลี่ยนแปลง

## Usage
รูปแบบพื้นฐานของคำสั่ง `vigr` คือ:

```bash
vigr [options] [arguments]
```

## Common Options
- `-h`, `--help`: แสดงข้อมูลช่วยเหลือเกี่ยวกับการใช้งานคำสั่ง
- `-o`, `--editor`: กำหนดโปรแกรมแก้ไขข้อความที่ต้องการใช้แทน `vi`
- `-s`, `--safe`: เปิดไฟล์ในโหมดที่ปลอดภัย ซึ่งจะทำการตรวจสอบความถูกต้องของไฟล์ก่อนที่จะบันทึก

## Common Examples
1. แก้ไขไฟล์ `/etc/hosts`:
   ```bash
   vigr /etc/hosts
   ```

2. แก้ไขไฟล์ `/etc/passwd`:
   ```bash
   vigr /etc/passwd
   ```

3. ใช้โปรแกรมแก้ไขข้อความอื่น เช่น `nano`:
   ```bash
   vigr -o nano /etc/hosts
   ```

4. เปิดในโหมดที่ปลอดภัย:
   ```bash
   vigr -s /etc/hosts
   ```

## Tips
- ควรใช้ `vigr` แทนการใช้ `vi` หรือ `vim` โดยตรงเมื่อทำการแก้ไขไฟล์ระบบ เพื่อให้แน่ใจว่ามีการตรวจสอบความถูกต้อง
- หากคุณไม่คุ้นเคยกับ `vi` หรือ `vim` คุณสามารถตั้งค่าให้ใช้โปรแกรมแก้ไขที่คุณถนัดมากกว่า
- ก่อนทำการแก้ไขไฟล์ระบบ ควรทำการสำรองข้อมูลไฟล์เดิมไว้เสมอ เพื่อป้องกันการสูญเสียข้อมูลที่สำคัญ
