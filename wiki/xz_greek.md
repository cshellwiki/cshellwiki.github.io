# [Λειτουργικό Σύστημα] Bash xz Χρήση: Συμπίεση και αποσυμπίεση αρχείων

## Overview
Η εντολή `xz` χρησιμοποιείται για τη συμπίεση και αποσυμπίεση αρχείων, προσφέροντας υψηλή αναλογία συμπίεσης. Είναι ιδανική για τη μείωση του μεγέθους των αρχείων, καθιστώντας τα πιο εύκολα στη μεταφορά και αποθήκευση.

## Usage
Η βασική σύνταξη της εντολής είναι:

```bash
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: Αποσυμπιέζει το αρχείο.
- `-k`, `--keep`: Διατηρεί το αρχικό αρχείο μετά τη συμπίεση.
- `-f`, `--force`: Υποχρεώνει την αντικατάσταση των αρχείων χωρίς προειδοποίηση.
- `-9`: Χρησιμοποιεί τη μέγιστη συμπίεση.
- `-1` έως `-9`: Ρυθμίζει το επίπεδο συμπίεσης από το ελάχιστο (1) έως το μέγιστο (9).

## Common Examples
### Συμπίεση ενός αρχείου
Για να συμπιέσετε ένα αρχείο με όνομα `example.txt`, χρησιμοποιήστε την παρακάτω εντολή:

```bash
xz example.txt
```

### Αποσυμπίεση ενός αρχείου
Για να αποσυμπιέσετε ένα αρχείο `example.txt.xz`, χρησιμοποιήστε:

```bash
xz -d example.txt.xz
```

### Συμπίεση με διατήρηση του αρχικού αρχείου
Για να συμπιέσετε το αρχείο `example.txt` και να διατηρήσετε το αρχικό, χρησιμοποιήστε:

```bash
xz -k example.txt
```

### Συμπίεση με μέγιστη αναλογία
Για να συμπιέσετε ένα αρχείο με τη μέγιστη αναλογία συμπίεσης, μπορείτε να χρησιμοποιήσετε:

```bash
xz -9 example.txt
```

## Tips
- Χρησιμοποιήστε την επιλογή `-k` αν θέλετε να διατηρήσετε το αρχικό αρχείο κατά τη διάρκεια της συμπίεσης.
- Για καλύτερη απόδοση, δοκιμάστε διαφορετικά επίπεδα συμπίεσης ανάλογα με τις ανάγκες σας.
- Να είστε προσεκτικοί με την επιλογή `-f`, καθώς μπορεί να αντικαταστήσει αρχεία χωρίς προειδοποίηση.
