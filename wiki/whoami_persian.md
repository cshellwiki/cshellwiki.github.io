# [لینوکس] Bash whoami استفاده: [نمایش نام کاربری فعلی]

## Overview
دستور `whoami` در Bash به شما این امکان را می‌دهد که نام کاربری فعلی خود را در سیستم عامل مشاهده کنید. این دستور به ویژه در مواقعی که در محیط‌های چند کاربره کار می‌کنید، مفید است.

## Usage
سینتکس پایه دستور `whoami` به صورت زیر است:

```bash
whoami [options] [arguments]
```

## Common Options
دستور `whoami` معمولاً گزینه‌های زیادی ندارد، اما چند گزینه رایج شامل موارد زیر است:

- `--help`: نمایش راهنمای استفاده از دستور.
- `--version`: نمایش نسخه‌ی فعلی دستور.

## Common Examples
در اینجا چند مثال عملی از استفاده‌ی دستور `whoami` آورده شده است:

1. **نمایش نام کاربری فعلی:**
   ```bash
   whoami
   ```

2. **نمایش راهنمای استفاده:**
   ```bash
   whoami --help
   ```

3. **نمایش نسخه‌ی دستور:**
   ```bash
   whoami --version
   ```

## Tips
- از دستور `whoami` در اسکریپت‌ها برای تأیید نام کاربری جاری استفاده کنید.
- اگر در محیط‌های چند کاربره کار می‌کنید، می‌توانید از این دستور برای شناسایی خود استفاده کنید.
- به یاد داشته باشید که این دستور فقط نام کاربری را نمایش می‌دهد و اطلاعات دیگری ارائه نمی‌دهد.
