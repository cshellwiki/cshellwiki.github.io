# [Linux] Bash xz 使用法: ファイルの圧縮と解凍

## Overview
`xz` コマンドは、ファイルを高圧縮率で圧縮および解凍するためのツールです。特に、データのサイズを小さくすることが求められる場合に便利です。

## Usage
基本的な構文は次のとおりです。

```
xz [オプション] [引数]
```

## Common Options
- `-d` または `--decompress`: 圧縮されたファイルを解凍します。
- `-k` または `--keep`: 元のファイルを保持したまま圧縮または解凍します。
- `-f` または `--force`: 既存のファイルを上書きします。
- `-z`: 圧縮を行います（デフォルトの動作）。
- `-9`: 最大圧縮率を指定します。

## Common Examples
以下に、`xz` コマンドのいくつかの実用的な例を示します。

### ファイルを圧縮する
```bash
xz myfile.txt
```

### 圧縮されたファイルを解凍する
```bash
xz -d myfile.txt.xz
```

### 元のファイルを保持して圧縮する
```bash
xz -k myfile.txt
```

### 最大圧縮率で圧縮する
```bash
xz -9 myfile.txt
```

### 複数のファイルを圧縮する
```bash
xz file1.txt file2.txt file3.txt
```

## Tips
- 圧縮率を調整するために、`-1` から `-9` のオプションを使用してみてください。`-1` は速いが圧縮率が低く、`-9` は遅いが圧縮率が高いです。
- 大きなファイルを圧縮する際は、`-k` オプションを使って元のファイルを保持することで、データの損失を防げます。
- スクリプトでの使用時には、エラーハンドリングを考慮して、コマンドの実行結果を確認することをお勧めします。
