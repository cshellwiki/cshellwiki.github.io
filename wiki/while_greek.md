# [Λειτουργικό Σύστημα] Bash while χρήση: Επαναλαμβάνει εντολές όσο μια συνθήκη είναι αληθής

## Overview
Η εντολή `while` στο Bash χρησιμοποιείται για να εκτελεί μια σειρά εντολών επανειλημμένα όσο μια συγκεκριμένη συνθήκη παραμένει αληθής. Είναι ιδιαίτερα χρήσιμη για επαναλαμβανόμενες διαδικασίες που απαιτούν έλεγχο συνθηκών κατά την εκτέλεση.

## Usage
Η βασική σύνταξη της εντολής `while` είναι η εξής:

```bash
while [ συνθήκη ]
do
    # εντολές προς εκτέλεση
done
```

## Common Options
Η εντολή `while` δεν έχει πολλές επιλογές, αλλά οι πιο κοινές συνθήκες που χρησιμοποιούνται περιλαμβάνουν:
- `true`: Η συνθήκη είναι πάντα αληθής.
- `false`: Η συνθήκη είναι πάντα ψευδής.
- `test`: Μπορείτε να χρησιμοποιήσετε την εντολή `test` ή την συντομογραφία `[` για να ελέγξετε διάφορες συνθήκες.

## Common Examples

### Παράδειγμα 1: Απλή επανάληψη
Αυτό το παράδειγμα εκτυπώνει τους αριθμούς από 1 έως 5.

```bash
count=1
while [ $count -le 5 ]
do
    echo $count
    count=$((count + 1))
done
```

### Παράδειγμα 2: Ανάγνωση αρχείου γραμμή-γραμμή
Αυτό το παράδειγμα διαβάζει ένα αρχείο και εκτυπώνει κάθε γραμμή.

```bash
while IFS= read -r line
do
    echo "$line"
done < myfile.txt
```

### Παράδειγμα 3: Επαναλαμβανόμενη εκτέλεση μέχρι να δοθεί η σωστή είσοδος
Αυτό το παράδειγμα ζητά από τον χρήστη να εισάγει έναν αριθμό μέχρι να εισάγει έναν έγκυρο αριθμό.

```bash
number=0
while ! [[ "$number" =~ ^[0-9]+$ ]]
do
    read -p "Εισάγετε έναν αριθμό: " number
done
echo "Εισάγατε τον αριθμό: $number"
```

## Tips
- Χρησιμοποιήστε τη συνθήκη `true` για να δημιουργήσετε ατέρμονες βρόχους, αλλά φροντίστε να έχετε έναν μηχανισμό διακοπής.
- Να είστε προσεκτικοί με τις συνθήκες που χρησιμοποιείτε, ώστε να αποφύγετε ατέρμονες βρόχους που μπορεί να κολλήσουν το σύστημα.
- Χρησιμοποιήστε την εντολή `break` για να βγείτε από τον βρόχο όταν πληρούνται συγκεκριμένες συνθήκες.
