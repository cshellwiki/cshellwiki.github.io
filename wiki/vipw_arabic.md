# [لينكس] Bash vipw الاستخدام: تعديل ملف كلمات المرور

## Overview
أمر `vipw` يُستخدم لتعديل ملف كلمات المرور (passwd) بطريقة آمنة. يقوم هذا الأمر بفتح ملف `/etc/passwd` و `/etc/shadow` في محرر نصوص محدد، مما يتيح للمستخدمين إجراء التعديلات اللازمة على معلومات المستخدمين.

## Usage
التركيب الأساسي للأمر هو:
```
vipw [options] [arguments]
```

## Common Options
- `-s` : يستخدم هذا الخيار لتعديل ملف shadow بدلاً من ملف passwd.
- `-f` : يتيح لك تحديد ملف مختلف لتعديله.
- `-m` : يفتح الملف في وضع القراءة فقط.

## Common Examples
- لتعديل ملف passwd:
  ```bash
  vipw
  ```

- لتعديل ملف shadow:
  ```bash
  vipw -s
  ```

- لتعديل ملف passwd مخصص:
  ```bash
  vipw -f /path/to/custom/passwd
  ```

## Tips
- تأكد من أنك تمتلك صلاحيات الجذر (root) قبل استخدام الأمر `vipw`، حيث يتطلب الأمر الوصول إلى ملفات النظام.
- استخدم الخيار `-s` بحذر، لأن تعديل ملف shadow يمكن أن يؤثر على أمان النظام.
- قم بعمل نسخة احتياطية من الملفات قبل إجراء أي تعديلات لتجنب فقدان البيانات.
