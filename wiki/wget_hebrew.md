# [לינוקס] Bash wget שימוש: הורדת קבצים מהאינטרנט

## Overview
הפקודה `wget` היא כלי עוצמתי להורדת קבצים מהאינטרנט. היא תומכת בפרוטוקולים כמו HTTP, HTTPS ו-FTP, ומאפשרת למשתמשים להוריד קבצים בקלות וביעילות.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
wget [אפשרויות] [ארגומנטים]
```

## Common Options
- `-O [file]`: שמור את הקובץ בשם שצוין במקום השם המקורי.
- `-q`: הפעל מצב שקט, ללא פלט למסך.
- `-c`: המשך הורדה של קובץ לא גמור.
- `--limit-rate=[rate]`: הגבל את מהירות ההורדה.
- `-r`: הורדה ריקורסיבית של דפי אינטרנט.

## Common Examples
הנה כמה דוגמאות שימושיות להורדת קבצים:

1. הורדת קובץ פשוט:
   ```bash
   wget https://example.com/file.zip
   ```

2. שמירת הקובץ בשם שונה:
   ```bash
   wget -O myfile.zip https://example.com/file.zip
   ```

3. הורדה ריקורסיבית של אתר:
   ```bash
   wget -r https://example.com
   ```

4. המשך הורדה של קובץ לא גמור:
   ```bash
   wget -c https://example.com/largefile.zip
   ```

5. הגבלת מהירות ההורדה:
   ```bash
   wget --limit-rate=200k https://example.com/file.zip
   ```

## Tips
- השתמש באפשרות `-q` כדי להוריד קבצים בשקט, במיוחד כשאתה עובד עם סקריפטים.
- אם אתה מוריד קבצים רבים מאתר, שקול להשתמש באפשרות `-r` עם `--no-parent` כדי למנוע הורדת קבצים מההורים של הדף.
- תמיד בדוק את הקישור לפני ההורדה כדי לוודא שהוא פעיל.
