# [لینوکس] Bash vigr استفاده: ویرایش فایل‌های پیکربندی

## Overview
دستور `vigr` یک ویرایشگر متنی است که به طور خاص برای ویرایش فایل‌های پیکربندی مانند `/etc/passwd` و `/etc/group` طراحی شده است. این دستور به کاربران این امکان را می‌دهد که به صورت ایمن و بدون خطر آسیب رساندن به فایل‌های سیستمی، تغییرات لازم را انجام دهند.

## Usage
برای استفاده از دستور `vigr`، می‌توانید از ساختار زیر استفاده کنید:

```bash
vigr [options] [arguments]
```

## Common Options
- `-c`: بررسی فایل برای خطاهای نحوی.
- `-h`: نمایش راهنما و گزینه‌های موجود.
- `-r`: بازگشت به حالت خواندن فقط.

## Common Examples
برای ویرایش فایل پیکربندی `/etc/passwd`، می‌توانید از دستور زیر استفاده کنید:

```bash
sudo vigr /etc/passwd
```

برای بررسی فایل پیکربندی `/etc/group` و نمایش خطاهای احتمالی، از دستور زیر استفاده کنید:

```bash
sudo vigr -c /etc/group
```

برای ویرایش فایل به صورت خواندن فقط، می‌توانید از گزینه `-r` استفاده کنید:

```bash
sudo vigr -r /etc/passwd
```

## Tips
- همیشه قبل از ویرایش فایل‌های سیستمی، از آن‌ها نسخه پشتیبان تهیه کنید.
- از گزینه `-c` برای بررسی خطاها استفاده کنید تا از صحت فایل‌های پیکربندی اطمینان حاصل کنید.
- در صورت عدم آشنایی با ویرایشگر پیش‌فرض، می‌توانید ویرایشگر مورد نظر خود را با تنظیم متغیر محیطی `EDITOR` مشخص کنید.
