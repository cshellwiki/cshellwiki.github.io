# [Linux] Bash wget cách sử dụng: Tải tệp từ internet

## Overview
คำสั่ง `wget` เป็นเครื่องมือที่ใช้ในการดาวน์โหลดไฟล์จากอินเทอร์เน็ต โดยสามารถดาวน์โหลดไฟล์จาก HTTP, HTTPS และ FTP ได้อย่างง่ายดาย

## Usage
คำสั่งพื้นฐานของ `wget` มีรูปแบบดังนี้:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [filename]`: กำหนดชื่อไฟล์ที่ดาวน์โหลด
- `-c`: ดำเนินการดาวน์โหลดต่อจากจุดที่หยุด
- `-q`: ทำให้การดาวน์โหลดเงียบ ไม่มีข้อความแสดงผล
- `--limit-rate=[rate]`: จำกัดอัตราการดาวน์โหลด
- `-r`: ดาวน์โหลดไฟล์ทั้งหมดในเว็บไซต์ (recursive)

## Common Examples
- ดาวน์โหลดไฟล์จาก URL:
```bash
wget https://example.com/file.zip
```

- ดาวน์โหลดไฟล์และกำหนดชื่อไฟล์:
```bash
wget -O myfile.zip https://example.com/file.zip
```

- ดาวน์โหลดไฟล์ต่อจากจุดที่หยุด:
```bash
wget -c https://example.com/largefile.zip
```

- ดาวน์โหลดเว็บไซต์ทั้งหมด:
```bash
wget -r https://example.com
```

- ดาวน์โหลดไฟล์โดยจำกัดอัตราการดาวน์โหลด:
```bash
wget --limit-rate=200k https://example.com/file.zip
```

## Tips
- ใช้ `-q` เพื่อทำให้การดาวน์โหลดเงียบเมื่อคุณไม่ต้องการให้มีข้อความแสดงผล
- หากคุณต้องการดาวน์โหลดไฟล์จำนวนมากจากเว็บไซต์ ให้ใช้ตัวเลือก `-r` ร่วมกับ `-np` (no parent) เพื่อป้องกันการดาวน์โหลดไฟล์จากระดับสูงกว่า
- ตรวจสอบ URL ก่อนดาวน์โหลดเพื่อให้แน่ใจว่าเป็นไฟล์ที่คุณต้องการ
