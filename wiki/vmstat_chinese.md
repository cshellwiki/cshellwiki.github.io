# [Linux] Bash vmstat 使用指南: 显示系统虚拟内存统计信息

## 概述
`vmstat` 命令用于报告关于系统虚拟内存、进程、CPU活动等的统计信息。它可以帮助用户监控系统性能，识别潜在的瓶颈。

## 用法
基本语法如下：
```
vmstat [options] [interval] [count]
```

## 常用选项
- `-a`：显示活动和非活动的内存。
- `-m`：显示内存页的统计信息。
- `-s`：以表格形式显示内存和交换空间的统计信息。
- `-d`：显示块设备的统计信息。
- `interval`：指定更新的时间间隔（以秒为单位）。
- `count`：指定更新的次数。

## 常见示例
1. 显示系统的虚拟内存统计信息：
   ```bash
   vmstat
   ```

2. 每2秒更新一次，显示5次统计信息：
   ```bash
   vmstat 2 5
   ```

3. 显示内存页的统计信息：
   ```bash
   vmstat -m
   ```

4. 显示块设备的统计信息：
   ```bash
   vmstat -d
   ```

5. 显示详细的内存和交换空间统计信息：
   ```bash
   vmstat -s
   ```

## 提示
- 使用 `vmstat` 时，结合其他监控工具（如 `top` 或 `iostat`）可以获得更全面的系统性能视图。
- 定期监控系统性能，尤其是在高负载情况下，可以帮助及时发现和解决问题。
- 在分析性能问题时，注意查看 CPU 使用率和内存使用情况，以便找到瓶颈所在。
