# [লিনাক্স] Bash cal ব্যবহার: মাসের ক্যালেন্ডার প্রদর্শন

## Overview
`cal` কমান্ডটি একটি নির্দিষ্ট মাস বা বছরের ক্যালেন্ডার প্রদর্শন করতে ব্যবহৃত হয়। এটি ব্যবহারকারীদের জন্য একটি সহজ উপায়ে তারিখ এবং দিনগুলি দেখতে সহায়ক।

## Usage
নিচে `cal` কমান্ডের মৌলিক সিনট্যাক্স দেওয়া হলো:

```bash
cal [options] [arguments]
```

## Common Options
- `-y`: বর্তমান বছরের ক্যালেন্ডার প্রদর্শন করে।
- `-m`: মাসের নামের প্রথম অক্ষর বড় করে প্রদর্শন করে।
- `-3`: বর্তমান মাস, পূর্ববর্তী মাস এবং পরবর্তী মাসের ক্যালেন্ডার একসাথে প্রদর্শন করে।
- `-j`: দিন সংখ্যা (জুলিয়ান দিন) সহ ক্যালেন্ডার প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. বর্তমান মাসের ক্যালেন্ডার দেখার জন্য:
   ```bash
   cal
   ```

2. নির্দিষ্ট মাস এবং বছর (যেমন 2023 সালের জানুয়ারী) ক্যালেন্ডার দেখার জন্য:
   ```bash
   cal 01 2023
   ```

3. বর্তমান বছরের ক্যালেন্ডার দেখার জন্য:
   ```bash
   cal -y
   ```

4. বর্তমান মাস, পূর্ববর্তী মাস এবং পরবর্তী মাসের ক্যালেন্ডার একসাথে দেখার জন্য:
   ```bash
   cal -3
   ```

5. ক্যালেন্ডারকে জুলিয়ান দিন সংখ্যা সহ প্রদর্শন করার জন্য:
   ```bash
   cal -j
   ```

## Tips
- `cal` কমান্ডটি দ্রুত তারিখের তথ্য পেতে সহায়ক। এটি ব্যবহার করে আপনি সহজেই কোনও নির্দিষ্ট মাসের দিনগুলি চিহ্নিত করতে পারেন।
- যদি আপনি একটি নির্দিষ্ট দিন বা তারিখের সাথে কাজ করছেন, তাহলে `cal` এর সাথে `-m` অপশন ব্যবহার করে মাসের নামের প্রথম অক্ষর বড় করে দেখতে পারেন।
- ক্যালেন্ডার দেখতে চাইলে, `man cal` কমান্ডটি ব্যবহার করে এর বিস্তারিত তথ্য পেতে পারেন।
