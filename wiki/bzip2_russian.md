# [Linux] Bash bzip2 использование: Сжатие файлов

## Обзор
Команда `bzip2` используется для сжатия файлов с помощью алгоритма сжатия Burrows-Wheeler. Она позволяет значительно уменьшить размер файлов, что полезно для экономии дискового пространства и ускорения передачи данных.

## Использование
Основной синтаксис команды выглядит следующим образом:

```bash
bzip2 [опции] [аргументы]
```

## Общие опции
- `-d`, `--decompress`: Распаковать сжатый файл.
- `-k`, `--keep`: Сохранить оригинальный файл после сжатия.
- `-f`, `--force`: Принудительно перезаписать существующие файлы.
- `-v`, `--verbose`: Вывести подробную информацию о процессе сжатия.

## Общие примеры
1. Сжатие файла:
   ```bash
   bzip2 файл.txt
   ```

2. Распаковка сжатого файла:
   ```bash
   bzip2 -d файл.txt.bz2
   ```

3. Сжатие файла с сохранением оригинала:
   ```bash
   bzip2 -k файл.txt
   ```

4. Принудительное сжатие, перезаписывающее существующий файл:
   ```bash
   bzip2 -f файл.txt
   ```

5. Сжатие с выводом подробной информации:
   ```bash
   bzip2 -v файл.txt
   ```

## Советы
- Используйте опцию `-k`, если хотите сохранить оригинальный файл после сжатия.
- Для распаковки файлов, сжатых с помощью `bzip2`, всегда используйте `-d` или просто `bunzip2`.
- Обратите внимание на размер файла после сжатия; иногда для небольших файлов сжатие может не дать значительного уменьшения размера.
