# [لینوکس] Bash xz استفاده: فشرده‌سازی و استخراج فایل‌ها

## Overview
دستور `xz` برای فشرده‌سازی و استخراج فایل‌ها به کار می‌رود. این ابزار به طور خاص برای کاهش حجم فایل‌ها با استفاده از الگوریتم فشرده‌سازی LZMA طراحی شده است و معمولاً برای فشرده‌سازی فایل‌های بزرگ و بهینه‌سازی فضای دیسک استفاده می‌شود.

## Usage
سینتکس پایه دستور `xz` به صورت زیر است:
```bash
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: برای استخراج فایل‌های فشرده شده.
- `-k`, `--keep`: نگه‌داشتن فایل اصلی بعد از فشرده‌سازی.
- `-v`, `--verbose`: نمایش اطلاعات بیشتر در مورد عملیات فشرده‌سازی.
- `-9`: استفاده از بالاترین سطح فشرده‌سازی (بیشترین زمان و منابع).
- `-e`, `--extreme`: استفاده از روش‌های فشرده‌سازی پیشرفته.

## Common Examples
### فشرده‌سازی یک فایل
برای فشرده‌سازی یک فایل به نام `file.txt`:
```bash
xz file.txt
```

### استخراج یک فایل فشرده
برای استخراج یک فایل فشرده به نام `file.txt.xz`:
```bash
xz -d file.txt.xz
```

### فشرده‌سازی و نگه‌داشتن فایل اصلی
برای فشرده‌سازی یک فایل و نگه‌داشتن نسخه اصلی:
```bash
xz -k file.txt
```

### فشرده‌سازی با بالاترین سطح
برای فشرده‌سازی با بالاترین سطح فشرده‌سازی:
```bash
xz -9 file.txt
```

## Tips
- اگر نیاز به فشرده‌سازی سریع دارید، از گزینه `-1` تا `-3` استفاده کنید که سرعت بیشتری دارند.
- برای فشرده‌سازی چندین فایل به صورت همزمان، می‌توانید از wildcard استفاده کنید:
  ```bash
  xz *.txt
  ```
- به یاد داشته باشید که فایل‌های فشرده شده با `xz` معمولاً با پسوند `.xz` ذخیره می‌شوند، بنابراین هنگام استخراج، این پسوند را در نظر بگیرید.
