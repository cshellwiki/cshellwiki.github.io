# [سیستم‌عامل‌های یونیکس] C Shell (csh) xz <استفاده معادل>: فشرده‌سازی و استخراج فایل‌ها

## Overview
دستور `xz` برای فشرده‌سازی و استخراج فایل‌ها استفاده می‌شود. این دستور به شما امکان می‌دهد تا فایل‌ها را با استفاده از الگوریتم فشرده‌سازی XZ فشرده کنید که معمولاً نسبت فشرده‌سازی بالایی دارد.

## Usage
سینتکس پایه دستور `xz` به صورت زیر است:

```csh
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: استخراج فایل‌های فشرده.
- `-k`, `--keep`: نگه‌داشتن فایل‌های اصلی بعد از فشرده‌سازی.
- `-v`, `--verbose`: نمایش اطلاعات بیشتر در مورد فرآیند فشرده‌سازی.
- `-9`: استفاده از حداکثر فشرده‌سازی (بیشترین زمان را می‌گیرد).

## Common Examples
- فشرده‌سازی یک فایل:
  ```csh
  xz myfile.txt
  ```

- استخراج یک فایل فشرده:
  ```csh
  xz -d myfile.txt.xz
  ```

- فشرده‌سازی و نگه‌داشتن فایل اصلی:
  ```csh
  xz -k myfile.txt
  ```

- فشرده‌سازی با حداکثر فشرده‌سازی:
  ```csh
  xz -9 myfile.txt
  ```

## Tips
- برای فشرده‌سازی فایل‌های بزرگ، از گزینه `-9` استفاده کنید تا بهترین نسبت فشرده‌سازی را بدست آورید.
- همیشه قبل از فشرده‌سازی فایل‌های مهم، یک نسخه پشتیبان از آن‌ها تهیه کنید.
- برای مشاهده اطلاعات بیشتر در مورد فایل‌های فشرده، از گزینه `-v` استفاده کنید.
