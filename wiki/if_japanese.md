# [Linux] Bash if コマンド: 条件分岐を実行する

## Overview
`if` コマンドは、指定した条件が真であるかどうかを評価し、その結果に基づいて異なるコマンドを実行するための構文を提供します。これにより、スクリプト内での条件分岐が可能になります。

## Usage
基本的な構文は以下の通りです。

```bash
if [ 条件 ]; then
    コマンド
fi
```

## Common Options
- `-e`: ファイルが存在するかどうかをチェックします。
- `-d`: 指定したパスがディレクトリであるかどうかを確認します。
- `-f`: 指定したパスが通常のファイルであるかどうかを確認します。
- `-z`: 文字列が空であるかどうかを確認します。

## Common Examples

### 例1: ファイルの存在確認
```bash
if [ -e "example.txt" ]; then
    echo "ファイルは存在します。"
else
    echo "ファイルは存在しません。"
fi
```

### 例2: ディレクトリの確認
```bash
if [ -d "/path/to/directory" ]; then
    echo "ディレクトリは存在します。"
else
    echo "ディレクトリは存在しません。"
fi
```

### 例3: 変数の値チェック
```bash
name="Alice"
if [ -z "$name" ]; then
    echo "名前は設定されていません。"
else
    echo "名前は $name です。"
fi
```

## Tips
- `if` 文の条件は、常に `[ ]` または `[[ ]]` で囲む必要があります。
- 複数の条件を組み合わせる場合は、`&&`（論理積）や `||`（論理和）を使用できます。
- `then` の後には必ず改行が必要です。1行で書く場合は、`then` の前にセミコロン `;` を追加してください。
