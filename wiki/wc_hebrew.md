# [לינוקס] Bash wc שימוש: ספירת שורות, מילים ותווים בקובץ

## Overview
הפקודה `wc` (ראשי תיבות של "word count") משמשת לספירת שורות, מילים ותווים בקובץ או בקלט מהשורה. היא כלי שימושי לניתוח טקסטים ולבדיקת גודל הקבצים.

## Usage
התחביר הבסיסי של הפקודה הוא:

```
wc [אפשרויות] [ארגומנטים]
```

## Common Options
- `-l`: סופר את מספר השורות בקובץ.
- `-w`: סופר את מספר המילים בקובץ.
- `-c`: סופר את מספר התווים בקובץ.
- `-m`: סופר את מספר התווים (כולל תווים מיוחדים).
- `-L`: מציג את אורך השורה הארוכה ביותר בקובץ.

## Common Examples
- ספירת שורות בקובץ:
  ```bash
  wc -l filename.txt
  ```

- ספירת מילים בקובץ:
  ```bash
  wc -w filename.txt
  ```

- ספירת תווים בקובץ:
  ```bash
  wc -c filename.txt
  ```

- ספירת שורות, מילים ותווים בו זמנית:
  ```bash
  wc filename.txt
  ```

- ספירת שורות בקלט מהשורה:
  ```bash
  echo "Hello World" | wc -l
  ```

## Tips
- ניתן לשלב מספר אפשרויות בפעם אחת, לדוגמה: `wc -l -w filename.txt`.
- אם אתם עובדים עם קבצים גדולים, שקלו להשתמש בפקודה `cat` יחד עם `wc` כדי לעבד קלט בצורה יעילה יותר.
- הפקודה `wc` יכולה להיות שימושית מאוד בשילוב עם פקודות אחרות כמו `grep` או `find` לניתוח טקסטים בצורה מתקדמת.
