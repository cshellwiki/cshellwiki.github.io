# [لينكس] Bash break الاستخدام: إنهاء الحلقات

## Overview
أمر `break` في Bash يُستخدم لإنهاء حلقة تكرار (مثل `for` أو `while`) قبل أن تنتهي بشكل طبيعي. يُعتبر هذا الأمر مفيدًا عندما تحتاج إلى الخروج من الحلقة بناءً على شرط معين.

## Usage
الصيغة الأساسية للأمر هي:

```bash
break [n]
```

حيث `n` هو عدد الحلقات التي تريد الخروج منها. إذا لم يتم تحديد `n`، فسيخرج الأمر من الحلقة الأقرب.

## Common Options
- `n`: عدد الحلقات التي ترغب في الخروج منها. إذا كانت القيمة 1، سيخرج من الحلقة الأقرب، وإذا كانت 2، سيخرج من الحلقة الثانية الأقرب، وهكذا.

## Common Examples

### مثال 1: الخروج من حلقة for
```bash
for i in {1..5}; do
  if [ $i -eq 3 ]; then
    break
  fi
  echo $i
done
```
**الناتج:**
```
1
2
```

### مثال 2: الخروج من حلقة while
```bash
count=1
while [ $count -le 5 ]; do
  if [ $count -eq 4 ]; then
    break
  fi
  echo $count
  ((count++))
done
```
**الناتج:**
```
1
2
3
```

### مثال 3: الخروج من حلقات متعددة
```bash
for i in {1..3}; do
  for j in {1..3}; do
    if [ $j -eq 2 ]; then
      break 2
    fi
    echo "i: $i, j: $j"
  done
done
```
**الناتج:**
```
i: 1, j: 1
```

## Tips
- استخدم `break` بحذر، حيث يمكن أن يؤدي إلى خروج غير متوقع من الحلقات إذا لم يتم استخدامه بشكل صحيح.
- تأكد من أن لديك شرطًا واضحًا للخروج من الحلقة لتجنب الحلقات اللانهائية.
- يمكنك استخدام `break` مع `case` في الحلقات لتحديد متى يجب الخروج بناءً على القيم المختلفة.
