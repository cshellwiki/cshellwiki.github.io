# [Linux] Bash script 使用法: コマンドの実行を記録する

## Overview
`script` コマンドは、シェルセッションのすべての出力をファイルに記録するためのツールです。このコマンドを使用すると、ターミナルでの作業を後で再確認したり、他のユーザーと共有したりすることができます。

## Usage
基本的な構文は以下の通りです。

```bash
script [options] [file]
```

## Common Options
- `-a`: 既存のファイルに追記します。
- `-c`: 指定したコマンドを実行し、その出力を記録します。
- `-f`: 出力をリアルタイムで表示します。
- `-q`: スクリプトの開始メッセージを表示しません。

## Common Examples
以下に、`script` コマンドのいくつかの実用的な例を示します。

### 例 1: 基本的なスクリプトの作成
ターミナルセッションを記録するために、`typescript` というファイルに出力を保存します。

```bash
script typescript
```

### 例 2: 既存のファイルに追記
既存の `logfile` に出力を追記する場合は、`-a` オプションを使用します。

```bash
script -a logfile
```

### 例 3: 特定のコマンドの実行を記録
`ls` コマンドの出力を `output.log` に記録します。

```bash
script -c "ls -l" output.log
```

### 例 4: リアルタイムで出力を表示
出力をリアルタイムで表示しながら記録する場合は、`-f` オプションを使用します。

```bash
script -f typescript
```

## Tips
- スクリプトを終了するには、`exit` コマンドを入力するか、`Ctrl+D` を押します。
- 記録したファイルは、後で `cat` や `less` コマンドを使って確認できます。
- スクリプトを使用する際は、重要な情報が含まれる可能性があるため、記録ファイルの管理に注意してください。
