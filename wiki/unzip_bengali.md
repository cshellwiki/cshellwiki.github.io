# [লিনাক্স] Bash unzip ব্যবহার: ফাইল আনজিপ করা

## Overview
`unzip` কমান্ডটি ZIP ফাইলগুলি আনজিপ করার জন্য ব্যবহৃত হয়। এটি একটি জনপ্রিয় ফাইল ফরম্যাট যা একাধিক ফাইল এবং ফোল্ডারকে একটি একক ফাইলে সংরক্ষণ করে।

## Usage
`unzip` কমান্ডের মৌলিক সিনট্যাক্স নিম্নরূপ:

```bash
unzip [options] [arguments]
```

## Common Options
- `-l`: ZIP ফাইলের ভিতরে কি কি ফাইল আছে তা তালিকা করে।
- `-d <directory>`: আনজিপ করা ফাইলগুলি নির্দিষ্ট ডিরেক্টরিতে সংরক্ষণ করে।
- `-o`: পূর্ববর্তী ফাইলগুলিকে ওভাররাইট করে আনজিপ করে, কোন প্রশ্ন ছাড়াই।
- `-q`: আনজিপ করার সময় কোন আউটপুট প্রদর্শন করে না।

## Common Examples
- একটি ZIP ফাইল আনজিপ করা:
  ```bash
  unzip example.zip
  ```

- একটি ZIP ফাইলের ভিতরে কি কি ফাইল আছে তা দেখা:
  ```bash
  unzip -l example.zip
  ```

- একটি ZIP ফাইল নির্দিষ্ট ডিরেক্টরিতে আনজিপ করা:
  ```bash
  unzip example.zip -d /path/to/directory
  ```

- পূর্ববর্তী ফাইলগুলিকে ওভাররাইট করে আনজিপ করা:
  ```bash
  unzip -o example.zip
  ```

## Tips
- নিশ্চিত করুন যে আপনি যে ডিরেক্টরিতে ফাইলগুলি আনজিপ করছেন সেখানে পর্যাপ্ত স্থান আছে।
- যদি আপনি একটি বড় ZIP ফাইল আনজিপ করছেন, তবে `-q` অপশন ব্যবহার করে আউটপুট কমাতে পারেন।
- নিয়মিত ব্যাকআপ রাখা নিশ্চিত করুন, বিশেষ করে যখন আপনি ফাইলগুলি ওভাররাইট করছেন।
