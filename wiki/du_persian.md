# [سیستم‌عامل‌های یونیکس] C Shell (csh) du <استفاده معادل>: اندازه‌گیری استفاده از دیسک

## Overview
دستور `du` (Disk Usage) برای نمایش میزان فضای دیسک استفاده شده توسط فایل‌ها و دایرکتوری‌ها در سیستم‌های یونیکس و لینوکس به کار می‌رود. این دستور به کاربران کمک می‌کند تا بفهمند کدام فایل‌ها و دایرکتوری‌ها بیشترین فضای دیسک را اشغال کرده‌اند.

## Usage
سینتکس پایه‌ی این دستور به صورت زیر است:

```csh
du [options] [arguments]
```

## Common Options
- `-h`: نمایش اندازه‌ها به صورت قابل خواندن برای انسان (مثلاً با واحدهای KB، MB، GB).
- `-s`: نمایش مجموع فضای استفاده شده توسط دایرکتوری‌ها به جای نمایش هر فایل به صورت جداگانه.
- `-a`: نمایش اندازه‌ی همه فایل‌ها و دایرکتوری‌ها.
- `-c`: نمایش مجموع کل اندازه‌ها در انتهای خروجی.

## Common Examples
- نمایش اندازه‌ی دیسک استفاده شده توسط دایرکتوری جاری به صورت قابل خواندن:
  ```csh
  du -h
  ```

- نمایش مجموع فضای استفاده شده توسط یک دایرکتوری خاص:
  ```csh
  du -sh /path/to/directory
  ```

- نمایش اندازه‌ی همه فایل‌ها و دایرکتوری‌ها در یک دایرکتوری:
  ```csh
  du -ah /path/to/directory
  ```

- نمایش مجموع کل اندازه‌ها:
  ```csh
  du -ch /path/to/directory
  ```

## Tips
- برای بهینه‌سازی زمان اجرا، می‌توانید از گزینه `-s` برای نمایش فقط مجموع استفاده از دیسک استفاده کنید.
- اگر می‌خواهید فقط دایرکتوری‌های خاصی را بررسی کنید، می‌توانید از wildcardها (مانند `*`) در آرگومان‌ها استفاده کنید.
- به یاد داشته باشید که اجرای `du` در دایرکتوری‌های بزرگ ممکن است زمان‌بر باشد، بنابراین در صورت نیاز، از گزینه‌های محدودکننده استفاده کنید.
