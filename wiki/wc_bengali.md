# [লিনাক্স] Bash wc ব্যবহার: ফাইলের শব্দ, লাইন এবং অক্ষর গণনা

## Overview
`wc` (word count) কমান্ডটি ফাইলের মধ্যে শব্দ, লাইন এবং অক্ষর গণনা করার জন্য ব্যবহৃত হয়। এটি টেক্সট ফাইলের বিভিন্ন পরিসংখ্যান দ্রুত পেতে সহায়ক।

## Usage
`wc` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
wc [options] [arguments]
```

## Common Options
- `-l`: ফাইলে মোট লাইন সংখ্যা দেখায়।
- `-w`: ফাইলে মোট শব্দ সংখ্যা দেখায়।
- `-c`: ফাইলে মোট অক্ষর সংখ্যা দেখায়।
- `-m`: ফাইলে মোট অক্ষর সংখ্যা দেখায় (UTF-8 অক্ষর গণনা)।
- `-L`: সবচেয়ে দীর্ঘ লাইনের দৈর্ঘ্য দেখায়।

## Common Examples
1. **লাইন সংখ্যা গণনা**:
   ```bash
   wc -l filename.txt
   ```

2. **শব্দ সংখ্যা গণনা**:
   ```bash
   wc -w filename.txt
   ```

3. **অক্ষর সংখ্যা গণনা**:
   ```bash
   wc -c filename.txt
   ```

4. **একাধিক ফাইলের জন্য পরিসংখ্যান**:
   ```bash
   wc file1.txt file2.txt
   ```

5. **সব তথ্য একসাথে দেখানো**:
   ```bash
   wc filename.txt
   ```

## Tips
- `wc` কমান্ডটি পাইপের মাধ্যমে অন্যান্য কমান্ডের সাথে সংযুক্ত করা যেতে পারে, যেমন `cat` বা `grep`, যাতে ফাইলের নির্দিষ্ট অংশের পরিসংখ্যান পাওয়া যায়।
- ফাইলের আকার এবং বিষয়বস্তু বুঝতে `wc` ব্যবহার করা একটি দ্রুত এবং কার্যকর উপায়।
- যদি আপনি কেবল একটি নির্দিষ্ট পরিসংখ্যান জানতে চান, তাহলে সেই অনুযায়ী অপশন ব্যবহার করুন, যেমন `wc -w` শুধুমাত্র শব্দ সংখ্যা দেখাবে।
