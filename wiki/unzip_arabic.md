# [لينكس] Bash unzip الاستخدام: استخراج الملفات المضغوطة

## Overview
أداة `unzip` تُستخدم لاستخراج الملفات من الأرشيفات المضغوطة بصيغة ZIP. هذه الأداة تُعتبر واحدة من أكثر الأدوات شيوعًا في بيئات لينكس وUnix، حيث تتيح للمستخدمين الوصول إلى محتويات الملفات المضغوطة بسهولة.

## Usage
الصيغة الأساسية لاستخدام الأمر `unzip` هي:

```bash
unzip [options] [arguments]
```

## Common Options
- `-l`: يعرض محتويات ملف ZIP دون استخراجها.
- `-d <directory>`: يحدد الدليل الذي سيتم استخراج الملفات فيه.
- `-o`: يستبدل الملفات الموجودة دون طلب تأكيد.
- `-q`: يعمل في وضع هادئ، مما يعني تقليل الرسائل المعروضة أثناء عملية الاستخراج.

## Common Examples
- لاستخراج محتويات ملف ZIP إلى الدليل الحالي:

```bash
unzip archive.zip
```

- لاستخراج محتويات ملف ZIP إلى دليل محدد:

```bash
unzip archive.zip -d /path/to/directory
```

- لعرض محتويات ملف ZIP دون استخراجها:

```bash
unzip -l archive.zip
```

- لاستخراج ملف ZIP مع استبدال الملفات الموجودة:

```bash
unzip -o archive.zip
```

## Tips
- تأكد من وجود مساحة كافية على القرص قبل استخراج الملفات، خاصة إذا كان الأرشيف كبيرًا.
- استخدم الخيار `-q` إذا كنت ترغب في تقليل الرسائل المعروضة أثناء عملية الاستخراج.
- تحقق من محتويات الأرشيف باستخدام الخيار `-l` قبل الاستخراج لتجنب استخراج ملفات غير ضرورية.
