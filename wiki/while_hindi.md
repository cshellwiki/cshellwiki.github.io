# [Linux] Bash while का उपयोग: लूपिंग के लिए

## Overview
`while` कमांड एक लूपिंग संरचना है जो तब तक एक या अधिक कमांड को चलाती है जब तक कि एक निर्दिष्ट शर्त सत्य (true) होती है। इसका उपयोग तब किया जाता है जब आपको किसी कार्य को तब तक दोहराना हो जब तक कि कोई विशेष स्थिति पूरी न हो जाए।

## Usage
`while` कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
while [condition]
do
    # commands to be executed
done
```

## Common Options
`while` कमांड के लिए कोई विशेष विकल्प नहीं होते हैं, लेकिन आप शर्त के लिए विभिन्न परीक्षणों का उपयोग कर सकते हैं, जैसे:
- `-e`: यह जांचता है कि फ़ाइल मौजूद है या नहीं।
- `-f`: यह जांचता है कि फ़ाइल एक सामान्य फ़ाइल है या नहीं।
- `-d`: यह जांचता है कि फ़ाइल एक निर्देशिका है या नहीं।

## Common Examples

### Example 1: साधारण लूप
```bash
count=1
while [ $count -le 5 ]
do
    echo "Count is: $count"
    ((count++))
done
```
यह स्क्रिप्ट 1 से 5 तक की गिनती प्रदर्शित करती है।

### Example 2: फ़ाइलों की जांच
```bash
file="example.txt"
while [ ! -e $file ]
do
    echo "$file फ़ाइल अभी तक मौजूद नहीं है।"
    sleep 1
done
echo "$file फ़ाइल अब मौजूद है।"
```
यह स्क्रिप्ट तब तक प्रतीक्षा करती है जब तक कि `example.txt` फ़ाइल मौजूद नहीं हो जाती।

### Example 3: उपयोगकर्ता इनपुट
```bash
input=""
while [ "$input" != "exit" ]
do
    read -p "कृपया 'exit' टाइप करें: " input
    echo "आपने टाइप किया: $input"
done
```
यह स्क्रिप्ट उपयोगकर्ता से इनपुट लेती है और तब तक चलती है जब तक उपयोगकर्ता "exit" नहीं टाइप करता।

## Tips
- सुनिश्चित करें कि लूप में कोई ऐसा तरीका हो जिससे वह अंततः समाप्त हो सके, अन्यथा यह अनंत लूप में चला जाएगा।
- लूप के अंदर उचित डिबगिंग संदेश शामिल करें ताकि आप समझ सकें कि लूप किस स्थिति में है।
- शर्तों को स्पष्ट और संक्षिप्त रखें ताकि कोड पढ़ने में आसान हो।
