# [লিনাক্স] Bash batch ব্যবহার: ব্যাচ প্রক্রিয়াকরণ

## Overview
`batch` কমান্ডটি লিনাক্স এবং ইউনিক্স সিস্টেমে ব্যবহৃত হয়, যা ব্যবহারকারীদের নির্দিষ্ট সময়ে কাজগুলো সম্পাদন করার জন্য একটি ব্যাচ প্রক্রিয়া তৈরি করতে সাহায্য করে। এটি সাধারণত ব্যাকগ্রাউন্ডে কাজ চালানোর জন্য ব্যবহৃত হয়, যা সিস্টেমের ব্যস্ততার সময়ে কাজগুলো সম্পন্ন করতে পারে।

## Usage
`batch` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
batch [options] [arguments]
```

## Common Options
- `-l` : লম্বা আউটপুট ফরম্যাটে কাজের তালিকা দেখায়।
- `-q` : নিরব মোডে কাজ সম্পন্ন করে, আউটপুট দেখায় না।
- `-f` : ফাইল থেকে কমান্ড গ্রহণ করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### উদাহরণ ১: একটি সাধারণ ব্যাচ কাজ যোগ করা
```bash
echo "date" | batch
```
এই কমান্ডটি বর্তমান তারিখ এবং সময়কে ব্যাচে যোগ করবে এবং নির্ধারিত সময়ে এটি সম্পন্ন হবে।

### উদাহরণ ২: একটি স্ক্রিপ্ট ব্যাচে যোগ করা
```bash
batch < myscript.sh
```
এটি `myscript.sh` নামক স্ক্রিপ্টটিকে ব্যাচে যোগ করবে এবং নির্ধারিত সময়ে এটি চালানো হবে।

### উদাহরণ ৩: একাধিক কমান্ড ব্যাচে যোগ করা
```bash
echo -e "echo 'Hello World'\necho 'This is a batch job'" | batch
```
এই কমান্ডটি দুটি আলাদা কমান্ডকে ব্যাচে যোগ করবে এবং নির্ধারিত সময়ে উভয় কমান্ড চালানো হবে।

## Tips
- ব্যাচ কাজগুলো সময়মতো সম্পন্ন হবে, তাই নিশ্চিত করুন যে আপনার কাজগুলো সঠিকভাবে লেখা হয়েছে।
- ব্যাচে কাজ যোগ করার আগে, আপনার স্ক্রিপ্ট বা কমান্ডগুলো পরীক্ষা করে নিন যাতে কোনো ত্রুটি না থাকে।
- ব্যাচ কাজের আউটপুট দেখতে `atq` কমান্ড ব্যবহার করতে পারেন।
