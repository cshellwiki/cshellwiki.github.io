# [לינוקס] Bash vipw שימוש: עריכת קובץ passwd בצורה בטוחה

## Overview
הפקודה `vipw` משמשת לעריכת קובץ המשתמשים במערכת, בדרך כלל `/etc/passwd`, בצורה בטוחה. הפקודה פותחת את הקובץ בעורך טקסט, ומוודאת שאין גישה מקבילה לקובץ בזמן העריכה, מה שמפחית את הסיכון לשגיאות.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
vipw [אפשרויות]
```

## Common Options
- `-s`: פותח את הקובץ `/etc/shadow` לעריכה.
- `-f <file>`: מציין קובץ אחר לעריכה במקום הקובץ המוגדר כברירת מחדל.

## Common Examples
1. עריכת קובץ passwd:
   ```bash
   vipw
   ```

2. עריכת קובץ shadow:
   ```bash
   vipw -s
   ```

3. עריכת קובץ מותאם אישית:
   ```bash
   vipw -f /path/to/custom/file
   ```

## Tips
- תמיד גבה את הקבצים החשובים לפני ביצוע שינויים.
- השתמש באפשרות `-s` כדי לערוך את קובץ ה-shadow בזהירות.
- ודא שאתה מבצע את השינויים הנדרשים רק כאשר אתה בטוח במידע שאתה מוסיף או משנה.
