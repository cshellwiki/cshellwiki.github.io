# [লিনাক্স] Bash break ব্যবহার: লুপ থেকে বের হওয়া

## Overview
`break` কমান্ডটি Bash স্ক্রিপ্টে ব্যবহৃত হয় লুপ থেকে বের হওয়ার জন্য। যখন এটি একটি লুপের মধ্যে ব্যবহৃত হয়, তখন এটি লুপটি তৎক্ষণাৎ বন্ধ করে দেয় এবং পরবর্তী নির্দেশে চলে যায়।

## Usage
`break` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
break [n]
```

এখানে `n` হল ঐ সংখ্যক স্তরের লুপ থেকে বের হওয়ার নির্দেশ। যদি `n` উল্লেখ না করা হয়, তবে এটি বর্তমান লুপ থেকে বের হয়ে যাবে।

## Common Options
- `n`: নির্দিষ্ট করে যে কতটি স্তরের লুপ থেকে বের হতে হবে। যদি `n` 1 হয়, তাহলে শুধুমাত্র বর্তমান লুপ থেকে বের হবে; যদি 2 হয়, তাহলে বাইরের লুপ থেকেও বের হবে।

## Common Examples

### উদাহরণ 1: সাধারণ ব্যবহার
```bash
for i in {1..5}; do
    if [ $i -eq 3 ]; then
        break
    fi
    echo $i
done
```
এই উদাহরণে, লুপটি 1 থেকে 5 পর্যন্ত সংখ্যা প্রদর্শন করবে, কিন্তু 3 এ পৌঁছালে লুপটি বন্ধ হয়ে যাবে।

### উদাহরণ 2: স্তরের লুপ থেকে বের হওয়া
```bash
for i in {1..3}; do
    for j in {1..3}; do
        if [ $j -eq 2 ]; then
            break 2
        fi
        echo "i: $i, j: $j"
    done
done
```
এখানে, যখন `j` 2 হয়, তখন উভয় লুপ থেকে বের হয়ে যাবে।

### উদাহরণ 3: while লুপে ব্যবহার
```bash
count=1
while [ $count -le 5 ]; do
    if [ $count -eq 4 ]; then
        break
    fi
    echo $count
    ((count++))
done
```
এই উদাহরণে, 1 থেকে 3 পর্যন্ত সংখ্যা প্রদর্শিত হবে এবং 4 এ পৌঁছালে লুপটি বন্ধ হয়ে যাবে।

## Tips
- `break` কমান্ডটি শুধুমাত্র লুপের মধ্যে ব্যবহার করুন; অন্য কোথাও এটি কার্যকর হবে না।
- লুপের স্তর সংখ্যা সঠিকভাবে উল্লেখ করুন যাতে আপনি সঠিক লুপ থেকে বের হতে পারেন।
- লুপের মধ্যে `break` ব্যবহার করার সময়, নিশ্চিত করুন যে লুপটি কার্যকরভাবে কাজ করছে এবং আপনার প্রত্যাশিত ফলাফল দিচ্ছে।
