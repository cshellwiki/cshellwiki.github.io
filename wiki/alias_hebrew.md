# [לינוקס] Bash alias שימוש: יצירת פקודות מותאמות אישית

## Overview
הפקודה `alias` ב-Bash משמשת ליצירת קיצורים לפקודות ארוכות או מורכבות. באמצעות `alias`, ניתן להקל על השימוש בפקודות נפוצות על ידי מתן שם קצר יותר או יותר אינטואיטיבי.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
alias [options] [arguments]
```

## Common Options
- `-p`: מציג את כל הקיצורים המוגדרים כרגע.
- `--help`: מציג מידע עזרה על השימוש בפקודה.

## Common Examples
להלן מספר דוגמאות שימושיות ליצירת קיצורים:

1. קיצור לפקודת `ls -la`:
   ```bash
   alias ll='ls -la'
   ```

2. קיצור לפקודת `git status`:
   ```bash
   alias gs='git status'
   ```

3. קיצור לפקודת `rm -rf` עם אישור:
   ```bash
   alias rmd='rm -i'
   ```

4. קיצור לפקודת `cd` לתיקייה מסוימת:
   ```bash
   alias proj='cd ~/projects/my_project'
   ```

## Tips
- כדי שהקיצורים יישמרו גם לאחר סגירת הטרמינל, הוסף את הפקודות לקובץ `.bashrc` או `.bash_profile`.
- השתמש בשמות קצרים וברורים כדי להקל על השימוש.
- הימנע משימוש בשמות קיצורים שכבר קיימים במערכת כדי למנוע בלבול.
