# [لینوکس] Bash wget استفاده: دانلود فایل‌ها از اینترنت

## Overview
دستور wget ابزاری قدرتمند برای دانلود فایل‌ها از اینترنت است. این دستور به شما این امکان را می‌دهد که به سادگی و به صورت خودکار فایل‌ها را از وب‌سایت‌ها و سرورها دریافت کنید.

## Usage
سینتکس پایه دستور wget به صورت زیر است:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [file]`: فایل دانلود شده را با نام مشخص شده ذخیره می‌کند.
- `-c`: ادامه دانلود فایل‌های ناقص.
- `-r`: دانلود به صورت بازگشتی (recursive) از وب‌سایت‌ها.
- `-q`: حالت سکوت (quiet mode) که خروجی را کاهش می‌دهد.
- `--limit-rate=[rate]`: محدود کردن سرعت دانلود.

## Common Examples
- دانلود یک فایل ساده:
```bash
wget http://example.com/file.zip
```

- دانلود یک فایل و ذخیره آن با نام متفاوت:
```bash
wget -O newfile.zip http://example.com/file.zip
```

- ادامه دانلود یک فایل ناقص:
```bash
wget -c http://example.com/file.zip
```

- دانلود به صورت بازگشتی از یک وب‌سایت:
```bash
wget -r http://example.com/
```

- محدود کردن سرعت دانلود:
```bash
wget --limit-rate=200k http://example.com/file.zip
```

## Tips
- برای دانلود فایل‌های بزرگ، از گزینه `-c` استفاده کنید تا در صورت قطع شدن دانلود، بتوانید ادامه دهید.
- برای دانلود محتویات یک وب‌سایت، از گزینه `-r` استفاده کنید، اما مراقب باشید که این کار ممکن است حجم زیادی از داده‌ها را دانلود کند.
- از گزینه `-q` برای کاهش خروجی در زمان‌های دانلود طولانی استفاده کنید تا فقط خطاها را مشاهده کنید.
