# [سیستم‌عامل] C Shell (csh) wget استفاده: دانلود فایل‌ها از اینترنت

## Overview
دستور wget ابزاری است برای دانلود فایل‌ها از اینترنت. این دستور به شما امکان می‌دهد تا به راحتی فایل‌ها را از وب‌سایت‌ها و سرورها بارگیری کنید.

## Usage
سینتکس پایه دستور wget به صورت زیر است:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [file]`: نام فایل خروجی را مشخص می‌کند.
- `-c`: ادامه دانلود فایل‌های ناتمام.
- `-q`: حالت بی‌صدا، بدون نمایش اطلاعات در حین دانلود.
- `--limit-rate=[speed]`: محدود کردن سرعت دانلود.
- `-r`: دانلود به صورت بازگشتی، برای دانلود دایرکتوری‌ها و زیر دایرکتوری‌ها.

## Common Examples
- دانلود یک فایل:
  ```bash
  wget http://example.com/file.zip
  ```

- دانلود یک فایل با نام مشخص:
  ```bash
  wget -O myfile.zip http://example.com/file.zip
  ```

- ادامه دانلود فایل ناتمام:
  ```bash
  wget -c http://example.com/largefile.zip
  ```

- دانلود به صورت بی‌صدا:
  ```bash
  wget -q http://example.com/file.zip
  ```

- دانلود یک دایرکتوری به صورت بازگشتی:
  ```bash
  wget -r http://example.com/directory/
  ```

## Tips
- برای دانلود فایل‌های بزرگ، از گزینه `-c` استفاده کنید تا بتوانید دانلود را ادامه دهید.
- اگر می‌خواهید فقط فایل‌های خاصی را دانلود کنید، از گزینه `--accept` استفاده کنید.
- برای مدیریت بهتر دانلودها، می‌توانید از گزینه `--limit-rate` برای محدود کردن سرعت استفاده کنید.
