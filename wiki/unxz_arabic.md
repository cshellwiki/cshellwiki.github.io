# [لينكس] Bash unxz الاستخدام: فك ضغط ملفات XZ

## Overview
يستخدم الأمر `unxz` لفك ضغط الملفات المضغوطة بتنسيق XZ. يعد هذا التنسيق شائعًا لتقليل حجم الملفات، مما يسهل تخزينها ونقلها.

## Usage
تكون الصيغة الأساسية للأمر كما يلي:

```bash
unxz [options] [arguments]
```

## Common Options
- `-k` أو `--keep`: يحتفظ بالملف المضغوط الأصلي بعد فك الضغط.
- `-f` أو `--force`: يجبر الأمر على استبدال الملفات الموجودة دون طلب تأكيد.
- `-v` أو `--verbose`: يعرض معلومات تفصيلية أثناء عملية فك الضغط.

## Common Examples
- لفك ضغط ملف XZ بسيط:
```bash
unxz file.xz
```

- لفك ضغط ملف مع الاحتفاظ بالملف الأصلي:
```bash
unxz -k file.xz
```

- لفك ضغط ملف مع استبدال ملف موجود:
```bash
unxz -f file.xz
```

- لفك ضغط ملف وعرض معلومات تفصيلية:
```bash
unxz -v file.xz
```

## Tips
- تأكد من وجود مساحة كافية على القرص قبل فك ضغط الملفات الكبيرة.
- استخدم الخيار `-k` إذا كنت ترغب في الاحتفاظ بالملف الأصلي، خاصة عند التعامل مع ملفات مهمة.
- تحقق من الملفات المضغوطة باستخدام الأمر `xz -t` قبل فك الضغط للتأكد من سلامتها.
