# [Linux] Bash vmstat 使用說明: 監控系統性能

## Overview
`vmstat` 是一個用於監控系統性能的命令，能夠顯示有關虛擬內存、進程、CPU 活動和系統 I/O 的統計數據。這些資訊對於系統管理員來說非常重要，可以幫助他們了解系統的運行狀態和性能瓶頸。

## Usage
基本語法如下：
```
vmstat [options] [arguments]
```

## Common Options
- `-a`：顯示所有的內存，包括空閒和使用中的內存。
- `-n`：只顯示一次輸出，而不持續更新。
- `-S`：指定顯示的內存單位，例如 `m` 表示 MB。
- `-t`：顯示時間戳。

## Common Examples
以下是一些常見的 `vmstat` 使用範例：

1. 顯示系統的基本性能統計：
   ```bash
   vmstat
   ```

2. 每 2 秒更新一次顯示，顯示 5 次：
   ```bash
   vmstat 2 5
   ```

3. 顯示所有內存的使用情況：
   ```bash
   vmstat -a
   ```

4. 使用 MB 作為內存單位顯示：
   ```bash
   vmstat -S m
   ```

5. 顯示帶有時間戳的統計數據：
   ```bash
   vmstat -t
   ```

## Tips
- 在進行性能調試時，將 `vmstat` 與其他工具（如 `top` 或 `iostat`）結合使用，可以獲得更全面的系統狀態。
- 定期監控系統性能，特別是在高負載情況下，可以及早發現潛在問題。
- 使用 `vmstat` 的輸出來分析內存使用情況，幫助優化應用程式的性能。
