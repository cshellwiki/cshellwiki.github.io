# [Λειτουργικό Σύστημα] Bash usermod Χρήση: Διαχείριση χρηστών στο σύστημα

## Overview
Η εντολή `usermod` χρησιμοποιείται για την τροποποίηση των χαρακτηριστικών ενός υπάρχοντος χρήστη στο σύστημα. Με αυτή την εντολή, μπορείτε να αλλάξετε διάφορες παραμέτρους του λογαριασμού χρήστη, όπως το όνομα, την ομάδα, το κωδικό πρόσβασης και άλλες ρυθμίσεις.

## Usage
Η βασική σύνταξη της εντολής `usermod` είναι η εξής:

```bash
usermod [options] [username]
```

## Common Options
- `-aG`: Προσθέτει τον χρήστη σε μια ή περισσότερες ομάδες χωρίς να αφαιρέσει άλλες ομάδες.
- `-d`: Αλλάζει το home directory του χρήστη.
- `-l`: Αλλάζει το όνομα χρήστη.
- `-p`: Ορίζει τον κωδικό πρόσβασης του χρήστη (σε κωδικοποιημένη μορφή).
- `-s`: Αλλάζει το shell του χρήστη.

## Common Examples
1. **Προσθήκη χρήστη σε ομάδα:**
   ```bash
   usermod -aG sudo username
   ```
   Αυτή η εντολή προσθέτει τον χρήστη `username` στην ομάδα `sudo`.

2. **Αλλαγή home directory:**
   ```bash
   usermod -d /new/home/directory username
   ```
   Αυτή η εντολή αλλάζει το home directory του χρήστη `username` στο `/new/home/directory`.

3. **Αλλαγή ονόματος χρήστη:**
   ```bash
   usermod -l newusername oldusername
   ```
   Αυτή η εντολή αλλάζει το όνομα του χρήστη από `oldusername` σε `newusername`.

4. **Αλλαγή shell:**
   ```bash
   usermod -s /bin/zsh username
   ```
   Αυτή η εντολή αλλάζει το default shell του χρήστη `username` σε `zsh`.

## Tips
- Πάντα να δημιουργείτε αντίγραφα ασφαλείας των ρυθμίσεων χρηστών πριν από σημαντικές αλλαγές.
- Χρησιμοποιήστε την επιλογή `-aG` προσεκτικά για να αποφύγετε την αφαίρεση άλλων ομάδων από τον χρήστη.
- Ελέγξτε τις αλλαγές που κάνατε με την εντολή `id username` για να επιβεβαιώσετε τις ομάδες και τα δικαιώματα του χρήστη.
