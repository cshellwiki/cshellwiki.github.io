# [ระบบปฏิบัติการ] C Shell (csh) watch การใช้งาน: ตรวจสอบการเปลี่ยนแปลงของคำสั่ง

## Overview
คำสั่ง `watch` ใน C Shell (csh) ใช้สำหรับตรวจสอบการเปลี่ยนแปลงของผลลัพธ์จากคำสั่งที่กำหนด โดยจะทำการเรียกคำสั่งนั้นซ้ำ ๆ ในช่วงเวลาที่กำหนด เพื่อให้ผู้ใช้สามารถติดตามการเปลี่ยนแปลงได้อย่างต่อเนื่อง

## Usage
รูปแบบพื้นฐานของคำสั่ง `watch` คือ:
```
watch [options] [arguments]
```

## Common Options
- `-n <seconds>`: กำหนดระยะเวลาในการรอระหว่างการเรียกคำสั่งใหม่ (ค่าเริ่มต้นคือ 2 วินาที)
- `-d`: เน้นการเปลี่ยนแปลงในผลลัพธ์ โดยจะทำการไฮไลต์ส่วนที่มีการเปลี่ยนแปลง
- `-t`: ปิดการแสดงเวลาที่ด้านบนของหน้าจอ

## Common Examples
1. ตรวจสอบการใช้ CPU ทุก 5 วินาที:
   ```csh
   watch -n 5 top
   ```

2. ตรวจสอบสถานะของบริการ SSH:
   ```csh
   watch systemctl status ssh
   ```

3. ตรวจสอบการเปลี่ยนแปลงในไฟล์ log:
   ```csh
   watch -d tail -n 10 /var/log/syslog
   ```

4. ตรวจสอบการใช้พื้นที่ดิสก์ทุก 10 วินาที:
   ```csh
   watch -n 10 df -h
   ```

## Tips
- ใช้ตัวเลือก `-d` เพื่อช่วยให้เห็นการเปลี่ยนแปลงได้ชัดเจนยิ่งขึ้น
- ปรับระยะเวลาใน `-n` ให้เหมาะสมกับความต้องการของคุณ เพื่อไม่ให้เกิดการโหลดระบบมากเกินไป
- หากคุณต้องการหยุดการทำงานของ `watch` ให้กด `Ctrl + C` เพื่อออกจากคำสั่ง
