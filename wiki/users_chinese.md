# [Linux] Bash 用户命令: 显示当前用户信息

## 概述
`users` 命令用于显示当前登录系统的用户列表。它会列出所有当前活动的用户，通常以空格分隔。

## 用法
基本语法如下：
```
users [选项] [参数]
```

## 常用选项
- `-n`：只显示用户的数量，而不列出具体的用户名。
- `-r`：只显示当前登录的用户的真实姓名。

## 常见示例
1. **显示当前登录的所有用户：**
   ```bash
   users
   ```

2. **显示当前登录用户的数量：**
   ```bash
   users -n
   ```

3. **显示当前登录用户的真实姓名：**
   ```bash
   users -r
   ```

## 小贴士
- `users` 命令的输出可以与其他命令结合使用，例如通过管道将结果传递给 `wc -w` 来计算用户数量。
- 该命令在多用户环境中特别有用，可以快速查看哪些用户当前在线。
