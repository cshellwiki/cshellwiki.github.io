# [리눅스] C Shell (csh) unxz 사용법: .xz 파일 압축 해제

## Overview
`unxz` 명령어는 `.xz` 형식으로 압축된 파일을 해제하는 데 사용됩니다. 이 명령어는 XZ 압축 알고리즘을 사용하여 생성된 파일의 압축을 풀어 원래의 파일로 복원합니다.

## Usage
기본 구문은 다음과 같습니다:
```
unxz [options] [arguments]
```

## Common Options
- `-k` : 압축 해제 후 원본 `.xz` 파일을 삭제하지 않고 유지합니다.
- `-v` : 압축 해제 과정에서 자세한 정보를 출력합니다.
- `-f` : 파일이 존재할 경우 강제로 덮어씁니다.

## Common Examples
다음은 `unxz` 명령어의 몇 가지 일반적인 사용 예입니다.

1. 기본적인 압축 해제:
   ```bash
   unxz example.xz
   ```

2. 원본 파일 유지하면서 압축 해제:
   ```bash
   unxz -k example.xz
   ```

3. 압축 해제 과정의 자세한 정보 출력:
   ```bash
   unxz -v example.xz
   ```

4. 기존 파일을 강제로 덮어쓰기:
   ```bash
   unxz -f example.xz
   ```

## Tips
- `.xz` 파일을 자주 사용할 경우, `-k` 옵션을 사용하여 원본 파일을 보관하는 것이 유용할 수 있습니다.
- 압축 해제 후 파일의 무결성을 확인하려면, `md5sum` 또는 `sha256sum`과 같은 도구를 사용해 보세요.
- 대량의 파일을 한 번에 압축 해제하려면, 와일드카드(`*`)를 사용하여 여러 파일을 지정할 수 있습니다. 예를 들어, `unxz *.xz`는 현재 디렉토리의 모든 `.xz` 파일을 해제합니다.
