# [리눅스] Bash unxz 사용법: XZ 압축 해제

## Overview
`unxz` 명령어는 XZ 형식으로 압축된 파일을 해제하는 데 사용됩니다. XZ는 높은 압축률을 제공하는 파일 압축 포맷으로, 주로 소프트웨어 배포 및 데이터 저장에 사용됩니다.

## Usage
기본 구문은 다음과 같습니다:

```
unxz [옵션] [파일]
```

## Common Options
- `-k`, `--keep`: 압축 해제 후 원본 파일을 유지합니다.
- `-f`, `--force`: 기존 파일을 덮어쓰도록 강제합니다.
- `-v`, `--verbose`: 압축 해제 진행 상황을 자세히 출력합니다.

## Common Examples
1. 기본적인 압축 해제:
   ```bash
   unxz example.xz
   ```

2. 원본 파일 유지하면서 압축 해제:
   ```bash
   unxz -k example.xz
   ```

3. 기존 파일 덮어쓰기:
   ```bash
   unxz -f example.xz
   ```

4. 압축 해제 진행 상황 출력:
   ```bash
   unxz -v example.xz
   ```

## Tips
- 압축 해제할 파일이 여러 개일 경우, 와일드카드를 사용하여 한 번에 여러 파일을 해제할 수 있습니다. 예를 들어, `unxz *.xz`는 현재 디렉토리의 모든 XZ 파일을 해제합니다.
- 대용량 파일을 다룰 때는 `-k` 옵션을 사용하여 원본 파일을 보존하는 것이 유용할 수 있습니다.
