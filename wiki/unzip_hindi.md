# [Linux] Bash unzip उपयोग: फ़ाइलों को अनज़िप करना

## Overview
`unzip` कमांड का उपयोग ZIP फ़ाइलों को अनज़िप करने के लिए किया जाता है। यह आपको संकुचित फ़ाइलों को निकालने और उनके अंदर की सामग्री को एक्सेस करने की अनुमति देता है।

## Usage
`unzip` कमांड का मूल सिंटैक्स निम्नलिखित है:

```
unzip [options] [arguments]
```

## Common Options
- `-l`: ZIP फ़ाइल की सामग्री की सूची दिखाता है।
- `-d <directory>`: निकाली गई फ़ाइलों को निर्दिष्ट निर्देशिका में डालता है।
- `-o`: बिना किसी पुष्टि के फ़ाइलों को ओवरराइट करता है।
- `-q`: चुपचाप अनज़िप करता है, बिना किसी आउटपुट के।

## Common Examples
1. एक ZIP फ़ाइल को अनज़िप करना:
   ```bash
   unzip example.zip
   ```

2. ZIP फ़ाइल की सामग्री की सूची देखना:
   ```bash
   unzip -l example.zip
   ```

3. फ़ाइलों को एक विशेष निर्देशिका में अनज़िप करना:
   ```bash
   unzip example.zip -d /path/to/directory
   ```

4. बिना पुष्टि के फ़ाइलों को ओवरराइट करना:
   ```bash
   unzip -o example.zip
   ```

5. चुपचाप अनज़िप करना:
   ```bash
   unzip -q example.zip
   ```

## Tips
- सुनिश्चित करें कि आप उस निर्देशिका में हैं जहाँ आप अनज़िप करना चाहते हैं, या `-d` विकल्प का उपयोग करें।
- यदि आप केवल कुछ फ़ाइलें अनज़िप करना चाहते हैं, तो फ़ाइलों के नाम को निर्दिष्ट करें:
  ```bash
  unzip example.zip file1.txt file2.txt
  ```
- नियमित रूप से ZIP फ़ाइलों का बैकअप लें, ताकि डेटा हानि से बचा जा सके।
