# [লিনাক্স] Bash vgextend ব্যবহারের সমতুল্য: লজিক্যাল ভলিউম গ্রুপে নতুন ভলিউম যোগ করা

## Overview
`vgextend` কমান্ডটি লিনাক্সে লজিক্যাল ভলিউম ম্যানেজমেন্ট (LVM) এর অংশ। এটি একটি বিদ্যমান ভলিউম গ্রুপে নতুন প্যার্টিশন বা ভলিউম যোগ করার জন্য ব্যবহৃত হয়, যা ডিস্ক স্পেস বাড়াতে সহায়ক।

## Usage
`vgextend` কমান্ডের মৌলিক সিনট্যাক্স নিম্নরূপ:

```bash
vgextend [options] [volume_group] [physical_volume...]
```

## Common Options
- `-d`, `--debug`: ডিবাগ তথ্য প্রদর্শন করে।
- `-h`, `--help`: সাহায্য তথ্য প্রদর্শন করে।
- `-v`, `--verbose`: বিস্তারিত আউটপুট দেখায়।

## Common Examples
1. একটি নতুন প্যার্টিশন যোগ করা:
   ```bash
   vgextend my_volume_group /dev/sdb1
   ```

2. একাধিক প্যার্টিশন যোগ করা:
   ```bash
   vgextend my_volume_group /dev/sdb1 /dev/sdc1
   ```

3. ডিবাগ মোডে কমান্ড চালানো:
   ```bash
   vgextend -d my_volume_group /dev/sdb1
   ```

## Tips
- নিশ্চিত করুন যে নতুন প্যার্টিশনটি LVM ফরম্যাটে ফরম্যাট করা হয়েছে।
- `vgdisplay` কমান্ড ব্যবহার করে ভলিউম গ্রুপের বর্তমান অবস্থা পরীক্ষা করুন।
- নতুন ভলিউম যোগ করার পর, `lvextend` ব্যবহার করে লজিক্যাল ভলিউমের আকার বাড়ান।
