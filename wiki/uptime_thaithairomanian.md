# [ไทย] Bash uptime การใช้งาน: แสดงเวลาที่ระบบทำงาน

## Overview
คำสั่ง `uptime` ใช้เพื่อแสดงเวลาที่ระบบปฏิบัติการทำงานอยู่ รวมถึงจำนวนผู้ใช้งานในปัจจุบันและข้อมูลเกี่ยวกับการโหลดของระบบในช่วงเวลาต่างๆ

## Usage
คำสั่ง `uptime` มีรูปแบบพื้นฐานดังนี้:

```bash
uptime [options]
```

## Common Options
- `-p`: แสดงเวลาที่ระบบทำงานในรูปแบบที่อ่านง่าย
- `-s`: แสดงเวลาที่ระบบเริ่มทำงาน (boot time)

## Common Examples
ตัวอย่างการใช้งานคำสั่ง `uptime` มีดังนี้:

1. แสดงเวลาที่ระบบทำงานและข้อมูลเกี่ยวกับผู้ใช้งาน:
   ```bash
   uptime
   ```

2. แสดงเวลาที่ระบบทำงานในรูปแบบที่อ่านง่าย:
   ```bash
   uptime -p
   ```

3. แสดงเวลาที่ระบบเริ่มทำงาน:
   ```bash
   uptime -s
   ```

## Tips
- ใช้คำสั่ง `uptime` เป็นประจำเพื่อตรวจสอบสถานะของระบบและการโหลดของ CPU
- คำนึงถึงการโหลดของระบบเมื่อมีผู้ใช้งานหลายคน เพื่อให้แน่ใจว่าระบบทำงานได้อย่างราบรื่น
- คำสั่งนี้สามารถใช้ร่วมกับคำสั่งอื่นๆ เช่น `top` เพื่อดูรายละเอียดเพิ่มเติมเกี่ยวกับการใช้งานระบบ
