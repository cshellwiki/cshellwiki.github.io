# [Linux] Bash whoami उपयोग: वर्तमान उपयोगकर्ता का नाम दिखाना

## Overview
`whoami` कमांड का उपयोग वर्तमान उपयोगकर्ता का नाम देखने के लिए किया जाता है। यह एक सरल और प्रभावी तरीका है यह जानने का कि आप किस उपयोगकर्ता के रूप में सिस्टम पर लॉग इन हैं।

## Usage
कमांड का मूल सिंटैक्स इस प्रकार है:
```bash
whoami [options] [arguments]
```

## Common Options
`whoami` कमांड के लिए कोई विशेष विकल्प नहीं होते हैं, लेकिन आप इसे सामान्य तरीके से उपयोग कर सकते हैं। 

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. **साधारण उपयोग**:
   ```bash
   whoami
   ```
   यह कमांड वर्तमान उपयोगकर्ता का नाम प्रदर्शित करेगा।

2. **स्क्रिप्ट में उपयोग**:
   ```bash
   echo "आप वर्तमान में $USER के रूप में लॉग इन हैं: $(whoami)"
   ```
   यह कमांड एक संदेश प्रदर्शित करेगा जिसमें वर्तमान उपयोगकर्ता का नाम शामिल होगा।

3. **सिस्टम जानकारी के साथ**:
   ```bash
   echo "आपका उपयोगकर्ता नाम: $(whoami) और होस्ट नाम: $(hostname)"
   ```
   यह कमांड उपयोगकर्ता का नाम और होस्ट नाम दोनों प्रदर्शित करेगा।

## Tips
- `whoami` कमांड का उपयोग स्क्रिप्टिंग में करें ताकि आप सुनिश्चित कर सकें कि आप सही उपयोगकर्ता के रूप में कार्य कर रहे हैं।
- यदि आप एकाधिक उपयोगकर्ताओं के बीच स्विच कर रहे हैं, तो `whoami` का उपयोग करके हमेशा यह जांचें कि आप किस उपयोगकर्ता के रूप में कार्य कर रहे हैं।
- इस कमांड को अन्य कमांड के साथ संयोजित करें जैसे `sudo` या `su` यह देखने के लिए कि आप किस उपयोगकर्ता के रूप में कार्य कर रहे हैं।
