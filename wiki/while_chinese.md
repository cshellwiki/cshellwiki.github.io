# [Linux] Bash while 使用方法: 用于循环执行命令

## 概述
`while` 命令用于在满足特定条件时重复执行一段命令。它通常用于需要持续检查某个条件并根据该条件执行操作的场景。

## 用法
基本语法如下：
```bash
while [条件]
do
    命令
done
```

## 常用选项
- `条件`：可以是任何返回状态的命令，只有当该命令返回真（0）时，循环才会继续执行。
- `命令`：在每次循环中要执行的命令。

## 常见示例

### 示例 1: 简单计数循环
```bash
count=1
while [ $count -le 5 ]
do
    echo "计数: $count"
    count=$((count + 1))
done
```
这个示例将输出从 1 到 5 的计数。

### 示例 2: 读取文件行
```bash
while IFS= read -r line
do
    echo "读取行: $line"
done < 文件名.txt
```
此示例逐行读取文件并输出每一行。

### 示例 3: 无限循环
```bash
while true
do
    echo "这是一个无限循环"
    sleep 1
done
```
这个示例将每秒输出一次“这是一个无限循环”。

## 提示
- 确保在循环中有条件可以终止，以避免无限循环。
- 使用 `sleep` 命令可以减少 CPU 使用率，特别是在无限循环中。
- 在处理文件时，使用 `IFS` 变量可以避免因空格或特殊字符导致的问题。
