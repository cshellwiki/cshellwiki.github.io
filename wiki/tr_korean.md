# [리눅스] Bash tr 사용법: 문자 변환 및 삭제

## Overview
`tr` 명령어는 입력된 텍스트에서 특정 문자나 문자열을 변환하거나 삭제하는 데 사용됩니다. 주로 파이프를 통해 다른 명령어의 출력을 처리할 때 유용합니다.

## Usage
기본 구문은 다음과 같습니다:
```bash
tr [옵션] [변환할 문자] [대체할 문자]
```

## Common Options
- `-d`: 지정한 문자를 삭제합니다.
- `-s`: 연속된 중복 문자를 하나로 줄입니다.
- `-c`: 지정한 문자 외의 모든 문자를 대상으로 작업합니다.

## Common Examples
1. **문자 변환**
   ```bash
   echo "hello world" | tr 'h' 'H'
   ```
   출력: `Hello world`

2. **문자 삭제**
   ```bash
   echo "hello world" | tr -d 'o'
   ```
   출력: `hell wrld`

3. **중복 문자 줄이기**
   ```bash
   echo "aaabbbccc" | tr -s 'a' 'b'
   ```
   출력: `bbbc`

4. **대문자를 소문자로 변환**
   ```bash
   echo "Hello World" | tr 'A-Z' 'a-z'
   ```
   출력: `hello world`

5. **특정 문자를 다른 문자로 변환**
   ```bash
   echo "123-456-789" | tr '-' ':'
   ```
   출력: `123:456:789`

## Tips
- `tr` 명령어는 입력이 필요하므로, 파일에서 직접 읽으려면 `cat`과 함께 사용하거나 리다이렉션을 활용하세요.
- 여러 문자를 동시에 변환하려면 변환할 문자와 대체할 문자를 동일한 길이로 지정해야 합니다.
- `tr`은 대소문자 변환, 문자 삭제 및 변환에 매우 유용하므로, 텍스트 처리 작업에서 자주 활용하세요.
