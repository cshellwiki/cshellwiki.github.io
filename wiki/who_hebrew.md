# [לינוקס] Bash who שימוש: הצגת משתמשים מחוברים

## Overview
הפקודה `who` משמשת להצגת רשימת המשתמשים המחוברים למערכת. היא מספקת מידע על כל משתמש, כולל שם המשתמש, זמן החיבור, והטרמינל שבו הם מחוברים.

## Usage
התחביר הבסיסי של הפקודה הוא:
```
who [אפשרויות] [ארגומנטים]
```

## Common Options
- `-a`: מציג מידע נוסף על המשתמשים, כולל זמני חיבור.
- `-b`: מציג את הזמן האחרון שבו המערכת הופעלה מחדש.
- `-q`: מציג רק את שמות המשתמשים המחוברים, ללא מידע נוסף.

## Common Examples
להלן מספר דוגמאות שימושיות לפקודת `who`:

1. הצגת כל המשתמשים המחוברים:
   ```bash
   who
   ```

2. הצגת מידע נוסף על המשתמשים:
   ```bash
   who -a
   ```

3. הצגת הזמן האחרון שבו המערכת הופעלה מחדש:
   ```bash
   who -b
   ```

4. הצגת שמות המשתמשים בלבד:
   ```bash
   who -q
   ```

## Tips
- השתמש בפקודה `who` יחד עם `grep` כדי לחפש משתמש מסוים:
   ```bash
   who | grep username
   ```
- אם אתה רוצה לדעת מי מחובר למערכת בזמן אמת, תוכל להשתמש בפקודה `watch` יחד עם `who`:
   ```bash
   watch who
   ```
- זכור כי הפקודה `who` מציגה רק משתמשים המחוברים באופן פעיל, ולא משתמשים שהיו מחוברים בעבר.
