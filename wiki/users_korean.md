# [리눅스] Bash 사용자 명령어: 사용자 정보를 표시합니다.

## Overview
`users` 명령어는 현재 로그인한 사용자들의 이름을 표시합니다. 이 명령어는 시스템의 사용자 세션을 모니터링하는 데 유용합니다.

## Usage
기본 구문은 다음과 같습니다:
```
users [options] [arguments]
```

## Common Options
- `-n`: 사용자 이름을 중복 없이 표시합니다.
- `-u`: 사용자 이름과 함께 로그인 세션의 정보를 표시합니다.

## Common Examples
1. 현재 로그인한 사용자 목록 보기:
   ```bash
   users
   ```

2. 중복 없이 사용자 이름 표시:
   ```bash
   users -n
   ```

3. 사용자 이름과 로그인 세션 정보 보기:
   ```bash
   users -u
   ```

## Tips
- `users` 명령어는 간단하지만, 시스템의 사용자 활동을 빠르게 확인하는 데 매우 유용합니다.
- 다른 사용자 관련 명령어인 `who`나 `w`와 함께 사용하면 더 많은 정보를 얻을 수 있습니다.
