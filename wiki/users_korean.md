# [리눅스] C Shell (csh) 사용자 명령어: 사용자 목록을 표시합니다.

## Overview
`users` 명령어는 현재 시스템에 로그인한 사용자들의 목록을 표시합니다. 이 명령어는 여러 사용자가 동시에 로그인한 경우, 각 사용자의 이름을 공백으로 구분하여 출력합니다.

## Usage
기본 구문은 다음과 같습니다:

```
users [options] [arguments]
```

## Common Options
- `-n`: 사용자 이름을 출력할 때, 각 이름의 개수를 제한합니다.

## Common Examples
다음은 `users` 명령어의 몇 가지 일반적인 사용 예입니다.

1. 현재 로그인한 사용자 목록 보기:
   ```csh
   users
   ```

2. 사용자 이름 개수 제한하여 보기:
   ```csh
   users -n 2
   ```

## Tips
- `users` 명령어는 간단하지만, 시스템의 현재 로그인 상태를 빠르게 확인하는 데 유용합니다.
- 여러 사용자가 동시에 로그인한 경우, `who` 명령어를 사용하여 더 많은 정보를 얻을 수 있습니다.
