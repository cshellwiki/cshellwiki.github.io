# [Linux] Bash w 使用法: 查看登入使用者資訊

## Overview
`w` 命令用來顯示目前登入系統的使用者資訊，以及他們正在執行的活動。這個命令可以幫助系統管理員監控系統的使用情況。

## Usage
基本語法如下：
```
w [options] [arguments]
```

## Common Options
- `-h`：不顯示標題。
- `-s`：以簡潔模式顯示，省略不必要的資訊。
- `-f`：顯示完整的登入時間。
- `-u`：顯示使用者的 idle 時間。

## Common Examples
以下是一些常見的使用範例：

1. **顯示所有登入使用者的資訊**
   ```bash
   w
   ```

2. **以簡潔模式顯示使用者資訊**
   ```bash
   w -s
   ```

3. **不顯示標題的使用者資訊**
   ```bash
   w -h
   ```

4. **顯示完整的登入時間**
   ```bash
   w -f
   ```

5. **顯示使用者的 idle 時間**
   ```bash
   w -u
   ```

## Tips
- 使用 `w` 命令時，可以搭配其他命令如 `grep` 來過濾特定使用者的資訊。
- 定期檢查登入使用者的活動，可以幫助你了解系統的使用情況，並及早發現潛在的問題。
- 如果你只需要特定的資訊，可以考慮使用 `-s` 選項來簡化輸出。
