# [লিনাক্স] Bash wget ব্যবহার: ফাইল ডাউনলোড করা

## Overview
`wget` হল একটি শক্তিশালী কমান্ড-লাইন টুল যা ইন্টারনেট থেকে ফাইল ডাউনলোড করার জন্য ব্যবহৃত হয়। এটি HTTP, HTTPS এবং FTP প্রোটোকল সমর্থন করে এবং ব্যবহারকারীদের জন্য সহজে ফাইলগুলি ডাউনলোড করার সুবিধা প্রদান করে।

## Usage
`wget` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [file]`: ডাউনলোড করা ফাইলের নাম নির্দিষ্ট করে।
- `-c`: অসম্পূর্ণ ডাউনলোড পুনরায় শুরু করতে ব্যবহৃত হয়।
- `-r`: রিকার্সিভ ডাউনলোড সক্ষম করে, যা একটি ওয়েবসাইটের সমস্ত লিঙ্ক অনুসরণ করে।
- `-q`: সাইলেন্ট মোডে ডাউনলোড করে, যেখানে কমান্ডের আউটপুট কম দেখানো হয়।
- `--limit-rate=[rate]`: ডাউনলোডের গতিকে সীমিত করে।

## Common Examples
1. একটি সাধারণ ফাইল ডাউনলোড করা:
   ```bash
   wget https://example.com/file.zip
   ```

2. ডাউনলোড করা ফাইলের নাম পরিবর্তন করা:
   ```bash
   wget -O newfile.zip https://example.com/file.zip
   ```

3. অসম্পূর্ণ ডাউনলোড পুনরায় শুরু করা:
   ```bash
   wget -c https://example.com/largefile.zip
   ```

4. একটি ওয়েবসাইটের সমস্ত লিঙ্ক রিকার্সিভভাবে ডাউনলোড করা:
   ```bash
   wget -r https://example.com/
   ```

5. সাইলেন্ট মোডে ডাউনলোড করা:
   ```bash
   wget -q https://example.com/file.zip
   ```

## Tips
- ডাউনলোডের গতিকে সীমিত করতে `--limit-rate` ব্যবহার করুন, যাতে আপনার ইন্টারনেট ব্যান্ডউইথ সংরক্ষণ হয়।
- রিকার্সিভ ডাউনলোড করার সময় সাবধানতা অবলম্বন করুন, কারণ এটি অনেক ফাইল ডাউনলোড করতে পারে।
- ডাউনলোডের অগ্রগতি দেখতে `-q` অপশন ব্যবহার না করার চেষ্টা করুন, কারণ এটি সাইলেন্ট মোডে চলে যাবে।
