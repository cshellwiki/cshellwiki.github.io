# [Linux] Bash stty 使用法: 端末の設定を変更する

## Overview
`stty` コマンドは、端末の設定を変更するために使用されます。このコマンドを使うことで、入力や出力の動作を制御したり、特定の機能を有効または無効にしたりすることができます。

## Usage
基本的な構文は次のとおりです。

```bash
stty [options] [arguments]
```

## Common Options
- `-a` : 現在の端末設定をすべて表示します。
- `-g` : 現在の設定を保存するための値を表示します。
- `-F <file>` : 指定したファイルの端末設定を変更します。
- `erase <char>` : バックスペースキーの動作を変更します。
- `intr <char>` : 割り込みキーの動作を変更します。

## Common Examples
以下は、`stty` コマンドのいくつかの実用的な例です。

### 1. 現在の設定を表示する
```bash
stty -a
```

### 2. バックスペースキーを設定する
```bash
stty erase ^H
```

### 3. 割り込みキーを設定する
```bash
stty intr ^C
```

### 4. 端末の設定をファイルに保存する
```bash
stty -g > settings.txt
```

### 5. 別の端末デバイスの設定を変更する
```bash
stty -F /dev/ttyUSB0 9600
```

## Tips
- `stty -a` を使用して、現在の端末設定を確認することができます。これにより、どの設定が有効かを把握できます。
- 設定を変更する前に、現在の設定を `stty -g` で保存しておくと、後で元に戻すのが簡単です。
- スクリプト内で `stty` を使用する際は、変更後に元の設定に戻すことを忘れないようにしましょう。
