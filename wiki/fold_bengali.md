# [লিনাক্স] Bash fold ব্যবহার: টেক্সট লাইনকে নির্দিষ্ট প্রস্থে ভাঙা

## Overview
`fold` কমান্ডটি টেক্সট ফাইলের লাইনগুলোকে নির্দিষ্ট প্রস্থে ভাঙার জন্য ব্যবহৃত হয়। এটি সাধারণত লম্বা লাইনগুলোকে ছোট ছোট অংশে বিভক্ত করতে সাহায্য করে, যাতে সেগুলো সহজে পড়া যায়।

## Usage
`fold` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```
fold [options] [arguments]
```

## Common Options
- `-w`, `--width`: নির্দিষ্ট প্রস্থ (ডিফল্ট 80 ক্যারেক্টার) সেট করতে ব্যবহৃত হয়।
- `-s`, `--split-only`: শব্দের মাঝখানে ভাঙার চেষ্টা করে, যাতে শব্দগুলো সম্পূর্ণ থাকে।
- `-b`, `--bytes`: বাইটের ভিত্তিতে প্রস্থ নির্ধারণ করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### উদাহরণ 1: ডিফল্ট প্রস্থে ভাঙা
```
fold myfile.txt
```
এটি `myfile.txt` ফাইলের লাইনগুলোকে ডিফল্ট 80 ক্যারেক্টারে ভাঙবে।

### উদাহরণ 2: নির্দিষ্ট প্রস্থে ভাঙা
```
fold -w 50 myfile.txt
```
এটি `myfile.txt` ফাইলের লাইনগুলোকে 50 ক্যারেক্টারে ভাঙবে।

### উদাহরণ 3: শব্দের মাঝখানে ভাঙা
```
fold -s -w 40 myfile.txt
```
এটি `myfile.txt` ফাইলের লাইনগুলোকে 40 ক্যারেক্টারে শব্দের মাঝখানে ভাঙবে।

## Tips
- দীর্ঘ টেক্সট ফাইলের জন্য `fold` ব্যবহার করা হলে, প্রস্থ নির্বাচন করার সময় আপনার টার্মিনালের প্রস্থ বিবেচনা করুন।
- `fold` কমান্ডের আউটপুটকে অন্য কমান্ডের সাথে পাইপ করে ব্যবহার করা যেতে পারে, যেমন `less` বা `more`, যাতে আউটপুটটি আরও সহজে পড়া যায়।
- ফাইলের পরিবর্তে stdin থেকে ইনপুট নিতে `fold` ব্যবহার করতে পারেন, যা স্ক্রিপ্টিংয়ে খুবই কার্যকর।
