# [Linux] Bash wait Χρήση: Περιμένει την ολοκλήρωση διεργασιών

## Overview
Η εντολή `wait` χρησιμοποιείται στο Bash για να περιμένει την ολοκλήρωση μιας ή περισσότερων διεργασιών. Είναι ιδιαίτερα χρήσιμη όταν θέλουμε να διασφαλίσουμε ότι μια διεργασία έχει ολοκληρωθεί πριν προχωρήσουμε σε επόμενα βήματα στο σενάριο μας.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
wait [options] [arguments]
```

## Common Options
- `-n`: Περιμένει την ολοκλήρωση οποιασδήποτε διεργασίας που έχει εκτελεστεί στο παρασκήνιο.
- `PID`: Μπορείτε να καθορίσετε έναν συγκεκριμένο αριθμό διεργασίας (PID) για να περιμένετε μόνο αυτήν τη διεργασία.

## Common Examples

### Παράδειγμα 1: Περιμένοντας μια διεργασία
Αν εκτελέσετε μια διεργασία στο παρασκήνιο και θέλετε να περιμένετε την ολοκλήρωσή της:

```bash
sleep 5 &
wait $!
echo "Η διεργασία ολοκληρώθηκε."
```

### Παράδειγμα 2: Περιμένοντας πολλές διεργασίες
Μπορείτε να περιμένετε πολλές διεργασίες ταυτόχρονα:

```bash
sleep 3 &
sleep 5 &
wait
echo "Όλες οι διεργασίες ολοκληρώθηκαν."
```

### Παράδειγμα 3: Χρήση της επιλογής -n
Αν θέλετε να περιμένετε την ολοκλήρωση οποιασδήποτε διεργασίας στο παρασκήνιο:

```bash
sleep 2 &
sleep 4 &
wait -n
echo "Μία από τις διεργασίες ολοκληρώθηκε."
```

## Tips
- Χρησιμοποιήστε την `wait` για να διασφαλίσετε ότι οι διεργασίες που εκτελούνται στο παρασκήνιο έχουν ολοκληρωθεί πριν προχωρήσετε σε επόμενα βήματα.
- Αν έχετε πολλές διεργασίες, μπορείτε να χρησιμοποιήσετε την επιλογή `-n` για να περιμένετε την ολοκλήρωση της πρώτης που θα τελειώσει.
- Είναι καλή πρακτική να ελέγχετε την έξοδο των διεργασιών σας για να διασφαλίσετε ότι δεν υπάρχουν σφάλματα πριν προχωρήσετε.
