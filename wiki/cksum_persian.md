# [لینوکس] Bash cksum استفاده: محاسبه چک‌سام فایل‌ها

## Overview
دستور `cksum` در Bash برای محاسبه و نمایش چک‌سام (checksum) و تعداد بایت‌های یک فایل استفاده می‌شود. این چک‌سام به عنوان یک کد منحصر به فرد برای شناسایی تغییرات در محتویات فایل‌ها به کار می‌رود.

## Usage
سینتکس پایه دستور `cksum` به صورت زیر است:

```bash
cksum [options] [arguments]
```

## Common Options
- `-a, --algorithm`: انتخاب الگوریتم چک‌سام (به عنوان مثال، `md5`).
- `-h, --help`: نمایش راهنمای استفاده از دستور.
- `-V, --version`: نمایش نسخه دستور.

## Common Examples
1. محاسبه چک‌سام یک فایل:
   ```bash
   cksum myfile.txt
   ```

2. محاسبه چک‌سام چندین فایل به طور همزمان:
   ```bash
   cksum file1.txt file2.txt file3.txt
   ```

3. استفاده از گزینه `-a` برای انتخاب الگوریتم:
   ```bash
   cksum -a md5 myfile.txt
   ```

4. نمایش راهنمای استفاده:
   ```bash
   cksum --help
   ```

## Tips
- برای بررسی تغییرات در فایل‌ها، چک‌سام آن‌ها را در زمان‌های مختلف محاسبه کنید و مقایسه کنید.
- از چک‌سام برای تأیید صحت دانلود فایل‌ها استفاده کنید.
- در هنگام کار با فایل‌های بزرگ، از گزینه‌های مناسب برای بهینه‌سازی عملکرد استفاده کنید.
