# [Linux] Bash whoami の使い方: 現在のユーザー名を表示する

## Overview
`whoami` コマンドは、現在ログインしているユーザーの名前を表示するためのシンプルなコマンドです。このコマンドを使用することで、システム上で自分がどのユーザーとして作業しているかを確認できます。

## Usage
基本的な構文は以下の通りです。

```
whoami [options] [arguments]
```

## Common Options
`whoami` コマンドには特に多くのオプションはありませんが、以下のようなものがあります。

- `--help` : コマンドの使い方を表示します。
- `--version` : `whoami` コマンドのバージョン情報を表示します。

## Common Examples
以下に、`whoami` コマンドのいくつかの実用的な例を示します。

### 例 1: 現在のユーザー名を表示
```bash
whoami
```

### 例 2: ヘルプを表示
```bash
whoami --help
```

### 例 3: バージョン情報を表示
```bash
whoami --version
```

## Tips
- `whoami` コマンドは、スクリプト内で現在のユーザーを確認する際に便利です。
- 他のユーザーに切り替えた後、再度 `whoami` を実行することで、正しいユーザー名を確認できます。
- `whoami` は、特に複数のユーザーが同じシステムを使用する場合に役立ちます。
