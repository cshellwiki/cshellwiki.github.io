# [리눅스] C Shell (csh) vmstat 사용법: 시스템 성능 모니터링

## Overview
`vmstat` 명령어는 시스템의 메모리, 프로세서, 스왑, I/O 및 시스템 활동에 대한 정보를 제공하여 시스템 성능을 모니터링하는 데 사용됩니다.

## Usage
기본 구문은 다음과 같습니다:

```
vmstat [options] [arguments]
```

## Common Options
- `-a`: 활성 메모리와 비활성 메모리를 모두 표시합니다.
- `-m`: 메모리 통계 정보를 표시합니다.
- `-s`: 시스템 통계 요약을 제공합니다.
- `-n`: 출력에서 헤더를 한 번만 표시합니다.

## Common Examples
다음은 `vmstat` 명령어의 몇 가지 일반적인 사용 예입니다:

1. 기본 시스템 상태 확인:
   ```bash
   vmstat
   ```

2. 2초마다 시스템 상태를 5번 출력:
   ```bash
   vmstat 2 5
   ```

3. 메모리 통계 정보 표시:
   ```bash
   vmstat -m
   ```

4. 시스템 통계 요약 표시:
   ```bash
   vmstat -s
   ```

## Tips
- `vmstat`의 출력을 정기적으로 모니터링하여 시스템의 성능 문제를 조기에 발견할 수 있습니다.
- 다른 성능 모니터링 도구와 함께 사용하면 더욱 유용한 정보를 얻을 수 있습니다.
- 시스템의 부하가 높은 경우, `vmstat`의 출력에서 프로세서와 메모리 사용량을 주의 깊게 살펴보세요.
