# [리눅스] Bash vmstat 사용법: 시스템 성능 모니터링

## Overview
`vmstat` 명령어는 시스템의 가상 메모리, 프로세스, CPU 활동 및 I/O 통계를 모니터링하는 데 사용됩니다. 이 명령어는 시스템의 성능을 분석하고 문제를 진단하는 데 유용한 정보를 제공합니다.

## Usage
기본 구문은 다음과 같습니다:
```bash
vmstat [options] [arguments]
```

## Common Options
- `-a`: 활성화된 메모리와 비활성화된 메모리를 포함한 메모리 통계를 표시합니다.
- `-m`: 메모리 페이지 통계를 보여줍니다.
- `-s`: 시스템의 통계를 요약하여 표시합니다.
- `-t`: 시간 스탬프를 포함하여 결과를 표시합니다.
- `delay`: 결과를 반복적으로 출력할 간격(초)을 설정합니다.

## Common Examples
1. 기본 시스템 통계 보기:
   ```bash
   vmstat
   ```

2. 2초 간격으로 5회 통계 출력:
   ```bash
   vmstat 2 5
   ```

3. 메모리 페이지 통계 보기:
   ```bash
   vmstat -m
   ```

4. 시스템 통계 요약 보기:
   ```bash
   vmstat -s
   ```

5. 시간 스탬프와 함께 통계 출력:
   ```bash
   vmstat -t
   ```

## Tips
- `vmstat` 명령어는 시스템의 성능 문제를 진단하는 데 유용하므로, 정기적으로 모니터링하는 것이 좋습니다.
- 여러 번의 샘플을 수집하여 평균값을 분석하면 더 정확한 성능 평가를 할 수 있습니다.
- 다른 성능 모니터링 도구와 함께 사용하면 시스템의 전반적인 상태를 더욱 잘 이해할 수 있습니다.
