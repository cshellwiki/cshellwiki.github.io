# [Linux] Bash yum उपयोग: पैकेज प्रबंधन के लिए एक उपकरण

## Overview
`yum` (Yellowdog Updater Modified) एक पैकेज प्रबंधन उपकरण है जो RPM (Red Hat Package Manager) आधारित वितरणों पर सॉफ्टवेयर पैकेजों को स्थापित, अपडेट और हटाने के लिए उपयोग किया जाता है। यह उपयोगकर्ताओं को सॉफ्टवेयर को आसानी से प्रबंधित करने की सुविधा देता है।

## Usage
`yum` का मूल सिंटैक्स निम्नलिखित है:

```bash
yum [options] [arguments]
```

## Common Options
- `install`: एक या अधिक पैकेजों को स्थापित करने के लिए।
- `remove`: एक या अधिक पैकेजों को हटाने के लिए।
- `update`: स्थापित पैकेजों को अपडेट करने के लिए।
- `search`: उपलब्ध पैकेजों में खोज करने के लिए।
- `info`: किसी विशेष पैकेज के बारे में जानकारी प्राप्त करने के लिए।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

### पैकेज स्थापित करना
```bash
yum install httpd
```
यह कमांड `httpd` (Apache वेब सर्वर) पैकेज को स्थापित करेगा।

### पैकेज हटाना
```bash
yum remove httpd
```
यह कमांड `httpd` पैकेज को हटाएगा।

### पैकेज अपडेट करना
```bash
yum update httpd
```
यह कमांड `httpd` पैकेज को नवीनतम संस्करण में अपडेट करेगा।

### पैकेज खोज करना
```bash
yum search nginx
```
यह कमांड `nginx` से संबंधित पैकेजों की खोज करेगा।

### पैकेज जानकारी प्राप्त करना
```bash
yum info httpd
```
यह कमांड `httpd` पैकेज के बारे में विस्तृत जानकारी प्रदान करेगा।

## Tips
- हमेशा `yum update` चलाकर अपने सिस्टम को अद्यतित रखें।
- पैकेज स्थापित करने से पहले `yum search` का उपयोग करके सुनिश्चित करें कि पैकेज उपलब्ध है।
- यदि आप एक ही बार में कई पैकेज स्थापित करना चाहते हैं, तो उनके नामों को एक साथ स्पेस से अलग करें: `yum install package1 package2`।
