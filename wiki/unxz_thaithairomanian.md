# [ระบบปฏิบัติการ] Bash unxz การใช้งาน: แยกไฟล์ .xz

## Overview
คำสั่ง `unxz` ใช้สำหรับการแยกไฟล์ที่ถูกบีบอัดด้วยรูปแบบ .xz ซึ่งเป็นรูปแบบการบีบอัดที่มีประสิทธิภาพสูง โดยคำสั่งนี้จะคืนค่าไฟล์ต้นฉบับที่ถูกบีบอัดกลับมาให้คุณ

## Usage
รูปแบบพื้นฐานของคำสั่ง `unxz` คือ:

```
unxz [options] [arguments]
```

## Common Options
- `-k`, `--keep`: เก็บไฟล์ต้นฉบับไว้หลังจากการแยก
- `-f`, `--force`: บังคับให้เขียนทับไฟล์ที่มีอยู่แล้ว
- `-v`, `--verbose`: แสดงรายละเอียดเพิ่มเติมในระหว่างการทำงาน

## Common Examples
- แยกไฟล์ .xz:
  ```bash
  unxz file.xz
  ```

- แยกไฟล์และเก็บไฟล์ต้นฉบับ:
  ```bash
  unxz -k file.xz
  ```

- บังคับให้เขียนทับไฟล์ที่มีอยู่แล้ว:
  ```bash
  unxz -f file.xz
  ```

- แสดงรายละเอียดในระหว่างการแยก:
  ```bash
  unxz -v file.xz
  ```

## Tips
- ตรวจสอบให้แน่ใจว่าคุณมีพื้นที่เพียงพอในดิสก์สำหรับไฟล์ที่ถูกแยก
- ใช้ตัวเลือก `-k` หากคุณต้องการเก็บไฟล์ต้นฉบับไว้เพื่อความปลอดภัย
- หากคุณทำงานกับหลายไฟล์ .xz ให้ใช้คำสั่งแบบกลุ่ม เช่น `unxz *.xz` เพื่อแยกไฟล์ทั้งหมดในโฟลเดอร์เดียวกัน
