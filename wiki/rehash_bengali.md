# [লিনাক্স] Bash rehash ব্যবহার: নতুন কমান্ড ক্যাশে আপডেট করা

## Overview
`rehash` কমান্ডটি Bash শেলের জন্য একটি গুরুত্বপূর্ণ ফাংশন। এটি নতুন কমান্ড বা স্ক্রিপ্ট যুক্ত করার পর, সেগুলির ক্যাশে আপডেট করতে ব্যবহৃত হয়। যখন আপনি নতুন প্রোগ্রাম ইনস্টল করেন বা নতুন স্ক্রিপ্ট তৈরি করেন, তখন `rehash` কমান্ডটি ব্যবহার করে শেলকে সেগুলি চিনতে সাহায্য করে।

## Usage
`rehash` কমান্ডের মৌলিক সিনট্যাক্স নিম্নরূপ:

```bash
rehash [options] [arguments]
```

## Common Options
`rehash` কমান্ডের জন্য সাধারণ অপশনগুলি হল:
- কোন বিশেষ অপশন নেই, এটি সাধারণত আর্গুমেন্ট ছাড়াই ব্যবহৃত হয়।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### উদাহরণ 1: নতুন কমান্ড ক্যাশে আপডেট করা
যখন আপনি নতুন একটি স্ক্রিপ্ট তৈরি করেন এবং সেটি ব্যবহার করতে চান, তখন `rehash` ব্যবহার করুন:

```bash
rehash
```

### উদাহরণ 2: নতুন ইনস্টল করা প্রোগ্রাম চিনতে সাহায্য করা
যদি আপনি নতুন একটি প্রোগ্রাম ইনস্টল করেন, তাহলে এটি চিনতে `rehash` ব্যবহার করুন:

```bash
rehash
```

## Tips
- `rehash` কমান্ডটি প্রায়শই ব্যবহার করা হয় যখন আপনি নতুন সফটওয়্যার ইনস্টল করেন বা নতুন স্ক্রিপ্ট তৈরি করেন।
- এটি নিশ্চিত করে যে আপনার শেল সর্বদা সর্বশেষ কমান্ডগুলির তালিকা জানে।
- যদি আপনি একটি নতুন শেল সেশন শুরু করেন, তাহলে স্বয়ংক্রিয়ভাবে ক্যাশে আপডেট হয়, তাই `rehash` ব্যবহার করার প্রয়োজন নাও হতে পারে।
