# [Linux] Bash watch उपयोग: एक आदेश को नियमित अंतराल पर चलाना

## Overview
`watch` कमांड का उपयोग किसी विशेष आदेश को नियमित अंतराल पर चलाने के लिए किया जाता है। यह आपको आउटपुट को लगातार देखने की सुविधा देता है, जिससे आप समय के साथ बदलावों पर नज़र रख सकते हैं।

## Usage
`watch` कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
watch [options] [arguments]
```

## Common Options
- `-n <seconds>`: यह विकल्प आदेश को कितने सेकंड के अंतराल पर चलाना है, यह निर्धारित करता है। डिफ़ॉल्ट रूप से, यह 2 सेकंड है।
- `-d`: यह विकल्प आउटपुट में बदलाव को हाइलाइट करता है, जिससे आपको अंतर देखना आसान होता है।
- `-t`: यह विकल्प टाइटल बार को छुपाता है, जिससे केवल आउटपुट दिखाई देता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. हर 5 सेकंड में `date` कमांड चलाना:
   ```bash
   watch -n 5 date
   ```

2. हर 2 सेकंड में `ls` कमांड चलाना और बदलाव को हाइलाइट करना:
   ```bash
   watch -d ls -l
   ```

3. सिस्टम की मेमोरी उपयोग को हर 10 सेकंड में देखना:
   ```bash
   watch -n 10 free -h
   ```

4. एक विशेष फ़ाइल के आकार को हर 3 सेकंड में देखना:
   ```bash
   watch -n 3 du -sh /path/to/file
   ```

## Tips
- `watch` कमांड का उपयोग करते समय, आप आउटपुट को अधिक स्पष्टता के लिए `-d` विकल्प के साथ हाइलाइट कर सकते हैं।
- यदि आप लंबे समय तक एक ही आदेश को चलाना चाहते हैं, तो `-n` विकल्प का उपयोग करके समय अंतराल को समायोजित करें।
- ध्यान रखें कि `watch` कमांड का उपयोग करते समय, यह टर्मिनल को ब्लॉक कर देता है, इसलिए इसे बंद करने के लिए `Ctrl + C` दबाएँ।
