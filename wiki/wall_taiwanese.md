# [Linux] Bash wall 使用法: 發送訊息給所有登入的使用者

## Overview
`wall` 命令用來將訊息發送給所有登入的使用者。這個命令通常用於系統管理員需要通知所有使用者時，例如系統維護或即將關閉的警告。

## Usage
基本語法如下：
```
wall [options] [arguments]
```

## Common Options
- `-n`：不顯示發送者的名稱。
- `-s`：以簡單模式發送訊息，避免格式化。
- `-f`：從檔案中讀取訊息。

## Common Examples
以下是幾個常見的使用範例：

1. 發送簡單的訊息：
   ```bash
   wall "系統將於晚上11點進行維護，請保存您的工作。"
   ```

2. 從檔案發送訊息：
   ```bash
   wall -f /path/to/message.txt
   ```

3. 發送不顯示發送者名稱的訊息：
   ```bash
   wall -n "所有使用者注意：伺服器將於明天早上進行重啟。"
   ```

## Tips
- 確保在發送訊息前，您有適當的權限，因為某些系統可能限制普通使用者使用 `wall` 命令。
- 使用 `wall` 時，訊息會立即顯示在所有登入的終端上，因此請確保訊息內容清晰且簡潔。
- 可以使用 `Ctrl+C` 來中止正在發送的訊息，但這不會影響已經發送的部分。
