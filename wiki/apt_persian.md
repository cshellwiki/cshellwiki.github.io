# [لینوکس] C Shell (csh) apt <استفاده معادل>: مدیریت بسته‌ها در سیستم‌عامل

## Overview
دستورات `apt` برای مدیریت بسته‌ها در سیستم‌عامل‌های مبتنی بر دبیان مانند اوبونتو و دبیان استفاده می‌شوند. این ابزار به کاربران امکان می‌دهد تا نرم‌افزارها را نصب، حذف و به‌روزرسانی کنند.

## Usage
سینتکس پایه دستور `apt` به صورت زیر است:

```bash
apt [options] [arguments]
```

## Common Options
- `install`: برای نصب بسته‌های جدید.
- `remove`: برای حذف بسته‌های نصب شده.
- `update`: برای به‌روزرسانی فهرست بسته‌ها.
- `upgrade`: برای به‌روزرسانی بسته‌های نصب شده به نسخه‌های جدید.
- `search`: برای جستجوی بسته‌ها بر اساس نام یا توضیحات.

## Common Examples
- برای به‌روزرسانی فهرست بسته‌ها:
  ```bash
  apt update
  ```

- برای نصب یک بسته جدید، به عنوان مثال `vim`:
  ```bash
  apt install vim
  ```

- برای حذف یک بسته، به عنوان مثال `vim`:
  ```bash
  apt remove vim
  ```

- برای به‌روزرسانی تمام بسته‌های نصب شده:
  ```bash
  apt upgrade
  ```

- برای جستجوی بسته‌ای خاص، به عنوان مثال `curl`:
  ```bash
  apt search curl
  ```

## Tips
- همیشه قبل از نصب یا به‌روزرسانی بسته‌ها، از دستور `apt update` استفاده کنید تا فهرست بسته‌ها به‌روز باشد.
- برای حذف بسته‌ها، می‌توانید از گزینه `--purge` استفاده کنید تا تنظیمات مربوط به بسته نیز حذف شود.
- برای مشاهده اطلاعات بیشتر در مورد یک بسته خاص، از دستور `apt show [package_name]` استفاده کنید.
