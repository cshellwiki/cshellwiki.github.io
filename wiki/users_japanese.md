# [Linux] Bash users 使用法: ユーザー情報の表示

## Overview
`users` コマンドは、現在ログインしているユーザーの名前を表示します。このコマンドは、システム上でアクティブなユーザーを簡単に確認するために使用されます。

## Usage
基本的な構文は次のとおりです。

```bash
users [options] [arguments]
```

## Common Options
- `-n` : ユーザー名をカンマで区切って表示します。
- `-r` : ログインしているユーザーのリストを表示しますが、重複を除外します。

## Common Examples
以下は、`users` コマンドのいくつかの実用的な例です。

### 例1: 現在のユーザーを表示
```bash
users
```
このコマンドを実行すると、現在ログインしているユーザーの名前が表示されます。

### 例2: ユーザー名をカンマで区切って表示
```bash
users -n
```
このコマンドは、ログインしているユーザー名をカンマで区切って表示します。

### 例3: 重複を除外してユーザーを表示
```bash
users -r
```
このコマンドは、重複するユーザー名を除外して表示します。

## Tips
- `users` コマンドは、システムの状態を把握するために非常に便利です。特に、複数のユーザーが同時にログインしている場合に役立ちます。
- スクリプト内で使用する際は、出力を変数に格納して、他の処理に利用することができます。
