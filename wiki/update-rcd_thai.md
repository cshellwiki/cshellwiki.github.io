# [Linux] C Shell (csh) update-rc.d การใช้งาน: จัดการการเริ่มต้นของบริการ

## Overview
คำสั่ง `update-rc.d` ใช้สำหรับจัดการการเริ่มต้นของบริการในระบบปฏิบัติการ Linux โดยช่วยในการเพิ่ม ลบ หรือปรับเปลี่ยนบริการที่ต้องการให้เริ่มทำงานอัตโนมัติเมื่อระบบบูตขึ้น

## Usage
การใช้งานคำสั่ง `update-rc.d` มีรูปแบบพื้นฐานดังนี้:

```
update-rc.d [options] [arguments]
```

## Common Options
- `defaults` : ใช้สำหรับตั้งค่าพื้นฐานในการเริ่มต้นบริการ
- `remove` : ใช้เพื่อลบบริการออกจากการเริ่มต้นอัตโนมัติ
- `enable` : ใช้เพื่อเปิดใช้งานบริการในการเริ่มต้น
- `disable` : ใช้เพื่อลบบริการออกจากการเริ่มต้น

## Common Examples
- เพิ่มบริการให้เริ่มทำงานอัตโนมัติ:
    ```bash
    update-rc.d myservice defaults
    ```
  
- ลบบริการออกจากการเริ่มต้น:
    ```bash
    update-rc.d myservice remove
    ```

- เปิดใช้งานบริการในการเริ่มต้น:
    ```bash
    update-rc.d myservice enable
    ```

- ปิดใช้งานบริการในการเริ่มต้น:
    ```bash
    update-rc.d myservice disable
    ```

## Tips
- ตรวจสอบสถานะของบริการก่อนทำการปรับเปลี่ยน เพื่อให้แน่ใจว่าบริการนั้นมีอยู่ในระบบ
- ใช้คำสั่ง `man update-rc.d` เพื่อดูรายละเอียดเพิ่มเติมเกี่ยวกับตัวเลือกและการใช้งาน
- ควรทำการสำรองข้อมูลการตั้งค่าก่อนที่จะทำการเปลี่ยนแปลงใดๆ เพื่อป้องกันปัญหาที่อาจเกิดขึ้น
