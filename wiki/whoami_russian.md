# [Linux] Bash whoami Использование: Определение текущего пользователя

## Обзор
Команда `whoami` в Bash используется для отображения имени текущего пользователя, под которым выполняется сессия. Это полезно для проверки, какой пользователь активен в данный момент, особенно в средах с несколькими пользователями или при работе с различными учетными записями.

## Использование
Основной синтаксис команды выглядит следующим образом:

```bash
whoami [options] [arguments]
```

## Общие опции
Команда `whoami` имеет несколько опций, хотя в большинстве случаев используется без них. Вот некоторые из них:

- `--help`: выводит справочную информацию о команде.
- `--version`: показывает версию команды.

## Общие примеры
Вот несколько практических примеров использования команды `whoami`:

1. **Вывод имени текущего пользователя:**
   ```bash
   whoami
   ```

2. **Получение справочной информации:**
   ```bash
   whoami --help
   ```

3. **Проверка версии команды:**
   ```bash
   whoami --version
   ```

## Советы
- Используйте `whoami` для быстрой проверки, под какой учетной записью вы работаете, особенно если вы переключались между пользователями с помощью `su` или `sudo`.
- Команда `whoami` может быть полезна в скриптах для определения текущего пользователя и выполнения действий в зависимости от этого.
- Помните, что `whoami` всегда возвращает имя пользователя, а не идентификатор пользователя (UID). Если вам нужен UID, используйте команду `id -u`.
