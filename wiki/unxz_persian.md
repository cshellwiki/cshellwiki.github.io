# [لینوکس] Bash unxz استفاده: استخراج فایل‌های فشرده شده با فرمت XZ

## Overview
دستور `unxz` برای استخراج فایل‌های فشرده شده با فرمت XZ استفاده می‌شود. این فرمت معمولاً برای فشرده‌سازی داده‌ها به کار می‌رود و با استفاده از این دستور می‌توان به راحتی فایل‌های فشرده را به حالت اولیه خود بازگرداند.

## Usage
سینتکس پایه دستور `unxz` به صورت زیر است:

```
unxz [options] [arguments]
```

## Common Options
- `-k` یا `--keep`: فایل فشرده‌شده را پس از استخراج نگه می‌دارد.
- `-f` یا `--force`: در صورتی که فایل خروجی وجود داشته باشد، آن را بدون هشدار بازنویسی می‌کند.
- `-v` یا `--verbose`: اطلاعات بیشتری در مورد فرآیند استخراج نمایش می‌دهد.

## Common Examples
### مثال ۱: استخراج یک فایل XZ
برای استخراج یک فایل XZ ساده، می‌توانید از دستور زیر استفاده کنید:

```bash
unxz file.xz
```

### مثال ۲: استخراج و نگه‌داشتن فایل فشرده
اگر می‌خواهید فایل فشرده را پس از استخراج نگه دارید، از گزینه `-k` استفاده کنید:

```bash
unxz -k file.xz
```

### مثال ۳: استخراج با اجبار
اگر می‌خواهید فایل خروجی را بدون توجه به وجود آن بازنویسی کنید، از گزینه `-f` استفاده کنید:

```bash
unxz -f file.xz
```

### مثال ۴: استخراج با نمایش جزئیات
برای مشاهده جزئیات بیشتر در حین استخراج، می‌توانید از گزینه `-v` استفاده کنید:

```bash
unxz -v file.xz
```

## Tips
- همیشه قبل از استفاده از گزینه `-f` مطمئن شوید که اطلاعات مهمی در فایل خروجی وجود ندارد، زیرا این گزینه فایل‌ها را بدون هشدار بازنویسی می‌کند.
- اگر با فایل‌های بزرگ کار می‌کنید، می‌توانید از گزینه `-k` استفاده کنید تا نسخه فشرده را برای مقایسه یا استفاده‌های بعدی نگه دارید.
- برای مشاهده اطلاعات بیشتر در مورد گزینه‌ها و نحوه استفاده از `unxz`، می‌توانید از دستور `man unxz` استفاده کنید.
