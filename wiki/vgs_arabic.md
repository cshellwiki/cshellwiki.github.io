# [لينكس] Bash vgs الاستخدام: عرض معلومات حول مجموعات التخزين

## Overview
يستخدم أمر `vgs` لعرض معلومات حول مجموعات التخزين (Volume Groups) في نظام إدارة التخزين المنطقي (LVM). يوفر هذا الأمر تفاصيل حول حالة مجموعات التخزين، مثل الحجم المستخدم والمتاح.

## Usage
يمكن استخدام الأمر `vgs` بالصيغة التالية:

```bash
vgs [options] [arguments]
```

## Common Options
- `-o`: تحديد الأعمدة التي سيتم عرضها.
- `--units`: تحديد وحدات القياس المستخدمة في الإخراج.
- `-h`: عرض الإخراج بطريقة قابلة للقراءة (human-readable).
- `--noheadings`: عدم عرض رؤوس الأعمدة في الإخراج.

## Common Examples
- **عرض معلومات جميع مجموعات التخزين:**

```bash
vgs
```

- **عرض معلومات مع تحديد الأعمدة:**

```bash
vgs -o vg_name,lv_count,attr
```

- **عرض المعلومات بوحدات قابلة للقراءة:**

```bash
vgs -h
```

- **عرض المعلومات بدون رؤوس الأعمدة:**

```bash
vgs --noheadings
```

## Tips
- استخدم الخيار `-o` لتخصيص الأعمدة المعروضة حسب احتياجاتك.
- تأكد من تشغيل الأمر كمسؤول (root) للحصول على معلومات دقيقة حول مجموعات التخزين.
- يمكنك دمج `vgs` مع أوامر أخرى مثل `grep` لتصفية النتائج حسب الحاجة.
