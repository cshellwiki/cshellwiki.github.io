# [لينكس] Bash cksum: حساب قيمة التحقق من سلامة الملفات

## نظرة عامة
يستخدم أمر `cksum` لحساب قيمة التحقق من سلامة الملفات، والتي تتضمن قيمة عددية (checksum) وطول الملف. هذه القيمة يمكن استخدامها للتحقق من سلامة البيانات عند نقل الملفات أو تخزينها.

## الاستخدام
الصيغة الأساسية للأمر هي:

```bash
cksum [خيارات] [وسائط]
```

## الخيارات الشائعة
- `-a, --algorithm=ALGO`: تحديد خوارزمية معينة لحساب قيمة التحقق.
- `-h, --help`: عرض معلومات المساعدة حول الأمر.
- `-V, --version`: عرض معلومات حول إصدار الأمر.

## أمثلة شائعة
- حساب قيمة التحقق لملف:
```bash
cksum myfile.txt
```

- حساب قيمة التحقق لعدة ملفات:
```bash
cksum file1.txt file2.txt
```

- استخدام خيار المساعدة:
```bash
cksum --help
```

## نصائح
- تأكد من استخدام `cksum` على الملفات التي تحتاج إلى التحقق من سلامتها.
- يمكنك استخدام `cksum` مع أدوات أخرى مثل `diff` لمقارنة الملفات.
- تذكر أن قيمة التحقق قد تتغير إذا تم تعديل الملف، لذا تأكد من حفظ النسخ الأصلية إذا كنت بحاجة إلى التحقق لاحقًا.
