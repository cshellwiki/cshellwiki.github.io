# [Linux] Bash kill 使用法: 終止進程

## Overview
`kill` 命令用於向正在運行的進程發送信號，通常用來終止進程。這是一個非常有用的工具，特別是在需要強制關閉不響應的應用程序時。

## Usage
基本語法如下：
```
kill [選項] [進程ID]
```

## Common Options
- `-l`：列出所有可用的信號名稱。
- `-s SIGNAL`：指定要發送的信號，SIGNAL 可以是信號名稱或數字。
- `-9`：強制終止進程，這是最強的終止信號。

## Common Examples
- 終止一個進程：
  ```bash
  kill 1234
  ```
  這裡的 `1234` 是進程的 ID。

- 使用強制終止信號：
  ```bash
  kill -9 1234
  ```
  這會強制終止進程 ID 為 `1234` 的進程。

- 列出所有可用的信號：
  ```bash
  kill -l
  ```

- 發送特定信號（例如，`SIGTERM`）：
  ```bash
  kill -s SIGTERM 1234
  ```

## Tips
- 在使用 `kill` 命令之前，建議先使用 `ps` 命令查看進程 ID，以確保你終止的是正確的進程。
- 使用 `kill -9` 時要小心，因為這會強制終止進程，可能導致數據丟失或其他問題。
- 若不確定進程 ID，可以使用 `pgrep` 命令來查找進程名稱對應的 ID，例如：
  ```bash
  pgrep firefox
  ```
