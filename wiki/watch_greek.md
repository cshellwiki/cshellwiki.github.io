# [Λίνουξ] Bash watch Χρήση: Παρακολούθηση εντολών σε πραγματικό χρόνο

## Overview
Η εντολή `watch` χρησιμοποιείται για να εκτελεί μια άλλη εντολή περιοδικά και να εμφανίζει την έξοδο της σε πραγματικό χρόνο. Είναι ιδιαίτερα χρήσιμη για την παρακολούθηση αλλαγών σε αρχεία ή καταστάσεις συστήματος.

## Usage
Η βασική σύνταξη της εντολής είναι:

```bash
watch [options] [arguments]
```

## Common Options
- `-n, --interval`: Ορίζει το διάστημα (σε δευτερόλεπτα) μεταξύ των εκτελέσεων της εντολής.
- `-d, --differences`: Επισημαίνει τις αλλαγές στην έξοδο της εντολής.
- `-t, --no-title`: Αφαιρεί την επικεφαλίδα που δείχνει την ώρα και την εντολή που εκτελείται.
- `-x, --exec`: Εκτελεί την εντολή με τα καθορισμένα ορίσματα.

## Common Examples
### 1. Παρακολούθηση της κατάστασης του συστήματος
Για να παρακολουθήσετε την κατάσταση της μνήμης κάθε 2 δευτερόλεπτα, μπορείτε να χρησιμοποιήσετε την εντολή:

```bash
watch -n 2 free -h
```

### 2. Παρακολούθηση ενός φακέλου για αλλαγές
Για να παρακολουθήσετε τις αλλαγές σε έναν φάκελο, μπορείτε να χρησιμοποιήσετε την εντολή `ls`:

```bash
watch -d ls -l /path/to/directory
```

### 3. Παρακολούθηση διεργασιών
Για να παρακολουθήσετε τις τρέχουσες διεργασίες στο σύστημα:

```bash
watch -n 1 ps aux
```

### 4. Παρακολούθηση ενός αρχείου καταγραφής
Για να παρακολουθήσετε τις τελευταίες γραμμές ενός αρχείου καταγραφής:

```bash
watch -n 5 tail -n 10 /var/log/syslog
```

## Tips
- Χρησιμοποιήστε την επιλογή `-d` για να δείτε γρήγορα τις αλλαγές στην έξοδο.
- Μπορείτε να συνδυάσετε την `watch` με άλλες εντολές για πιο σύνθετες παρακολουθήσεις.
- Αν χρειάζεστε να παρακολουθήσετε μια εντολή που απαιτεί χρόνο εκτέλεσης, αυξήστε το διάστημα με την επιλογή `-n`.
