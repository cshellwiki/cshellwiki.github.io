# [ระบบปฏิบัติการ] Bash watch การใช้งาน: ตรวจสอบคำสั่งในช่วงเวลาที่กำหนด

## Overview
คำสั่ง `watch` ใน Bash ใช้เพื่อเรียกดูผลลัพธ์ของคำสั่งที่กำหนดในช่วงเวลาที่กำหนด โดยจะทำการอัปเดตผลลัพธ์บนหน้าจออย่างต่อเนื่อง ทำให้ผู้ใช้สามารถติดตามการเปลี่ยนแปลงได้อย่างง่ายดาย

## Usage
การใช้งานคำสั่ง `watch` มีรูปแบบพื้นฐานดังนี้:

```bash
watch [options] [arguments]
```

## Common Options
- `-n, --interval`: กำหนดระยะเวลาที่จะรอระหว่างการเรียกคำสั่งใหม่ (ค่าเริ่มต้นคือ 2 วินาที)
- `-d, --differences`: เน้นความแตกต่างระหว่างผลลัพธ์ของการเรียกคำสั่งในแต่ละครั้ง
- `-t, --no-title`: ไม่แสดงหัวเรื่องในหน้าต่าง watch
- `-x, --exec`: ใช้คำสั่งที่ระบุและส่งอาร์กิวเมนต์ไปยังคำสั่งนั้น

## Common Examples
1. ตรวจสอบการใช้ CPU ทุก 5 วินาที:
   ```bash
   watch -n 5 mpstat
   ```

2. ตรวจสอบสถานะของการทำงานของเซิร์ฟเวอร์:
   ```bash
   watch -d systemctl status apache2
   ```

3. ตรวจสอบการเปลี่ยนแปลงในไฟล์ log:
   ```bash
   watch tail -n 10 /var/log/syslog
   ```

4. ตรวจสอบการใช้พื้นที่ดิสก์:
   ```bash
   watch -n 10 df -h
   ```

## Tips
- ใช้ตัวเลือก `-d` เพื่อช่วยให้เห็นความแตกต่างระหว่างการอัปเดตได้ชัดเจนยิ่งขึ้น
- ปรับระยะเวลาใน `-n` ให้เหมาะสมกับความถี่ที่คุณต้องการตรวจสอบ
- หากต้องการหยุดการทำงานของ `watch` ให้กด `Ctrl + C` เพื่อออกจากโปรแกรม
