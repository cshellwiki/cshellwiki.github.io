# [Linux] Bash dstat: [แสดงข้อมูลการใช้งานระบบแบบเรียลไทม์]

## Overview
คำสั่ง `dstat` ใช้สำหรับแสดงข้อมูลการใช้งานระบบในรูปแบบเรียลไทม์ โดยสามารถแสดงข้อมูลเกี่ยวกับ CPU, หน่วยความจำ, การใช้งานดิสก์, และเครือข่ายได้ในเวลาเดียวกัน ซึ่งช่วยให้ผู้ใช้สามารถตรวจสอบประสิทธิภาพของระบบได้อย่างมีประสิทธิภาพ

## Usage
รูปแบบพื้นฐานของคำสั่ง `dstat` คือ:

```bash
dstat [options] [arguments]
```

## Common Options
- `-c`: แสดงข้อมูลการใช้งาน CPU
- `-d`: แสดงข้อมูลการใช้งานดิสก์
- `-n`: แสดงข้อมูลการใช้งานเครือข่าย
- `-m`: แสดงข้อมูลการใช้งานหน่วยความจำ
- `-t`: แสดงเวลาในแต่ละบรรทัด

## Common Examples
1. แสดงข้อมูลการใช้งาน CPU และหน่วยความจำ:
   ```bash
   dstat -c -m
   ```

2. แสดงข้อมูลการใช้งานดิสก์และเครือข่าย:
   ```bash
   dstat -d -n
   ```

3. แสดงข้อมูลทั้งหมดพร้อมเวลา:
   ```bash
   dstat -t -c -d -n -m
   ```

4. แสดงข้อมูลการใช้งานทุก 2 วินาที:
   ```bash
   dstat 2
   ```

## Tips
- ใช้ `dstat` ร่วมกับคำสั่งอื่น ๆ เช่น `grep` เพื่อกรองข้อมูลที่ต้องการ
- ลองใช้ `dstat --help` เพื่อดูตัวเลือกและคำอธิบายเพิ่มเติมเกี่ยวกับคำสั่ง
- การใช้ `dstat` ในการตรวจสอบระบบในช่วงเวลาที่มีการใช้งานสูงจะช่วยให้คุณเข้าใจปัญหาที่เกิดขึ้นได้ดีขึ้น
