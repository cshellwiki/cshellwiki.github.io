# [লিনাক্স] Bash df ব্যবহার: ডিস্ক স্পেসের তথ্য দেখানো

## Overview
`df` কমান্ডটি ডিস্কের ব্যবহৃত এবং অব্যবহৃত স্পেসের তথ্য প্রদর্শন করে। এটি সিস্টেমে সংযুক্ত ফাইল সিস্টেমগুলোর স্ট্যাটাস সম্পর্কে ধারণা দেয়, যা ব্যবহারকারীদের তাদের ডিস্ক স্পেসের ব্যবস্থাপনা করতে সহায়তা করে।

## Usage
`df` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
df [options] [arguments]
```

## Common Options
- `-h`: মানব-পঠনযোগ্য ফরম্যাটে আউটপুট দেখায় (যেমন KB, MB, GB)।
- `-T`: ফাইল সিস্টেমের টাইপ প্রদর্শন করে।
- `-a`: সমস্ত ফাইল সিস্টেমের তথ্য দেখায়, এমনকি যেগুলো ব্যবহার হচ্ছে না।
- `-i`: ইনোডের তথ্য প্রদর্শন করে, যা ফাইল সিস্টেমের ইনোড সংখ্যা এবং ব্যবহৃত ইনোড সংখ্যা দেখায়।

## Common Examples
1. **ডিস্ক স্পেসের সাধারণ তথ্য দেখানো:**
   ```bash
   df
   ```

2. **মানব-পঠনযোগ্য ফরম্যাটে ডিস্ক স্পেস দেখানো:**
   ```bash
   df -h
   ```

3. **ফাইল সিস্টেমের টাইপ সহ তথ্য দেখানো:**
   ```bash
   df -T
   ```

4. **সব ফাইল সিস্টেমের তথ্য দেখানো:**
   ```bash
   df -a
   ```

5. **ইনোডের তথ্য দেখানো:**
   ```bash
   df -i
   ```

## Tips
- `df -h` ব্যবহার করে আপনি সহজেই ডিস্ক স্পেসের অবস্থা বুঝতে পারবেন।
- নিয়মিত `df` কমান্ড চালিয়ে আপনার ডিস্ক স্পেসের ব্যবস্থাপনা করুন, বিশেষ করে যখন আপনি বড় ফাইল বা অ্যাপ্লিকেশন ইনস্টল করছেন।
- যদি আপনি একটি নির্দিষ্ট ফাইল সিস্টেমের তথ্য দেখতে চান, তাহলে সেই ফাইল সিস্টেমের পাথ উল্লেখ করুন, যেমন: `df /home`।
