# [리눅스] C Shell (csh) wait 사용법: 프로세스 종료 대기

## Overview
`wait` 명령은 C Shell에서 백그라운드에서 실행 중인 프로세스가 종료될 때까지 대기하는 데 사용됩니다. 이 명령은 주로 스크립트에서 여러 프로세스를 동시에 실행하고, 특정 프로세스가 완료될 때까지 기다릴 필요가 있을 때 유용합니다.

## Usage
기본 구문은 다음과 같습니다:

```csh
wait [options] [arguments]
```

## Common Options
- `pid`: 특정 프로세스 ID에 대해 대기합니다. 이 인자를 제공하지 않으면, 모든 자식 프로세스가 종료될 때까지 대기합니다.

## Common Examples

1. **모든 자식 프로세스 대기하기**
   ```csh
   command1 &
   command2 &
   wait
   ```
   위의 예에서 `command1`과 `command2`가 백그라운드에서 실행되며, `wait` 명령은 두 프로세스가 모두 종료될 때까지 대기합니다.

2. **특정 프로세스 ID 대기하기**
   ```csh
   command1 &
   pid=$!
   command2 &
   wait $pid
   ```
   여기서는 `command1`의 프로세스 ID를 저장한 후, `command1`이 종료될 때까지 대기합니다.

3. **여러 프로세스 대기하기**
   ```csh
   command1 &
   command2 &
   command3 &
   wait $!
   ```
   이 예에서는 마지막으로 시작된 프로세스가 종료될 때까지 대기합니다.

## Tips
- `wait` 명령은 스크립트에서 프로세스 간의 의존성을 관리하는 데 매우 유용합니다.
- 프로세스 ID를 저장하고 특정 프로세스에 대해 대기하는 방식은 복잡한 스크립트에서 유용하게 사용될 수 있습니다.
- `wait` 명령을 사용하여 자식 프로세스의 종료 상태를 확인할 수 있으며, 이를 통해 오류 처리를 할 수 있습니다.
