# [리눅스] Bash wait 사용법: 프로세스 종료 대기

## Overview
`wait` 명령어는 현재 셸에서 실행 중인 백그라운드 프로세스가 종료될 때까지 대기하는 데 사용됩니다. 이 명령어는 주로 스크립트에서 여러 프로세스를 동시에 실행하고, 그 결과를 기다릴 때 유용합니다.

## Usage
기본 구문은 다음과 같습니다:
```bash
wait [options] [arguments]
```

## Common Options
- `-n`: 다음으로 종료되는 프로세스가 완료될 때까지 대기합니다.
- `PID`: 특정 프로세스 ID(PID)를 지정하여 해당 프로세스가 종료될 때까지 대기합니다.

## Common Examples
1. **백그라운드 프로세스 대기**
   ```bash
   sleep 5 &
   wait
   echo "모든 백그라운드 프로세스가 완료되었습니다."
   ```

2. **특정 프로세스 대기**
   ```bash
   sleep 5 &
   PID=$!
   echo "프로세스 ID: $PID"
   wait $PID
   echo "프로세스가 완료되었습니다."
   ```

3. **여러 프로세스 대기**
   ```bash
   sleep 3 &
   sleep 5 &
   wait
   echo "모든 프로세스가 완료되었습니다."
   ```

4. **다음으로 종료되는 프로세스 대기**
   ```bash
   sleep 3 &
   sleep 5 &
   wait -n
   echo "다음으로 종료된 프로세스가 완료되었습니다."
   ```

## Tips
- `wait` 명령어는 스크립트의 흐름을 제어하는 데 매우 유용하므로, 프로세스의 종료를 정확히 확인하고 싶을 때 사용하세요.
- 여러 프로세스를 동시에 실행할 때는 각 프로세스의 PID를 저장하고, 필요에 따라 개별적으로 대기할 수 있습니다.
- `wait`를 사용하여 프로세스가 종료된 후 후속 작업을 수행하는 것이 좋습니다.
