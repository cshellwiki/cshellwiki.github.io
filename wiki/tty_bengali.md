# [লিনাক্স] Bash tty ব্যবহার: টার্মিনাল ডিভাইসের নাম প্রদর্শন

## Overview
`tty` কমান্ডটি ব্যবহারকারীর টার্মিনাল ডিভাইসের নাম প্রদর্শন করে। এটি মূলত একটি টেক্সট ইন্টারফেসে কাজ করার সময় আপনার বর্তমান টার্মিনাল সেশনের তথ্য জানার জন্য ব্যবহৃত হয়।

## Usage
`tty` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
tty [options] [arguments]
```

## Common Options
- `-s`: আউটপুট ছাড়া, শুধুমাত্র সফল হলে 0 রিটার্ন কোড প্রদান করে।
- `--help`: কমান্ডের সাহায্য তথ্য প্রদর্শন করে।
- `--version`: `tty` কমান্ডের সংস্করণ তথ্য প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. **সাধারণ ব্যবহার:**
   ```bash
   tty
   ```
   আউটপুট: `/dev/pts/0` (বা আপনার টার্মিনালের নাম)

2. **সফল হলে কিছু না দেখানো:**
   ```bash
   tty -s && echo "This is a terminal"
   ```
   যদি এটি একটি টার্মিনাল হয়, তাহলে "This is a terminal" প্রদর্শিত হবে।

3. **সহায়তা তথ্য দেখা:**
   ```bash
   tty --help
   ```

4. **সংস্করণ তথ্য দেখা:**
   ```bash
   tty --version
   ```

## Tips
- `tty` কমান্ডটি স্ক্রিপ্টে ব্যবহার করে নিশ্চিত করুন যে স্ক্রিপ্টটি টার্মিনাল থেকে চালানো হচ্ছে।
- যখন আপনি SSH এর মাধ্যমে একটি রিমোট সার্ভারে লগইন করেন, তখন `tty` কমান্ড ব্যবহার করে নিশ্চিত করুন যে আপনি একটি টার্মিনাল সেশনে আছেন।
- `tty` এর আউটপুট ব্যবহার করে আপনার স্ক্রিপ্টে শর্তাধীন লজিক তৈরি করতে পারেন।
