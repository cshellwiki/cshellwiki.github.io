# [লিনাক্স] Bash xz ব্যবহার: ফাইল সংকোচন ও ডিকম্প্রেস করা

## Overview
`xz` কমান্ডটি একটি শক্তিশালী টুল যা ফাইল সংকোচনের জন্য ব্যবহৃত হয়। এটি উচ্চ সংকোচন অনুপাত প্রদান করে এবং সাধারণত `.xz` ফাইল এক্সটেনশনের সাথে ব্যবহৃত হয়।

## Usage
`xz` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: ফাইলটি ডিকম্প্রেস করে।
- `-k`, `--keep`: সংকোচনের পর মূল ফাইলটি রেখে দেয়।
- `-v`, `--verbose`: সংকোচনের সময় বিস্তারিত তথ্য প্রদর্শন করে।
- `-z`, `--compress`: ফাইল সংকোচনের জন্য (ডিফল্ট অপশন)।
- `-f`, `--force`: ফাইল সংকোচন বা ডিকম্প্রেস করতে বাধ্য করে, যদি ফাইলটি বিদ্যমান থাকে।

## Common Examples
- একটি ফাইল সংকোচন করা:

```bash
xz myfile.txt
```

- একটি `.xz` ফাইল ডিকম্প্রেস করা:

```bash
xz -d myfile.txt.xz
```

- সংকোচনের সময় মূল ফাইলটি রেখে দেওয়া:

```bash
xz -k myfile.txt
```

- সংকোচন প্রক্রিয়া সম্পর্কে বিস্তারিত তথ্য দেখা:

```bash
xz -v myfile.txt
```

## Tips
- সংকোচন করার আগে নিশ্চিত হয়ে নিন যে আপনার কাছে পর্যাপ্ত ডিস্ক স্পেস আছে।
- `-k` অপশন ব্যবহার করে মূল ফাইলটি সংরক্ষণ করা ভাল, বিশেষ করে যখন আপনি নিশ্চিত নন যে সংকোচন সফল হবে কিনা।
- সংকোচনের সময় ফাইলের নাম পরিবর্তন করতে চাইলে `-f` অপশন ব্যবহার করুন, তবে সাবধান থাকুন কারণ এটি মূল ফাইলটি মুছে ফেলতে পারে।
