# [Linux] Bash expr 用法等价: 计算和字符串操作

## 概述
`expr` 命令用于计算表达式的值，支持整数运算和字符串操作。它可以在 Bash 脚本中执行简单的数学计算和字符串处理。

## 用法
基本语法如下：
```bash
expr [options] [arguments]
```

## 常用选项
- `+` : 加法运算
- `-` : 减法运算
- `*` : 乘法运算（需使用反斜杠转义）
- `/` : 除法运算
- `%` : 取模运算
- `=` : 字符串相等比较
- `!=` : 字符串不相等比较
- `:` : 字符串匹配

## 常见示例
1. **整数加法**
   ```bash
   expr 5 + 3
   ```
   输出：`8`

2. **字符串连接**
   ```bash
   expr "Hello, " : "Hello, " && expr "World!"
   ```
   输出：`World!`

3. **取模运算**
   ```bash
   expr 10 % 3
   ```
   输出：`1`

4. **字符串比较**
   ```bash
   expr "abc" = "abc"
   ```
   输出：`1`（表示相等）

5. **使用变量进行计算**
   ```bash
   a=10
   b=20
   expr $a + $b
   ```
   输出：`30`

## 提示
- 使用 `expr` 时，确保在运算符周围留有空格。
- 对于乘法运算，记得使用反斜杠转义，例如 `expr 5 \* 3`。
- 在 Bash 脚本中，考虑使用 `$(( ))` 进行数学运算，因为它更简洁且功能更强大。
