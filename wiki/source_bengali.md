# [লিনাক্স] Bash source ব্যবহার: স্ক্রিপ্ট চালানো

## Overview
`source` কমান্ডটি Bash শেলের মধ্যে একটি স্ক্রিপ্ট বা ফাইলের নির্দেশাবলী কার্যকর করার জন্য ব্যবহৃত হয়। এটি মূলত একটি স্ক্রিপ্টের কন্টেন্টকে বর্তমান শেলের প্রসঙ্গে লোড করে, ফলে স্ক্রিপ্টে সংজ্ঞায়িত পরিবর্তনশীল এবং ফাংশনগুলি বর্তমান শেলে উপলব্ধ হয়।

## Usage
`source` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
source [options] [arguments]
```

## Common Options
- `-h`, `--help`: সাহায্য তথ্য প্রদর্শন করে।
- `-v`, `--verbose`: স্ক্রিপ্টের প্রতিটি কমান্ডের আগে তা প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হল:

### উদাহরণ 1: একটি স্ক্রিপ্ট চালানো
```bash
source myscript.sh
```
এই কমান্ডটি `myscript.sh` নামক স্ক্রিপ্টটি বর্তমান শেলে চালাবে।

### উদাহরণ 2: পরিবেশ পরিবর্তনশীল সেট করা
```bash
source env.sh
```
এই কমান্ডটি `env.sh` ফাইল থেকে পরিবেশ পরিবর্তনশীলগুলি লোড করবে।

### উদাহরণ 3: ফাংশন সংজ্ঞায়িত করা
```bash
source functions.sh
```
এটি `functions.sh` ফাইল থেকে ফাংশনগুলি লোড করে, যাতে সেগুলি বর্তমান শেলে ব্যবহার করা যায়।

## Tips
- `source` কমান্ডটি ব্যবহার করার সময় নিশ্চিত হন যে স্ক্রিপ্টে সঠিক পারমিশন রয়েছে।
- স্ক্রিপ্টে পরিবর্তনশীল এবং ফাংশনগুলি সংজ্ঞায়িত করার সময়, সেগুলি সঠিকভাবে কাজ করবে কিনা তা পরীক্ষা করুন।
- স্ক্রিপ্টের মধ্যে ত্রুটি থাকলে, তা শেলের মধ্যে প্রভাব ফেলতে পারে, তাই স্ক্রিপ্টটি পরীক্ষা করা গুরুত্বপূর্ণ।
