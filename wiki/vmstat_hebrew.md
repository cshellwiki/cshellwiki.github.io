# [לינוקס] Bash vmstat שימוש: הצגת מידע על ביצועי מערכת

## Overview
הפקודה `vmstat` (Virtual Memory Statistics) משמשת להצגת מידע על ביצועי מערכת, כולל נתונים על זיכרון, מעבד, תהליכים, קלט/פלט ועוד. היא מספקת תובנות חשובות על מצב המערכת בזמן אמת.

## Usage
התחביר הבסיסי של הפקודה הוא:
```
vmstat [options] [arguments]
```

## Common Options
- `-a`: מציג מידע על זיכרון פעיל ולא פעיל.
- `-m`: מציג מידע על זיכרון מטמון.
- `-s`: מציג סיכום של סטטיסטיקות מערכת.
- `-t`: מציג תאריך ושעה בכל שורה.
- `delay`: מספר השניות בין כל עדכון של המידע.

## Common Examples
- **הצגת מידע בסיסי על ביצועי מערכת:**
  ```bash
  vmstat
  ```

- **הצגת מידע כל 2 שניות:**
  ```bash
  vmstat 2
  ```

- **הצגת מידע על זיכרון פעיל ולא פעיל:**
  ```bash
  vmstat -a
  ```

- **הצגת סיכום סטטיסטיקות מערכת:**
  ```bash
  vmstat -s
  ```

## Tips
- השתמש באופציה `-t` כדי לקבל תאריך ושעה בכל עדכון, מה שיכול לעזור בניתוח נתונים לאורך זמן.
- כדאי להריץ את הפקודה עם `delay` כדי לעקוב אחרי שינויים בביצועי המערכת בזמן אמת.
- ניתוח התוצאות יכול לסייע בזיהוי בעיות בביצועים, כמו חוסר בזיכרון או עומס על המעבד.
