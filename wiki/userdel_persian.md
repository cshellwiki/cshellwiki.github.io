# [لینوکس] Bash userdel استفاده: حذف کاربر از سیستم

## Overview
دستور `userdel` برای حذف یک کاربر از سیستم استفاده می‌شود. این دستور می‌تواند به همراه گزینه‌های مختلفی اجرا شود تا رفتار آن را تنظیم کند.

## Usage
سینتکس پایه این دستور به شکل زیر است:

```
userdel [options] [arguments]
```

## Common Options
- `-r`: این گزینه باعث می‌شود که دایرکتوری خانگی کاربر و تمام فایل‌های آن نیز حذف شوند.
- `-f`: این گزینه حذف کاربر را بدون بررسی اینکه آیا کاربر در حال حاضر وارد سیستم شده است، انجام می‌دهد.
- `-Z`: این گزینه برای حذف کاربر با توجه به SELinux context استفاده می‌شود.

## Common Examples
برای حذف یک کاربر به نام `exampleuser`:

```bash
userdel exampleuser
```

برای حذف کاربر و دایرکتوری خانگی‌اش:

```bash
userdel -r exampleuser
```

برای حذف کاربر بدون بررسی وضعیت ورود:

```bash
userdel -f exampleuser
```

## Tips
- قبل از حذف یک کاربر، مطمئن شوید که هیچ فرآیند فعالی از آن کاربر در حال اجرا نیست.
- از گزینه `-r` با احتیاط استفاده کنید، زیرا این گزینه تمام داده‌های کاربر را حذف می‌کند.
- همیشه یک نسخه پشتیبان از داده‌های مهم تهیه کنید قبل از حذف کاربران.
