# [লিনাক্স] Bash lsmod ব্যবহার: মডিউল তালিকা দেখানো

## Overview
`lsmod` কমান্ডটি লিনাক্স সিস্টেমে লোড করা মডিউলগুলির তালিকা প্রদর্শন করে। এটি সিস্টেমের বর্তমান মডিউল কনফিগারেশন সম্পর্কে তথ্য প্রদান করে, যা ড্রাইভার এবং অন্যান্য কনফিগারেশন সম্পর্কিত।

## Usage
`lsmod` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
lsmod [options] [arguments]
```

## Common Options
`lsmod` কমান্ডের জন্য কিছু সাধারণ অপশন নিম্নলিখিত:

- **--help**: সাহায্য তথ্য দেখায়।
- **--version**: কমান্ডের সংস্করণ তথ্য দেখায়।

## Common Examples
এখানে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. **মডিউল তালিকা দেখানো**:
   ```bash
   lsmod
   ```

2. **মডিউল তালিকা সহ সাহায্য তথ্য দেখা**:
   ```bash
   lsmod --help
   ```

3. **মডিউল সংস্করণ তথ্য দেখা**:
   ```bash
   lsmod --version
   ```

## Tips
- `lsmod` কমান্ডটি সাধারণত রুট ব্যবহারকারীর অধিকার প্রয়োজন হয় না, তাই এটি সাধারণ ব্যবহারকারীর দ্বারা সহজেই ব্যবহার করা যায়।
- মডিউলগুলির মধ্যে নির্ভরতা বুঝতে `lsmod` এর আউটপুট ব্যবহার করুন, যা মডিউলগুলির মধ্যে সম্পর্ক নির্দেশ করে।
- যদি আপনি মডিউল লোড বা আনলোড করতে চান, তবে `modprobe` এবং `rmmod` কমান্ডগুলি ব্যবহার করুন।
