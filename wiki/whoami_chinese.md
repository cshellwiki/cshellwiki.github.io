# [Linux] Bash whoami 使用方法: 显示当前用户的用户名

## 概述
`whoami` 命令用于显示当前登录用户的用户名。它是一个简单而实用的命令，常用于确认当前用户身份。

## 用法
基本语法如下：
```
whoami [选项] [参数]
```

## 常用选项
- `--help`：显示帮助信息。
- `--version`：显示版本信息。

## 常见示例
1. **显示当前用户的用户名**
   ```bash
   whoami
   ```

2. **显示帮助信息**
   ```bash
   whoami --help
   ```

3. **显示版本信息**
   ```bash
   whoami --version
   ```

## 小贴士
- 使用 `whoami` 可以快速确认你在终端中以哪个用户身份运行命令，特别是在使用 `sudo` 或切换用户时。
- 在脚本中使用 `whoami` 可以帮助调试和记录当前用户信息。
