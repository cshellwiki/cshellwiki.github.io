# [लिनक्स] Bash update-rc.d उपयोग: सेवा प्रबंधन के लिए स्क्रिप्ट जोड़ना या हटाना

## Overview
`update-rc.d` एक Bash कमांड है जिसका उपयोग सिस्टम सेवाओं को प्रबंधित करने के लिए किया जाता है। यह कमांड विशेष रूप से सिस्टम स्टार्टअप और शटडाउन प्रक्रियाओं में सेवाओं को जोड़ने या हटाने के लिए उपयोगी है।

## Usage
बुनियादी सिंटैक्स इस प्रकार है:
```
update-rc.d [options] [arguments]
```

## Common Options
- `defaults`: डिफ़ॉल्ट रनलेवल के लिए स्क्रिप्ट को जोड़ता है।
- `remove`: सेवा को सिस्टम से हटाने के लिए उपयोग किया जाता है।
- `enable`: सेवा को सक्षम करने के लिए।
- `disable`: सेवा को अक्षम करने के लिए।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. **सेवा को जोड़ना**:
   ```bash
   sudo update-rc.d myservice defaults
   ```

2. **सेवा को हटाना**:
   ```bash
   sudo update-rc.d myservice remove
   ```

3. **सेवा को सक्षम करना**:
   ```bash
   sudo update-rc.d myservice enable
   ```

4. **सेवा को अक्षम करना**:
   ```bash
   sudo update-rc.d myservice disable
   ```

## Tips
- सुनिश्चित करें कि आप `sudo` का उपयोग कर रहे हैं, क्योंकि `update-rc.d` को प्रशासनिक विशेषाधिकारों की आवश्यकता होती है।
- सेवा स्क्रिप्ट को `/etc/init.d/` में सही तरीके से स्थापित करें ताकि `update-rc.d` इसे पहचान सके।
- सेवाओं को जोड़ने या हटाने से पहले, हमेशा बैकअप लेना न भूलें।
