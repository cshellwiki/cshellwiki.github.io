# [Linux] Bash unxz の使い方: XZ 圧縮ファイルを解凍する

## Overview
`unxz` コマンドは、XZ 形式で圧縮されたファイルを解凍するためのツールです。このコマンドを使用すると、圧縮されたデータを元の状態に戻すことができます。

## Usage
基本的な構文は以下の通りです。

```
unxz [オプション] [引数]
```

## Common Options
- `-k` または `--keep`: 解凍後も元の圧縮ファイルを保持します。
- `-f` または `--force`: 既存のファイルを上書きします。
- `-v` または `--verbose`: 解凍の進行状況を表示します。

## Common Examples
以下に、`unxz` コマンドのいくつかの実用的な例を示します。

### 例1: 基本的な解凍
XZ 形式のファイルを解凍します。
```bash
unxz example.xz
```

### 例2: 解凍後に元のファイルを保持
元の圧縮ファイルを保持しながら解凍します。
```bash
unxz -k example.xz
```

### 例3: 既存のファイルを上書き
既存のファイルがあっても強制的に解凍します。
```bash
unxz -f example.xz
```

### 例4: 解凍の進行状況を表示
解凍の進行状況を確認しながら解凍します。
```bash
unxz -v example.xz
```

## Tips
- 解凍する前に、ファイルのバックアップを取ることをお勧めします。
- 大きなファイルを解凍する場合は、十分なディスクスペースがあることを確認してください。
- `unxz` コマンドは、他の圧縮形式（例: `.gz` や `.zip`）には対応していないため、適切なコマンドを使用してください。
