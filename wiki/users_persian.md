# [لینوکس] کاربران Bash: [مدیریت کاربران]

## Overview
دستور `users` در Bash به کاربران این امکان را می‌دهد که نام‌های کاربرانی که در حال حاضر وارد سیستم شده‌اند را مشاهده کنند. این دستور به سادگی لیستی از نام‌های کاربری را که در حال حاضر در سیستم فعال هستند، نمایش می‌دهد.

## Usage
سینتکس پایه دستور `users` به صورت زیر است:

```bash
users [options]
```

## Common Options
- `-H`, `--help`: نمایش راهنمای استفاده از دستور.
- `-V`, `--version`: نمایش نسخه دستور.

## Common Examples
در زیر چند مثال رایج از استفاده از دستور `users` آورده شده است:

1. **نمایش نام‌های کاربری فعال:**
   ```bash
   users
   ```

2. **نمایش راهنمای دستور:**
   ```bash
   users --help
   ```

3. **نمایش نسخه دستور:**
   ```bash
   users --version
   ```

## Tips
- برای مشاهده نام‌های کاربری فعال در زمان‌های مختلف، می‌توانید دستور `users` را در یک اسکریپت یا به صورت دوره‌ای اجرا کنید.
- این دستور معمولاً با سایر دستورات مدیریتی مانند `who` و `w` ترکیب می‌شود تا اطلاعات بیشتری درباره کاربران فعال به دست آورید.
