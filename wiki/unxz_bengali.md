# [লিনাক্স] Bash unxz ব্যবহার: ফাইলের এক্সটেনশন কম্প্রেস করা

## Overview
`unxz` কমান্ডটি `.xz` ফাইল ফরম্যাটের কম্প্রেসড ফাইলগুলোকে আনজিপ বা ডিকম্প্রেস করার জন্য ব্যবহৃত হয়। এটি XZ কম্প্রেশন প্রযুক্তি ব্যবহার করে তৈরি করা ফাইলগুলোকে সহজে ব্যবহারের জন্য পুনরুদ্ধার করে।

## Usage
`unxz` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
unxz [options] [arguments]
```

## Common Options
- `-k` বা `--keep`: ডিকম্প্রেস করার পর মূল `.xz` ফাইলটি রেখে দেয়।
- `-f` বা `--force`: যদি ফাইলটি ইতিমধ্যেই বিদ্যমান থাকে, তাহলে সেটিকে ওভাররাইট করে।
- `-v` বা `--verbose`: প্রক্রিয়ার সময় বিস্তারিত তথ্য প্রদর্শন করে।

## Common Examples
1. একটি `.xz` ফাইল ডিকম্প্রেস করা:
   ```bash
   unxz example.xz
   ```

2. মূল ফাইলটি রেখে ডিকম্প্রেস করা:
   ```bash
   unxz -k example.xz
   ```

3. একটি বিদ্যমান ফাইলকে ওভাররাইট করে ডিকম্প্রেস করা:
   ```bash
   unxz -f example.xz
   ```

4. ডিকম্প্রেস করার সময় বিস্তারিত তথ্য দেখা:
   ```bash
   unxz -v example.xz
   ```

## Tips
- যদি আপনি নিশ্চিত না হন যে ফাইলটি ডিকম্প্রেস করার পর হারিয়ে যাবে কিনা, তাহলে `-k` অপশন ব্যবহার করুন।
- একাধিক `.xz` ফাইল একসাথে ডিকম্প্রেস করতে, ফাইলগুলোর নাম স্পেস দিয়ে আলাদা করে লিখুন:
  ```bash
  unxz file1.xz file2.xz file3.xz
  ```
- ডিকম্প্রেস করার পর ফাইলের আকার এবং স্থানীয় ফাইল সিস্টেমে স্থান সম্পর্কে সচেতন থাকুন, কারণ বড় ফাইলগুলো স্থান দখল করতে পারে।
