# [لينكس] Bash userdel الاستخدام: حذف مستخدم من النظام

## Overview
يستخدم الأمر `userdel` لحذف حسابات المستخدمين من نظام التشغيل. عند تنفيذ هذا الأمر، يتم إزالة معلومات المستخدم من ملفات النظام، مما يمنع المستخدم من تسجيل الدخول إلى النظام.

## Usage
تكون الصيغة الأساسية للأمر كالتالي:
```
userdel [options] [arguments]
```

## Common Options
- `-r`: يحذف الدليل الرئيسي للمستخدم والملفات المرتبطة به.
- `-f`: يجبر الحذف حتى لو كان المستخدم متصلاً.
- `-Z`: يحدد سياق SELinux للمستخدم المحذوف.

## Common Examples
- لحذف مستخدم بدون حذف الدليل الرئيسي:
  ```bash
  userdel username
  ```

- لحذف مستخدم مع حذف الدليل الرئيسي:
  ```bash
  userdel -r username
  ```

- لحذف مستخدم حتى لو كان متصلاً:
  ```bash
  userdel -f username
  ```

## Tips
- تأكد من أنك تمتلك صلاحيات كافية (مثل مستخدم الجذر) قبل تنفيذ الأمر `userdel`.
- تحقق من وجود أي عمليات نشطة للمستخدم قبل حذفه لتجنب فقدان البيانات.
- استخدم الخيار `-r` بحذر، حيث أنه سيؤدي إلى حذف جميع الملفات المرتبطة بالمستخدم.
