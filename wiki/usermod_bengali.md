# [লিনাক্স] Bash usermod ব্যবহার: ব্যবহারকারীর তথ্য পরিবর্তন

## Overview
`usermod` কমান্ডটি লিনাক্সে ব্যবহারকারীর তথ্য পরিবর্তন করার জন্য ব্যবহৃত হয়। এটি ব্যবহারকারীর নাম, গোষ্ঠী, এবং অন্যান্য বৈশিষ্ট্য পরিবর্তন করতে সহায়তা করে।

## Usage
নিচে `usermod` কমান্ডের মৌলিক সিনট্যাক্স দেওয়া হলো:

```bash
usermod [options] [arguments]
```

## Common Options
- `-aG`: একটি নতুন গোষ্ঠীতে ব্যবহারকারীকে যুক্ত করে, পূর্ববর্তী গোষ্ঠীগুলি বজায় রাখে।
- `-d`: ব্যবহারকারীর হোম ডিরেক্টরি পরিবর্তন করে।
- `-l`: ব্যবহারকারীর নাম পরিবর্তন করে।
- `-s`: ব্যবহারকারীর শেলের ধরন পরিবর্তন করে।
- `-u`: ব্যবহারকারীর UID পরিবর্তন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### 1. ব্যবহারকারীকে একটি নতুন গোষ্ঠীতে যুক্ত করা
```bash
usermod -aG sudo username
```

### 2. ব্যবহারকারীর হোম ডিরেক্টরি পরিবর্তন করা
```bash
usermod -d /new/home/directory username
```

### 3. ব্যবহারকারীর নাম পরিবর্তন করা
```bash
usermod -l newusername oldusername
```

### 4. ব্যবহারকারীর শেল পরিবর্তন করা
```bash
usermod -s /bin/zsh username
```

### 5. ব্যবহারকারীর UID পরিবর্তন করা
```bash
usermod -u 1001 username
```

## Tips
- ব্যবহারকারী তথ্য পরিবর্তন করার আগে ব্যাকআপ নেওয়া সর্বদা ভাল।
- পরিবর্তনগুলি কার্যকর করতে ব্যবহারকারীর লগআউট করা প্রয়োজন হতে পারে।
- সঠিক অনুমতি সহ কমান্ড চালানো নিশ্চিত করুন, সাধারণত `sudo` ব্যবহার করা হয়।
