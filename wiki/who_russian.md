# [Linux] Bash who использование: отображает пользователей, вошедших в систему

## Обзор
Команда `who` используется в системах на базе Unix для отображения списка пользователей, которые в настоящее время вошли в систему. Она предоставляет информацию о том, кто активен в системе, а также время их входа.

## Использование
Базовый синтаксис команды `who` выглядит следующим образом:

```bash
who [опции] [аргументы]
```

## Общие опции
- `-a` — отображает всю доступную информацию о пользователях.
- `-b` — показывает время последней перезагрузки системы.
- `-q` — выводит только список пользователей и количество активных сессий.
- `--help` — выводит справочную информацию по команде.

## Общие примеры
1. **Вывод списка всех пользователей, вошедших в систему:**
   ```bash
   who
   ```

2. **Вывод информации о последней перезагрузке системы:**
   ```bash
   who -b
   ```

3. **Вывод только имен пользователей и количества активных сессий:**
   ```bash
   who -q
   ```

4. **Вывод полной информации о пользователях:**
   ```bash
   who -a
   ```

## Советы
- Используйте `who` вместе с другими командами, такими как `grep`, для фильтрации результатов. Например, чтобы найти конкретного пользователя:
  ```bash
  who | grep username
  ```
- Команда `who` может быть полезна для администраторов системы для мониторинга активности пользователей.
- Обратите внимание, что команда `who` может не отображать пользователей, которые вошли в систему через графический интерфейс, если они не создали терминальную сессию.
