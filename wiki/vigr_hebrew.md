# [לינוקס] Bash vigr שימוש: עריכת קבצי קונפיגורציה של מערכת

## Overview
הפקודה `vigr` משמשת לעריכת קבצי הקונפיגורציה של המערכת, במיוחד את הקובץ `/etc/group` ואת הקובץ `/etc/passwd`. היא מציעה ממשק נוח ובטוח יותר לעריכת קבצים אלו, תוך כדי שמירה על שלמותם.

## Usage
הסינטקס הבסיסי של הפקודה הוא:
```
vigr [אפשרויות] [ארגומנטים]
```

## Common Options
- `-c` : בודק את הקובץ לפני העריכה.
- `-s` : מציג את הקובץ במצב קריאה בלבד.
- `-f` : מציין קובץ אחר לעריכה במקום הקובץ המוגדר כברירת מחדל.

## Common Examples
- עריכת הקובץ `/etc/passwd`:
```bash
vigr /etc/passwd
```

- עריכת הקובץ `/etc/group`:
```bash
vigr /etc/group
```

- בדיקת הקובץ לפני עריכה:
```bash
vigr -c /etc/passwd
```

- עריכת קובץ אחר:
```bash
vigr -f /path/to/your/file
```

## Tips
- תמיד השתמש בפקודה `vigr` במקום `vi` או `vim` כדי לערוך קבצי קונפיגורציה חשובים, כך תוכל להימנע מטעויות קריטיות.
- הקפד לבדוק את הקובץ לפני שמירה, במיוחד אם אתה עובד עם קבצים רגישים כמו `/etc/passwd`.
- השתמש באפשרות `-s` אם אתה רק רוצה לבדוק את התוכן מבלי לערוך אותו.
