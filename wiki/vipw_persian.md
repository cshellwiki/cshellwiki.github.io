# [لینوکس] Bash vipw استفاده: ویرایش فایل passwd

## Overview
دستور `vipw` برای ویرایش فایل‌های مربوط به کاربران سیستم، به ویژه فایل `passwd`، استفاده می‌شود. این دستور به شما این امکان را می‌دهد که اطلاعات کاربران را به صورت ایمن و بدون خطر تداخل با دیگر فرآیندها ویرایش کنید.

## Usage
سینتکس پایه دستور `vipw` به صورت زیر است:

```bash
vipw [options]
```

## Common Options
- `-s` : این گزینه به شما اجازه می‌دهد تا فایل `shadow` را نیز ویرایش کنید.
- `-m` : با استفاده از این گزینه، می‌توانید فایل `passwd` را در حالت ویرایش قرار دهید.
- `-h` : نمایش راهنما و اطلاعات مربوط به استفاده از دستور.

## Common Examples
برای ویرایش فایل `passwd`، می‌توانید به سادگی دستور زیر را اجرا کنید:

```bash
vipw
```

اگر بخواهید فایل `shadow` را نیز ویرایش کنید، از گزینه `-s` استفاده کنید:

```bash
vipw -s
```

برای ویرایش فایل `passwd` در حالت خاص، می‌توانید از گزینه `-m` استفاده کنید:

```bash
vipw -m
```

## Tips
- همیشه قبل از ویرایش فایل‌های سیستمی، از آن‌ها پشتیبان تهیه کنید.
- از ویرایش همزمان چند کاربر در فایل‌های `passwd` و `shadow` خودداری کنید تا از بروز مشکلات جلوگیری شود.
- برای اطمینان از صحت تغییرات، پس از ویرایش، از دستور `vipw` برای بررسی صحت فایل استفاده کنید.
