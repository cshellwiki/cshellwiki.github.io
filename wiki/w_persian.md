# [لینوکس] Bash w استفاده: نمایش کاربران و فعالیت‌های آن‌ها

## Overview
دستور `w` در Bash برای نمایش اطلاعات مربوط به کاربران فعال در سیستم و فعالیت‌های آن‌ها استفاده می‌شود. این دستور به شما اجازه می‌دهد تا ببینید چه کسانی در حال حاضر وارد سیستم شده‌اند و هر کاربر چه کاری انجام می‌دهد.

## Usage
سینتکس پایه دستور `w` به صورت زیر است:

```bash
w [options] [arguments]
```

## Common Options
- `-h`: حذف هدر از خروجی.
- `-s`: نمایش اطلاعات به صورت مختصر.
- `-u`: نمایش زمان آخرین فعالیت کاربر.

## Common Examples
### مثال 1: نمایش کاربران فعال
برای نمایش لیست کاربران فعال و فعالیت‌های آن‌ها، کافیست دستور زیر را اجرا کنید:

```bash
w
```

### مثال 2: نمایش اطلاعات به صورت مختصر
اگر می‌خواهید خروجی را به صورت مختصر مشاهده کنید، از گزینه `-s` استفاده کنید:

```bash
w -s
```

### مثال 3: حذف هدر از خروجی
برای حذف هدر از خروجی، می‌توانید از گزینه `-h` استفاده کنید:

```bash
w -h
```

### مثال 4: نمایش زمان آخرین فعالیت کاربر
برای نمایش زمان آخرین فعالیت هر کاربر، از گزینه `-u` استفاده کنید:

```bash
w -u
```

## Tips
- برای مشاهده اطلاعات به روز، می‌توانید دستور `w` را به صورت مکرر اجرا کنید.
- استفاده از گزینه‌های مختلف می‌تواند به شما کمک کند تا اطلاعات دقیق‌تری درباره کاربران و فعالیت‌های آن‌ها به دست آورید.
- این دستور به ویژه در سرورهای چند کاربره بسیار مفید است، زیرا به شما اجازه می‌دهد تا نظارت دقیقی بر فعالیت کاربران داشته باشید.
