# [لينكس] Bash uptime الاستخدام المعادل: عرض مدة تشغيل النظام

## Overview
يستخدم أمر `uptime` في نظام التشغيل لعرض مدة تشغيل النظام، بالإضافة إلى معلومات حول عدد المستخدمين المتصلين والتحميل الحالي للنظام. يساعد هذا الأمر في تقييم أداء النظام واستقراره.

## Usage
تكون الصيغة الأساسية للأمر كالتالي:

```bash
uptime [options] [arguments]
```

## Common Options
- `-p` : عرض مدة التشغيل بشكل مختصر.
- `-s` : عرض الوقت الذي بدأ فيه تشغيل النظام.
- `-h` : عرض مساعدة حول الأمر.

## Common Examples
- لعرض مدة تشغيل النظام والمعلومات ذات الصلة:
  ```bash
  uptime
  ```

- لعرض مدة التشغيل بشكل مختصر:
  ```bash
  uptime -p
  ```

- لعرض الوقت الذي بدأ فيه تشغيل النظام:
  ```bash
  uptime -s
  ```

## Tips
- استخدم الأمر `uptime` بانتظام لمراقبة أداء النظام.
- يمكنك دمج الأمر مع أدوات أخرى مثل `grep` لتصفية النتائج حسب الحاجة.
- تأكد من استخدام الخيارات المناسبة للحصول على المعلومات التي تحتاجها بشكل أسرع.
