# [ระบบปฏิบัติการ] C Shell (csh) xz การบีบอัดไฟล์: บีบอัดและขยายไฟล์

## Overview
คำสั่ง `xz` ใช้สำหรับบีบอัดและขยายไฟล์ในระบบ Unix/Linux โดยให้การบีบอัดที่มีประสิทธิภาพสูงและสามารถลดขนาดไฟล์ได้อย่างมาก

## Usage
รูปแบบพื้นฐานของคำสั่งคือ:
```
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: ขยายไฟล์ที่ถูกบีบอัด
- `-k`, `--keep`: เก็บไฟล์ต้นฉบับไว้หลังจากบีบอัด
- `-f`, `--force`: บังคับให้บีบอัดไฟล์แม้ว่าจะมีไฟล์ที่มีชื่อเดียวกันอยู่แล้ว
- `-z`, `--compress`: บีบอัดไฟล์ (เป็นค่าเริ่มต้น)
- `-9`: ใช้ระดับการบีบอัดสูงสุด

## Common Examples
- บีบอัดไฟล์:
```bash
xz file.txt
```

- ขยายไฟล์ที่ถูกบีบอัด:
```bash
xz -d file.txt.xz
```

- บีบอัดไฟล์และเก็บไฟล์ต้นฉบับ:
```bash
xz -k file.txt
```

- บังคับบีบอัดไฟล์แม้ว่าจะมีไฟล์ที่มีชื่อเดียวกัน:
```bash
xz -f file.txt
```

- บีบอัดไฟล์ด้วยระดับการบีบอัดสูงสุด:
```bash
xz -9 file.txt
```

## Tips
- ใช้ตัวเลือก `-k` หากคุณต้องการเก็บไฟล์ต้นฉบับไว้เพื่อป้องกันการสูญหาย
- ระดับการบีบอัดที่สูงขึ้นจะใช้เวลานานขึ้น แต่จะลดขนาดไฟล์ได้มากขึ้น
- ตรวจสอบขนาดไฟล์หลังการบีบอัดเพื่อให้แน่ใจว่ามีการบีบอัดที่มีประสิทธิภาพ
