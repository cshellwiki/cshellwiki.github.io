# [Linux] Bash w w: แสดงข้อมูลผู้ใช้ที่เข้าสู่ระบบ

## Overview
คำสั่ง `w` ใช้เพื่อแสดงข้อมูลเกี่ยวกับผู้ใช้ที่เข้าสู่ระบบในระบบ Linux รวมถึงเวลาที่ผู้ใช้เข้าสู่ระบบ, ระยะเวลาที่ใช้งาน, และคำสั่งที่กำลังทำงานอยู่ในขณะนั้น

## Usage
คำสั่ง `w` มีรูปแบบพื้นฐานดังนี้:
```bash
w [options] [arguments]
```

## Common Options
- `-h` : ไม่แสดงส่วนหัวของข้อมูล
- `-s` : แสดงข้อมูลในรูปแบบที่ย่อ
- `-u` : แสดงข้อมูลเกี่ยวกับผู้ใช้ที่เข้าสู่ระบบเท่านั้น

## Common Examples
1. แสดงข้อมูลผู้ใช้ทั้งหมดที่เข้าสู่ระบบ:
   ```bash
   w
   ```

2. แสดงข้อมูลโดยไม่แสดงส่วนหัว:
   ```bash
   w -h
   ```

3. แสดงข้อมูลในรูปแบบที่ย่อ:
   ```bash
   w -s
   ```

4. แสดงข้อมูลเกี่ยวกับผู้ใช้ที่เข้าสู่ระบบเท่านั้น:
   ```bash
   w -u
   ```

## Tips
- ใช้คำสั่ง `w` ร่วมกับ `grep` เพื่อค้นหาผู้ใช้เฉพาะ:
  ```bash
  w | grep username
  ```
- คำสั่ง `w` เป็นเครื่องมือที่ดีในการตรวจสอบการใช้งานระบบและการวิเคราะห์ปัญหาเกี่ยวกับผู้ใช้ที่เข้าสู่ระบบ
- คำสั่งนี้สามารถใช้ในการติดตามการทำงานของผู้ใช้ในระบบได้อย่างมีประสิทธิภาพ
