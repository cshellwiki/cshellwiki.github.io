# [לינוקס] Bash write שימוש: שליחת הודעות למשתמשים

## Overview
הפקודה `write` משמשת לשליחת הודעות טקסט למשתמשים אחרים במערכת לינוקס. כאשר אתה שולח הודעה, המשתמש המקבל יכול לראות את ההודעה בזמן אמת במסוף שלו.

## Usage
התחביר הבסיסי של הפקודה הוא:

```
write [options] [username] [tty]
```

## Common Options
- `-n`: שלח הודעה למשתמש מבלי להמתין לתגובה.
- `-h`: הצג עזרה עם מידע על אפשרויות נוספות.

## Common Examples
1. שליחת הודעה למשתמש בשם "user1":
   ```bash
   write user1
   ```

2. שליחת הודעה למשתמש בשם "user2" ב-tty ספציפי:
   ```bash
   write user2 /dev/pts/1
   ```

3. שליחת הודעה עם אפשרות `-n`:
   ```bash
   write -n user3
   ```

## Tips
- ודא שהמשתמש שאליו אתה שולח הודעה מחובר למערכת, אחרת ההודעה לא תגיע.
- השתמש בפקודה `who` כדי לבדוק מי מחובר ולגלות את ה-tty שלהם.
- אם אתה רוצה להפסיק לשלוח הודעה, פשוט לחץ על `Ctrl + D`.
