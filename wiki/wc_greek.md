# [Λειτουργικό Σύστημα] Bash wc Χρήση: Μετράει γραμμές, λέξεις και χαρακτήρες

## Overview
Η εντολή `wc` (word count) χρησιμοποιείται για να μετρήσει τον αριθμό των γραμμών, των λέξεων και των χαρακτήρων σε ένα ή περισσότερα αρχεία. Είναι ένα χρήσιμο εργαλείο για την ανάλυση περιεχομένου κειμένου.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
wc [options] [arguments]
```

## Common Options
- `-l`: Μετράει μόνο τις γραμμές.
- `-w`: Μετράει μόνο τις λέξεις.
- `-c`: Μετράει μόνο τους χαρακτήρες.
- `-m`: Μετράει μόνο τους χαρακτήρες (συμπεριλαμβανομένων των multibyte χαρακτήρων).
- `-L`: Εμφανίζει το μήκος της μεγαλύτερης γραμμής.

## Common Examples
### 1. Μέτρηση γραμμών, λέξεων και χαρακτήρων
```bash
wc myfile.txt
```

### 2. Μέτρηση μόνο γραμμών
```bash
wc -l myfile.txt
```

### 3. Μέτρηση μόνο λέξεων
```bash
wc -w myfile.txt
```

### 4. Μέτρηση μόνο χαρακτήρων
```bash
wc -c myfile.txt
```

### 5. Μέτρηση μεγαλύτερης γραμμής
```bash
wc -L myfile.txt
```

## Tips
- Χρησιμοποιήστε την εντολή `wc` σε συνδυασμό με άλλες εντολές, όπως `cat` ή `grep`, για πιο σύνθετες αναλύσεις.
- Μπορείτε να μετρήσετε πολλαπλά αρχεία ταυτόχρονα, απλά προσθέτοντας περισσότερα ονόματα αρχείων στην εντολή.
- Για να αποθηκεύσετε την έξοδο σε ένα αρχείο, μπορείτε να χρησιμοποιήσετε την ανακατεύθυνση εξόδου, π.χ. `wc myfile.txt > output.txt`.
