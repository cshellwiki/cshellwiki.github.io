# [लिनक्स] Bash uptime उपयोग: सिस्टम की सक्रियता की जानकारी

## Overview
`uptime` कमांड आपके सिस्टम की सक्रियता की जानकारी प्रदान करता है। यह बताता है कि सिस्टम कब से चालू है, वर्तमान समय, और पिछले 1, 5, और 15 मिनटों में कितने उपयोगकर्ता लॉग इन हैं।

## Usage
कमांड का मूल सिंटैक्स इस प्रकार है:
```
uptime [options]
```

## Common Options
- `-p` : सिस्टम की सक्रियता को एक सरल वाक्य में प्रदर्शित करता है।
- `-s` : सिस्टम के प्रारंभ होने का समय दिखाता है।
- `-h` : मदद प्रदर्शित करता है।

## Common Examples
1. **सामान्य उपयोग**:
   ```bash
   uptime
   ```
   यह कमांड सिस्टम की सक्रियता, वर्तमान समय, और लॉग इन उपयोगकर्ताओं की संख्या दिखाएगा।

2. **सक्रियता का सरल वाक्य**:
   ```bash
   uptime -p
   ```
   यह कमांड सिस्टम की सक्रियता को एक सरल वाक्य में प्रदर्शित करेगा, जैसे "उपकरण 5 दिन, 3 घंटे, 20 मिनट से चालू है।"

3. **प्रारंभ होने का समय**:
   ```bash
   uptime -s
   ```
   यह कमांड सिस्टम के प्रारंभ होने का समय दिखाएगा।

## Tips
- नियमित रूप से `uptime` कमांड चलाने से आप अपने सिस्टम की स्थिरता और प्रदर्शन की निगरानी कर सकते हैं।
- यदि आप सर्वर पर काम कर रहे हैं, तो `uptime` का उपयोग करके यह जान सकते हैं कि कितने उपयोगकर्ता सक्रिय हैं, जिससे आप संसाधनों का बेहतर प्रबंधन कर सकते हैं।
- `uptime` कमांड को स्क्रिप्ट में शामिल करने से आप सिस्टम की सक्रियता की जानकारी को स्वचालित रूप से लॉग कर सकते हैं।
