# [Linux] Bash wget использование: Загрузка файлов из интернета

## Обзор
Команда `wget` используется для загрузки файлов из интернета. Она поддерживает различные протоколы, такие как HTTP, HTTPS и FTP, и может работать в фоновом режиме, что делает её удобной для автоматизации загрузок.

## Использование
Основной синтаксис команды выглядит следующим образом:

```bash
wget [опции] [аргументы]
```

## Общие опции
- `-O [файл]` — сохранить загружаемый файл под указанным именем.
- `-c` — продолжить прерванную загрузку.
- `-q` — тихий режим, не выводит информацию о процессе загрузки.
- `--limit-rate=[скорость]` — ограничить скорость загрузки.
- `-r` — рекурсивная загрузка, позволяет загружать страницы и связанные с ними файлы.

## Общие примеры
1. Загрузка файла по URL:
   ```bash
   wget https://example.com/file.zip
   ```

2. Сохранение файла под другим именем:
   ```bash
   wget -O myfile.zip https://example.com/file.zip
   ```

3. Продолжение прерванной загрузки:
   ```bash
   wget -c https://example.com/largefile.zip
   ```

4. Ограничение скорости загрузки до 100КБ/с:
   ```bash
   wget --limit-rate=100k https://example.com/file.zip
   ```

5. Рекурсивная загрузка сайта:
   ```bash
   wget -r https://example.com/
   ```

## Советы
- Используйте опцию `-q`, если вам не нужна информация о процессе загрузки, чтобы сделать вывод более чистым.
- При использовании рекурсивной загрузки, будьте осторожны с большим количеством данных, чтобы не перегрузить сервер.
- Если вы часто загружаете файлы с одного и того же сайта, рассмотрите возможность использования файла конфигурации для автоматизации процесса.
