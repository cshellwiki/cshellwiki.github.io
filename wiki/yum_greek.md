# [Linux] Bash yum χρήση: Διαχείριση πακέτων σε συστήματα Linux

## Overview
Η εντολή `yum` (Yellowdog Updater, Modified) είναι ένα εργαλείο διαχείρισης πακέτων που χρησιμοποιείται σε διανομές Linux, όπως η CentOS και η Fedora. Επιτρέπει στους χρήστες να εγκαθιστούν, να ενημερώνουν και να διαγράφουν πακέτα λογισμικού από τα αποθετήρια.

## Usage
Η βασική σύνταξη της εντολής `yum` είναι η εξής:

```bash
yum [options] [arguments]
```

## Common Options
- `install`: Εγκαθιστά ένα ή περισσότερα πακέτα.
- `remove`: Αφαιρεί ένα ή περισσότερα πακέτα.
- `update`: Ενημερώνει όλα τα εγκατεστημένα πακέτα στην τελευταία τους έκδοση.
- `search`: Αναζητά πακέτα με βάση ένα όνομα ή περιγραφή.
- `info`: Παρέχει πληροφορίες σχετικά με ένα συγκεκριμένο πακέτο.

## Common Examples
### Εγκατάσταση πακέτου
Για να εγκαταστήσετε ένα πακέτο, χρησιμοποιήστε την εντολή:

```bash
yum install httpd
```

### Αφαίρεση πακέτου
Για να αφαιρέσετε ένα πακέτο, μπορείτε να χρησιμοποιήσετε την εντολή:

```bash
yum remove httpd
```

### Ενημέρωση πακέτων
Για να ενημερώσετε όλα τα εγκατεστημένα πακέτα, εκτελέστε:

```bash
yum update
```

### Αναζήτηση πακέτου
Για να αναζητήσετε ένα πακέτο, χρησιμοποιήστε την εντολή:

```bash
yum search nginx
```

### Πληροφορίες για πακέτο
Για να δείτε πληροφορίες σχετικά με ένα συγκεκριμένο πακέτο:

```bash
yum info httpd
```

## Tips
- Πάντα να ελέγχετε τις διαθέσιμες ενημερώσεις με την εντολή `yum update` για να διατηρείτε το σύστημα ασφαλές και ενημερωμένο.
- Χρησιμοποιήστε την επιλογή `-y` για να επιβεβαιώσετε αυτόματα τις ενέργειες, π.χ., `yum install -y httpd`.
- Ελέγξτε τα αποθετήρια που έχετε ενεργοποιήσει με την εντολή `yum repolist` για να βεβαιωθείτε ότι έχετε πρόσβαση σε όλα τα απαραίτητα πακέτα.
