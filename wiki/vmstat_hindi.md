# [लिनक्स] Bash vmstat उपयोग: सिस्टम प्रदर्शन की निगरानी

## Overview
`vmstat` एक उपयोगी Bash कमांड है जो सिस्टम की मेमोरी, प्रोसेस, और इनपुट/आउटपुट गतिविधियों की जानकारी प्रदान करता है। यह कमांड सिस्टम के प्रदर्शन को मॉनिटर करने में मदद करती है और विभिन्न संसाधनों के उपयोग का विश्लेषण करने के लिए उपयोगी होती है।

## Usage
`vmstat` कमांड का मूल सिंटैक्स निम्नलिखित है:

```bash
vmstat [options] [arguments]
```

## Common Options
यहाँ कुछ सामान्य विकल्प दिए गए हैं जो `vmstat` के साथ उपयोग किए जा सकते हैं:

- `-a` : सभी सक्रिय और निष्क्रिय मेमोरी को दिखाता है।
- `-m` : मेमोरी की उपयोगिता को दिखाता है, जैसे कि कैश और बफर।
- `-s` : सिस्टम के विभिन्न संसाधनों के बारे में संक्षिप्त सांख्यिकी प्रदान करता है।
- `-t` : समय की मुहर के साथ आउटपुट दिखाता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. **बुनियादी vmstat कमांड:**
   ```bash
   vmstat
   ```

2. **हर 2 सेकंड में प्रदर्शन डेटा प्राप्त करना:**
   ```bash
   vmstat 2
   ```

3. **सभी सक्रिय और निष्क्रिय मेमोरी दिखाना:**
   ```bash
   vmstat -a
   ```

4. **सिस्टम संसाधनों की सांख्यिकी दिखाना:**
   ```bash
   vmstat -s
   ```

5. **समय की मुहर के साथ डेटा दिखाना:**
   ```bash
   vmstat -t
   ```

## Tips
- `vmstat` का उपयोग करते समय, यह सुनिश्चित करें कि आप उचित समय अंतराल का चयन करें ताकि आप सिस्टम के प्रदर्शन का सही विश्लेषण कर सकें।
- यदि आप लंबे समय तक प्रदर्शन की निगरानी कर रहे हैं, तो आउटपुट को एक फ़ाइल में रीडायरेक्ट करना सहायक हो सकता है:
  ```bash
  vmstat 5 > vmstat_output.txt
  ```
- विभिन्न विकल्पों के साथ प्रयोग करें ताकि आप अपनी आवश्यकताओं के अनुसार सबसे उपयुक्त जानकारी प्राप्त कर सकें।
