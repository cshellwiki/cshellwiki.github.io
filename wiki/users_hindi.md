# [Linux] Bash users उपयोग: उपयोगकर्ताओं की जानकारी प्राप्त करना

## Overview
`users` कमांड का उपयोग वर्तमान में लॉग इन किए गए उपयोगकर्ताओं की सूची देखने के लिए किया जाता है। यह कमांड सिस्टम पर सक्रिय उपयोगकर्ताओं के नाम प्रदर्शित करता है।

## Usage
कमांड का मूल स्वरूप इस प्रकार है:

```
users [options]
```

## Common Options
`users` कमांड के लिए कुछ सामान्य विकल्प निम्नलिखित हैं:

- `-n`: उपयोगकर्ताओं की संख्या प्रदर्शित करता है।
- `-r`: केवल रूट उपयोगकर्ताओं की सूची दिखाता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. **सभी वर्तमान उपयोगकर्ताओं की सूची प्राप्त करना:**
   ```bash
   users
   ```

2. **उपयोगकर्ताओं की संख्या दिखाना:**
   ```bash
   users -n
   ```

3. **रूट उपयोगकर्ताओं की सूची प्राप्त करना:**
   ```bash
   users -r
   ```

## Tips
- `users` कमांड का उपयोग करते समय, यह ध्यान रखें कि यह केवल उन उपयोगकर्ताओं के नाम दिखाता है जो वर्तमान में लॉग इन हैं।
- यदि आप अधिक विस्तृत जानकारी चाहते हैं, तो `who` या `w` कमांड का उपयोग करें, जो उपयोगकर्ताओं के बारे में अधिक जानकारी प्रदान करते हैं।
