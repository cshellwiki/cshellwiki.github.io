# [লিনাক্স] Bash vigr ব্যবহার: ফাইলের জন্য নিরাপদ সম্পাদনা

## Overview
`vigr` কমান্ডটি লিনাক্স এবং ইউনিক্স সিস্টেমে `/etc/passwd` এবং `/etc/group` ফাইলের মতো গুরুত্বপূর্ণ ফাইলগুলি নিরাপদে সম্পাদনা করার জন্য ব্যবহৃত হয়। এটি ফাইলটি সম্পাদনা করার সময় সঠিকতা নিশ্চিত করে এবং ভুলভাবে সম্পাদনা হওয়া থেকে রক্ষা করে।

## Usage
বেসিক সিনট্যাক্স হল:
```
vigr [options] [arguments]
```

## Common Options
- `-c`: ফাইলের সিনট্যাক্স চেক করে।
- `-f FILE`: নির্দিষ্ট ফাইল সম্পাদনা করতে ব্যবহার করুন।
- `-r`: সম্পাদনা করার জন্য নির্দিষ্ট ফাইলের রিজার্ভেশন।

## Common Examples
1. সাধারণভাবে `vigr` ব্যবহার করা:
   ```bash
   vigr
   ```

2. নির্দিষ্ট ফাইল সম্পাদনা করা:
   ```bash
   vigr -f /etc/group
   ```

3. সিনট্যাক্স চেক করার জন্য:
   ```bash
   vigr -c
   ```

## Tips
- `vigr` ব্যবহার করার সময়, নিশ্চিত করুন যে আপনি সঠিক ফাইল সম্পাদনা করছেন।
- ফাইল সম্পাদনার আগে ব্যাকআপ রাখা সর্বদা একটি ভাল অভ্যাস।
- `vigr` কমান্ডটি ব্যবহার করার সময়, এটি একটি টেক্সট এডিটর খুলবে, তাই আপনার এডিটর সম্পর্কে কিছু ধারণা থাকা উচিত।
