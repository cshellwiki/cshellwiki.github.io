# [לינוקס] Bash vgcreate שימוש: יצירת קבוצת נפח חדשה

## Overview
הפקודה `vgcreate` משמשת ליצירת קבוצת נפח (Volume Group) חדשה במערכת ניהול דיסקים לוגיים (LVM). קבוצת נפח מאפשרת למשתמשים לאחד מספר מחיצות פיזיות לדיסק לוגי אחד, מה שמקל על ניהול שטח האחסון.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
vgcreate [options] [arguments]
```

## Common Options
- `-s, --stripes <number>`: מגדיר את מספר ה-stripes לקבוצת הנפח.
- `-p, --max-pv <number>`: מגדיר את המספר המרבי של מחיצות פיזיות שיכולות להיות בקבוצת הנפח.
- `-n, --name <name>`: מגדיר שם לקבוצת הנפח.

## Common Examples
1. יצירת קבוצת נפח חדשה בשם `myvg` עם מחיצה פיזית `/dev/sda1`:
   ```bash
   vgcreate myvg /dev/sda1
   ```

2. יצירת קבוצת נפח עם מספר מחיצות פיזיות:
   ```bash
   vgcreate myvg /dev/sda1 /dev/sdb1
   ```

3. יצירת קבוצת נפח עם הגדרת מספר ה-stripes:
   ```bash
   vgcreate -s 64k myvg /dev/sda1
   ```

## Tips
- תמיד ודא שהמחיצות הפיזיות שאתה משתמש בהן זמינות ולא בשימוש על ידי מערכת אחרת.
- שקול להשתמש בפקודות `pvcreate` לפני `vgcreate` כדי להכין את המחיצות הפיזיות לשימוש.
- ניהול קבוצות נפח יכול להיות מורכב, לכן מומלץ לקרוא את התיעוד המלא של LVM לפני ביצוע שינויים משמעותיים.
