# [লিনাক্স] Bash update-rc.d ব্যবহার: সিস্টেম সার্ভিস পরিচালনা

## Overview
`update-rc.d` কমান্ডটি লিনাক্স সিস্টেমে সার্ভিস স্ক্রিপ্টগুলিকে ইনস্টল এবং মুছে ফেলার জন্য ব্যবহৃত হয়। এটি সিস্টেমের স্টার্টআপ এবং শাটডাউন প্রক্রিয়ার সময় নির্দিষ্ট সার্ভিসগুলিকে স্বয়ংক্রিয়ভাবে চালু বা বন্ধ করতে সাহায্য করে।

## Usage
কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
update-rc.d [options] [arguments]
```

## Common Options
- `defaults`: ডিফল্ট রানলেভেলগুলিতে সার্ভিসটি যুক্ত করে।
- `remove`: সার্ভিসটি রানলেভেল থেকে মুছে ফেলে।
- `enable`: সার্ভিসটি সক্রিয় করে।
- `disable`: সার্ভিসটি নিষ্ক্রিয় করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হল:

### সার্ভিস যুক্ত করা
```bash
sudo update-rc.d myservice defaults
```

### সার্ভিস মুছে ফেলা
```bash
sudo update-rc.d myservice remove
```

### সার্ভিস সক্রিয় করা
```bash
sudo update-rc.d myservice enable
```

### সার্ভিস নিষ্ক্রিয় করা
```bash
sudo update-rc.d myservice disable
```

## Tips
- সার্ভিস স্ক্রিপ্টগুলি `/etc/init.d/` ডিরেক্টরিতে থাকা উচিত।
- নিশ্চিত করুন যে আপনার স্ক্রিপ্টগুলি সঠিকভাবে লিখিত এবং কার্যকরী।
- `update-rc.d` ব্যবহার করার সময় সর্বদা `sudo` ব্যবহার করুন, কারণ এটি প্রশাসনিক অধিকার প্রয়োজন।
