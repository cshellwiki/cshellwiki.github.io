# [লিনাক্স] Bash uptime ব্যবহার: সিস্টেমের কার্যকাল প্রদর্শন

## Overview
`uptime` কমান্ডটি সিস্টেমের কার্যকাল, বর্তমান সময় এবং লগইন করা ব্যবহারকারীদের সংখ্যা প্রদর্শন করে। এটি সিস্টেমের স্থিতিশীলতা এবং কর্মক্ষমতা সম্পর্কে ধারণা দেয়।

## Usage
`uptime` কমান্ডের মৌলিক সিনট্যাক্স হল:

```
uptime [options]
```

## Common Options
- `-p`: সিস্টেমের কার্যকালকে একটি সংক্ষিপ্ত আকারে প্রদর্শন করে।
- `-s`: সিস্টেমের শুরু হওয়ার সময় প্রদর্শন করে।

## Common Examples
1. সাধারণ `uptime` কমান্ড চালানো:
   ```bash
   uptime
   ```

2. কার্যকাল সংক্ষিপ্ত আকারে দেখতে:
   ```bash
   uptime -p
   ```

3. সিস্টেমের শুরু হওয়ার সময় দেখতে:
   ```bash
   uptime -s
   ```

## Tips
- `uptime` কমান্ডটি সিস্টেমের স্থিতিশীলতা পর্যবেক্ষণের জন্য নিয়মিত ব্যবহার করুন।
- সিস্টেমের কার্যকাল দীর্ঘ হলে, এটি একটি স্থিতিশীল পরিবেশ নির্দেশ করে।
- লগইন করা ব্যবহারকারীদের সংখ্যা পর্যবেক্ষণ করে সিস্টেমের লোড সম্পর্কে ধারণা পেতে পারেন।
