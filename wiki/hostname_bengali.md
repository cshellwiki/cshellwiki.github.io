# [লিনাক্স] Bash hostname ব্যবহার: হোস্টনেম দেখানো এবং সেট করা

## Overview
`hostname` কমান্ডটি আপনার সিস্টেমের হোস্টনেম দেখানোর এবং সেট করার জন্য ব্যবহৃত হয়। এটি একটি গুরুত্বপূর্ণ কমান্ড যা নেটওয়ার্কিং এবং সিস্টেম প্রশাসনের জন্য প্রয়োজনীয়।

## Usage
নিচে `hostname` কমান্ডের মৌলিক সিনট্যাক্স দেওয়া হলো:

```bash
hostname [options] [arguments]
```

## Common Options
- `-f` : সম্পূর্ণ ফক্টর নাম দেখায়।
- `-i` : হোস্টনেমের সাথে সম্পর্কিত আইপি ঠিকানা দেখায়।
- `-s` : সংক্ষিপ্ত হোস্টনেম দেখায়।
- `-V` : কমান্ডের সংস্করণ তথ্য দেখায়।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### 1. বর্তমান হোস্টনেম দেখানো
```bash
hostname
```

### 2. সম্পূর্ণ ফক্টর নাম দেখানো
```bash
hostname -f
```

### 3. হোস্টনেমের আইপি ঠিকানা দেখানো
```bash
hostname -i
```

### 4. হোস্টনেম পরিবর্তন করা
```bash
sudo hostname new-hostname
```

## Tips
- হোস্টনেম পরিবর্তন করার পর, সিস্টেম রিবুট করা প্রয়োজন হতে পারে যাতে পরিবর্তন কার্যকর হয়।
- সঠিক হোস্টনেম নির্বাচন করুন, কারণ এটি নেটওয়ার্কে সনাক্তকরণে সহায়তা করে।
- হোস্টনেম পরিবর্তন করার সময়, প্রশাসনিক অনুমতি প্রয়োজন হতে পারে, তাই `sudo` ব্যবহার করুন।
