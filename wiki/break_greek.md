# [Λειτουργικό Σύστημα] Bash break Χρήση: Τερματισμός βρόχων

## Overview
Η εντολή `break` χρησιμοποιείται σε σενάρια Bash για να τερματίσει έναν βρόχο (loop) πρόωρα. Όταν εκτελείται, η `break` θα σταματήσει την εκτέλεση του βρόχου και θα μεταφέρει τον έλεγχο στην επόμενη εντολή μετά τον βρόχο.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
break [n]
```

Όπου το `n` είναι προαιρετικό και καθορίζει πόσους βρόχους θα τερματίσει. Αν δεν καθοριστεί, θα τερματίσει τον πιο εσωτερικό βρόχο.

## Common Options
- `n`: Καθορίζει τον αριθμό των βρόχων που θα τερματιστούν. Αν `n=1`, τερματίζεται μόνο ο πιο εσωτερικός βρόχος. Αν `n=2`, τερματίζονται οι δύο πιο εσωτερικοί βρόχοι, και ούτω καθεξής.

## Common Examples

### Παράδειγμα 1: Τερματισμός ενός βρόχου
```bash
for i in {1..5}; do
  if [ $i -eq 3 ]; then
    break
  fi
  echo $i
done
```
Αυτό το παράδειγμα θα εκτυπώσει τους αριθμούς 1 και 2, και θα τερματίσει τον βρόχο όταν `i` είναι 3.

### Παράδειγμα 2: Τερματισμός πολλών βρόχων
```bash
for i in {1..3}; do
  for j in {1..3}; do
    if [ $j -eq 2 ]; then
      break 2
    fi
    echo "$i, $j"
  done
done
```
Αυτό το παράδειγμα θα τερματίσει και τους δύο βρόχους όταν `j` είναι 2, χωρίς να εκτυπώσει τίποτα.

### Παράδειγμα 3: Χρήση με while
```bash
count=1
while true; do
  echo $count
  if [ $count -eq 5 ]; then
    break
  fi
  ((count++))
done
```
Αυτό το παράδειγμα θα εκτυπώσει τους αριθμούς από 1 έως 5 και θα τερματίσει τον βρόχο όταν `count` είναι 5.

## Tips
- Χρησιμοποιήστε την `break` για να αποφύγετε την εκτέλεση περιττών επαναλήψεων σε βρόχους.
- Να είστε προσεκτικοί με το `n`, καθώς μπορεί να τερματίσει περισσότερους βρόχους από αυτούς που περιμένατε.
- Η `break` είναι χρήσιμη σε συνδυασμό με συνθήκες (if statements) για να ελέγξετε πότε πρέπει να σταματήσετε την εκτέλεση ενός βρόχου.
