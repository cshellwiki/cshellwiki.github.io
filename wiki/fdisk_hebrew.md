# [לינוקס] Bash fdisk שימוש: ניהול מחיצות דיסק

## Overview
הפקודה `fdisk` משמשת לניהול מחיצות דיסק במערכות לינוקס. היא מאפשרת למשתמש ליצור, למחוק ולשנות מחיצות על כוננים קשיחים או כונני SSD.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
fdisk [options] [device]
```

## Common Options
- `-l`: מציג את כל המחיצות על הכוננים המותקנים במערכת.
- `-u`: מציג את גודל המחיצות בבלוקים.
- `-n`: יוצר מחיצה חדשה.
- `-d`: מוחק מחיצה קיימת.
- `-t`: משנה את סוג המחיצה.

## Common Examples
1. **הצגת כל המחיצות:**
   ```bash
   fdisk -l
   ```

2. **יצירת מחיצה חדשה:**
   ```bash
   fdisk /dev/sda
   ```
   לאחר מכן, השתמש באופציה `n` כדי ליצור מחיצה חדשה.

3. **מחיקת מחיצה:**
   ```bash
   fdisk /dev/sda
   ```
   לאחר מכן, השתמש באופציה `d` כדי למחוק מחיצה קיימת.

4. **שינוי סוג מחיצה:**
   ```bash
   fdisk /dev/sda
   ```
   לאחר מכן, השתמש באופציה `t` כדי לשנות את סוג המחיצה.

## Tips
- תמיד גבה את הנתונים שלך לפני ביצוע שינויים במחיצות.
- השתמש בפקודה `man fdisk` כדי לקבל מידע נוסף על הפקודה ואופציותיה.
- ודא שאתה פועל על הכונן הנכון כדי למנוע אובדן נתונים.
