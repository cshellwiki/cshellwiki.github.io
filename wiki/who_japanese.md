# [Linux] Bash who 使用法: 現在ログインしているユーザーの表示

## 概要
`who` コマンドは、システムに現在ログインしているユーザーの情報を表示します。このコマンドを使用することで、誰がシステムにアクセスしているかを簡単に確認できます。

## 使用法
基本的な構文は以下の通りです。

```
who [オプション] [引数]
```

## 一般的なオプション
- `-a`: すべての情報を表示します。
- `-b`: 最後のブート時刻を表示します。
- `-q`: ログインしているユーザーの名前を簡潔に表示します。
- `--help`: コマンドのヘルプを表示します。

## 一般的な例
以下に、`who` コマンドのいくつかの実用的な例を示します。

### 例1: 現在のログインユーザーを表示
```bash
who
```

### 例2: すべての情報を表示
```bash
who -a
```

### 例3: 最後のブート時刻を表示
```bash
who -b
```

### 例4: 簡潔なユーザーリストを表示
```bash
who -q
```

## ヒント
- `who` コマンドは、システムのセキュリティ監視に役立ちます。定期的に実行して、誰がログインしているかを確認しましょう。
- `who` の出力は、他のコマンドと組み合わせて使用することができます。例えば、`grep` を使って特定のユーザーをフィルタリングすることができます。
- スクリプト内で使用する場合は、出力をファイルにリダイレクトすることで、後で確認できるように保存することができます。
