# [Λειτουργικό Σύστημα] Bash who Χρήση: Εμφάνιση χρηστών που είναι συνδεδεμένοι

## Overview
Η εντολή `who` χρησιμοποιείται για να εμφανίσει μια λίστα με τους χρήστες που είναι συνδεδεμένοι στο σύστημα. Παρέχει πληροφορίες όπως το όνομα χρήστη, η τρέχουσα κονσόλα και η ώρα σύνδεσης.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```
who [options] [arguments]
```

## Common Options
- `-a`: Εμφανίζει όλες τις διαθέσιμες πληροφορίες για τους χρήστες.
- `-b`: Δείχνει την τελευταία ώρα εκκίνησης του συστήματος.
- `-q`: Εμφανίζει μόνο τους χρήστες που είναι συνδεδεμένοι, μαζί με τον αριθμό τους.
- `--help`: Εμφανίζει βοήθεια σχετικά με τη χρήση της εντολής.

## Common Examples
### Παράδειγμα 1: Εμφάνιση όλων των συνδεδεμένων χρηστών
```bash
who
```

### Παράδειγμα 2: Εμφάνιση λεπτομερειών για όλους τους χρήστες
```bash
who -a
```

### Παράδειγμα 3: Εμφάνιση της τελευταίας ώρας εκκίνησης του συστήματος
```bash
who -b
```

### Παράδειγμα 4: Εμφάνιση μόνο των χρηστών που είναι συνδεδεμένοι
```bash
who -q
```

## Tips
- Χρησιμοποιήστε την επιλογή `-a` για να αποκτήσετε μια πλήρη εικόνα των χρηστών και των συνδέσεών τους.
- Συνδυάστε την εντολή `who` με άλλες εντολές όπως `grep` για να φιλτράρετε συγκεκριμένα αποτελέσματα.
- Ελέγξτε τα δικαιώματα πρόσβασης αν δεν μπορείτε να δείτε πληροφορίες για άλλους χρήστες.
