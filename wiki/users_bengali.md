# [লিনাক্স] Bash ব্যবহারকারীদের ব্যবহার: ব্যবহারকারীদের তালিকা দেখুন

## Overview
`users` কমান্ডটি বর্তমানে লগ ইন করা ব্যবহারকারীদের তালিকা প্রদর্শন করে। এটি সিস্টেমে একাধিক ব্যবহারকারী লগ ইন থাকলে তাদের নামগুলো দেখাতে সহায়ক।

## Usage
নিচে `users` কমান্ডের মৌলিক সিনট্যাক্স দেওয়া হলো:

```
users [options]
```

## Common Options
- `-n` : ব্যবহারকারীদের নামের সংখ্যা সীমাবদ্ধ করতে ব্যবহৃত হয়।
- `-r` : শুধুমাত্র রুট ব্যবহারকারীর নাম দেখায়।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### উদাহরণ 1: লগ ইন করা ব্যবহারকারীদের তালিকা দেখানো
```bash
users
```
এই কমান্ডটি সিস্টেমে বর্তমানে লগ ইন করা সকল ব্যবহারকারীর নাম দেখাবে।

### উদাহরণ 2: ব্যবহারকারীদের সংখ্যা সীমাবদ্ধ করা
```bash
users -n 2
```
এই কমান্ডটি সর্বাধিক ২ জন ব্যবহারকারীর নাম প্রদর্শন করবে।

### উদাহরণ 3: শুধুমাত্র রুট ব্যবহারকারীর নাম দেখানো
```bash
users -r
```
এই কমান্ডটি শুধুমাত্র রুট ব্যবহারকারীর নাম দেখাবে।

## Tips
- `users` কমান্ডটি সাধারণত `who` বা `w` কমান্ডের সাথে ব্যবহার করা হয়, যা আরও বিস্তারিত তথ্য প্রদান করে।
- যদি আপনি দেখতে চান যে কোন ব্যবহারকারী কোন টার্মিনালে লগ ইন করেছেন, তাহলে `who` কমান্ড ব্যবহার করা ভালো।
- সিস্টেম প্রশাসক হিসেবে, ব্যবহারকারীদের কার্যকলাপ ট্র্যাক করার জন্য এই কমান্ডটি কার্যকর হতে পারে।
