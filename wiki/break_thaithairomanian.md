# [Bash] Bash break utilizare: [interrompe un ciclu]

## Overview
คำสั่ง `break` ใน Bash ใช้เพื่อหยุดการทำงานของลูปในสคริปต์หรือคำสั่งที่กำลังทำงานอยู่ โดยจะออกจากลูปที่กำหนดและดำเนินการต่อไปยังคำสั่งถัดไปในสคริปต์

## Usage
- ไวยากรณ์พื้นฐานของคำสั่ง `break` คือ:

```bash
break [n]
```

โดยที่ `n` คือจำนวนระดับของลูปที่ต้องการหยุด (ถ้าไม่ระบุจะหยุดที่ลูปที่อยู่ในระดับหนึ่ง)

## Common Options
- `n` : จำนวนระดับของลูปที่ต้องการหยุด (ถ้าไม่ระบุ จะหยุดที่ลูปที่อยู่ในระดับหนึ่ง)

## Common Examples
- ตัวอย่างที่ 1: หยุดลูป `for` เมื่อถึงค่าที่กำหนด

```bash
for i in {1..10}; do
  if [ $i -eq 5 ]; then
    break
  fi
  echo $i
done
```
ผลลัพธ์: 
```
1
2
3
4
```

- ตัวอย่างที่ 2: หยุดลูป `while` เมื่อเงื่อนไขเป็นจริง

```bash
count=1
while true; do
  if [ $count -gt 3 ]; then
    break
  fi
  echo $count
  ((count++))
done
```
ผลลัพธ์:
```
1
2
3
```

- ตัวอย่างที่ 3: ใช้ `break` ในลูปซ้อน

```bash
for i in {1..3}; do
  for j in {1..3}; do
    if [ $j -eq 2 ]; then
      break 2
    fi
    echo "i: $i, j: $j"
  done
done
```
ผลลัพธ์:
```
i: 1, j: 1
```

## Tips
- ใช้ `break` เพื่อควบคุมการทำงานของลูปให้มีประสิทธิภาพมากขึ้น โดยเฉพาะเมื่อคุณต้องการหยุดการทำงานเมื่อเงื่อนไขบางอย่างเป็นจริง
- ควรใช้ `break` อย่างระมัดระวังในลูปซ้อน เพื่อหลีกเลี่ยงการหยุดลูปที่ไม่ต้องการ
- การใช้ `break` ร่วมกับ `if` จะช่วยให้คุณสามารถควบคุมการทำงานของลูปได้อย่างยืดหยุ่น
