# [لينكس] Bash watch الاستخدام: عرض الأوامر بشكل دوري

## Overview
أمر `watch` في Bash يُستخدم لمراقبة تنفيذ الأوامر بشكل دوري. يسمح لك بمشاهدة نتائج الأوامر المتكررة في فترات زمنية محددة، مما يسهل متابعة التغييرات في البيانات أو الحالة.

## Usage
الصيغة الأساسية للأمر هي:

```bash
watch [options] [arguments]
```

## Common Options
- `-n <seconds>`: يحدد الفترة الزمنية بين تنفيذ الأوامر بالثواني. القيمة الافتراضية هي 2 ثانية.
- `-d`: يميز التغييرات في المخرجات بين كل تنفيذ.
- `-t`: يمنع عرض عنوان الأمر، مما يجعل العرض أكثر نظافة.
- `-x`: يسمح بتشغيل الأمر مع أي خيارات أو معلمات إضافية.

## Common Examples
- لمراقبة استخدام الذاكرة:
```bash
watch -n 5 free -h
```

- لمراقبة حالة خدمة معينة مثل Apache:
```bash
watch systemctl status apache2
```

- لمراقبة تغييرات الملفات في دليل معين:
```bash
watch -d ls -l /path/to/directory
```

- لمراقبة استخدام المعالج:
```bash
watch -n 1 top -b -n 1
```

## Tips
- استخدم الخيار `-d` لتسهيل رؤية التغييرات بين كل تنفيذ.
- يمكنك تغيير الفترة الزمنية باستخدام الخيار `-n` لتناسب احتياجاتك.
- تذكر أن استخدام `watch` مع الأوامر الثقيلة قد يؤثر على أداء النظام، لذا استخدمه بحذر.
