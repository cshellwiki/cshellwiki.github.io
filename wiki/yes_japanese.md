# [Linux] Bash yes 使用法: 繰り返し文字列を出力する

## Overview
`yes` コマンドは、指定した文字列を無限に繰り返し出力するためのコマンドです。デフォルトでは「y」を繰り返しますが、他の文字列を指定することもできます。このコマンドは、特に他のコマンドの入力として使用する際に便利です。

## Usage
基本的な構文は次の通りです。

```bash
yes [options] [arguments]
```

## Common Options
- `-h`, `--help`: ヘルプメッセージを表示します。
- `-V`, `--version`: バージョン情報を表示します。

## Common Examples
以下は、`yes` コマンドのいくつかの実用的な例です。

### 例1: デフォルトの「y」を出力する
```bash
yes
```
このコマンドは、「y」を無限に出力します。

### 例2: 特定の文字列を出力する
```bash
yes "Hello, World!"
```
このコマンドは、「Hello, World!」を無限に出力します。

### 例3: 他のコマンドと組み合わせる
```bash
yes | head -n 5
```
このコマンドは、「y」を5回出力します。

### 例4: 繰り返しの入力を必要とするコマンドに使用
```bash
yes | rm -i *.tmp
```
このコマンドは、すべての `.tmp` ファイルを削除する際に、毎回「y」と答えます。

## Tips
- `yes` コマンドは無限に出力を続けるため、使用する際は注意が必要です。必要に応じて、`head` や `tail` と組み合わせて使用してください。
- `yes` の出力をファイルにリダイレクトすることも可能です。例えば、`yes "OK" > output.txt` とすることで、ファイルに「OK」を繰り返し書き込むことができます。
