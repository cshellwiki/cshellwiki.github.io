# [لينكس] Bash pkg الاستخدام: إدارة الحزم

## Overview
أمر `pkg` هو أداة تستخدم لإدارة الحزم في أنظمة التشغيل المستندة إلى Unix، مثل FreeBSD. يتيح لك هذا الأمر تثبيت، تحديث، وإزالة الحزم البرمجية بسهولة.

## Usage
تكون الصيغة الأساسية لأمر `pkg` كما يلي:

```bash
pkg [options] [arguments]
```

## Common Options
- `install`: لتثبيت حزمة جديدة.
- `remove`: لإزالة حزمة مثبتة.
- `update`: لتحديث قاعدة بيانات الحزم.
- `upgrade`: لترقية الحزم المثبتة إلى أحدث إصدار.
- `search`: للبحث عن حزم معينة.

## Common Examples
- لتثبيت حزمة جديدة:
```bash
pkg install package_name
```

- لإزالة حزمة مثبتة:
```bash
pkg remove package_name
```

- لتحديث قاعدة بيانات الحزم:
```bash
pkg update
```

- لترقية جميع الحزم المثبتة:
```bash
pkg upgrade
```

- للبحث عن حزمة معينة:
```bash
pkg search package_name
```

## Tips
- تأكد من تحديث قاعدة بيانات الحزم بانتظام باستخدام `pkg update` قبل تثبيت أو ترقية أي حزمة.
- استخدم `pkg info package_name` للحصول على معلومات مفصلة حول حزمة معينة.
- يمكنك استخدام `pkg autoremove` لإزالة الحزم غير المستخدمة التي تم تثبيتها كاعتماديات.
