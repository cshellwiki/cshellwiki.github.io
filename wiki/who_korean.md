# [리눅스] Bash who 사용법: 현재 로그인한 사용자 목록 보기

## Overview
`who` 명령어는 현재 시스템에 로그인한 사용자들의 목록을 보여줍니다. 이 명령어는 사용자 이름, 로그인 시간, 터미널 정보 등을 포함하여 시스템의 현재 상태를 파악하는 데 유용합니다.

## Usage
기본 구문은 다음과 같습니다:

```
who [옵션] [인수]
```

## Common Options
- `-a`: 모든 정보를 표시합니다.
- `-b`: 시스템의 마지막 부팅 시간을 보여줍니다.
- `-q`: 로그인한 사용자 수와 사용자 이름만 표시합니다.
- `--help`: 사용법을 보여줍니다.

## Common Examples
다음은 `who` 명령어의 몇 가지 일반적인 사용 예입니다.

1. 현재 로그인한 사용자 목록 보기:
   ```bash
   who
   ```

2. 시스템의 마지막 부팅 시간 확인:
   ```bash
   who -b
   ```

3. 로그인한 사용자 수와 이름만 표시하기:
   ```bash
   who -q
   ```

4. 모든 정보 표시하기:
   ```bash
   who -a
   ```

## Tips
- `who` 명령어는 시스템 관리자가 현재 로그인한 사용자와 그들의 활동을 모니터링하는 데 유용합니다.
- `who`와 `w` 명령어를 함께 사용하면 더 많은 사용자 활동 정보를 확인할 수 있습니다.
- 주기적으로 `who` 명령어를 실행하여 시스템의 보안을 유지하는 것도 좋은 습관입니다.
