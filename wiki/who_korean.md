# [리눅스] C Shell (csh) who 사용법: 현재 로그인한 사용자 목록 보기

## 개요
`who` 명령어는 현재 시스템에 로그인한 사용자들의 목록을 표시합니다. 이 명령어를 사용하면 각 사용자의 로그인 시간과 터미널 정보를 확인할 수 있습니다.

## 사용법
기본 구문은 다음과 같습니다:
```
who [옵션] [인수]
```

## 일반 옵션
- `-a`: 모든 정보를 표시합니다.
- `-b`: 시스템의 마지막 부팅 시간을 표시합니다.
- `-q`: 로그인한 사용자 수와 사용자 목록만 표시합니다.
- `-r`: 현재 실행 중인 런레벨을 표시합니다.

## 일반 예제
1. 현재 로그인한 사용자 목록 보기:
   ```csh
   who
   ```

2. 모든 정보 표시:
   ```csh
   who -a
   ```

3. 시스템의 마지막 부팅 시간 확인:
   ```csh
   who -b
   ```

4. 로그인한 사용자 수와 목록만 보기:
   ```csh
   who -q
   ```

5. 현재 실행 중인 런레벨 확인:
   ```csh
   who -r
   ```

## 팁
- `who` 명령어는 시스템 관리자가 사용자 활동을 모니터링하는 데 유용합니다.
- `who` 명령어의 출력을 `grep`과 함께 사용하여 특정 사용자를 찾을 수 있습니다.
- 주기적으로 `who` 명령어를 실행하여 시스템의 사용자 상태를 확인하는 것이 좋습니다.
