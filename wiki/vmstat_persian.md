# [لینوکس] Bash vmstat استفاده: [نظارت بر وضعیت سیستم]

## Overview
دستور `vmstat` در لینوکس برای نظارت بر وضعیت سیستم و نمایش اطلاعات مربوط به حافظه، پردازنده و ورودی/خروجی استفاده می‌شود. این دستور به کاربران کمک می‌کند تا عملکرد سیستم را بررسی کرده و مشکلات احتمالی را شناسایی کنند.

## Usage
سینتکس پایه دستور `vmstat` به صورت زیر است:

```bash
vmstat [options] [arguments]
```

## Common Options
- `-a`: نمایش اطلاعات مربوط به حافظه فعال و غیر فعال.
- `-m`: نمایش اطلاعات مربوط به حافظه‌های صفحه.
- `-s`: نمایش آمار کلی سیستم.
- `-n`: جلوگیری از نمایش نام سیستم در خروجی.
- `interval`: زمان بین هر بار نمایش اطلاعات (به ثانیه).

## Common Examples
### مثال 1: نمایش وضعیت سیستم به صورت پیش‌فرض
```bash
vmstat
```

### مثال 2: نمایش وضعیت سیستم هر 5 ثانیه
```bash
vmstat 5
```

### مثال 3: نمایش آمار حافظه
```bash
vmstat -a
```

### مثال 4: نمایش آمار کلی سیستم
```bash
vmstat -s
```

## Tips
- برای تجزیه و تحلیل بهتر، می‌توانید خروجی `vmstat` را با سایر ابزارهای نظارتی مانند `top` یا `htop` ترکیب کنید.
- به یاد داشته باشید که استفاده از `vmstat` در زمان‌های مختلف می‌تواند به شناسایی الگوهای مصرف منابع کمک کند.
- برای مشاهده تغییرات در عملکرد سیستم، از گزینه `interval` استفاده کنید تا خروجی به صورت پیوسته نمایش داده شود.
