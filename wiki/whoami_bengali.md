# [লিনাক্স] Bash whoami ব্যবহার: বর্তমান ব্যবহারকারীর নাম দেখান

## Overview
`whoami` কমান্ডটি বর্তমান লগ ইন করা ব্যবহারকারীর নাম প্রদর্শন করে। এটি বিশেষ করে তখন উপকারী যখন আপনি নিশ্চিত হতে চান যে আপনি কোন ব্যবহারকারী হিসেবে লগ ইন আছেন।

## Usage
`whoami` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
whoami [options] [arguments]
```

## Common Options
`whoami` কমান্ডের জন্য সাধারণ কিছু অপশন হলো:

- `--help`: এই অপশনটি ব্যবহার করলে কমান্ডের সাহায্য তথ্য দেখাবে।
- `--version`: এই অপশনটি ব্যবহার করলে কমান্ডের সংস্করণ তথ্য দেখাবে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. **বর্তমান ব্যবহারকারীর নাম দেখানো:**
   ```bash
   whoami
   ```

2. **কমান্ডের সাহায্য তথ্য দেখা:**
   ```bash
   whoami --help
   ```

3. **কমান্ডের সংস্করণ তথ্য দেখা:**
   ```bash
   whoami --version
   ```

## Tips
- `whoami` কমান্ডটি স্ক্রিপ্টে ব্যবহার করা যেতে পারে যাতে নিশ্চিত হওয়া যায় যে স্ক্রিপ্টটি সঠিক ব্যবহারকারীর অধীনে চলছে।
- এটি ব্যবহার করে আপনি যদি বিভিন্ন ব্যবহারকারীর অধীনে কাজ করেন তবে আপনার কাজের পরিবেশ বুঝতে সাহায্য করে।
