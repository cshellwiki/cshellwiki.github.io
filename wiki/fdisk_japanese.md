# [Linux] Bash fdisk の使い方: パーティションを管理する

## 概要
`fdisk` コマンドは、Linux システムにおいてディスクパーティションを管理するためのツールです。このコマンドを使用することで、パーティションの作成、削除、変更、表示などが行えます。

## 使用法
基本的な構文は以下の通りです。

```bash
fdisk [オプション] [デバイス]
```

## 一般的なオプション
- `-l`: システム上のすべてのパーティションテーブルをリスト表示します。
- `-u`: セクタ単位でのサイズを表示します。
- `-n`: 新しいパーティションを作成します。
- `-d`: 既存のパーティションを削除します。
- `-p`: 現在のパーティションテーブルを表示します。

## 一般的な例
以下は、`fdisk` コマンドのいくつかの実用的な例です。

### 1. パーティションテーブルの表示
```bash
fdisk -l /dev/sda
```

### 2. 新しいパーティションの作成
```bash
fdisk /dev/sda
```
このコマンドを実行後、インタラクティブなメニューが表示されます。`n` を入力して新しいパーティションを作成します。

### 3. パーティションの削除
```bash
fdisk /dev/sda
```
インタラクティブメニューで `d` を選択し、削除したいパーティションの番号を指定します。

### 4. 現在のパーティションテーブルの表示
```bash
fdisk -p /dev/sda
```

## ヒント
- `fdisk` を使用する際は、必ず重要なデータのバックアップを取ってください。パーティションの変更はデータ損失を引き起こす可能性があります。
- `fdisk` は主にMBR（Master Boot Record）パーティションテーブルを扱います。GPT（GUID Partition Table）を使用する場合は、`gdisk` や `parted` コマンドを検討してください。
- コマンドを実行する際は、管理者権限が必要です。`sudo` を使用してコマンドを実行してください。
