# [लिनक्स] Bash xz उपयोग: फ़ाइलों को संकुचित और अनसंकुचित करना

## Overview
`xz` एक कमांड-लाइन टूल है जिसका उपयोग फ़ाइलों को संकुचित करने और अनसंकुचित करने के लिए किया जाता है। यह उच्च संकुचन अनुपात प्रदान करता है, जिससे फ़ाइलों का आकार काफी कम हो जाता है।

## Usage
`xz` कमांड का मूल सिंटैक्स इस प्रकार है:

```
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: फ़ाइल को अनसंकुचित करें।
- `-k`, `--keep`: मूल फ़ाइल को बनाए रखें, संकुचित फ़ाइल के साथ।
- `-v`, `--verbose`: प्रक्रिया के दौरान विस्तृत जानकारी दिखाएं।
- `-9`: अधिकतम संकुचन के लिए।
- `-1` से `-9`: संकुचन के स्तर को निर्धारित करें (1 सबसे कम और 9 सबसे अधिक)।

## Common Examples
### फ़ाइल को संकुचित करना
एक फ़ाइल को संकुचित करने के लिए, निम्नलिखित कमांड का उपयोग करें:

```bash
xz filename.txt
```

### फ़ाइल को अनसंकुचित करना
संकुचित फ़ाइल को अनसंकुचित करने के लिए:

```bash
xz -d filename.txt.xz
```

या

```bash
unxz filename.txt.xz
```

### मूल फ़ाइल को बनाए रखना
यदि आप मूल फ़ाइल को संकुचित करते समय बनाए रखना चाहते हैं:

```bash
xz -k filename.txt
```

### अधिकतम संकुचन के साथ संकुचन
अधिकतम संकुचन के लिए:

```bash
xz -9 filename.txt
```

## Tips
- संकुचित फ़ाइलों का उपयोग करते समय, सुनिश्चित करें कि आपके पास अनसंकुचन के लिए `xz` स्थापित है।
- यदि आप कई फ़ाइलों को संकुचित करना चाहते हैं, तो आप वाइल्डकार्ड का उपयोग कर सकते हैं, जैसे `xz *.txt`।
- संकुचित फ़ाइलों को प्रबंधित करने के लिए, `-v` विकल्प का उपयोग करें ताकि आप प्रक्रिया के दौरान जानकारी प्राप्त कर सकें।
