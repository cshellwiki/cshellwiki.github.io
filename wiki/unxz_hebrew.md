# [לינוקס] Bash unxz שימוש: דחיסת קבצים בפורמט XZ

## Overview
הפקודה `unxz` משמשת לדחיסת קבצים בפורמט XZ. היא מאפשרת למשתמשים לפתוח קבצים דחוסים ולהחזירם למצבם המקורי.

## Usage
התחביר הבסיסי של הפקודה הוא:

```
unxz [אפשרויות] [ארגומנטים]
```

## Common Options
- `-k`, `--keep`: שומר את הקובץ המקורי לאחר הדחיסה.
- `-f`, `--force`: מכריח את הפקודה לפעול גם אם הקובץ המפוענח קיים.
- `-v`, `--verbose`: מציג מידע מפורט על תהליך הפענוח.

## Common Examples
- פענוח קובץ דחוס בשם `file.xz`:
  ```bash
  unxz file.xz
  ```

- פענוח קובץ דחוס ושמירה על הקובץ המקורי:
  ```bash
  unxz -k file.xz
  ```

- פענוח קובץ דחוס עם אפשרות להכריח את הפקודה לפעול:
  ```bash
  unxz -f file.xz
  ```

- פענוח קובץ דחוס והצגת מידע מפורט על התהליך:
  ```bash
  unxz -v file.xz
  ```

## Tips
- תמיד כדאי לבדוק אם יש לכם גיבוי של הקובץ המקורי לפני הפענוח, במיוחד אם אתם משתמשים באפשרות `-f`.
- השתמשו באפשרות `-k` אם אתם רוצים לשמור את הקובץ המקורי לצורך השוואה או גיבוי.
- אם אתם עובדים עם מספר קבצים, ניתן להשתמש בפקודה על מספר קבצים בו זמנית, לדוגמה:
  ```bash
  unxz file1.xz file2.xz
  ```
