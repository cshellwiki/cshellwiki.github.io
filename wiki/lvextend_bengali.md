# [লিনাক্স] Bash lvextend ব্যবহার: লজিক্যাল ভলিউমের আকার বৃদ্ধি করা

## Overview
`lvextend` কমান্ডটি লিনাক্সে লজিক্যাল ভলিউমের আকার বৃদ্ধি করতে ব্যবহৃত হয়। এটি মূলত লজিক্যাল ভলিউম ম্যানেজমেন্ট (LVM) এর একটি অংশ, যা ডিস্ক স্পেসের কার্যকর ব্যবস্থাপনা করতে সহায়তা করে।

## Usage
`lvextend` কমান্ডের মৌলিক সিনট্যাক্স হল:

```
lvextend [options] [arguments]
```

## Common Options
- `-L` : নির্দিষ্ট আকারে লজিক্যাল ভলিউম বাড়াতে ব্যবহৃত হয়।
- `-l` : লজিক্যাল একক সংখ্যা (LE) দ্বারা আকার বাড়ানোর জন্য।
- `-r` : ফাইল সিস্টেমকে স্বয়ংক্রিয়ভাবে প্রসারিত করতে ব্যবহৃত হয়।
- `--resizefs` : ফাইল সিস্টেমের আকার বাড়ানোর জন্য।

## Common Examples
1. একটি লজিক্যাল ভলিউমের আকার 10GB বৃদ্ধি করতে:
   ```bash
   lvextend -L +10G /dev/vg_name/lv_name
   ```

2. একটি লজিক্যাল ভলিউমের আকার 50 লজিক্যাল একক (LE) দ্বারা বাড়াতে:
   ```bash
   lvextend -l +50 /dev/vg_name/lv_name
   ```

3. ফাইল সিস্টেমকে স্বয়ংক্রিয়ভাবে প্রসারিত করতে:
   ```bash
   lvextend -r -L +5G /dev/vg_name/lv_name
   ```

4. একটি লজিক্যাল ভলিউমের আকার 20GB নির্দিষ্ট করতে:
   ```bash
   lvextend -L 20G /dev/vg_name/lv_name
   ```

## Tips
- `lvextend` চালানোর আগে নিশ্চিত করুন যে আপনার কাছে পর্যাপ্ত ফ্রি স্পেস রয়েছে।
- `-r` অপশন ব্যবহার করলে ফাইল সিস্টেম স্বয়ংক্রিয়ভাবে প্রসারিত হবে, যা সময় সাশ্রয়ী।
- আকার পরিবর্তনের পর, ফাইল সিস্টেমের স্বাস্থ্য পরীক্ষা করতে `fsck` কমান্ড ব্যবহার করুন।
