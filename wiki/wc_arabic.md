# [لينكس] Bash wc الاستخدام: عد الكلمات والأسطر والحروف

## Overview
أداة `wc` (Word Count) هي أداة تستخدم في نظام لينكس لعد الكلمات والأسطر والحروف في الملفات أو المدخلات النصية. تعتبر هذه الأداة مفيدة لتحليل المحتوى النصي بشكل سريع.

## Usage
الصيغة الأساسية لاستخدام الأمر `wc` هي:

```bash
wc [options] [arguments]
```

## Common Options
- `-l`: عد الأسطر فقط.
- `-w`: عد الكلمات فقط.
- `-c`: عد الحروف فقط.
- `-m`: عد الحروف (بما في ذلك الأحرف متعددة البايت).
- `-L`: عرض طول أطول سطر في الملف.

## Common Examples
- عد الأسطر في ملف نصي:
  ```bash
  wc -l filename.txt
  ```

- عد الكلمات في ملف نصي:
  ```bash
  wc -w filename.txt
  ```

- عد الحروف في ملف نصي:
  ```bash
  wc -c filename.txt
  ```

- استخدام الأمر مع عدة ملفات:
  ```bash
  wc -l file1.txt file2.txt
  ```

- عرض طول أطول سطر في ملف:
  ```bash
  wc -L filename.txt
  ```

## Tips
- يمكنك دمج الخيارات معًا، مثل `wc -lw filename.txt` لعد الأسطر والكلمات في نفس الوقت.
- استخدم `wc` مع أنابيب الأوامر (`|`) لمعالجة المدخلات من أوامر أخرى، مثل:
  ```bash
  cat filename.txt | wc -w
  ```
- تأكد من استخدام `wc` مع الملفات النصية فقط للحصول على نتائج دقيقة.
