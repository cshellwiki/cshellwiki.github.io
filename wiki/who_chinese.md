# [Linux] Bash who 用法: 显示当前登录用户信息

## 概述
`who` 命令用于显示当前登录系统的用户信息，包括用户名、终端、登录时间等。这对于系统管理员和用户来说，了解系统的使用情况非常有用。

## 用法
基本语法如下：
```
who [options] [arguments]
```

## 常用选项
- `-a`：显示所有信息，包括用户、终端、登录时间等。
- `-b`：显示系统最后一次启动的时间。
- `-q`：仅显示当前登录的用户数量和用户名。
- `--help`：显示帮助信息。

## 常见示例
1. 显示当前登录的所有用户信息：
   ```bash
   who
   ```

2. 显示所有信息，包括用户和登录时间：
   ```bash
   who -a
   ```

3. 显示系统最后一次启动的时间：
   ```bash
   who -b
   ```

4. 仅显示当前登录的用户数量和用户名：
   ```bash
   who -q
   ```

## 小贴士
- 使用 `who` 命令时，可以结合 `grep` 命令过滤特定用户的信息。
- 定期检查登录用户可以帮助及时发现未授权的访问。
- 结合 `w` 命令使用，可以获取更详细的用户活动信息。
