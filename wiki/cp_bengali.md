# [লিনাক্স] Bash cp ব্যবহার: ফাইল কপি করা

## Overview
`cp` কমান্ডটি লিনাক্স এবং ইউনিক্স ভিত্তিক অপারেটিং সিস্টেমে ফাইল এবং ডিরেক্টরি কপি করার জন্য ব্যবহৃত হয়। এটি ব্যবহারকারীদের একটি ফাইলের একটি কপি তৈরি করতে সহায়তা করে, যা মূল ফাইলের সাথে একই তথ্য ধারণ করে।

## Usage
`cp` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
cp [options] [source] [destination]
```

## Common Options
- `-r`: রিকার্সিভভাবে ডিরেক্টরি কপি করে।
- `-i`: যদি গন্তব্যে একটি ফাইল ইতিমধ্যে থাকে, তাহলে ব্যবহারকারীকে নিশ্চিতকরণের জন্য জিজ্ঞাসা করে।
- `-u`: কেবলমাত্র সেই ফাইলগুলি কপি করে যা গন্তব্যে নেই বা উৎসের চেয়ে নতুন।
- `-v`: কপি করার সময় প্রক্রিয়াটি প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. একটি ফাইল কপি করা:
   ```bash
   cp file1.txt file2.txt
   ```

2. একটি ডিরেক্টরি এবং এর সমস্ত ফাইল কপি করা:
   ```bash
   cp -r myfolder/ newfolder/
   ```

3. কপি করার সময় নিশ্চিতকরণের জন্য জিজ্ঞাসা করা:
   ```bash
   cp -i file1.txt file2.txt
   ```

4. কেবলমাত্র নতুন ফাইলগুলি কপি করা:
   ```bash
   cp -u file1.txt file2.txt
   ```

5. কপি করার সময় প্রক্রিয়া প্রদর্শন করা:
   ```bash
   cp -v file1.txt file2.txt
   ```

## Tips
- নিশ্চিত করুন যে আপনি কপি করার আগে গন্তব্যের অবস্থান সঠিকভাবে চেক করেছেন।
- `-i` অপশন ব্যবহার করা ভাল, যাতে ভুলবশত ফাইল ওভাররাইট না হয়।
- বড় ফাইল বা ডিরেক্টরি কপি করার সময়, `-v` অপশন ব্যবহার করে প্রক্রিয়া পর্যবেক্ষণ করুন।
