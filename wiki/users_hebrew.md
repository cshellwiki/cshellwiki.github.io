# [לינוקס] Bash users שימושים: הצגת מידע על משתמשים במערכת

## Overview
הפקודה `users` משמשת להצגת שמות המשתמשים המחוברים כעת למערכת. היא מספקת מידע פשוט ויעיל על מי נמצא במערכת בזמן אמת.

## Usage
התחביר הבסיסי של הפקודה הוא:

```bash
users [options] [arguments]
```

## Common Options
- `-n`: מציג את מספר המשתמשים המחוברים.
- `-r`: מציג את שמות המשתמשים המחוברים עם מספרי מזהה (UID).

## Common Examples
הנה כמה דוגמאות לשימוש בפקודת `users`:

1. **הצגת שמות המשתמשים המחוברים:**
   ```bash
   users
   ```

2. **הצגת מספר המשתמשים המחוברים:**
   ```bash
   users -n
   ```

3. **הצגת שמות המשתמשים עם UID:**
   ```bash
   users -r
   ```

## Tips
- השתמש בפקודה `who` כדי לקבל מידע מפורט יותר על המשתמשים המחוברים, כולל זמני כניסה ומידע על המכשירים.
- אם אתה מנהל מערכת, תוכל לשלב את הפקודה `users` עם פקודות נוספות כדי לנתח את השימוש במערכת בצורה טובה יותר.
