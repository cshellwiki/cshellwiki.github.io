# [লিনাক্স] Bash groups ব্যবহার: ব্যবহারকারীর গ্রুপ তথ্য দেখানো

## Overview
`groups` কমান্ডটি ব্যবহারকারীর গ্রুপ সদস্যপদ সম্পর্কিত তথ্য প্রদর্শন করে। এটি একটি নির্দিষ্ট ব্যবহারকারীর জন্য বা বর্তমান লগ ইন করা ব্যবহারকারীর জন্য গ্রুপগুলির তালিকা দেখায়।

## Usage
`groups` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
groups [options] [arguments]
```

## Common Options
- `-h`, `--help`: সাহায্য তথ্য দেখায়।
- `-v`, `--version`: কমান্ডের সংস্করণ তথ্য দেখায়।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হল:

1. বর্তমান ব্যবহারকারীর গ্রুপগুলি দেখানোর জন্য:
   ```bash
   groups
   ```

2. নির্দিষ্ট ব্যবহারকারীর গ্রুপগুলি দেখানোর জন্য:
   ```bash
   groups username
   ```

3. সাহায্য তথ্য দেখানোর জন্য:
   ```bash
   groups --help
   ```

## Tips
- `groups` কমান্ডটি ব্যবহার করে আপনি সহজেই জানতে পারবেন কোন গ্রুপগুলির সদস্য আপনি।
- যদি আপনি একটি নির্দিষ্ট ব্যবহারকারীর গ্রুপ দেখতে চান, নিশ্চিত হন যে আপনি সঠিক ব্যবহারকারীর নাম ব্যবহার করছেন।
- গ্রুপ সদস্যপদ পরিবর্তন হলে, `groups` কমান্ডটি পুনরায় চালিয়ে নতুন তথ্য যাচাই করুন।
