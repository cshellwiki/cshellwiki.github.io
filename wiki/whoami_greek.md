# [Λειτουργικό Σύστημα] Bash whoami Χρήση: [Εμφάνιση του ονόματος χρήστη]

## Overview
Η εντολή `whoami` χρησιμοποιείται για να εμφανίσει το όνομα του τρέχοντος χρήστη που είναι συνδεδεμένος στο σύστημα. Είναι χρήσιμη για να επιβεβαιώσετε ποιος είστε όταν εργάζεστε σε ένα περιβάλλον πολλαπλών χρηστών.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```bash
whoami [options] [arguments]
```

## Common Options
- `--help`: Εμφανίζει πληροφορίες βοήθειας για την εντολή.
- `--version`: Εμφανίζει την έκδοση της εντολής `whoami`.

## Common Examples
### Παράδειγμα 1: Απλή χρήση
Για να δείτε το όνομα του τρέχοντος χρήστη, απλά πληκτρολογήστε:

```bash
whoami
```

### Παράδειγμα 2: Χρήση με επιλογές
Για να δείτε την έκδοση της εντολής, μπορείτε να χρησιμοποιήσετε:

```bash
whoami --version
```

### Παράδειγμα 3: Βοήθεια
Για να λάβετε βοήθεια σχετικά με την εντολή, πληκτρολογήστε:

```bash
whoami --help
```

## Tips
- Χρησιμοποιήστε την εντολή `whoami` για να επιβεβαιώσετε ότι εργάζεστε με τον σωστό λογαριασμό χρήστη, ειδικά σε συστήματα με πολλούς χρήστες.
- Συνδυάστε την `whoami` με άλλες εντολές για να δημιουργήσετε πιο σύνθετα σενάρια, όπως η καταγραφή δραστηριοτήτων με το όνομα χρήστη.
