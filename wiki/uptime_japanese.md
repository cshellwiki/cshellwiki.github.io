# [Linux] Bash uptime の使い方: システムの稼働時間を表示

## Overview
`uptime` コマンドは、システムの稼働時間、現在のユーザー数、およびシステムの負荷平均を表示します。この情報は、サーバーのパフォーマンスを監視する際に非常に役立ちます。

## Usage
基本的な構文は以下の通りです。

```
uptime [options]
```

## Common Options
- `-p` : システムの稼働時間を人間が読みやすい形式で表示します。
- `-s` : システムが最後に起動した日時を表示します。

## Common Examples
以下にいくつかの実用的な例を示します。

1. シンプルに稼働時間を表示する:
   ```bash
   uptime
   ```

2. 人間が読みやすい形式で稼働時間を表示する:
   ```bash
   uptime -p
   ```

3. システムの起動日時を表示する:
   ```bash
   uptime -s
   ```

## Tips
- 定期的に `uptime` コマンドを実行して、システムの負荷を監視しましょう。
- スクリプトに組み込むことで、システムの状態をログに記録することもできます。
- `uptime` の出力を他のコマンドと組み合わせて、より詳細な分析を行うことが可能です。
