# [Linux] Bash w χρήση: εμφάνιση χρηστών και δραστηριοτήτων

## Overview
Η εντολή `w` χρησιμοποιείται για να εμφανίζει πληροφορίες σχετικά με τους συνδεδεμένους χρήστες και τις δραστηριότητές τους στο σύστημα. Παρέχει μια γρήγορη εικόνα του ποιοι είναι συνδεδεμένοι, τι κάνουν και πόσο καιρό είναι συνδεδεμένοι.

## Usage
Η βασική σύνταξη της εντολής είναι:

```bash
w [options] [arguments]
```

## Common Options
- `-h`: Αφαιρεί την κεφαλίδα από την έξοδο.
- `-s`: Εμφανίζει λιγότερες πληροφορίες, με συμπυκνωμένη μορφή.
- `-f`: Εμφανίζει τη διεύθυνση IP των χρηστών.
- `-u`: Εμφανίζει την τελευταία δραστηριότητα των χρηστών.

## Common Examples

### Παράδειγμα 1: Βασική χρήση
Για να δείτε τους συνδεδεμένους χρήστες και τις δραστηριότητές τους, απλά πληκτρολογήστε:

```bash
w
```

### Παράδειγμα 2: Χρήση με επιλογή -h
Για να δείτε την έξοδο χωρίς κεφαλίδα:

```bash
w -h
```

### Παράδειγμα 3: Χρήση με επιλογή -s
Για να δείτε μια πιο συμπυκνωμένη μορφή της εξόδου:

```bash
w -s
```

### Παράδειγμα 4: Χρήση με επιλογή -f
Για να δείτε τη διεύθυνση IP των χρηστών:

```bash
w -f
```

## Tips
- Χρησιμοποιήστε την εντολή `w` σε τακτική βάση για να παρακολουθείτε τη δραστηριότητα των χρηστών στο σύστημα.
- Συνδυάστε την εντολή `w` με άλλες εντολές, όπως `grep`, για να φιλτράρετε συγκεκριμένους χρήστες.
- Μάθετε τις επιλογές που σας ενδιαφέρουν καλύτερα για να προσαρμόσετε την έξοδο σύμφωνα με τις ανάγκες σας.
