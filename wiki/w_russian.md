# [Linux] Bash w: [показать пользователей и их активность]

## Обзор
Команда `w` в Bash используется для отображения информации о текущих пользователях, которые вошли в систему, а также о том, что они делают. Она предоставляет данные о времени входа, активности и загрузке системы.

## Использование
Основной синтаксис команды выглядит следующим образом:

```
w [опции] [аргументы]
```

## Общие параметры
- `-h` — отключает заголовок в выводе.
- `-s` — выводит сокращённый формат, убирая некоторые колонки.
- `-u` — показывает только пользователей, которые активно используют систему.

## Общие примеры

1. **Основной вывод информации о пользователях:**
   ```bash
   w
   ```

2. **Вывод без заголовка:**
   ```bash
   w -h
   ```

3. **Сокращённый формат:**
   ```bash
   w -s
   ```

4. **Показать только активных пользователей:**
   ```bash
   w -u
   ```

## Советы
- Используйте `w` для мониторинга активности пользователей в многопользовательских системах.
- Команда может быть полезна для администраторов для выявления неактивных сессий.
- Регулярно проверяйте активность пользователей, чтобы оптимизировать ресурсы системы.
