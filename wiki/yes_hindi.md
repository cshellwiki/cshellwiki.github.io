# [Linux] Bash yes उपयोग: निरंतर 'yes' प्रिंट करना

## Overview
`yes` कमांड एक साधारण टूल है जो निरंतर "yes" शब्द को आउटपुट करता है। इसका मुख्य उपयोग अन्य कमांड के साथ पाइपिंग के लिए किया जाता है, जहाँ आपको लगातार "yes" इनपुट की आवश्यकता होती है।

## Usage
`yes` कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
yes [options] [arguments]
```

## Common Options
- `-n` : एक विशेष स्ट्रिंग को आउटपुट करने के लिए। 
- `--help` : मदद संदेश दिखाता है।
- `--version` : संस्करण जानकारी दिखाता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. **साधारण उपयोग**:
   ```bash
   yes
   ```
   यह कमांड लगातार "yes" प्रिंट करेगा।

2. **विशेष स्ट्रिंग का उपयोग**:
   ```bash
   yes "Hello, World!"
   ```
   यह "Hello, World!" को लगातार प्रिंट करेगा।

3. **पाइपिंग के साथ उपयोग**:
   ```bash
   yes | head -n 5
   ```
   यह पहले 5 "yes" आउटपुट करेगा।

4. **फाइल में लिखना**:
   ```bash
   yes "Data" > output.txt
   ```
   यह "Data" को output.txt फाइल में निरंतर लिखेगा।

## Tips
- `yes` का उपयोग तब करें जब आपको किसी कमांड के लिए लगातार "yes" इनपुट की आवश्यकता हो।
- ध्यान रखें कि `yes` कमांड अनंत रूप से आउटपुट करता है, इसलिए इसे सावधानी से उपयोग करें।
- यदि आप किसी विशेष स्ट्रिंग का उपयोग कर रहे हैं, तो सुनिश्चित करें कि वह स्ट्रिंग आपके उपयोग के लिए उपयुक्त है।
