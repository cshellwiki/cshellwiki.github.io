# [Linux] Bash userdel <Utilizarea echivalentă>: [şterge utilizatori]

## Overview
คำสั่ง `userdel` ใช้สำหรับลบผู้ใช้ในระบบ Linux โดยจะลบข้อมูลผู้ใช้ที่เกี่ยวข้องจากฐานข้อมูลผู้ใช้ในระบบ

## Usage
คำสั่ง `userdel` มีรูปแบบพื้นฐานดังนี้:

```bash
userdel [options] [username]
```

## Common Options
- `-r`: ลบไดเรกทอรีบ้านของผู้ใช้และไฟล์ที่เกี่ยวข้อง
- `-f`: บังคับลบผู้ใช้แม้ว่าผู้ใช้นั้นจะเข้าสู่ระบบอยู่
- `-Z`: ลบ SELinux user mapping ของผู้ใช้

## Common Examples
- ลบผู้ใช้ที่ชื่อว่า `john`:
```bash
userdel john
```

- ลบผู้ใช้และไดเรกทอรีบ้านของเขา:
```bash
userdel -r john
```

- บังคับลบผู้ใช้ที่ชื่อว่า `mary`:
```bash
userdel -f mary
```

## Tips
- ควรตรวจสอบว่าผู้ใช้ที่ต้องการลบไม่ได้เข้าสู่ระบบอยู่ก่อนทำการลบ
- ใช้ตัวเลือก `-r` เพื่อให้แน่ใจว่าไฟล์ที่เกี่ยวข้องกับผู้ใช้จะถูกลบออกด้วย
- ควรทำการสำรองข้อมูลก่อนที่จะลบผู้ใช้เพื่อป้องกันการสูญหายของข้อมูลที่สำคัญ
