# [লিনাক্স] Bash free ব্যবহার: মেমোরি ব্যবহারের তথ্য দেখার জন্য

## Overview
`free` কমান্ডটি লিনাক্স সিস্টেমে মেমোরি ব্যবহারের তথ্য প্রদর্শন করে। এটি RAM এবং Swap মেমোরির অবস্থা সম্পর্কে সংক্ষিপ্ত তথ্য সরবরাহ করে, যা সিস্টেমের কর্মক্ষমতা বিশ্লেষণে সহায়ক।

## Usage
`free` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
free [options]
```

## Common Options
- `-h`: মানব-পঠনযোগ্য ফরম্যাটে আউটপুট দেখায় (যেমন KB, MB, GB)।
- `-m`: মেমোরি তথ্য মেগাবাইটে দেখায়।
- `-g`: মেমোরি তথ্য গিগাবাইটে দেখায়।
- `-s <seconds>`: নির্দিষ্ট সময় অন্তর (সেকেন্ডে) আউটপুট আপডেট করে।
- `-t`: মোট মেমোরি এবং ব্যবহৃত মেমোরির তথ্য দেখায়।

## Common Examples

### 1. সাধারণ তথ্য দেখানো
```bash
free
```

### 2. মানব-পঠনযোগ্য ফরম্যাটে তথ্য দেখানো
```bash
free -h
```

### 3. মেগাবাইটে তথ্য দেখানো
```bash
free -m
```

### 4. গিগাবাইটে তথ্য দেখানো
```bash
free -g
```

### 5. প্রতি 5 সেকেন্ডে তথ্য আপডেট করা
```bash
free -s 5
```

### 6. মোট মেমোরি সহ তথ্য দেখানো
```bash
free -t
```

## Tips
- `free -h` ব্যবহার করা সর্বদা ভালো, কারণ এটি তথ্যকে আরও সহজে বোঝার জন্য উপযোগী করে।
- নিয়মিতভাবে `free` কমান্ড চালিয়ে সিস্টেমের মেমোরি ব্যবহারের উপর নজর রাখা ভালো।
- যদি আপনার সিস্টেমে Swap মেমোরি ব্যবহার হচ্ছে, তবে এটি সিস্টেমের কর্মক্ষমতা সম্পর্কে গুরুত্বপূর্ণ তথ্য দিতে পারে।
