# [لينكس] أوامر Bash للمستخدمين: إدارة المستخدمين على النظام

## Overview
أمر `users` يُستخدم لعرض أسماء المستخدمين الذين قاموا بتسجيل الدخول حاليًا إلى النظام. هذا الأمر مفيد لمراقبة النشاطات الحالية على النظام ومعرفة من يستخدمه في الوقت الفعلي.

## Usage
الصيغة الأساسية للأمر هي:

```bash
users [options]
```

## Common Options
- لا توجد خيارات شائعة مع أمر `users`، حيث أنه أمر بسيط يقوم فقط بعرض أسماء المستخدمين.

## Common Examples
إليك بعض الأمثلة العملية لاستخدام الأمر `users`:

1. **عرض أسماء المستخدمين المتصلين حاليًا:**
   ```bash
   users
   ```

2. **استخدام الأمر مع `echo` لطباعة أسماء المستخدمين:**
   ```bash
   echo "المستخدمون المتصلون: $(users)"
   ```

3. **دمج الأمر مع `wc` لحساب عدد المستخدمين المتصلين:**
   ```bash
   users | wc -w
   ```

## Tips
- استخدم الأمر `who` للحصول على معلومات أكثر تفصيلاً حول المستخدمين المتصلين، مثل وقت تسجيل الدخول.
- يمكن دمج الأمر `users` مع أوامر أخرى في Bash للحصول على معلومات إضافية، مثل استخدام `grep` لتصفية النتائج.
- تأكد من استخدام الأمر في بيئة مناسبة، حيث أن النتائج قد تختلف حسب عدد المستخدمين المتصلين في الوقت الحالي.
