# [לינוקס] Bash break שימוש: לסיים לולאה

## Overview
הפקודה `break` ב-Bash משמשת לסיים לולאה מוקדם. כאשר `break` מופעל בתוך לולאה, הוא מפסיק את ביצוע הלולאה הנוכחית וממשיך עם הפקודות הבאות לאחר הלולאה.

## Usage
התחביר הבסיסי של הפקודה הוא:
```
break [options]
```

## Common Options
- `n`: מספר הלולאה שברצונך לסיים. ברירת המחדל היא 1, כלומר תסיים את הלולאה הפנימית ביותר.

## Common Examples

### דוגמה 1: סיום לולאת for
```bash
for i in {1..5}; do
  if [ $i -eq 3 ]; then
    break
  fi
  echo $i
done
```
במקרה זה, הפלט יהיה:
```
1
2
```
הלולאה תפסיק לפעול כאשר `i` שווה ל-3.

### דוגמה 2: סיום לולאת while
```bash
count=1
while [ $count -le 5 ]; do
  if [ $count -eq 4 ]; then
    break
  fi
  echo $count
  ((count++))
done
```
הפלט יהיה:
```
1
2
3
```
הלולאה תפסיק לפעול כאשר `count` שווה ל-4.

### דוגמה 3: סיום לולאת until
```bash
num=0
until [ $num -ge 3 ]; do
  echo $num
  ((num++))
  if [ $num -eq 2 ]; then
    break
  fi
done
```
הפלט יהיה:
```
0
1
```
הלולאה תפסיק לפעול כאשר `num` שווה ל-2.

## Tips
- השתמש ב-`break` כאשר אתה רוצה לשלוט על זרימת הלולאות שלך ולמנוע ביצוע מיותר של פקודות.
- אם אתה עובד עם לולאות מקוננות, תוכל להשתמש באופציה `n` כדי לציין איזו לולאה לסיים.
- תמיד כדאי לבדוק את התנאים שלך לפני השימוש ב-`break` כדי למנוע סיום לא צפוי של הלולאות.
