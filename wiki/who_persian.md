# [لینوکس] Bash who معادل استفاده: نمایش کاربران وارد شده به سیستم

## Overview
دستورات `who` برای نمایش لیست کاربرانی که در حال حاضر به سیستم وارد شده‌اند، استفاده می‌شود. این دستور اطلاعاتی مانند نام کاربری، زمان ورود و ترمینالی که کاربر از آن استفاده می‌کند را نمایش می‌دهد.

## Usage
سینتکس پایه دستور `who` به صورت زیر است:

```bash
who [options] [arguments]
```

## Common Options
- `-a`: نمایش تمام اطلاعات مربوط به کاربران.
- `-b`: نمایش زمان آخرین بوت سیستم.
- `-q`: فقط نمایش نام‌های کاربری و تعداد کاربران وارد شده.
- `--help`: نمایش راهنمای استفاده از دستور.

## Common Examples
- نمایش لیست کاربران وارد شده:
  ```bash
  who
  ```

- نمایش تمام اطلاعات مربوط به کاربران:
  ```bash
  who -a
  ```

- نمایش زمان آخرین بوت سیستم:
  ```bash
  who -b
  ```

- نمایش فقط نام‌های کاربری و تعداد کاربران:
  ```bash
  who -q
  ```

## Tips
- برای مشاهده اطلاعات دقیق‌تر در مورد کاربران، می‌توانید از گزینه `-a` استفاده کنید.
- اگر به دنبال زمان آخرین بوت سیستم هستید، گزینه `-b` بسیار مفید است.
- برای بررسی سریع تعداد کاربران وارد شده، گزینه `-q` را امتحان کنید.
