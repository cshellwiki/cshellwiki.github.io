# [سیستم‌عامل] C Shell (csh) who معادل استفاده: نمایش کاربران وارد شده

## Overview
دستورات `who` در C Shell (csh) برای نمایش لیستی از کاربرانی که در حال حاضر وارد سیستم شده‌اند، استفاده می‌شود. این دستور اطلاعاتی مانند نام کاربری، زمان ورود و ترمینال مورد استفاده را نمایش می‌دهد.

## Usage
سینتکس پایه دستور `who` به شکل زیر است:

```csh
who [options] [arguments]
```

## Common Options
- `-a`: نمایش تمامی اطلاعات مربوط به کاربران، شامل زمان ورود و وضعیت.
- `-b`: نمایش زمان آخرین راه‌اندازی سیستم.
- `-q`: نمایش فقط نام کاربران و تعداد آنها.
- `--help`: نمایش راهنمای استفاده از دستور.

## Common Examples
1. نمایش لیست کاربران وارد شده:
   ```csh
   who
   ```

2. نمایش تمامی اطلاعات مربوط به کاربران:
   ```csh
   who -a
   ```

3. نمایش زمان آخرین راه‌اندازی سیستم:
   ```csh
   who -b
   ```

4. نمایش فقط نام کاربران و تعداد آنها:
   ```csh
   who -q
   ```

## Tips
- برای مشاهده اطلاعات دقیق‌تر، می‌توانید از گزینه `-a` استفاده کنید.
- اگر فقط به نام کاربران نیاز دارید، گزینه `-q` می‌تواند مفید باشد.
- این دستور می‌تواند در اسکریپت‌های مدیریتی برای نظارت بر کاربران فعال استفاده شود.
