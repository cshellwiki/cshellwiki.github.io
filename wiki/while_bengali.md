# [লিনাক্স] Bash while ব্যবহার: একটি লুপের মাধ্যমে কার্যক্রম চালানো

## Overview
`while` কমান্ডটি একটি লুপ তৈরি করতে ব্যবহৃত হয় যা একটি নির্দিষ্ট শর্ত সত্য থাকাকালীন কার্যক্রম চালায়। এটি সাধারণত স্ক্রিপ্টিংয়ে ব্যবহৃত হয় যেখানে একটি কাজ বারবার সম্পন্ন করতে হয় যতক্ষণ না শর্তটি মিথ্যা হয়।

## Usage
`while` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
while [condition]
do
    [commands]
done
```

## Common Options
`while` কমান্ডের জন্য সাধারণ অপশনগুলি হলো:
- `[condition]`: এটি একটি শর্ত যা সত্য হলে লুপটি চালানো হবে।
- `[commands]`: এই অংশে আপনি যে সমস্ত কমান্ড চালাতে চান তা উল্লেখ করবেন।

## Common Examples

### উদাহরণ 1: সংখ্যা মুদ্রণ
```bash
count=1
while [ $count -le 5 ]
do
    echo $count
    ((count++))
done
```
এই উদাহরণে, ১ থেকে ৫ পর্যন্ত সংখ্যা মুদ্রণ করা হবে।

### উদাহরণ 2: ব্যবহারকারীর ইনপুট নেওয়া
```bash
input=""
while [ "$input" != "exit" ]
do
    read -p "Type 'exit' to quit: " input
done
```
এখানে, ব্যবহারকারী 'exit' টাইপ না করা পর্যন্ত লুপটি চলতে থাকবে।

### উদাহরণ 3: ফাইলের লাইন পড়া
```bash
while IFS= read -r line
do
    echo "$line"
done < filename.txt
```
এই উদাহরণে, `filename.txt` ফাইলের প্রতিটি লাইন পড়া হবে এবং মুদ্রণ করা হবে।

## Tips
- শর্তটি যতটা সম্ভব সহজ রাখুন যাতে লুপটি দ্রুত সম্পন্ন হয়।
- লুপের ভিতরে একটি `sleep` কমান্ড ব্যবহার করলে CPU ব্যবহার কমতে পারে।
- লুপের শেষে একটি `break` কমান্ড ব্যবহার করে প্রয়োজনীয় সময়ে লুপটি বন্ধ করতে পারেন।
