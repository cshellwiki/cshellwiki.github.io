# [لينكس] Bash usermod الاستخدام: إدارة خصائص المستخدمين

## Overview
يستخدم الأمر `usermod` لتعديل خصائص المستخدمين في نظام لينكس. يمكن من خلاله تغيير معلومات مثل اسم المستخدم، مجموعة المستخدم، وبيانات أخرى متعلقة بالحساب.

## Usage
تكون الصيغة الأساسية للأمر كما يلي:

```bash
usermod [options] [arguments]
```

## Common Options
- `-l, --login NEW_LOGIN`: تغيير اسم المستخدم إلى الاسم الجديد المحدد.
- `-d, --home NEW_HOME`: تغيير مسار الدليل الرئيسي للمستخدم.
- `-g, --gid GROUP`: تغيير مجموعة المستخدم الأساسية.
- `-a, --append`: إضافة المستخدم إلى مجموعة دون إزالة من المجموعات الأخرى.
- `-s, --shell SHELL`: تغيير قشرة المستخدم.

## Common Examples
- تغيير اسم المستخدم:
```bash
usermod -l newusername oldusername
```

- تغيير الدليل الرئيسي للمستخدم:
```bash
usermod -d /new/home/directory username
```

- إضافة مستخدم إلى مجموعة:
```bash
usermod -a -G groupname username
```

- تغيير قشرة المستخدم:
```bash
usermod -s /bin/bash username
```

## Tips
- تأكد من أنك تمتلك صلاحيات الجذر (root) أو أنك مستخدم يمتلك صلاحيات كافية لتنفيذ الأمر.
- استخدم الخيار `-l` بحذر، حيث يمكن أن يؤثر على ملفات المستخدم.
- تحقق من التغييرات باستخدام الأمر `id username` بعد تنفيذ `usermod` للتأكد من أن التعديلات تمت بنجاح.
