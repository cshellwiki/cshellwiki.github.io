# [Linux] Bash vgcreate Использование: Создание группы томов

## Обзор
Команда `vgcreate` используется для создания группы томов (VG) в системе управления логическими томами (LVM). Группа томов объединяет несколько физических томов (PV) в единое логическое пространство, что упрощает управление дисковым пространством.

## Использование
Базовый синтаксис команды выглядит следующим образом:

```
vgcreate [options] [имя_группы_томов] [физические_тома]
```

## Общие параметры
- `-s, --stripes <количество>`: Указывает количество полос для создания.
- `-p, --stripesize <размер>`: Устанавливает размер полосы в байтах.
- `-f, --force`: Принудительное создание группы томов, даже если некоторые физические тома уже используются.

## Общие примеры
1. Создание группы томов с именем `myvg` из физических томов `/dev/sda1` и `/dev/sdb1`:

   ```bash
   vgcreate myvg /dev/sda1 /dev/sdb1
   ```

2. Создание группы томов с указанием количества полос:

   ```bash
   vgcreate -s 64k myvg /dev/sda1 /dev/sdb1
   ```

3. Принудительное создание группы томов:

   ```bash
   vgcreate -f myvg /dev/sda1 /dev/sdb1
   ```

## Советы
- Перед созданием группы томов убедитесь, что физические тома не содержат данных, которые вам нужны, так как они могут быть перезаписаны.
- Используйте команду `vgs` после создания группы томов, чтобы проверить, была ли она успешно создана.
- Регулярно создавайте резервные копии данных, особенно перед изменениями в конфигурации LVM.
