# [লিনাক্স] Bash vmstat ব্যবহার: সিস্টেমের কর্মক্ষমতা পর্যবেক্ষণ

## Overview
`vmstat` কমান্ডটি সিস্টেমের ভার্চুয়াল মেমরি, প্রসেস, এবং CPU ব্যবহারের তথ্য প্রদর্শন করে। এটি সিস্টেমের কর্মক্ষমতা বিশ্লেষণ করতে সাহায্য করে এবং বিভিন্ন সিস্টেমের অবস্থা সম্পর্কে ধারণা দেয়।

## Usage
`vmstat` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
vmstat [options] [arguments]
```

## Common Options
- `-a`: সমস্ত মেমরি তথ্য দেখায়, যেমন অ্যাক্টিভ এবং ইনঅ্যাক্টিভ মেমরি।
- `-m`: মেমরি ব্যবহার সম্পর্কিত তথ্য দেখায়।
- `-s`: সিস্টেমের বিভিন্ন পরিসংখ্যানের সারসংক্ষেপ দেখায়।
- `-d`: ব্লক ডিভাইসের তথ্য দেখায়।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হল:

1. সিস্টেমের বর্তমান কর্মক্ষমতা দেখার জন্য:
   ```bash
   vmstat
   ```

2. প্রতি 2 সেকেন্ড অন্তর তথ্য আপডেট করার জন্য:
   ```bash
   vmstat 2
   ```

3. মেমরি ব্যবহারের বিস্তারিত তথ্য দেখানোর জন্য:
   ```bash
   vmstat -a
   ```

4. সিস্টেমের পরিসংখ্যানের সারসংক্ষেপ দেখানোর জন্য:
   ```bash
   vmstat -s
   ```

## Tips
- `vmstat` ব্যবহার করার সময়, সময়সীমা উল্লেখ করা হলে এটি সিস্টেমের কর্মক্ষমতা পরিবর্তনের গতিশীলতা বুঝতে সাহায্য করে।
- নিয়মিতভাবে `vmstat` চালানো হলে, সিস্টেমের দীর্ঘমেয়াদী প্রবণতা বিশ্লেষণ করা সহজ হয়।
- অন্যান্য সিস্টেম মনিটরিং টুলের সাথে মিলিয়ে `vmstat` ব্যবহার করলে আরও বিস্তারিত তথ্য পাওয়া যায়।
