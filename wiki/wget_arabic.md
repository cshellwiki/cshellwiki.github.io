# [لينكس] Bash wget الاستخدام: تحميل الملفات من الإنترنت

## Overview
يعتبر أمر `wget` أداة قوية ومفيدة لتحميل الملفات من الإنترنت. يمكن استخدامه لتنزيل الملفات بشكل غير تفاعلي، مما يعني أنه يمكن تشغيله في الخلفية دون الحاجة إلى التفاعل المباشر مع المستخدم. يدعم `wget` بروتوكولات HTTP وHTTPS وFTP، مما يجعله خيارًا شائعًا لتحميل المحتوى من الويب.

## Usage
يمكن استخدام الأمر `wget` بالشكل التالي:

```bash
wget [options] [arguments]
```

## Common Options
- `-O [filename]`: يحدد اسم الملف الذي سيتم حفظ المحتوى فيه.
- `-c`: يستأنف تحميل ملف تم إيقافه سابقًا.
- `-q`: يعمل في وضع هادئ (quiet mode) دون عرض المعلومات على الشاشة.
- `--limit-rate=[amount]`: يحدد الحد الأقصى لسرعة التحميل.
- `-r`: يقوم بتحميل الملفات بشكل متكرر (recursive) من موقع ويب.

## Common Examples
- لتحميل ملف واحد:
```bash
wget https://example.com/file.zip
```

- لتحميل ملف مع تحديد اسم مختلف:
```bash
wget -O myfile.zip https://example.com/file.zip
```

- لاستئناف تحميل ملف تم إيقافه:
```bash
wget -c https://example.com/file.zip
```

- لتحميل موقع ويب بالكامل:
```bash
wget -r https://example.com
```

- لتحديد سرعة التحميل:
```bash
wget --limit-rate=200k https://example.com/file.zip
```

## Tips
- تأكد من استخدام الخيار `-c` إذا كنت تحتاج إلى استئناف تحميل الملفات الكبيرة.
- استخدم الخيار `-q` إذا كنت ترغب في تقليل الرسائل المعروضة أثناء التحميل.
- تحقق من الروابط قبل التحميل، خاصة إذا كنت تقوم بتحميل محتوى من مواقع غير موثوقة.
