# [리눅스] Bash df 사용법: 디스크 공간 확인

## Overview
`df` 명령어는 파일 시스템의 디스크 공간 사용량을 확인하는 데 사용됩니다. 이 명령어를 통해 각 파일 시스템의 총 용량, 사용 중인 용량, 남은 용량 및 마운트 지점을 쉽게 확인할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:
```
df [옵션] [인수]
```

## Common Options
- `-h`: 사람이 읽기 쉬운 형식으로 출력합니다 (예: KB, MB, GB).
- `-T`: 파일 시스템의 유형을 함께 표시합니다.
- `-a`: 모든 파일 시스템을 표시합니다, 사용 중이지 않은 파일 시스템도 포함됩니다.
- `-i`: inode 사용량을 표시합니다.

## Common Examples
1. 기본 사용법:
   ```bash
   df
   ```

2. 사람이 읽기 쉬운 형식으로 출력:
   ```bash
   df -h
   ```

3. 파일 시스템 유형과 함께 출력:
   ```bash
   df -T
   ```

4. 모든 파일 시스템 표시:
   ```bash
   df -a
   ```

5. inode 사용량 확인:
   ```bash
   df -i
   ```

## Tips
- `df -h` 옵션을 사용하면 디스크 공간을 쉽게 이해할 수 있는 형식으로 확인할 수 있습니다.
- 정기적으로 `df` 명령어를 사용하여 디스크 공간을 모니터링하면 시스템의 성능을 유지하는 데 도움이 됩니다.
- 특정 마운트 지점의 사용량을 확인하려면 `df [마운트 지점]` 형식으로 명령어를 입력하세요.
