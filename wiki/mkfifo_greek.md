# [Λειτουργικό Σύστημα] Bash mkfifo Χρήση: Δημιουργία ονομάτων σωλήνων

## Overview
Η εντολή `mkfifo` χρησιμοποιείται για τη δημιουργία ειδικών αρχείων τύπου FIFO (First In, First Out), τα οποία επιτρέπουν την επικοινωνία μεταξύ διεργασιών στο Unix/Linux περιβάλλον. Αυτά τα αρχεία λειτουργούν ως σωλήνες, επιτρέποντας σε μία διεργασία να γράφει δεδομένα και σε άλλη να τα διαβάζει.

## Usage
Η βασική σύνταξη της εντολής `mkfifo` είναι η εξής:

```bash
mkfifo [options] [arguments]
```

## Common Options
- `-m, --mode=MODE`: Ορίζει τα δικαιώματα πρόσβασης για το νέο FIFO.
- `--help`: Εμφανίζει βοήθεια σχετικά με τη χρήση της εντολής.
- `--version`: Εμφανίζει την έκδοση της εντολής.

## Common Examples
### Δημιουργία ενός απλού FIFO
Για να δημιουργήσετε ένα FIFO με το όνομα `myfifo`, μπορείτε να χρησιμοποιήσετε την παρακάτω εντολή:

```bash
mkfifo myfifo
```

### Δημιουργία FIFO με συγκεκριμένα δικαιώματα
Για να δημιουργήσετε ένα FIFO με δικαιώματα 644, μπορείτε να χρησιμοποιήσετε την επιλογή `-m`:

```bash
mkfifo -m 644 myfifo
```

### Χρήση FIFO για επικοινωνία μεταξύ διεργασιών
Ακολουθεί ένα παράδειγμα όπου μία διεργασία γράφει σε ένα FIFO και μία άλλη το διαβάζει:

1. Σε ένα τερματικό, εκτελέστε:

   ```bash
   cat myfifo
   ```

2. Σε ένα άλλο τερματικό, εκτελέστε:

   ```bash
   echo "Hello, World!" > myfifo
   ```

Αυτό θα εμφανίσει "Hello, World!" στο πρώτο τερματικό.

## Tips
- Βεβαιωθείτε ότι οι διεργασίες που χρησιμοποιούν το FIFO είναι σωστά συγχρονισμένες για να αποφύγετε προβλήματα με την ανάγνωση και την εγγραφή.
- Χρησιμοποιήστε τα δικαιώματα πρόσβασης για να ελέγξετε ποιοι χρήστες μπορούν να διαβάσουν ή να γράψουν στο FIFO.
- Μην ξεχνάτε να διαγράφετε τα FIFO όταν δεν τα χρειάζεστε πια, χρησιμοποιώντας την εντολή `rm`.
