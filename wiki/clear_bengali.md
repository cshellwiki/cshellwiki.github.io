# [লিনাক্স] Bash clear ব্যবহার: স্ক্রীন পরিষ্কার করা

## Overview
`clear` কমান্ডটি টার্মিনালের স্ক্রীন পরিষ্কার করতে ব্যবহৃত হয়। এটি স্ক্রীনে প্রদর্শিত সমস্ত টেক্সট মুছে ফেলে এবং একটি পরিষ্কার পৃষ্ঠায় ফিরে আসে, যা ব্যবহারকারীর জন্য নতুন কমান্ডগুলি চালানোর সময় সুবিধাজনক।

## Usage
`clear` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
clear [options] [arguments]
```

## Common Options
- `-x` : স্ক্রীন পরিষ্কার করার সময় কার্সরকে উপরের বাম কোণে নিয়ে যায়।
- `-V` : কমান্ডের সংস্করণ তথ্য প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. **স্ক্রীন পরিষ্কার করা:**
   ```bash
   clear
   ```

2. **স্ক্রীন পরিষ্কার করে কার্সরকে উপরের বাম কোণে নিয়ে যাওয়া:**
   ```bash
   clear -x
   ```

3. **কমান্ডের সংস্করণ তথ্য দেখা:**
   ```bash
   clear -V
   ```

## Tips
- নিয়মিতভাবে স্ক্রীন পরিষ্কার করা আপনার কাজের পরিবেশকে আরও সংগঠিত রাখতে সাহায্য করে।
- যদি আপনি টার্মিনালে অনেক তথ্য দেখছেন এবং কিছু সময় পরে আবার পরিষ্কার করতে চান, তাহলে `Ctrl + L` শর্টকাট ব্যবহার করতে পারেন, যা `clear` কমান্ডের সমান।
- স্ক্রীন পরিষ্কার করার আগে গুরুত্বপূর্ণ তথ্য সংরক্ষণ করতে ভুলবেন না, কারণ এটি সমস্ত দৃশ্যমান টেক্সট মুছে ফেলবে।
