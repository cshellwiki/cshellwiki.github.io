# [लिनक्स] Bash @ उपयोग: फ़ाइलों और निर्देशिकाओं की सूची बनाना

## Overview
@ कमांड का उपयोग फ़ाइलों और निर्देशिकाओं की सूची बनाने के लिए किया जाता है। यह उपयोगकर्ताओं को अपने सिस्टम में मौजूद फ़ाइलों और फ़ोल्डरों को आसानी से देखने की सुविधा देता है।

## Usage
@ कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
@ [विकल्प] [आर्गुमेंट्स]
```

## Common Options
- `-l`: विस्तृत सूची प्रदर्शित करता है जिसमें फ़ाइलों के अधिकार, स्वामी, आकार आदि शामिल होते हैं।
- `-a`: सभी फ़ाइलें दिखाता है, जिसमें छिपी हुई फ़ाइलें भी शामिल हैं।
- `-h`: फ़ाइल के आकार को मानव-पठनीय प्रारूप में दिखाता है (जैसे KB, MB)।

## Common Examples
यहां कुछ सामान्य उदाहरण दिए गए हैं:

1. वर्तमान निर्देशिका में सभी फ़ाइलों और फ़ोल्डरों की सूची दिखाने के लिए:
   ```bash
   @ 
   ```

2. विस्तृत सूची के साथ फ़ाइलों को दिखाने के लिए:
   ```bash
   @ -l
   ```

3. सभी फ़ाइलें, जिसमें छिपी हुई फ़ाइलें भी शामिल हैं, दिखाने के लिए:
   ```bash
   @ -a
   ```

4. फ़ाइलों के आकार को मानव-पठनीय प्रारूप में दिखाने के लिए:
   ```bash
   @ -lh
   ```

## Tips
- हमेशा `-l` विकल्प का उपयोग करें जब आप फ़ाइलों के बारे में अधिक जानकारी प्राप्त करना चाहते हैं।
- छिपी हुई फ़ाइलों को देखने के लिए `-a` विकल्प का उपयोग करें, जैसे कि `.bashrc`।
- फ़ाइलों को क्रमबद्ध करने के लिए `-t` विकल्प का उपयोग करें, जिससे आप नवीनतम फ़ाइलें सबसे पहले देख सकें।
