# [Linux] Bash w उपयोग: उपयोगकर्ताओं की जानकारी दिखाना

## Overview
`w` कमांड का उपयोग वर्तमान में सिस्टम पर लॉग इन किए गए उपयोगकर्ताओं की जानकारी दिखाने के लिए किया जाता है। यह कमांड यह दर्शाता है कि कौन से उपयोगकर्ता सिस्टम पर हैं, वे क्या कर रहे हैं, और उनके लॉगिन समय के बारे में जानकारी प्रदान करता है।

## Usage
कमांड की मूल संरचना इस प्रकार है:
```
w [options] [arguments]
```

## Common Options
- `-h`: हेडर को छिपाता है।
- `-s`: संक्षिप्त आउटपुट दिखाता है।
- `-f`: फुल-पाथ नाम दिखाता है।
- `-u`: उपयोगकर्ता के नाम को दिखाता है।

## Common Examples
1. सभी लॉग इन उपयोगकर्ताओं की जानकारी देखने के लिए:
   ```bash
   w
   ```

2. संक्षिप्त आउटपुट के साथ जानकारी देखने के लिए:
   ```bash
   w -s
   ```

3. हेडर को छिपाते हुए जानकारी देखने के लिए:
   ```bash
   w -h
   ```

4. विशेष उपयोगकर्ता के लिए जानकारी देखने के लिए:
   ```bash
   w username
   ```

## Tips
- `w` कमांड का उपयोग करते समय, यह सुनिश्चित करें कि आप उचित अनुमतियों के साथ लॉग इन हैं, ताकि आप सभी उपयोगकर्ताओं की जानकारी देख सकें।
- यदि आप नियमित रूप से उपयोगकर्ताओं की गतिविधियों की निगरानी कर रहे हैं, तो आप `watch` कमांड के साथ `w` का उपयोग कर सकते हैं:
  ```bash
  watch w
  ```
- `w` कमांड का आउटपुट समझने में मदद के लिए, आप इसे `man w` कमांड के माध्यम से और अधिक विस्तार से पढ़ सकते हैं।
