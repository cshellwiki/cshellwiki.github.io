# [Linux] C Shell (csh) yum การใช้งาน: จัดการแพ็กเกจซอฟต์แวร์

## Overview
คำสั่ง `yum` (Yellowdog Updater, Modified) เป็นเครื่องมือที่ใช้ในการจัดการแพ็กเกจซอฟต์แวร์ในระบบปฏิบัติการ Linux ที่ใช้ RPM (Red Hat Package Manager) โดยช่วยให้ผู้ใช้สามารถติดตั้ง อัปเดต และลบแพ็กเกจซอฟต์แวร์ได้อย่างง่ายดาย

## Usage
รูปแบบพื้นฐานของคำสั่ง `yum` มีดังนี้:
```
yum [options] [arguments]
```

## Common Options
- `install` : ติดตั้งแพ็กเกจใหม่
- `remove` : ลบแพ็กเกจที่ติดตั้งอยู่
- `update` : อัปเดตแพ็กเกจที่ติดตั้งอยู่ให้เป็นเวอร์ชันล่าสุด
- `list` : แสดงรายการแพ็กเกจที่ติดตั้งและแพ็กเกจที่สามารถติดตั้งได้
- `search` : ค้นหาแพ็กเกจตามชื่อหรือคำอธิบาย

## Common Examples
- ติดตั้งแพ็กเกจ:
    ```bash
    yum install httpd
    ```
- ลบแพ็กเกจ:
    ```bash
    yum remove httpd
    ```
- อัปเดตแพ็กเกจ:
    ```bash
    yum update httpd
    ```
- แสดงรายการแพ็กเกจที่ติดตั้ง:
    ```bash
    yum list installed
    ```
- ค้นหาแพ็กเกจ:
    ```bash
    yum search nginx
    ```

## Tips
- ใช้คำสั่ง `yum clean all` เพื่อล้างแคชและพื้นที่เก็บข้อมูลที่ไม่จำเป็น
- ตรวจสอบการอัปเดตทั้งหมดโดยใช้ `yum update` เป็นประจำเพื่อให้ระบบของคุณปลอดภัยและทันสมัย
- ใช้ `yum info [package_name]` เพื่อดูรายละเอียดของแพ็กเกจก่อนติดตั้ง
