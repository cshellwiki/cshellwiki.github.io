# [לינוקס] Bash whoami שימוש: הצגת שם המשתמש הנוכחי

## Overview
הפקודה `whoami` משמשת להצגת שם המשתמש הנוכחי שמבצע את הפקודות במערכת. זהו כלי פשוט אך שימושי, במיוחד כאשר עובדים עם מספר חשבונות משתמשים או כאשר יש צורך לאמת את זהות המשתמש.

## Usage
התחביר הבסיסי של הפקודה הוא:

```
whoami [אפשרויות] [ארגומנטים]
```

## Common Options
- אין אפשרויות נוספות נפוצות לפקודה `whoami`. הפקודה פועלת בצורה פשוטה ומחזירה את שם המשתמש בלבד.

## Common Examples
להלן מספר דוגמאות לשימוש בפקודה `whoami`:

1. **הצגת שם המשתמש הנוכחי:**
   ```bash
   whoami
   ```

2. **שימוש בפקודה בתוך סקריפט:**
   ```bash
   echo "המשתמש הנוכחי הוא: $(whoami)"
   ```

3. **שילוב עם פקודות נוספות:**
   ```bash
   if [ "$(whoami)" = "root" ]; then
       echo "אתה מחובר כמנהל מערכת."
   fi
   ```

## Tips
- השתמש בפקודה `whoami` כדי לבדוק את שם המשתמש הנוכחי לפני ביצוע פעולות רגישות במערכת.
- ניתן לשלב את הפקודה `whoami` עם פקודות אחרות כדי לבצע בדיקות תנאים בסקריפטים.
- אם אתה עובד עם מספר חשבונות משתמשים, הקפד לבדוק את שם המשתמש הנוכחי כדי למנוע טעויות.
