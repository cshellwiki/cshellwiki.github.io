# [ระบบปฏิบัติการ] C Shell (csh) who การใช้งาน: แสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบ

## Overview
คำสั่ง `who` ใช้เพื่อแสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบในระบบปฏิบัติการ Unix หรือ Linux พร้อมกับข้อมูลเกี่ยวกับเวลาและสถานที่ที่ผู้ใช้เข้าสู่ระบบ

## Usage
รูปแบบพื้นฐานของคำสั่ง `who` คือ:

```
who [options] [arguments]
```

## Common Options
- `-b`: แสดงเวลาที่ระบบเริ่มทำงาน
- `-q`: แสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบและจำนวนผู้ใช้
- `-H`: แสดงหัวข้อสำหรับข้อมูลที่แสดง
- `--help`: แสดงข้อมูลช่วยเหลือเกี่ยวกับคำสั่ง

## Common Examples
1. แสดงรายชื่อผู้ใช้ทั้งหมดที่เข้าสู่ระบบ:
   ```csh
   who
   ```

2. แสดงเวลาที่ระบบเริ่มทำงาน:
   ```csh
   who -b
   ```

3. แสดงรายชื่อผู้ใช้และจำนวนผู้ใช้ที่เข้าสู่ระบบ:
   ```csh
   who -q
   ```

4. แสดงข้อมูลพร้อมหัวข้อ:
   ```csh
   who -H
   ```

## Tips
- ใช้คำสั่ง `who` ร่วมกับคำสั่ง `grep` เพื่อค้นหาผู้ใช้เฉพาะ:
  ```csh
  who | grep username
  ```
- คำสั่ง `who` สามารถใช้ในสคริปต์เพื่อการตรวจสอบสถานะของผู้ใช้ในระบบได้
- ตรวจสอบการเข้าสู่ระบบของผู้ใช้โดยใช้ `last` เพื่อดูประวัติการเข้าสู่ระบบก่อนหน้านี้
