# [Linux] Bash uname 使用方法: 显示系统信息

## 概述
`uname` 命令用于显示系统信息，包括操作系统名称、版本、内核版本等。它是了解当前系统环境的重要工具。

## 用法
基本语法如下：
```
uname [选项] [参数]
```

## 常用选项
- `-a`：显示所有可用的信息。
- `-s`：显示操作系统名称。
- `-n`：显示网络节点主机名。
- `-r`：显示内核版本。
- `-v`：显示内核版本信息。
- `-m`：显示机器硬件名称。
- `-p`：显示处理器类型（如果可用）。
- `-i`：显示硬件平台（如果可用）。
- `-o`：显示操作系统类型。

## 常见示例
以下是一些常用的 `uname` 命令示例：

1. 显示所有系统信息：
   ```bash
   uname -a
   ```

2. 显示操作系统名称：
   ```bash
   uname -s
   ```

3. 显示内核版本：
   ```bash
   uname -r
   ```

4. 显示机器硬件名称：
   ```bash
   uname -m
   ```

5. 显示网络节点主机名：
   ```bash
   uname -n
   ```

## 小贴士
- 使用 `uname -a` 可以快速获取系统的全面信息，适合初学者。
- 在脚本中使用 `uname` 可以帮助判断当前运行环境，便于进行条件判断。
- 结合其他命令使用，例如 `grep`，可以筛选出特定的信息：
  ```bash
  uname -a | grep Linux
  ```
