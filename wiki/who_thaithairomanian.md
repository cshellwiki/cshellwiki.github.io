# [ระบบปฏิบัติการ] Bash who การใช้งาน: แสดงข้อมูลผู้ใช้ที่เข้าสู่ระบบ

## Overview
คำสั่ง `who` ใช้เพื่อแสดงข้อมูลเกี่ยวกับผู้ใช้ที่เข้าสู่ระบบในระบบ Unix/Linux รวมถึงชื่อผู้ใช้, terminal ที่ใช้งาน, วันที่และเวลาที่เข้าสู่ระบบ และข้อมูลอื่น ๆ ที่เกี่ยวข้อง

## Usage
รูปแบบพื้นฐานของคำสั่งคือ:
```
who [options] [arguments]
```

## Common Options
- `-a`: แสดงข้อมูลทั้งหมดรวมถึงผู้ใช้ที่เข้าสู่ระบบและผู้ใช้ที่ไม่ใช้งาน
- `-b`: แสดงเวลาที่ระบบบูตล่าสุด
- `-q`: แสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบและจำนวนผู้ใช้ทั้งหมด
- `--help`: แสดงข้อมูลช่วยเหลือเกี่ยวกับคำสั่ง

## Common Examples
- แสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบ:
  ```bash
  who
  ```

- แสดงข้อมูลทั้งหมดรวมถึงผู้ใช้ที่ไม่ใช้งาน:
  ```bash
  who -a
  ```

- แสดงเวลาที่ระบบบูตล่าสุด:
  ```bash
  who -b
  ```

- แสดงรายชื่อผู้ใช้ที่เข้าสู่ระบบและจำนวนผู้ใช้ทั้งหมด:
  ```bash
  who -q
  ```

## Tips
- ใช้ `who` ร่วมกับคำสั่งอื่น เช่น `grep` เพื่อค้นหาผู้ใช้เฉพาะในผลลัพธ์
- คำสั่ง `who` สามารถใช้ในสคริปต์เพื่อเก็บข้อมูลเกี่ยวกับการใช้งานระบบ
- ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์ในการเข้าถึงข้อมูลผู้ใช้ในระบบเพื่อให้คำสั่งทำงานได้อย่างถูกต้อง
