# [리눅스] Bash uptime 사용법: 시스템 가동 시간 확인

## Overview
`uptime` 명령어는 시스템의 가동 시간, 현재 로그인한 사용자 수, 그리고 시스템의 평균 부하를 보여줍니다. 이 정보를 통해 시스템의 상태를 간단히 파악할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:

```
uptime [options]
```

## Common Options
- `-p`: 시스템의 가동 시간을 간단하게 표시합니다.
- `-s`: 시스템이 시작된 시간을 표시합니다.

## Common Examples
1. 기본 사용법:
   ```bash
   uptime
   ```

2. 간단한 가동 시간 표시:
   ```bash
   uptime -p
   ```

3. 시스템 시작 시간 확인:
   ```bash
   uptime -s
   ```

## Tips
- `uptime` 명령어는 주기적으로 시스템 상태를 모니터링할 때 유용합니다.
- 시스템의 부하를 확인하여 성능 문제를 조기에 발견할 수 있습니다.
- 스크립트에서 `uptime`을 사용하여 시스템의 가동 시간을 자동으로 기록할 수 있습니다.
