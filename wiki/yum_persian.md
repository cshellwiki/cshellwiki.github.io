# [لینوکس] Bash yum استفاده: مدیریت بسته‌ها در سیستم‌های مبتنی بر RPM

## Overview
دستور `yum` (Yellowdog Updater Modified) یک ابزار مدیریت بسته برای توزیع‌های لینوکسی است که از فرمت بسته RPM استفاده می‌کنند. این ابزار به کاربران اجازه می‌دهد تا بسته‌ها را نصب، حذف و به‌روزرسانی کنند و همچنین وابستگی‌های لازم را به‌طور خودکار مدیریت کند.

## Usage
سینتکس پایه دستور `yum` به شکل زیر است:

```
yum [options] [arguments]
```

## Common Options
- `install`: نصب یک یا چند بسته.
- `remove`: حذف یک یا چند بسته.
- `update`: به‌روزرسانی بسته‌های نصب‌شده به آخرین نسخه.
- `search`: جستجوی بسته‌ها بر اساس نام یا توضیحات.
- `info`: نمایش اطلاعات مربوط به یک بسته خاص.
- `list`: نمایش لیست بسته‌های موجود یا نصب‌شده.

## Common Examples
- نصب یک بسته:
  ```bash
  yum install httpd
  ```
  
- حذف یک بسته:
  ```bash
  yum remove httpd
  ```

- به‌روزرسانی بسته‌ها:
  ```bash
  yum update
  ```

- جستجوی بسته:
  ```bash
  yum search nginx
  ```

- نمایش اطلاعات یک بسته:
  ```bash
  yum info vim
  ```

- نمایش لیست بسته‌های نصب‌شده:
  ```bash
  yum list installed
  ```

## Tips
- همیشه قبل از به‌روزرسانی سیستم، از داده‌های مهم خود پشتیبان تهیه کنید.
- برای به‌روز نگه‌داشتن سیستم، به‌طور منظم از دستور `yum update` استفاده کنید.
- برای جستجوی بسته‌های خاص، از گزینه `search` استفاده کنید تا زمان خود را صرفه‌جویی کنید.
- از گزینه `info` برای بررسی جزئیات بسته‌ها قبل از نصب یا حذف آن‌ها استفاده کنید.
