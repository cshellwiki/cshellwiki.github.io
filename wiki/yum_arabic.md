# [لينكس] Bash yum الاستخدام: إدارة الحزم في نظام لينكس

## Overview
يعتبر أمر `yum` أداة قوية لإدارة الحزم في أنظمة تشغيل لينكس، حيث يُستخدم لتثبيت، تحديث، وإزالة البرمجيات. يعتمد `yum` على مستودعات الحزم، مما يسهل عملية إدارة البرمجيات بشكل فعال.

## Usage
تكون الصيغة الأساسية لأمر `yum` كالتالي:

```bash
yum [options] [arguments]
```

## Common Options
- `install`: لتثبيت حزمة جديدة.
- `remove`: لإزالة حزمة مثبتة.
- `update`: لتحديث الحزم المثبتة إلى أحدث إصدار.
- `search`: للبحث عن حزم معينة في المستودعات.
- `list`: لعرض الحزم المتاحة أو المثبتة.

## Common Examples
- لتثبيت حزمة جديدة:
```bash
yum install package_name
```

- لإزالة حزمة مثبتة:
```bash
yum remove package_name
```

- لتحديث جميع الحزم المثبتة:
```bash
yum update
```

- للبحث عن حزمة معينة:
```bash
yum search package_name
```

- لعرض جميع الحزم المثبتة:
```bash
yum list installed
```

## Tips
- تأكد من تحديث قاعدة بيانات الحزم بانتظام باستخدام `yum update` للحفاظ على أمان النظام.
- استخدم خيار `-y` لتجنب طلب التأكيد عند تثبيت أو إزالة الحزم:
```bash
yum install -y package_name
```
- تحقق من وجود الحزم المكررة أو غير المستخدمة باستخدام `yum autoremove` لتنظيف النظام.
