# [לינוקס] Bash xz שימוש: דחיסת קבצים

## Overview
הפקודה `xz` משמשת לדחיסת קבצים בפורמט XZ, שהיא טכניקת דחיסה מתקדמת המציעה יחס דחיסה גבוה. הפקודה יכולה לשמש גם לדחיסת קבצים וגם לפענוח קבצים דחוסים.

## Usage
התחביר הבסיסי של הפקודה הוא:
```bash
xz [options] [arguments]
```

## Common Options
- `-d`, `--decompress`: מפענח קובץ דחוס.
- `-k`, `--keep`: שומר על הקובץ המקורי לאחר הדחיסה.
- `-f`, `--force`: מכריח דחיסה או פענוח גם אם הקובץ קיים.
- `-9`: משתמש ברמת דחיסה מקסימלית (1-9).
- `-t`, `--test`: בודק אם הקובץ הדחוס תקין.

## Common Examples
- דחיסת קובץ:
```bash
xz file.txt
```
- דחיסת קובץ ושמירה על המקורי:
```bash
xz -k file.txt
```
- פענוח קובץ דחוס:
```bash
xz -d file.txt.xz
```
- בדיקת תקינות של קובץ דחוס:
```bash
xz -t file.txt.xz
```
- דחיסת קובץ עם רמת דחיסה מקסימלית:
```bash
xz -9 file.txt
```

## Tips
- השתמש באופציה `-k` אם אתה רוצה לשמור על הקובץ המקורי בזמן הדחיסה.
- אם אתה עובד עם קבצים גדולים, שקול להשתמש באופציה `-9` כדי להשיג יחס דחיסה טוב יותר, אך שים לב שזה עשוי לקחת יותר זמן.
- תמיד בדוק את תקינות הקבצים הדחוסים שלך עם האופציה `-t` כדי למנוע בעיות בעת הפענוח.
