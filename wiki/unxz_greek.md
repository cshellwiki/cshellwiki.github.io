# [Linux] Bash unxz Χρήση: Αποσυμπίεση αρχείων XZ

## Overview
Η εντολή `unxz` χρησιμοποιείται για την αποσυμπίεση αρχείων που έχουν συμπιεστεί με τη μέθοδο XZ. Είναι ένα εργαλείο που επιτρέπει στους χρήστες να επαναφέρουν τα δεδομένα στην αρχική τους μορφή από τα συμπιεσμένα αρχεία.

## Usage
Η βασική σύνταξη της εντολής `unxz` είναι η εξής:

```bash
unxz [options] [arguments]
```

## Common Options
- `-k`, `--keep`: Διατηρεί το αρχείο συμπιεσμένο μετά την αποσυμπίεση.
- `-f`, `--force`: Αναγκάζει την αποσυμπίεση, ακόμα και αν το αρχείο εξόδου υπάρχει ήδη.
- `-v`, `--verbose`: Εμφανίζει λεπτομέρειες σχετικά με τη διαδικασία αποσυμπίεσης.

## Common Examples
Ακολουθούν μερικά παραδείγματα χρήσης της εντολής `unxz`:

1. Αποσυμπίεση ενός αρχείου:
   ```bash
   unxz αρχείο.xz
   ```

2. Αποσυμπίεση και διατήρηση του αρχείου:
   ```bash
   unxz -k αρχείο.xz
   ```

3. Αποσυμπίεση με αναγκαστική επιλογή:
   ```bash
   unxz -f αρχείο.xz
   ```

4. Αποσυμπίεση με λεπτομέρειες:
   ```bash
   unxz -v αρχείο.xz
   ```

## Tips
- Χρησιμοποιήστε την επιλογή `-k` αν θέλετε να διατηρήσετε το αρχείο XZ για μελλοντική χρήση.
- Ελέγξτε πάντα αν το αρχείο εξόδου υπάρχει πριν από την αποσυμπίεση, ειδικά αν δεν χρησιμοποιείτε την επιλογή `-f`.
- Για μεγάλες αρχεία, η αποσυμπίεση μπορεί να διαρκέσει λίγο, οπότε να είστε υπομονετικοί.
