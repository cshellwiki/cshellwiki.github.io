# [লিনাক্স] Bash who ব্যবহার: সিস্টেমে লগ ইন করা ব্যবহারকারীদের তালিকা দেখানো

## Overview
`who` কমান্ডটি সিস্টেমে বর্তমানে লগ ইন করা ব্যবহারকারীদের তালিকা প্রদর্শন করে। এটি ব্যবহারকারীর নাম, লগ ইন করার সময় এবং লগ ইন করা টার্মিনাল সম্পর্কে তথ্য সরবরাহ করে।

## Usage
`who` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```
who [options] [arguments]
```

## Common Options
- `-a`: সমস্ত তথ্য প্রদর্শন করে, যেমন লগ ইন সময়, লগ আউট সময় ইত্যাদি।
- `-b`: সিস্টেমের শেষ বুট সময় দেখায়।
- `-q`: শুধুমাত্র লগ ইন করা ব্যবহারকারীদের সংখ্যা এবং তাদের নাম দেখায়।

## Common Examples

### 1. সাধারণ ব্যবহার
সাধারণভাবে `who` কমান্ড চালালে লগ ইন করা ব্যবহারকারীদের তালিকা দেখা যাবে:
```bash
who
```

### 2. সমস্ত তথ্য দেখানো
`-a` অপশন ব্যবহার করে সমস্ত তথ্য দেখতে:
```bash
who -a
```

### 3. সিস্টেমের শেষ বুট সময় দেখা
সিস্টেমের শেষ বুট সময় জানতে:
```bash
who -b
```

### 4. লগ ইন করা ব্যবহারকারীদের সংখ্যা এবং নাম
শুধুমাত্র লগ ইন করা ব্যবহারকারীদের সংখ্যা এবং নাম দেখতে:
```bash
who -q
```

## Tips
- `who` কমান্ডের আউটপুটের মাধ্যমে আপনি সিস্টেমের ব্যবহারকারীদের কার্যকলাপ সম্পর্কে ধারণা পেতে পারেন।
- যদি আপনি একটি নির্দিষ্ট সময়ে লগ ইন করা ব্যবহারকারীদের দেখতে চান, তাহলে `last` কমান্ড ব্যবহার করা আরও কার্যকর হতে পারে।
- `who` কমান্ডের আউটপুটকে `grep` এর সাথে ব্যবহার করে নির্দিষ্ট ব্যবহারকারীর তথ্য খুঁজে বের করা সম্ভব।
