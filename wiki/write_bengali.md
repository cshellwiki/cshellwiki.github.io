# [লিনাক্স] Bash write ব্যবহার: ব্যবহারকারীদের মধ্যে বার্তা পাঠানো

## Overview
`write` কমান্ডটি লিনাক্স এবং ইউনিক্স সিস্টেমে ব্যবহারকারীদের মধ্যে সরাসরি বার্তা পাঠানোর জন্য ব্যবহৃত হয়। এটি একটি ব্যবহারকারীকে অন্য ব্যবহারকারীর টার্মিনালে বার্তা পাঠাতে সক্ষম করে, যা সাধারণত সিস্টেমে লগ ইন করা ব্যবহারকারীদের মধ্যে যোগাযোগের জন্য ব্যবহৃত হয়।

## Usage
`write` কমান্ডের মৌলিক সিনট্যাক্স নিম্নরূপ:

```bash
write [options] [username] [tty]
```

## Common Options
- `-n`: ব্যবহারকারীর নামের আগে একটি নতুন লাইন ছাপা হবে।
- `-h`: বার্তা পাঠানোর সময় হেডার প্রদর্শন করবে না।
- `-s`: বার্তা পাঠানোর সময় একটি সাইলেন্ট মোডে থাকবে, অর্থাৎ, ব্যবহারকারীকে কিছু জানানো হবে না।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. অন্য ব্যবহারকারীকে বার্তা পাঠানো:
   ```bash
   write john
   Hello John, how are you?
   ```

2. নির্দিষ্ট টার্মিনালে বার্তা পাঠানো:
   ```bash
   write john pts/1
   This is a message for you.
   ```

3. সাইলেন্ট মোডে বার্তা পাঠানো:
   ```bash
   write -s john
   Please check your email.
   ```

## Tips
- নিশ্চিত করুন যে আপনি যে ব্যবহারকারীকে বার্তা পাঠাচ্ছেন, তিনি লগ ইন আছেন এবং তার টার্মিনালটি খোলা আছে।
- বার্তা পাঠানোর পরে, `Ctrl + D` চাপুন বার্তা শেষ করতে।
- ব্যবহারকারীর টার্মিনালে বার্তা পাঠানোর সময় তাদের মনোযোগ আকর্ষণ করার জন্য একটি স্পষ্ট এবং সংক্ষিপ্ত বার্তা লিখুন।
