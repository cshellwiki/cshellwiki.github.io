# [Linux] Bash unxz использование: Распаковка файлов .xz

## Обзор
Команда `unxz` используется для распаковки файлов, сжатых с помощью алгоритма XZ. Она удаляет сжатие и восстанавливает оригинальный файл, что делает её полезной для работы с архивами, которые занимают меньше места на диске.

## Использование
Основной синтаксис команды `unxz` выглядит следующим образом:

```bash
unxz [опции] [аргументы]
```

## Общие опции
- `-k`, `--keep`: Сохраняет оригинальный файл после распаковки.
- `-f`, `--force`: Принудительно перезаписывает существующие файлы без запроса.
- `-v`, `--verbose`: Выводит подробную информацию о процессе распаковки.

## Общие примеры
Вот несколько практических примеров использования команды `unxz`:

1. **Распаковка файла**:
   ```bash
   unxz файл.xz
   ```

2. **Распаковка с сохранением оригинала**:
   ```bash
   unxz -k файл.xz
   ```

3. **Принудительная распаковка**:
   ```bash
   unxz -f файл.xz
   ```

4. **Вывод подробной информации**:
   ```bash
   unxz -v файл.xz
   ```

## Советы
- Используйте опцию `-k`, если хотите сохранить оригинальный сжатый файл для дальнейшего использования.
- Перед распаковкой всегда проверяйте, достаточно ли у вас места на диске, чтобы избежать ошибок.
- Если вы работаете с несколькими файлами, рассмотрите возможность использования подстановочных знаков, например: `unxz *.xz`.
