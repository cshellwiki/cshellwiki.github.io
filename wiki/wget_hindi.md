# [लिनक्स] Bash wget उपयोग: फ़ाइलें डाउनलोड करने के लिए

## Overview
`wget` एक शक्तिशाली कमांड-लाइन उपकरण है जिसका उपयोग वेब से फ़ाइलें डाउनलोड करने के लिए किया जाता है। यह HTTP, HTTPS और FTP प्रोटोकॉल का समर्थन करता है और बैकग्राउंड में काम करने की क्षमता रखता है।

## Usage
`wget` कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
wget [options] [arguments]
```

## Common Options
- `-O <file>`: डाउनलोड की गई फ़ाइल का नाम निर्दिष्ट करता है।
- `-q`: चुपचाप डाउनलोड करता है, बिना किसी आउटपुट के।
- `-c`: डाउनलोड को फिर से शुरू करने की अनुमति देता है यदि यह पहले से अधूरा है।
- `--limit-rate=<rate>`: डाउनलोड की गति को सीमित करता है।
- `-r`: रेकर्सिव डाउनलोडिंग के लिए उपयोग किया जाता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

### 1. एक साधारण फ़ाइल डाउनलोड करना
```bash
wget https://example.com/file.zip
```

### 2. फ़ाइल का नाम बदलकर डाउनलोड करना
```bash
wget -O newfile.zip https://example.com/file.zip
```

### 3. डाउनलोड को फिर से शुरू करना
```bash
wget -c https://example.com/largefile.zip
```

### 4. रेकर्सिव डाउनलोड करना
```bash
wget -r https://example.com/directory/
```

### 5. डाउनलोड की गति सीमित करना
```bash
wget --limit-rate=200k https://example.com/file.zip
```

## Tips
- जब आप बड़े फ़ाइलों को डाउनलोड कर रहे हों, तो `-c` विकल्प का उपयोग करें ताकि आप डाउनलोड को फिर से शुरू कर सकें।
- `-q` विकल्प का उपयोग करें जब आप बैकग्राउंड में काम कर रहे हों और आउटपुट नहीं देखना चाहते।
- रेकर्सिव डाउनलोड करते समय, सुनिश्चित करें कि आप सही URL का उपयोग कर रहे हैं ताकि अनावश्यक फ़ाइलें डाउनलोड न हों।
