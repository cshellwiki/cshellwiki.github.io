# [리눅스] Bash userdel 사용법: 사용자 계정 삭제

## Overview
`userdel` 명령어는 리눅스 시스템에서 사용자 계정을 삭제하는 데 사용됩니다. 이 명령어를 통해 특정 사용자를 시스템에서 제거할 수 있으며, 관련된 파일이나 디렉토리도 함께 삭제할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:

```bash
userdel [options] [username]
```

## Common Options
- `-r`: 사용자 계정과 함께 해당 사용자의 홈 디렉토리와 메일 스풀을 삭제합니다.
- `-f`: 사용자가 로그인 중일 경우에도 강제로 삭제합니다.
- `-Z`: SELinux 사용자 맵을 삭제합니다.

## Common Examples
사용자 계정을 삭제하는 몇 가지 예시는 다음과 같습니다:

1. 기본 사용자 삭제:
   ```bash
   userdel username
   ```

2. 사용자와 홈 디렉토리 삭제:
   ```bash
   userdel -r username
   ```

3. 로그인 중인 사용자 강제 삭제:
   ```bash
   userdel -f username
   ```

4. SELinux 사용자 맵 삭제:
   ```bash
   userdel -Z username
   ```

## Tips
- 사용자 계정을 삭제하기 전에 해당 사용자가 시스템에 로그인 중인지 확인하는 것이 좋습니다.
- `-r` 옵션을 사용할 때는 삭제할 홈 디렉토리의 중요 파일이 있는지 미리 확인하세요.
- 사용자 계정을 삭제한 후에는 관련된 프로세스나 서비스가 정상적으로 작동하는지 점검하는 것이 좋습니다.
