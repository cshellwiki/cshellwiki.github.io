# [لينكس] Bash w: عرض معلومات المستخدمين النشطين

## Overview
يستخدم الأمر `w` في نظام لينكس لعرض معلومات حول المستخدمين النشطين على النظام، بما في ذلك الوقت الذي قضوه في النظام، والأوامر التي يقومون بتنفيذها، ومعلومات أخرى مفيدة.

## Usage
يمكن استخدام الأمر `w` بالصيغة التالية:

```bash
w [options] [arguments]
```

## Common Options
- `-h`: إخفاء رأس الجدول.
- `-s`: عرض معلومات مختصرة.
- `-u`: عرض معلومات عن المستخدمين فقط.

## Common Examples
### عرض معلومات المستخدمين النشطين
لعرض قائمة بالمستخدمين النشطين على النظام، يمكنك استخدام الأمر التالي:

```bash
w
```

### عرض معلومات مختصرة
إذا كنت ترغب في الحصول على معلومات مختصرة فقط، يمكنك استخدام الخيار `-s`:

```bash
w -s
```

### إخفاء رأس الجدول
لإخفاء رأس الجدول عند عرض المعلومات، يمكنك استخدام الخيار `-h`:

```bash
w -h
```

## Tips
- استخدم الأمر `w` بشكل دوري لمراقبة النشاط على النظام.
- يمكن دمج الأمر `w` مع أوامر أخرى مثل `grep` لتصفية النتائج حسب المستخدمين المحددين.
- تذكر أن المعلومات المعروضة تعكس الحالة الحالية للنظام، لذا قد تتغير بسرعة.
