# [لينكس] Bash write الاستخدام: إرسال الرسائل إلى المستخدمين

## Overview
أمر `write` يُستخدم لإرسال رسائل نصية إلى مستخدمين آخرين على نفس النظام. يمكن أن يكون هذا مفيدًا للتواصل السريع بين المستخدمين في بيئة متعددة المستخدمين.

## Usage
الصيغة الأساسية للأمر هي:

```bash
write [options] [username] [tty]
```

- `username`: اسم المستخدم الذي تريد إرسال الرسالة إليه.
- `tty`: (اختياري) رقم الجهاز الطرفي الذي يعمل عليه المستخدم.

## Common Options
- `-n`: لا تنتظر حتى يتم استلام الرسالة.
- `-h`: عرض مساعدة مختصرة حول الأمر.

## Common Examples

### إرسال رسالة إلى مستخدم
لإرسال رسالة إلى مستخدم باسم "ahmed":
```bash
write ahmed
```
ثم يمكنك كتابة الرسالة، ولإرسالها اضغط على `Ctrl+D` لإنهاء.

### إرسال رسالة إلى مستخدم على جهاز طرفي محدد
لإرسال رسالة إلى مستخدم "mohamed" على الجهاز الطرفي `pts/1`:
```bash
write mohamed pts/1
```

### استخدام الخيار -n
لإرسال رسالة دون انتظار استلامها:
```bash
write -n ahmed
```

## Tips
- تأكد من أن المستخدم الذي تريد إرسال الرسالة إليه متصل، وإلا فلن تصل الرسالة.
- يمكنك استخدام الأمر `who` لمعرفة المستخدمين المتصلين وأجهزتهم الطرفية.
- تذكر أن الرسائل التي ترسلها عبر `write` يمكن أن تكون مزعجة، لذا استخدمها بحذر.
