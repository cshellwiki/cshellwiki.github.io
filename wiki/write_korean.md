# [리눅스] C Shell (csh) write 사용법: 사용자 간 메시지 전송

## Overview
`write` 명령은 현재 로그인한 사용자에게 메시지를 전송하는 데 사용됩니다. 이 명령을 통해 다른 사용자와 실시간으로 소통할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:
```
write [사용자] [터미널]
```

## Common Options
- `-n`: 메시지를 보낼 때 사용자에게 알림을 하지 않습니다.
- `-h`: 메시지를 보낼 때 수신자의 터미널을 숨깁니다.

## Common Examples
다음은 `write` 명령을 사용하는 몇 가지 예입니다.

1. 특정 사용자에게 메시지 보내기:
   ```bash
   write username
   ```
   이 명령을 입력한 후 메시지를 입력하고 `Ctrl+D`를 눌러 전송합니다.

2. 특정 터미널에 메시지 보내기:
   ```bash
   write username pts/1
   ```
   `pts/1`은 메시지를 받을 사용자의 터미널입니다.

3. 알림 없이 메시지 보내기:
   ```bash
   write -n username
   ```

## Tips
- 메시지를 보내기 전에 상대방이 로그인해 있는지 확인하세요.
- `write` 명령을 사용하기 전에 수신자가 `mesg y` 명령을 실행하여 메시지를 받을 수 있도록 설정되어 있어야 합니다.
- 대화가 끝난 후 `Ctrl+D`를 눌러 세션을 종료하세요.
