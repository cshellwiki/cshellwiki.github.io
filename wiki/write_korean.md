# [리눅스] Bash write 사용법: 다른 사용자에게 메시지 전송

## Overview
`write` 명령은 리눅스 시스템에서 다른 사용자에게 실시간으로 메시지를 전송하는 데 사용됩니다. 이 명령은 주로 터미널 세션에서 다른 사용자의 세션에 직접 메시지를 보낼 때 유용합니다.

## Usage
기본 구문은 다음과 같습니다:
```bash
write [사용자] [터미널]
```

## Common Options
- `-n`: 메시지를 보낼 때 사용자에게 알림을 표시하지 않습니다.
- `-h`: 메시지를 보낼 때 수신자의 터미널이 활성화되어 있는지 확인합니다.

## Common Examples
- 특정 사용자에게 메시지 전송하기:
```bash
write username
```
이 명령을 입력한 후, 입력한 메시지는 `username`의 터미널에 표시됩니다.

- 특정 터미널에 메시지 전송하기:
```bash
write username pts/1
```
이 명령은 `username`의 `pts/1` 터미널에 메시지를 전송합니다.

- 메시지 전송 후 종료하기:
```bash
write username
Hello, how are you?
```
메시지를 입력한 후, `Ctrl+D`를 눌러 전송을 종료할 수 있습니다.

## Tips
- `write` 명령을 사용하기 전에 수신자가 메시지를 받을 수 있는 상태인지 확인하세요. 수신자가 로그인하지 않거나 터미널이 비활성화되어 있으면 메시지를 받을 수 없습니다.
- 여러 사용자에게 동시에 메시지를 보내고 싶다면, 각 사용자에게 개별적으로 `write` 명령을 사용해야 합니다.
- 수신자가 메시지를 받는 동안, 그들의 터미널 화면에 메시지가 표시되므로, 개인적인 내용을 보낼 때는 주의하세요.
