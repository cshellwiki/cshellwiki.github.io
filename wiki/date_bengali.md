# [লিনাক্স] Bash date ব্যবহার: বর্তমান তারিখ ও সময় দেখানো

## Overview
`date` কমান্ডটি লিনাক্স এবং ইউনিক্স সিস্টেমে বর্তমান তারিখ এবং সময় প্রদর্শনের জন্য ব্যবহৃত হয়। এটি বিভিন্ন ফরম্যাটে তারিখ ও সময় দেখানোর ক্ষমতা রাখে, যা ব্যবহারকারীদের জন্য খুবই উপকারী।

## Usage
`date` কমান্ডের মৌলিক সিনট্যাক্স হলো:

```bash
date [options] [arguments]
```

## Common Options
নিচে `date` কমান্ডের কিছু সাধারণ অপশন উল্লেখ করা হলো:

- `+FORMAT` : নির্দিষ্ট ফরম্যাটে তারিখ ও সময় প্রদর্শন করতে ব্যবহৃত হয়।
- `-u` : ইউনিভার্সাল টাইম (UTC) অনুযায়ী তারিখ ও সময় দেখায়।
- `-d STRING` : একটি নির্দিষ্ট তারিখ ও সময়ের জন্য প্রদর্শন করে, যেখানে STRING একটি তারিখের বর্ণনা।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. বর্তমান তারিখ ও সময় দেখানো:
   ```bash
   date
   ```

2. নির্দিষ্ট ফরম্যাটে তারিখ ও সময় দেখানো (যেমন: YYYY-MM-DD):
   ```bash
   date +"%Y-%m-%d"
   ```

3. UTC অনুযায়ী বর্তমান সময় দেখানো:
   ```bash
   date -u
   ```

4. একটি নির্দিষ্ট তারিখের জন্য সময় দেখানো (যেমন: "2023-10-01"):
   ```bash
   date -d "2023-10-01"
   ```

## Tips
- `date` কমান্ডের সাথে ফরম্যাট অপশন ব্যবহার করে আপনার প্রয়োজন অনুযায়ী সময় ও তারিখের আউটপুট কাস্টমাইজ করতে পারেন।
- সময় অঞ্চল পরিবর্তন করতে `TZ` পরিবেশ পরিবর্তনশীল ব্যবহার করতে পারেন।
- স্ক্রিপ্টে `date` ব্যবহার করে লগ ফাইলের জন্য টাইমস্ট্যাম্প তৈরি করতে পারেন, যা ডিবাগিংয়ে সহায়ক হতে পারে।
