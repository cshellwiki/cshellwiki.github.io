# [Linux] Bash write 使用法: 他のユーザーにメッセージを送信する

## Overview
`write` コマンドは、システム上の他のユーザーにメッセージを送信するために使用されます。このコマンドを使うことで、リアルタイムでコミュニケーションを取ることができます。

## Usage
基本的な構文は次のとおりです。

```
write [ユーザー名] [端末]
```

## Common Options
- `-n` : メッセージを送信する前に、相手の端末がアクティブであるか確認します。
- `-h` : ヘルプメッセージを表示します。

## Common Examples
以下は、`write` コマンドの一般的な使用例です。

### 例1: 特定のユーザーにメッセージを送信
```bash
write taro pts/1
```
このコマンドは、`taro` ユーザーの `pts/1` 端末にメッセージを送信します。

### 例2: メッセージを送信する
メッセージを入力し、`Ctrl+D` で送信を終了します。
```bash
write hanako
こんにちは、元気ですか？
```

### 例3: ユーザーがログインしているか確認
```bash
write -n taro
```
このコマンドは、`taro` ユーザーがログインしているかどうかを確認します。

## Tips
- `write` コマンドを使用する前に、相手がメッセージを受信できる状態であることを確認してください。
- メッセージを送信する際は、簡潔に伝えることを心がけましょう。
- 相手がメッセージを受信している場合、相手の端末にメッセージが表示されますので、プライバシーに配慮してください。
