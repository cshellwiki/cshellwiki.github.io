# [Linux] Bash w コマンド: ユーザーのログイン情報を表示する

## Overview
`w` コマンドは、システムにログインしているユーザーの情報を表示します。このコマンドは、各ユーザーのログイン時間、実行中のプロセス、CPU使用率などを確認するのに役立ちます。

## Usage
基本的な構文は次のとおりです。

```bash
w [options] [arguments]
```

## Common Options
- `-h` : ヘッダーを表示しない。
- `-s` : 簡潔な形式で表示する。
- `-f` : ホスト名を表示する。

## Common Examples
以下は、`w` コマンドのいくつかの実用的な例です。

### 1. 基本的な使用法
現在ログインしているユーザーの情報を表示します。

```bash
w
```

### 2. ヘッダーを表示しない
ヘッダーを省略して、ユーザー情報のみを表示します。

```bash
w -h
```

### 3. 簡潔な形式で表示
簡潔な形式でユーザー情報を表示します。

```bash
w -s
```

### 4. ホスト名を表示
ホスト名を含めてユーザー情報を表示します。

```bash
w -f
```

## Tips
- `w` コマンドは、システムのリソース使用状況を監視する際に非常に便利です。
- 定期的に `w` コマンドを実行して、どのユーザーがシステムを使用しているかを確認すると良いでしょう。
- スクリプト内で `w` コマンドを使用して、特定のユーザーのアクティビティを監視することも可能です。
