# [লিনাক্স] Bash watch ব্যবহার: কমান্ডের আউটপুট নিয়মিতভাবে দেখুন

## Overview
`watch` কমান্ডটি লিনাক্সে একটি খুবই কার্যকরী টুল, যা আপনাকে একটি নির্দিষ্ট কমান্ডের আউটপুট নিয়মিতভাবে পর্যবেক্ষণ করতে দেয়। এটি একটি নির্দিষ্ট সময় অন্তর কমান্ডটি পুনরায় চালায় এবং আউটপুটকে স্ক্রীনে দেখায়, যা আপনাকে পরিবর্তনগুলি তাত্ক্ষণিকভাবে দেখতে সাহায্য করে।

## Usage
`watch` কমান্ডের মৌলিক সিনট্যাক্স নিম্নরূপ:

```bash
watch [options] [arguments]
```

## Common Options
- `-n <seconds>`: প্রতি `<seconds>` সেকেন্ডে কমান্ডটি পুনরায় চালানোর জন্য সময় নির্ধারণ করে।
- `-d`: আউটপুটে পরিবর্তনগুলি হাইলাইট করে।
- `-t`: টাইটেল বারটি সরিয়ে দেয়, যাতে স্ক্রীনটি পরিষ্কার থাকে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

1. প্রতি 2 সেকেন্ডে `date` কমান্ডের আউটপুট দেখুন:
   ```bash
   watch -n 2 date
   ```

2. প্রতি 5 সেকেন্ডে `df -h` কমান্ডের আউটপুট দেখুন, যেখানে ডিস্ক ব্যবহারের তথ্য প্রদর্শিত হয়:
   ```bash
   watch -n 5 df -h
   ```

3. `ls` কমান্ডের আউটপুট পরিবর্তনগুলি হাইলাইট করে দেখুন:
   ```bash
   watch -d ls
   ```

4. প্রতি 10 সেকেন্ডে একটি নির্দিষ্ট ফোল্ডারের বিষয়বস্তু দেখুন:
   ```bash
   watch -n 10 ls /path/to/directory
   ```

## Tips
- `watch` ব্যবহার করার সময়, সময়সীমা নির্বাচন করুন যা আপনার পর্যবেক্ষণের প্রয়োজন অনুযায়ী উপযুক্ত।
- পরিবর্তনগুলি সহজে দেখতে `-d` অপশন ব্যবহার করুন।
- দীর্ঘ সময় ধরে চলমান কমান্ডের জন্য `-t` অপশন ব্যবহার করে স্ক্রীনকে পরিষ্কার রাখুন।
