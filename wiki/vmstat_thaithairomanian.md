# [ระบบปฏิบัติการ] Bash vmstat การใช้งาน: แสดงข้อมูลการใช้งานระบบ

## Overview
คำสั่ง `vmstat` ใช้เพื่อแสดงข้อมูลเกี่ยวกับสถานะของระบบ เช่น การใช้งานหน่วยความจำ การประมวลผล และการทำงานของ I/O โดยจะช่วยให้ผู้ใช้สามารถตรวจสอบประสิทธิภาพของระบบได้อย่างรวดเร็ว

## Usage
คำสั่ง `vmstat` มีรูปแบบพื้นฐานดังนี้:
```bash
vmstat [options] [arguments]
```

## Common Options
- `-a` : แสดงข้อมูลเกี่ยวกับหน่วยความจำทั้งหมด
- `-m` : แสดงข้อมูลเกี่ยวกับหน่วยความจำที่จัดสรรให้กับแอปพลิเคชัน
- `-s` : แสดงสถิติการใช้งานหน่วยความจำ
- `-d` : แสดงข้อมูลเกี่ยวกับการทำงานของดิสก์
- `-t` : แสดงเวลาในผลลัพธ์

## Common Examples
1. แสดงข้อมูลสถานะของระบบในรูปแบบพื้นฐาน:
   ```bash
   vmstat
   ```

2. แสดงข้อมูลหน่วยความจำทั้งหมด:
   ```bash
   vmstat -a
   ```

3. แสดงสถิติการใช้งานหน่วยความจำ:
   ```bash
   vmstat -s
   ```

4. แสดงข้อมูลเกี่ยวกับการทำงานของดิสก์:
   ```bash
   vmstat -d
   ```

5. แสดงข้อมูลทุก ๆ 2 วินาที:
   ```bash
   vmstat 2
   ```

## Tips
- ใช้ `vmstat` ร่วมกับคำสั่งอื่น ๆ เช่น `grep` เพื่อกรองข้อมูลที่ต้องการ
- ควรใช้ตัวเลือก `-t` เพื่อให้เห็นเวลาในผลลัพธ์ ซึ่งจะช่วยในการวิเคราะห์ข้อมูลได้ดียิ่งขึ้น
- หากต้องการติดตามการเปลี่ยนแปลงของระบบในระยะยาว ให้ใช้ `vmstat` ในการบันทึกข้อมูลลงไฟล์เพื่อวิเคราะห์ภายหลัง
