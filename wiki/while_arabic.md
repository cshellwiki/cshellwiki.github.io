# [لينكس] Bash while الاستخدام: تنفيذ حلقة تكرار حتى يتحقق شرط معين

## Overview
أمر `while` في Bash يُستخدم لتنفيذ حلقة تكرار تستمر طالما أن الشرط المحدد يُعتبر صحيحًا. يُعتبر هذا الأمر مفيدًا في العديد من السيناريوهات، مثل معالجة البيانات أو تنفيذ الأوامر بشكل متكرر حتى يتم الوصول إلى حالة معينة.

## Usage
التركيب الأساسي لأمر `while` هو كما يلي:

```bash
while [شرط]
do
    # الأوامر التي سيتم تنفيذها
done
```

## Common Options
- لا توجد خيارات محددة لأمر `while`، ولكن يمكن استخدامه مع أي تعبير شرطي يُرجع قيمة صحيحة أو خاطئة.

## Common Examples

### مثال 1: عد من 1 إلى 5
```bash
count=1
while [ $count -le 5 ]
do
    echo "العدد: $count"
    ((count++))
done
```

### مثال 2: قراءة مدخلات المستخدم حتى يتم إدخال "exit"
```bash
input=""
while [ "$input" != "exit" ]
do
    read -p "أدخل نص (اكتب 'exit' للخروج): " input
    echo "أنت أدخلت: $input"
done
```

### مثال 3: تنفيذ أمر حتى تنجح عملية معينة
```bash
success=1
while [ $success -ne 0 ]
do
    echo "محاولة تنفيذ الأمر..."
    # استبدل الأمر التالي بالأمر الفعلي
    some_command
    success=$?
done
```

## Tips
- تأكد من أن الشرط داخل حلقة `while` سيصل في النهاية إلى حالة تجعل الحلقة تتوقف، لتجنب الحلقات اللانهائية.
- استخدم `break` للخروج من الحلقة في حالة معينة إذا كنت بحاجة إلى ذلك.
- يمكنك استخدام `continue` لتخطي تكرار معين في الحلقة والانتقال إلى التكرار التالي.
