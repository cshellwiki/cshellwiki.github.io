# [Linux] Bash 写入命令: 发送消息给其他用户

## 概述
`write` 命令用于向其他登录用户发送消息。它允许用户在终端之间进行实时通信，适用于需要快速交流的场景。

## 用法
基本语法如下：
```bash
write [用户名] [终端]
```

## 常用选项
- `-n`：不显示消息的发送者。
- `-s`：静默模式，不显示提示信息。

## 常见示例
1. 向用户 `alice` 发送消息：
   ```bash
   write alice
   你好，Alice！我们稍后见面。
   ```
   输入完消息后，按 `Ctrl+D` 结束。

2. 向特定终端的用户发送消息：
   ```bash
   write bob pts/1
   你好，Bob！请查看你的终端。
   ```

3. 使用静默模式发送消息：
   ```bash
   write -s alice
   这是一条静默消息。
   ```

## 提示
- 在发送消息前，确保接收用户的终端处于活动状态。
- 使用 `who` 命令查看当前登录用户及其终端信息。
- 发送消息时，保持简洁明了，以便接收者快速理解。
