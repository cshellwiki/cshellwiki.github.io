# [리눅스] Bash w 사용법: 현재 로그인한 사용자 정보 보기

## Overview
`w` 명령어는 현재 시스템에 로그인한 사용자와 그들의 활동 정보를 표시합니다. 이 명령어는 사용자의 로그인 시간, 현재 실행 중인 프로세스, 그리고 CPU 사용량 등을 보여줍니다.

## Usage
기본 구문은 다음과 같습니다:
```
w [옵션] [인수]
```

## Common Options
- `-h`: 헤더를 생략하고 출력합니다.
- `-s`: 간단한 형식으로 출력합니다.
- `-f`: 포맷을 변경하여 더 많은 정보를 표시합니다.

## Common Examples
- 현재 로그인한 사용자 정보 보기:
  ```bash
  w
  ```

- 헤더 없이 사용자 정보 보기:
  ```bash
  w -h
  ```

- 간단한 형식으로 출력하기:
  ```bash
  w -s
  ```

- 포맷을 변경하여 더 많은 정보 보기:
  ```bash
  w -f
  ```

## Tips
- `w` 명령어는 시스템 모니터링에 유용하며, 서버 관리 시 현재 사용자 활동을 확인하는 데 도움을 줍니다.
- 주기적으로 `w` 명령어를 실행하여 시스템의 부하 상태를 파악할 수 있습니다.
- 다른 사용자와의 협업 시, `w` 명령어를 사용하여 누가 시스템에 로그인했는지 확인할 수 있습니다.
