# [لينكس] Bash whoami الاستخدام: معرفة اسم المستخدم الحالي

## Overview
يستخدم الأمر `whoami` في Bash لعرض اسم المستخدم الحالي الذي تم تسجيل الدخول به في النظام. هو أمر بسيط ولكنه مفيد لتأكيد الهوية في بيئات متعددة المستخدمين.

## Usage
التركيب الأساسي للأمر هو كما يلي:

```bash
whoami [options] [arguments]
```

## Common Options
- `--help`: يعرض معلومات المساعدة حول الأمر.
- `--version`: يعرض معلومات حول إصدار الأمر.

## Common Examples
إليك بعض الأمثلة العملية لاستخدام الأمر `whoami`:

1. **عرض اسم المستخدم الحالي:**
   ```bash
   whoami
   ```

2. **عرض اسم المستخدم مع معلومات إضافية:**
   ```bash
   whoami --help
   ```

3. **عرض إصدار الأمر:**
   ```bash
   whoami --version
   ```

## Tips
- استخدم الأمر `whoami` في السكريبتات للتحقق من هوية المستخدم قبل تنفيذ عمليات حساسة.
- يمكنك دمج `whoami` مع أوامر أخرى، مثل `echo`، لطباعة رسالة مخصصة:
  ```bash
  echo "مرحبًا، أنت مسجل كـ $(whoami)"
  ```
