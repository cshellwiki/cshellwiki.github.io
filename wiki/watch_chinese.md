# [Linux] Bash watch 使用方法: 实时监控命令输出

## 概述
`watch` 命令用于定期执行指定的命令，并实时显示其输出。它可以帮助用户监控命令的变化，适用于需要持续观察某些系统状态或进程的场景。

## 用法
基本语法如下：
```
watch [options] [arguments]
```

## 常用选项
- `-n, --interval <秒数>`: 设置命令执行的时间间隔，单位为秒，默认为 2 秒。
- `-d, --differences`: 高亮显示输出之间的差异。
- `-t, --no-title`: 不显示标题行。
- `-x, --exec`: 直接执行指定的命令，而不是使用默认的 shell。

## 常见示例
1. 每 5 秒查看当前目录的文件列表：
   ```bash
   watch -n 5 ls -l
   ```

2. 实时监控系统的内存使用情况：
   ```bash
   watch free -h
   ```

3. 高亮显示 `df` 命令的变化，监控磁盘使用情况：
   ```bash
   watch -d df -h
   ```

4. 监控特定进程的状态，例如 `nginx`：
   ```bash
   watch -n 1 'ps aux | grep nginx'
   ```

## 小贴士
- 使用 `-n` 选项调整命令执行的频率，以避免过于频繁的输出影响可读性。
- 结合 `-d` 选项，可以快速识别输出中的变化，特别是在监控动态数据时。
- 使用 `-t` 选项可以减少屏幕上的干扰，专注于命令输出本身。
