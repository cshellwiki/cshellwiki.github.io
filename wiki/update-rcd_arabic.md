# [لينكس] Bash update-rc.d الاستخدام: إدارة خدمات النظام

## Overview
يستخدم الأمر `update-rc.d` لإدارة خدمات النظام في توزيعات لينكس. يتيح لك هذا الأمر إضافة أو إزالة أو تعديل خدمات النظام التي يتم تشغيلها تلقائيًا عند بدء التشغيل.

## Usage
تكون الصيغة الأساسية للأمر كما يلي:
```
update-rc.d [options] [arguments]
```

## Common Options
- `defaults`: يستخدم لإضافة الخدمة إلى مستويات التشغيل الافتراضية.
- `remove`: يستخدم لإزالة الخدمة من مستويات التشغيل.
- `enable`: يتيح تفعيل الخدمة.
- `disable`: يتيح تعطيل الخدمة.

## Common Examples
- **إضافة خدمة جديدة:**
```bash
sudo update-rc.d myservice defaults
```

- **إزالة خدمة:**
```bash
sudo update-rc.d myservice remove
```

- **تفعيل خدمة:**
```bash
sudo update-rc.d myservice enable
```

- **تعطيل خدمة:**
```bash
sudo update-rc.d myservice disable
```

## Tips
- تأكد من أن الخدمة لديك تم إعدادها بشكل صحيح قبل إضافتها باستخدام `update-rc.d`.
- استخدم الأمر `ls /etc/rc*.d/` للتحقق من الخدمات المتاحة في مستويات التشغيل المختلفة.
- من الأفضل إجراء نسخ احتياطي لملفات التكوين قبل إجراء تغييرات على الخدمات.
