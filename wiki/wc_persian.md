# [لینوکس] Bash wc استفاده: [شمارش خطوط، کلمات و کاراکترها]

## Overview
دستور `wc` (word count) در Bash برای شمارش خطوط، کلمات و کاراکترها در فایل‌ها یا ورودی‌های متنی استفاده می‌شود. این دستور به کاربران کمک می‌کند تا اطلاعات مفیدی از محتویات فایل‌ها به دست آورند.

## Usage
سینتکس پایه دستور `wc` به صورت زیر است:

```bash
wc [options] [arguments]
```

## Common Options
- `-l`: شمارش فقط خطوط.
- `-w`: شمارش فقط کلمات.
- `-c`: شمارش فقط کاراکترها.
- `-m`: شمارش کاراکترها (با در نظر گرفتن کاراکترهای چندبایتی).
- `-L`: نمایش طول بلندترین خط.

## Common Examples
شمارش خطوط، کلمات و کاراکترها در یک فایل متنی:

```bash
wc myfile.txt
```

فقط شمارش خطوط در یک فایل:

```bash
wc -l myfile.txt
```

فقط شمارش کلمات در یک فایل:

```bash
wc -w myfile.txt
```

شمارش کاراکترها در یک فایل:

```bash
wc -c myfile.txt
```

شمارش کاراکترها با در نظر گرفتن کاراکترهای چندبایتی:

```bash
wc -m myfile.txt
```

نمایش طول بلندترین خط در یک فایل:

```bash
wc -L myfile.txt
```

## Tips
- برای بررسی چند فایل به طور همزمان، می‌توانید نام چند فایل را به دستور اضافه کنید.
- برای استفاده از خروجی `wc` در اسکریپت‌ها، می‌توانید از قابلیت‌های پردازش متن مانند `awk` یا `sed` استفاده کنید.
- با استفاده از `-h` می‌توانید خروجی را به صورت خوانا (با واحدهای مناسب) مشاهده کنید.
