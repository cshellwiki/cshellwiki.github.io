# [Linux] Bash xz 用法: 压缩和解压缩文件

## 概述
`xz` 命令用于压缩和解压缩文件，采用高效的 LZMA 算法。它通常用于减少文件的大小，以便于存储和传输。

## 用法
基本语法如下：
```bash
xz [选项] [参数]
```

## 常用选项
- `-d` 或 `--decompress`: 解压缩文件。
- `-k` 或 `--keep`: 在压缩或解压缩后保留原始文件。
- `-f` 或 `--force`: 强制覆盖已存在的文件。
- `-z` 或 `--compress`: 压缩文件（默认操作）。
- `-t` 或 `--test`: 测试压缩文件的完整性。

## 常见示例
1. 压缩文件：
   ```bash
   xz file.txt
   ```
   这将创建一个名为 `file.txt.xz` 的压缩文件，并删除原始的 `file.txt`。

2. 解压缩文件：
   ```bash
   xz -d file.txt.xz
   ```
   这将解压缩 `file.txt.xz` 并恢复原始的 `file.txt`。

3. 保留原始文件：
   ```bash
   xz -k file.txt
   ```
   这将压缩 `file.txt`，同时保留原始文件。

4. 强制覆盖：
   ```bash
   xz -f file.txt
   ```
   如果 `file.txt.xz` 已存在，这将强制覆盖它。

5. 测试压缩文件：
   ```bash
   xz -t file.txt.xz
   ```
   这将检查 `file.txt.xz` 的完整性。

## 小贴士
- 使用 `-k` 选项可以在压缩时保留原始文件，避免意外丢失数据。
- 在处理大文件时，可以考虑使用 `-9` 选项以获得更高的压缩比，但会消耗更多的时间和资源。
- 定期测试压缩文件的完整性，确保数据未损坏。
