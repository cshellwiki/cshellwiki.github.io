# [لينكس] Bash xz الاستخدام: ضغط وفك ضغط الملفات

## Overview
أداة `xz` هي أداة تستخدم لضغط وفك ضغط الملفات باستخدام خوارزمية XZ. توفر هذه الأداة مستوى عالٍ من الضغط، مما يجعلها مثالية لتقليل حجم الملفات الكبيرة.

## Usage
يمكن استخدام الأمر `xz` مع مجموعة من الخيارات لتحديد كيفية ضغط الملفات. الصيغة الأساسية هي:

```bash
xz [options] [arguments]
```

## Common Options
- `-d` أو `--decompress`: لفك ضغط الملفات.
- `-k` أو `--keep`: يحتفظ بالملف الأصلي بعد الضغط.
- `-f` أو `--force`: يجبر العملية حتى إذا كان الملف موجودًا.
- `-z` أو `--compress`: لضغط الملفات (وهو الخيار الافتراضي).
- `-9`: يحدد مستوى الضغط الأقصى.

## Common Examples
- لضغط ملف:
```bash
xz filename.txt
```

- لفك ضغط ملف:
```bash
xz -d filename.txt.xz
```

- لضغط ملف مع الاحتفاظ بالملف الأصلي:
```bash
xz -k filename.txt
```

- لضغط ملف مع مستوى ضغط أقصى:
```bash
xz -9 filename.txt
```

## Tips
- تأكد من أن لديك مساحة كافية على القرص عند استخدام `xz`، حيث قد تحتاج إلى مساحة إضافية أثناء عملية الضغط.
- استخدم الخيار `-k` إذا كنت بحاجة إلى الاحتفاظ بالملف الأصلي، خاصةً عند العمل على ملفات مهمة.
- يمكن استخدام `xz` مع ملفات الأرشيف مثل tar، مما يجعل من السهل ضغط وفك ضغط المجلدات الكبيرة.
