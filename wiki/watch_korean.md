# [리눅스] Bash watch 사용법: 주기적으로 명령어 실행 결과 보기

## Overview
`watch` 명령어는 지정한 명령어를 주기적으로 실행하여 그 결과를 화면에 갱신해주는 유용한 도구입니다. 이를 통해 시스템 모니터링이나 작업 상태 확인을 쉽게 할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:

```bash
watch [options] [arguments]
```

## Common Options
- `-n, --interval`: 명령어를 실행할 간격(초)을 설정합니다. 기본값은 2초입니다.
- `-d, --differences`: 이전 출력과의 차이를 강조 표시합니다.
- `-t, --no-title`: 제목 표시줄을 숨깁니다.
- `-h, --help`: 사용법을 보여줍니다.

## Common Examples
1. **기본 사용법**: 현재 디렉토리의 파일 목록을 2초마다 갱신하여 보여줍니다.
   ```bash
   watch ls
   ```

2. **간격 설정**: 5초마다 시스템의 메모리 사용량을 확인합니다.
   ```bash
   watch -n 5 free -h
   ```

3. **차이 강조**: CPU 사용량을 1초마다 확인하고 이전 결과와의 차이를 강조합니다.
   ```bash
   watch -d -n 1 top -b -n 1 | head -n 20
   ```

4. **제목 숨기기**: 제목 없이 디스크 사용량을 10초마다 확인합니다.
   ```bash
   watch -t df -h
   ```

## Tips
- `watch` 명령어를 사용할 때는 자주 변경되는 정보를 모니터링하는 데 유용합니다.
- `-d` 옵션을 사용하여 변화가 있는 부분을 쉽게 확인할 수 있습니다.
- 긴 명령어를 사용할 경우, 작은 스크립트 파일로 저장한 후 `watch`로 실행하는 것이 편리합니다.
