# [لينكس] Bash zip الاستخدام: ضغط الملفات والمجلدات

## Overview
أداة `zip` تُستخدم لضغط الملفات والمجلدات في نظام لينكس. تقوم هذه الأداة بإنشاء أرشيف مضغوط يمكن تخزينه بسهولة أو نقله، مما يساعد في توفير المساحة على القرص.

## Usage
تكون الصيغة الأساسية للأمر كما يلي:

```bash
zip [options] [arguments]
```

## Common Options
- `-r`: ضغط مجلدات بشكل متكرر.
- `-e`: تشفير الأرشيف باستخدام كلمة مرور.
- `-u`: تحديث الملفات الموجودة في الأرشيف.
- `-d`: حذف ملفات من الأرشيف.
- `-q`: تشغيل الأمر في وضع هادئ (بدون رسائل).

## Common Examples
- **ضغط ملف واحد:**
  ```bash
  zip myarchive.zip file.txt
  ```

- **ضغط مجلد بالكامل:**
  ```bash
  zip -r myarchive.zip myfolder/
  ```

- **ضغط عدة ملفات:**
  ```bash
  zip myarchive.zip file1.txt file2.txt file3.txt
  ```

- **تحديث ملف في الأرشيف:**
  ```bash
  zip -u myarchive.zip updatedfile.txt
  ```

- **حذف ملف من الأرشيف:**
  ```bash
  zip -d myarchive.zip file.txt
  ```

## Tips
- تأكد من استخدام الخيار `-r` عند ضغط مجلدات لضمان تضمين جميع الملفات الفرعية.
- استخدم الخيار `-e` إذا كنت بحاجة إلى حماية الأرشيف بكلمة مرور.
- تحقق من محتويات الأرشيف باستخدام الأمر `unzip -l myarchive.zip` قبل نقله أو مشاركته.
