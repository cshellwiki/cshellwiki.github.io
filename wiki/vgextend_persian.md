# [سیستم‌عامل لینوکس] C Shell (csh) vgextend: افزودن فضای جدید به گروه‌های ولوم

## Overview
دستور `vgextend` در سیستم‌عامل‌های مبتنی بر لینوکس برای افزودن فضای جدید به یک گروه ولوم (Volume Group) استفاده می‌شود. این دستور به شما این امکان را می‌دهد که دیسک‌های جدید را به گروه ولوم موجود اضافه کنید تا فضای ذخیره‌سازی بیشتری برای مدیریت و استفاده در دسترس باشد.

## Usage
سینتکس پایه دستور `vgextend` به شکل زیر است:

```
vgextend [options] [arguments]
```

## Common Options
- `-l, --extents`: تعداد اکستنت‌ها (extents) را مشخص می‌کند که باید به گروه ولوم اضافه شود.
- `-n, --noheadings`: خروجی را بدون سرصفحه نمایش می‌دهد.
- `-f, --force`: این گزینه برای اجبار به افزودن دیسک‌ها به گروه ولوم استفاده می‌شود، حتی اگر برخی از شرایط رعایت نشده باشد.

## Common Examples
در اینجا چند مثال عملی از استفاده از دستور `vgextend` آورده شده است:

### مثال ۱: افزودن یک دیسک به گروه ولوم
```bash
vgextend myvg /dev/sdb1
```
در این مثال، دیسک `/dev/sdb1` به گروه ولوم `myvg` اضافه می‌شود.

### مثال ۲: افزودن چند دیسک به گروه ولوم
```bash
vgextend myvg /dev/sdb1 /dev/sdc1
```
در اینجا، دو دیسک `/dev/sdb1` و `/dev/sdc1` به گروه ولوم `myvg` افزوده می‌شوند.

### مثال ۳: استفاده از گزینه `--force`
```bash
vgextend -f myvg /dev/sdd1
```
این مثال دیسک `/dev/sdd1` را به گروه ولوم `myvg` اضافه می‌کند و از گزینه `--force` برای اجبار استفاده می‌کند.

## Tips
- قبل از استفاده از `vgextend`، اطمینان حاصل کنید که دیسک‌های جدید به درستی پارتیشن‌بندی و فرمت شده‌اند.
- همیشه از داده‌های خود پشتیبان‌گیری کنید قبل از انجام تغییرات در گروه‌های ولوم.
- برای بررسی وضعیت گروه ولوم بعد از افزودن دیسک، از دستور `vgs` استفاده کنید.
