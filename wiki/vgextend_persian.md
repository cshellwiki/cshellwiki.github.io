# [لینوکس] Bash vgextend استفاده: افزودن فضای ذخیره‌سازی به گروه‌های منطقی

## Overview
دستور `vgextend` در لینوکس برای افزودن فضای ذخیره‌سازی جدید به یک گروه منطقی (Logical Volume Group) استفاده می‌شود. این دستور به شما این امکان را می‌دهد که با اضافه کردن دیسک‌های جدید، ظرفیت گروه منطقی خود را افزایش دهید.

## Usage
سینتکس پایه دستور `vgextend` به صورت زیر است:

```bash
vgextend [options] [arguments]
```

## Common Options
- `-l, --extents`: تعیین تعداد اکستنت‌ها (extents) برای افزودن.
- `-n, --name`: نام گروه منطقی که می‌خواهید به آن فضای جدید اضافه کنید.
- `--test`: فقط شبیه‌سازی می‌کند و هیچ تغییری ایجاد نمی‌کند.
- `-d, --debug`: فعال‌سازی حالت اشکال‌زدایی برای نمایش اطلاعات بیشتر.

## Common Examples
### مثال ۱: افزودن یک دیسک جدید به گروه منطقی
برای افزودن یک دیسک جدید به گروه منطقی با نام `vg01`، می‌توانید از دستور زیر استفاده کنید:

```bash
vgextend vg01 /dev/sdb1
```

### مثال ۲: افزودن چند دیسک به گروه منطقی
اگر بخواهید چند دیسک را به گروه منطقی اضافه کنید، می‌توانید به این صورت عمل کنید:

```bash
vgextend vg01 /dev/sdb1 /dev/sdc1
```

### مثال ۳: استفاده از گزینه‌های اضافی
برای افزودن فضای ذخیره‌سازی با تعیین تعداد اکستنت‌ها، می‌توانید از گزینه `-l` استفاده کنید:

```bash
vgextend -l +100 vg01 /dev/sdb1
```

## Tips
- قبل از استفاده از `vgextend`، اطمینان حاصل کنید که دیسک‌های جدید به درستی پارتیشن‌بندی و فرمت شده‌اند.
- همیشه از داده‌های مهم خود پشتیبان‌گیری کنید، زیرا تغییرات در گروه‌های منطقی می‌تواند خطرناک باشد.
- از گزینه `--test` برای بررسی تغییرات قبل از اعمال آن‌ها استفاده کنید.
