# [লিনাক্স] Bash true ব্যবহার: সত্য মান প্রদান করে

## Overview
`true` কমান্ডটি একটি খুব সহজ Bash কমান্ড যা সর্বদা সফলভাবে (exit status 0) ফিরে আসে। এটি সাধারণত স্ক্রিপ্ট বা কমান্ডের মধ্যে একটি সফল অবস্থার প্রতিনিধিত্ব করতে ব্যবহৃত হয়।

## Usage
`true` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
true [options] [arguments]
```

## Common Options
`true` কমান্ডের জন্য সাধারণত কোন অপশন নেই, কারণ এটি শুধুমাত্র সফলভাবে ফিরে আসে এবং কোন অতিরিক্ত কার্যকলাপ সম্পাদন করে না।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হলো:

### উদাহরণ 1: একটি স্ক্রিপ্টে ব্যবহার
```bash
#!/bin/bash
if true; then
    echo "এটি সবসময় সফল হবে।"
fi
```

### উদাহরণ 2: লুপে ব্যবহার
```bash
while true; do
    echo "এটি একটি অসীম লুপ।"
    sleep 1
done
```

### উদাহরণ 3: কমান্ডের সাথে ব্যবহার
```bash
true && echo "এই কমান্ড সফল হয়েছে।"
```

## Tips
- `true` কমান্ডটি স্ক্রিপ্টে অবস্থার পরীক্ষা করার জন্য খুবই কার্যকরী। এটি বিশেষ করে যখন আপনি একটি শর্তের ভিত্তিতে কিছু করতে চান।
- এটি একটি অসীম লুপ তৈরি করতে ব্যবহার করা যেতে পারে, কিন্তু সতর্ক থাকুন কারণ এটি CPU ব্যবহার বাড়াতে পারে।
- `true` কমান্ডটি `false` কমান্ডের বিপরীত, যা সর্বদা ব্যর্থ (exit status 1) ফিরে আসে।
