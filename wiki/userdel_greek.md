# [Linux] Bash userdel <Χρήση: Διαγραφή χρηστών από το σύστημα>

## Overview
Η εντολή `userdel` χρησιμοποιείται για τη διαγραφή χρηστών από το σύστημα Linux. Αυτή η εντολή αφαιρεί τον λογαριασμό του χρήστη και, προαιρετικά, τα αρχεία του χρήστη.

## Usage
Η βασική σύνταξη της εντολής είναι η εξής:

```
userdel [options] [username]
```

## Common Options
- `-r`: Διαγράφει τον χρήστη και το αρχείο του home directory.
- `-f`: Εξαναγκάζει τη διαγραφή του χρήστη, ακόμη και αν είναι συνδεδεμένος.
- `-Z`: Αφαιρεί την SELinux context του χρήστη.

## Common Examples
### Διαγραφή χρήστη χωρίς να διαγραφούν τα αρχεία του
```bash
userdel username
```

### Διαγραφή χρήστη και του home directory του
```bash
userdel -r username
```

### Εξαναγκαστική διαγραφή χρήστη
```bash
userdel -f username
```

### Διαγραφή χρήστη με SELinux context
```bash
userdel -Z username
```

## Tips
- Πάντα να ελέγχετε αν ο χρήστης είναι συνδεδεμένος πριν τη διαγραφή του, για να αποφύγετε προβλήματα.
- Χρησιμοποιήστε την επιλογή `-r` μόνο αν είστε σίγουροι ότι θέλετε να διαγράψετε και τα αρχεία του χρήστη.
- Κάντε backup σημαντικών δεδομένων πριν τη διαγραφή χρηστών, ειδικά αν χρησιμοποιείτε την επιλογή `-r`.
