# [লিনাক্স] Bash uname ব্যবহার: সিস্টেমের তথ্য প্রদর্শন

## Overview
`uname` কমান্ডটি লিনাক্স এবং ইউনিক্স ভিত্তিক সিস্টেমে ব্যবহৃত হয়। এটি সিস্টেমের বিভিন্ন তথ্য যেমন কনফিগারেশন, কনসোলের নাম, এবং কার্নেল সংস্করণ প্রদর্শন করে।

## Usage
`uname` কমান্ডের মৌলিক সিনট্যাক্স হল:

```bash
uname [options] [arguments]
```

## Common Options
- `-a`: সমস্ত উপলব্ধ তথ্য প্রদর্শন করে।
- `-s`: কার্নেলের নাম প্রদর্শন করে।
- `-n`: নেটওয়ার্কের হোস্টনেম প্রদর্শন করে।
- `-r`: কার্নেলের সংস্করণ প্রদর্শন করে।
- `-v`: কার্নেলের সংস্করণের বিবরণ প্রদর্শন করে।
- `-m`: মেশিনের আর্কিটেকচার প্রদর্শন করে।

## Common Examples
নিচে কিছু সাধারণ উদাহরণ দেওয়া হল:

### 1. সমস্ত তথ্য প্রদর্শন
```bash
uname -a
```
এই কমান্ডটি সিস্টেমের সমস্ত তথ্য যেমন কার্নেল নাম, সংস্করণ, এবং আর্কিটেকচার প্রদর্শন করবে।

### 2. কার্নেলের নাম দেখুন
```bash
uname -s
```
এটি শুধুমাত্র কার্নেলের নাম প্রদর্শন করবে।

### 3. নেটওয়ার্কের হোস্টনেম দেখুন
```bash
uname -n
```
এই কমান্ডটি সিস্টেমের নেটওয়ার্কের হোস্টনেম দেখাবে।

### 4. কার্নেলের সংস্করণ দেখুন
```bash
uname -r
```
এটি কার্নেলের সংস্করণ প্রদর্শন করবে।

### 5. মেশিনের আর্কিটেকচার দেখুন
```bash
uname -m
```
এই কমান্ডটি মেশিনের আর্কিটেকচার যেমন x86_64 বা arm64 দেখাবে।

## Tips
- `uname -a` ব্যবহার করে সব তথ্য একসাথে পেতে পারেন, যা সমস্যা সমাধানে সহায়ক হতে পারে।
- সঠিক তথ্য পাওয়ার জন্য কমান্ডটি রুট প্রিভিলেজে চালানো হতে পারে, বিশেষ করে কিছু সিস্টেমের ক্ষেত্রে।
- সিস্টেমের আপডেট বা পরিবর্তনের পরে `uname` কমান্ডটি চালিয়ে নিশ্চিত করুন যে সবকিছু সঠিকভাবে কাজ করছে।
